# <a name="install-the-windows-server-essentials-log-collector"></a>Установка сборщика журналов Windows Server Essentials

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Мастер установки сборщика журналов Windows Server Essentials устанавливает сборщик журналов как запуска Add-in. Можно установить и использовать сборщик данных журнала на сетевых компьютерах и сервере. После установки сборщик журналов появится на панели мониторинга.  
  
###  <a name="BKMK_ToInstall"></a>Чтобы установить сборщик журналов  
  
1.  Загрузите пакет установки сборщика журналов на любой сервер или компьютер в сети.  
  
    > [!NOTE]
    >  Вы можете [загрузить пакет установки сборщика журналов](https://go.microsoft.com/fwlink/p/?LinkId=255470) от корпорации Майкрософт.  
  
2.  Дважды щелкните значок сборщика журналов.  
  
3.  Если установка выполняется с компьютера в сети, введите учетные данные администратора сервера при появлении соответствующего запроса.  
  
4.  Выберите принять условия лицензионного соглашения на использование программного обеспечения Майкрософт.  
  
5.  Чтобы установить сборщик журналов только на сервере, установите **только на сервере** флажок. Чтобы установить сборщик журналов на всех компьютерах сети, выберите **на сервере и на всех компьютерах в сети** флажок.  
  
6.  Нажмите кнопку **установить надстройку**.  
  
###  <a name="BKMK_Reinstall"></a>Переустановка сборщика журналов  
 Если это необходимо переустановить сборщик журналов, необходимо удалить и переустановить сборщик журналов на сервере и на всех компьютерах в сети. При удалении сборщика журналов на сервере с помощью панели мониторинга, всех компьютерах сети будет автоматически удален сборщик журналов.  
  
##### <a name="to-uninstall-and-reinstall-the-log-collector"></a>Чтобы удалить и переустановить сборщик журналов  
  
1.  Откройте панель мониторинга.  
  
2.  Нажмите кнопку **надстройки** выберите **сборщика журналов** из списка и нажмите кнопку **удалить**.  
  

3.  Скачайте и установите сборщик журналов, выполнив действия, описанные в предыдущей процедуре [для установки сборщика журналов](Install-the-Windows-Server-Essentials-Log-Collector.md#BKMK_ToInstall).  

3.  Скачайте и установите сборщик журналов, выполнив действия, описанные в предыдущей процедуре [для установки сборщика журналов](../support/Install-the-Windows-Server-Essentials-Log-Collector.md#BKMK_ToInstall).  

  
### <a name="manually-install-the-log-collector"></a>Установка сборщика журналов вручную  
 Если мастер установки не удалось установить сборщик журналов, можно установить сборщик журналов на одном компьютере, используя описанную ниже процедуру.  
  
##### <a name="to-manually-install-the-log-collector"></a>Установка сборщика журналов вручную  
  
1.  Измените расширение скачанного файла установки с .wssx на .cab.  
  
2.  Дважды щелкните имя файла установки.  
  
3.  Нажмите кнопку **ОК** при появлении.  
  
4.  Дважды щелкните имя файла, заканчивая ˜.msi и выберите папку для его извлечения.  
  
5.  Перейдите к папке с извлеченным файлом и дважды щелкните файл установки, чтобы завершить установку с помощью мастера.

# <a name="install-add-ins"></a>Установка Add-Ins

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Надстройки на всех серверных и клиентских компьютерах можно включить, установив их перед созданием образа. Эти надстройки будут автоматически включены на всех компьютерах, где устанавливается данный образ. Можно установить надстройку либо запустив файл .wssx, или можно установить отдельные файлы надстроек, следуя инструкциям, приведенным в [документации по SDK](https://go.microsoft.com/fwlink/?LinkID=248648)для каждого типа надстроек. При установке с помощью wssx-файла надстройку можно удалить с помощью диспетчера надстроек. Если установить отдельные файлы, надстройка не является управляемым с помощью диспетчера надстроек.  
  
> [!NOTE]
>  Любое программное обеспечение, установленное или скачанное на сервере (в том числе вкладки панели мониторинга и уведомления о работоспособности) должны иметь локализованные интерфейс, который соответствует языку операционной системы, установленной на сервере.  
  
#### <a name="to-install-an-add-in"></a>Чтобы установить надстройку  
  
1.  (Необязательно) При установке надстройки с помощью wssx-файла, выполните следующие действия:  
  
    1.  Сохраните файл .wssx < AddinName\ > на компьютере-образце.  
  
    2.  Дважды щелкните файл .wssx, чтобы открыть мастер установки надстройки.  
  
    3.  Следуйте инструкциям мастера, чтобы завершить установку.  
  
2.  (Необязательно) Установите отдельные файлы надстроек в соответствующие каталоги, как определено в пакете SDK для каждого типа надстройки.

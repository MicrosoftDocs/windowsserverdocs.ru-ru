# <a name="configure-support-for-a-wireless-network"></a>Настройка поддержки беспроводных сетей

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Можно настроить операционную систему для поддержки беспроводной сети. Для включения поддержки беспроводной сети на сервере, необходимо выполнить следующие требования.  
  
-   Сервер должен иметь проводные сетевые адаптеры.  
  
-   Должны быть предварительно установлены правильный драйвер адаптера беспроводной сети, если сетевой адаптер не поддерживается операционной системой.  
  
-   Возможность включения и отключения адаптер беспроводной сети должны быть доступны. Методы для выполнения этого может включать физическую кнопку на сервере или пользовательского интерфейса на панели мониторинга. Руководство пользователя должен предоставить действия для включения и отключения беспроводного сетевого адаптера.  
  
-   Возможность выбрать беспроводную сеть и подключиться к ней должны быть доступны. Это следует сделать, добавив пользовательский интерфейс панели мониторинга. Руководство пользователя должен предоставить этапы Выбор и подключение к беспроводной сети.  
  
-   Если необходима возможность поддержки соединением беспроводной сети, необходимо предоставить Расширенный пользовательский интерфейс на информационной панели. Пользовательский интерфейс может быть кнопку или ссылку, которая запускает настройки мастера беспроводной сетью с прямым соединением сети в панели управления в Windows Server 2008 R2.  
  
## <a name="additional-considerations"></a>Дополнительные рекомендации  
 При настройке поддержки беспроводной сети, следует также рассматривать следующие сведения:  
  
-   Сервер должен быть подключен к сети с помощью провода для запуска программы установки.  
  
-   К сетевому компьютеру, на котором выполняется восстановление исходного состояния системы необходимо подключение к сети с помощью провода.  
  
-   Сервер должен быть подключен к сети с сеть, чтобы выполнить восстановление исходного состояния системы сервера.  
  
-   Если на сервере создается сеть с прямым соединением, адаптер беспроводной сети предназначенный для сети компьютер компьютер, пользователь всегда должен подключите сетевой кабель к серверу для получения подключение к Интернету.

# <a name="prepare-the-technician-computer"></a>Подготовка обслуживающего компьютера

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

На обслуживающем компьютере содержатся средства и документация комплект средств для развертывания и оценки Windows (ADK), на нем также настраивается образ развертываемой операционной системы. Для подготовки обслуживающего компьютера выполните следующие действия.

-   Подготовьте компьютер с Windows 7, Windows 8 или Windows 10.

-   Установите Windows ADK на компьютере под управлением Windows 7, Windows 8 или Windows 10 с [веб-сайта Майкрософт](https://go.microsoft.com/fwlink/?LinkID=248647).

# <a name="change-the-order-and-grouping-of-tabs"></a>Изменение порядка и группировки вкладок

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Можно изменить порядок вкладок на панели мониторинга, чтобы пользовательская вкладка стала первой (слева) в строке вкладок. Для этого необходимо добавить запись в реестр. Также может повлиять на группировки вкладок путем добавления записей в реестр. Последовательность вкладок может быть вашу основной вкладку следуют встроенных вкладок Майкрософт, следуют какие-либо дополнительные вкладки и затем с вкладки сторонних производителей.  
  
## <a name="change-the-order-of-the-tabs-in-the-dashboard"></a>Изменить порядок вкладок на панели мониторинга  
 Необходимо добавить идентификатор вкладка в реестр, чтобы определить порядок надстройки.  
  
#### <a name="to-display-your-tab-first-in-the-list-of-tabs"></a>Для отображения на вкладке сначала в списке вкладок  
  
1.  На компьютере-образце нажмите кнопку **запустить**, введите **regedit**, а затем нажмите клавишу **ввод**.  
  
2.  В левой области разверните **HKEY_LOCAL_MACHINE**, разверните **программного обеспечения**, разверните **Microsoft**и затем разверните **Windows Server**. Если **OEM** раздел не существует, необходимо выполнить следующие действия, чтобы создать его:  
  
    1.  Щелкните правой кнопкой мыши **Windows Server**, наведите курсор на **New**, а затем нажмите кнопку **ключ**.  
  
    2.  Тип **OEM** для имени ключа.  
  
3.  Щелкните правой кнопкой мыши **OEM**, наведите курсор на **New**, а затем нажмите кнопку **строковое значение**.  
  
4.  Тип **DashboardMainTabID** как имя строки и нажмите клавишу **ввод**.  
  
5.  Щелкните правой кнопкой мыши новую строку в области справа, а затем нажмите кнопку **изменить**.  
  
6.  Введите идентификатор GUID, который был определен для вкладки верхнего уровня и нажмите клавишу **ввод**.  
  
     Дополнительные сведения о создании и идентификации вкладок верхнего уровня см. в разделе [Создание вкладки верхнего уровня](https://msdn.microsoft.com/library/gg513957) в пакет SDK для решений Windows Server.  
  
7.  Сохраните изменения в реестр.  
  
8.  Необходимо также включить GUID для вашей основной вкладки верхнего уровня в списке идентификаторов для группировки вкладок. Чтобы сделать это, выполните действия, приведенные в **изменит условие группирования вкладок на панели мониторинга**.  
  
## <a name="change-the-grouping-of-tabs-in-the-dashboard"></a>Изменение группировки вкладок на панели мониторинга  
 Вы убедитесь, что ваши вкладки были сгруппированы вместе и включены в список встроенных вкладок Майкрософт, добавив соответствующие идентификаторы в реестр.  
  
#### <a name="to-change-the-grouping-of-tabs"></a>Чтобы изменить группировки вкладок  
  
1.  Если regedit не открыт, нажмите кнопку **запустить**, введите **regedit**и нажмите клавишу **ввод**.  
  
2.  В левой области разверните **HKEY_LOCAL_MACHINE**, разверните **программного обеспечения**, разверните **Microsoft**и затем разверните **Windows Server**.  
  
3.  Щелкните правой кнопкой мыши **OEM**, наведите курсор на **New**, а затем нажмите кнопку **ключ**.  
  
4.  Тип **DashboardAddins** как имя раздела и нажмите клавишу **ввод**.  
  
5.  Щелкните правой кнопкой мыши **DashboardAddins**, наведите курсор на **New**, а затем нажмите кнопку **строковое значение**.  
  
6.  Введите идентификатор GUID для вашу вкладку в качестве имени строки. Значение не требуется.  
  
7.  Повторите шаги 5 и 6 для каждой из вкладок и вкладки.  
  
8.  Сохраните изменения в реестр.

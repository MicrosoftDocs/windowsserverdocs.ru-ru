 Этот документ содержит пошаговые инструкции по установке и настройке Windows Server Essentials. Перед началом установки проверьте и выполнить действия, описанные в [Подготовка к установке Windows Server Essentials](../install/Before-You-Install-Windows-Server-Essentials.md).

2.  [Шаг 2: Настройка операционной системы Windows Server Essentials](../install/Install-and-Configure-Windows-Server-Essentials.md#BKMK_Step2Configure) на этом шаге завершения установки, предоставляя информацию о вашей компании, параметры домена и администратору сети. Эта информация используется для подготовки сервера к использованию.  

  
###  <a name="BKMK_ManualInstallation"></a>Шаг 1: Установка операционной системы Windows Server Essentials  
  
> [!IMPORTANT]
>  После установки операционной системы не настраивайте сервер до завершения [шаг 2: Настройка операционной системы Windows Server Essentials](#BKMK_Step2Configure).  
  
 **Ожидаемое время завершения:** около 30 минут.  
  
> [!NOTE]
>  Время выполнения данной процедуры основаны на требованиях к оборудованию.  
  
##### <a name="to-install-the-operating-system"></a>Для установки операционной системы  
  
1.  Подключите компьютер к сети с помощью сетевого кабеля.  
  
    > [!IMPORTANT]
    >  Не отключайте компьютер от сети во время установки. Это может привести к сбою установки.  
  
2.  Включите компьютер и вставьте в дисковод DVD-диска Windows Server Essentials.  
  
     Подключите съемный носитель (дискету или флэш-накопитель USB), содержащий файлы ответов, при выполнении автоматической установки. В зависимости от содержимого файлов ответов может не появиться некоторые или любой из описанных ниже установочных окон.  
  
3.  Перезагрузите компьютер. При сообщении **нажмите любую клавишу для загрузки с компакт-диска или DVD-диска** появится, нажмите любую клавишу.  
  
    > [!NOTE]
    >  Если компьютер не загружается с DVD-диска, убедитесь, что дисковод компакт-дисков стоит первым в последовательности загрузки BIOS. Дополнительные сведения о последовательности загрузки BIOS см. в документации изготовителя компьютера.  
  
4.  Выберите **язык**, вы хотите установить, **формат времени и денежных единиц**, и **клавиатуры или метод ввода**, а затем нажмите кнопку **Далее**.  
  
5.  Нажмите кнопку **установить сейчас**.  
  
6.  В **ввести ключ продукта**, введите ключ продукта.  
  
7.  Чтение **условия лицензионного соглашения**. Если вы согласны, выберите **я принимаю условия лицензионного соглашения** флажок и нажмите кнопку **Далее**.  
  
    > [!NOTE]
    >  Если вы не принять условия лицензионного соглашения, установку невозможно продолжить.  
  
8.  В **какого типа установки вы хотите? **, нажмите кнопку **Выборочная: установить только Windows (Дополнительно)**  
  
9. В **где вы хотите установить Windows? **, выберите жесткий диск, где вы хотите установить операционной системы Windows. Убедитесь, что все внутренние жесткие диски будут доступны для установки.  
  
    > [!IMPORTANT]
    >   Windows Server Essentials должен быть установлен как диск c:, а размер тома должен быть не менее 60 ГБ. Рекомендуется создать два раздела на диске с операционной системой и не использовать C: (системный раздел) для хранения деловой информации.  
  
    > [!NOTE]
    >  Если жесткого диска нет в списке (например, на жестком диске Serial дополнительные технологии вложение (SATA)), необходимо загрузить драйверы устройства для этого жесткого диска. Приобретите драйвер устройства от производителя и сохраните его на съемный носитель (например, на гибком диске или USB-устройство флэш-накопителя). Подключите съемный носитель к компьютеру, а затем нажмите кнопку **нагрузки драйвера**.  
  
     Если вам необходимо удалить и/или создать разделы, см. следующие действия:  
  
    1.  Чтобы удалить раздел, выберите раздел, нажмите кнопку **Настройка диска (Дополнительно)**, а затем нажмите кнопку **удалить**. После удаления системного раздела, создать новый раздел, следуя инструкциям этапа **b** шаг или **c**.  
  
        > [!NOTE]
        >  После нажатия кнопки **Настройка диска (Дополнительно)**, что параметр больше не появляется. В этом случае пропустите все указания, касающиеся параметров диска.  
  
    2.  Чтобы создать раздел из неразмеченной области, выберите жесткий диск, необходимо создать раздел, выберите **Настройка диска (Дополнительно)**, нажмите кнопку **New**, а затем в **размер** текст введите размер раздела, чтобы создать. Например, если вы используете Рекомендуемые разделы размером 120 гигабайт (ГБ), введите **122880**, а затем нажмите кнопку **применить**. После создания раздела нажмите кнопку **Далее**. Раздел форматируется перед продолжением установки.  
  
    3.  Чтобы создать раздел, занимающий всю неразмеченную, выберите жесткий диск, необходимо создать раздел, выберите **Настройка диска (Дополнительно)**, нажмите кнопку **New**, а затем нажмите кнопку **применить** принять размер раздела по умолчанию. После создания раздела нажмите кнопку **Далее**. Раздел форматируется перед продолжением установки.  
  
        > [!IMPORTANT]
        >  После завершения этого этапа, невозможно переместить операционной системы в другой раздел.  
  
 Во время установки временные файлы копируются в установочную папку на компьютере, что занимает около 30 минут. После установки операционной системы Windows Server Essentials, компьютер будет перезагружен. Теперь все готово для настройки операционной системы Windows Server Essentials.  
  
###  <a name="BKMK_Step2Configure"></a>Шаг 2: Настройка операционной системы Windows Server Essentials  
  
> [!IMPORTANT]
>  Если выполняется миграция с предыдущей версии Windows Small Business Server на Windows Server Essentials, выполните другим процессом. Дополнительные сведения об установке с миграцией см. в следующих:  
>   
>  -   [Миграция с Windows SBS 2003](../migrate/Migrate-Windows-Small-Business-Server-2003-to-Windows-Server-Essentials.md)  
> -   [Миграция с Windows SBS 2008](../migrate/Migrate-Windows-Small-Business-Server-2008-to-Windows-Server-Essentials.md)  
  
 На этом этапе установки предлагается ответить на несколько вопросов о вашей организации. Эта информация используется для настройки операционной системы.  
  
> [!IMPORTANT]
>  Перед началом этого шага убедитесь, что локальный сетевой адаптер подключен к маршрутизатору или к коммутатору, который будет включен и нормально функционируют должным образом.  
  
 **Ожидаемое время завершения:** около 30 минут  
  
##### <a name="to-configure-the-operating-system"></a>Чтобы настроить операционную систему  
  
1.  На **Проверьте настройку даты и времени** щелкните **изменить системную дату и время параметры** для выбора параметров Дата, время и часовой пояс для сервера. Когда вы закончите, нажмите кнопку **Далее**.  
  
    > [!IMPORTANT]
    >  При установке Windows Server Essentials на виртуальной машине, убедитесь, что выбрать одинаковые параметры часового пояса, используемые операционной системой узла. Если настройки часового пояса отличаются, может произойти сбой установки.  
  
2.  На **Выбор режима установки сервера** выполните одно из следующих действий:  
  
    1.  Выберите **чистой установки** настроить полную новую установку программного обеспечения сервера Windows Server Essentials.  
  
    2.  Выберите **миграции сервера** для установки Windows Server Essentials и присоединить этот сервер к существующему домену Windows.  
  
3.  На **персонализации сервера** введите название организации, внутреннее имя домена и имя сервера.  
  
    > [!IMPORTANT]
    >  Имя сервера должно быть уникальным в сети. После завершения этого шага нельзя изменить имя сервера или внутреннее имя домена.  
  
4.  Нажмите кнопку **Далее**.  
  
5.  На **укажите сведения об учетной записи администратора** введите сведения для учетной записи администратора.  
  
    > [!CAUTION]
    >  Не Называйте учетную запись сетевого администратора администратор или администратор сети. Эти имена учетной записи используются для использования системой.  
  
6.  На **предоставляют данные учетной записи обычного пользователя** введите сведения для новой учетной записи обычного пользователя и нажмите кнопку **Далее**.  
  
7.  На **автоматическое обновление сервера** выберите, как вы желаете получать обновления Windows для сервера, а затем нажмите кнопку **Далее**.  
  
8.  **Обновление и Подготовка сервера** страница отображается ход выполнения завершающего этапа установки. Это занимает время для выполнения и несколько раз выполнит перезагрузку компьютера.  
  
9. После перезагрузки сервера последнего, **сервер готов к использованию** откроется страница. Нажмите кнопку **закрыть**.  
  
10. Щелкните иконку панели мониторинга на **запустить** экране, а затем на панели мониторинга выполните **Установка сервера** задачи на **Главная** страницы. Эти задачи необходимо выполнить сразу же после завершения установки Windows Server Essentials.  
  
> [!NOTE]
>  По завершении установки автоматически будет выполнен сервер с учетной записью администратора, который вы добавили во время установки. Пароль встроенной учетной записи администратора устанавливается на один и тот же пароль в качестве учетной записи администратора, а затем встроенную учетную запись администратора отключается.  
  
 Установленный сервер можно использовать ограниченного промежутка времени (известных как периода ознакомления) без ввода ключа продукта. После истечения ознакомительного периода необходимо ввести ключ продукта для активации сервера, либо продлить ознакомительный период. Можно продлить ознакомительный период не более двух раз. При достижении максимальное число допустимого срока ознакомительного периода необходимо активировать сервер с помощью ключа продукта.  
  
### <a name="customize-windows-server-essentials"></a>Настройка Windows Server Essentials  
 **Главная** странице панели мониторинга Windows Server Essentials ссылки на **установки** задачи, которые необходимо выполнить сразу же после установки сервера. Выполняя эти задачи, вы помогаете защитить информацию, которая хранится на сервере и функции, доступные в Windows Server Essentials.  
  
 Если вы решите не выполнять задачи, пользователи не могут получить доступ к некоторым сетевым компонентам. Чтобы выполнить задачи позже, вернитесь панели мониторинга Windows Server Essentials **Главная** страницы.

|----------|-----------------|

|Добавление учетных записей пользователей|При выборе этой задачи отображается краткая информация о добавлении учетных записей пользователей. |Настройка уведомлений по электронной почте|При выборе этой задачи отображается краткая информация об электронных уведомлениях для оповещений.

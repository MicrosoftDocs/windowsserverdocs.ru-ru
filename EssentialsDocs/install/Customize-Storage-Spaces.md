# <a name="customize-storage-spaces"></a>Настройка дисковых пространств

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Чтобы выполнить предустановку дисковых пространств для удобства работы пользователя, выполнить сценарий конфигурации дисковых пространств из PostIC.cmd. Сценарий конфигурации дисковых пространств необходимо выполнить на конечном компьютере после завершения установки Windows Server Essentials.

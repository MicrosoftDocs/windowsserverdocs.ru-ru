# <a name="customize-storage-spaces"></a>Настройка дисковых пространств

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Чтобы выполнить предустановку дисковых пространств для удобства пользователя, можно запустить скрипт PostIC.cmd для конфигурации дисковых пространств. После завершения установки Windows Server Essentials на конечном компьютере должен быть запущен сценарий настройки дискового пространства.

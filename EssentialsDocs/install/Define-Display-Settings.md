# <a name="define-display-settings"></a>Определение параметров отображения

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Разрешение экрана следует установить в соответствии с оборудованием, подключенным к серверу. Для оптимального отображения панели администрирования требуется разрешение 1024 x 768 или выше. Дополнительные сведения об установке разрешения экрана см. в документации к [комплекту средств для развертывания и оценки Windows](https://go.microsoft.com/fwlink/?LinkId=248694) .

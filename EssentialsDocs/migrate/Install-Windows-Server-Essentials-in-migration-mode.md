# <a name="install-windows-server-essentials-in-migration-mode1"></a>Установка Windows Server Essentials в mode1 миграции

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Имеется только один сервер в сети, на котором работает Windows Server Essentials, и этот сервер должен быть контроллером домена для сети.  
  
 При установке Windows Server Essentials в режиме миграции мастер установки выполняет следующие задачи:  
  
1.  Устанавливает и настраивает серверное программное обеспечение Windows Server Essentials на конечном сервере.  
  
2.  Обновляет схему домена до самой последней версии.  
  
3.  Присоединяет конечный сервер к существующему домену. Исходный сервер и конечный сервер могут быть членами одного домена до завершения процесса миграции. После завершения миграции необходимо удалить исходный сервер из сети в течение 21 дня.  
  
    > [!WARNING]
    >  Сообщение об ошибке добавляется в журнал событий каждый день в течение 21-дневного льготного периода, пока не удалите исходный сервер из сети. Текст для чтения сообщений, «проверка роли FSMO обнаружила условие в вашей среде, которое не соответствует требованиям политики лицензирования. Сервер управления должен выполнять основной контроллер домена и роли Active Directory хозяина именования доменов. Перенесите роли Active Directory на сервере управления теперь. Этот сервер будет автоматически завершит работу, если проблема не устранена в течение 21 дня с момента первого обнаружения этого условия.» По истечении 21-дневного льготного периода исходного сервера завершается.  
  
4.  Переносит роли хозяев операций (также известные гибкие операции с единым хозяином или FSMO) с исходного сервера на конечный сервер. Роли хозяина операций — задачи специализированных контроллера домена, которые используются, когда стандартные методы обновлений и передачи данных неэффективны. Когда конечный сервер становится контроллером домена, он должен выполнять операции роли хозяина.  
  
5.  Настраивает конечный сервер в качестве сервера глобального каталога. Сервер глобального каталога — это контроллер домена, который управляет репозиторием распределенных данных. Он содержит репрезентация с возможностью поиска частичного каждого объекта в каждом домене леса Active Directory.  
  
6.  Настраивает конечный сервер в качестве сервера лицензирования.  
  
##  <a name="BKMK_Install"></a>Установка Windows Server Essentials на конечном сервере  
 Установка и настройка Windows Server Essentials на конечном сервере в режиме миграции, выполните следующую процедуру.  
  
#### <a name="to-install-windows-server-essentials-on-the-destination-server"></a>Чтобы установить Windows Server Essentials на конечном сервере  
  
1.  Включите конечный сервер и вставьте Windows Server Essentials DVD1 в дисковод DVD-диска. Если вы видите сообщение с запросом, если вы хотите загрузку с компакт-диска или DVD-диска, нажмите любую клавишу для этого.  
  
    > [!NOTE]
    >  Если конечный сервер поддерживает загрузку с USB-накопитель, можно использовать **Windows 7 USB/DVD Download Tool** Чтобы создать загрузочный флэш-накопитель USB из файла Windows Server Essentials ISO. С помощью USB-накопитель можно существенно ускорить процесс установки, так как флэш-накопители чтения данных намного быстрее, чем у DVD-дисков. После создания загрузочного накопителя USB можно добавить файл ответов на флэш-накопитель. Вы можете [загрузить средство Windows 7 USB/DVD Download Tool](https://go.microsoft.com/fwlink/p/?LinkId=248282) бесплатно на веб-сайте магазина Майкрософт.  
  
    > [!NOTE]
    >  Если конечный сервер не загружается с DVD-диска, перезагрузите компьютер и проверьте настройку BIOS, чтобы убедиться, что **DVD-ROM** стоит первым в последовательности загрузки. Дополнительные сведения о способах изменения последовательности загрузки BIOS см. в документации изготовителя оборудования.  
  
2.  Нажмите кнопку **новой установки**.  
  
3.  Если у вас есть внутренний жесткий диск, который не отображается в списке, нажмите кнопку **загрузить драйверы** и установите необходимые драйверы перед продолжением.  
  
4.  Установите флажок, используемый для проверки всех файлов и папок на основном жестком диске будут удалены и нажмите кнопку **установить**.  
  
5.  На **Выбор режима установки сервера** щелкните **миграции сервера**и предоставьте необходимые сведения о миграции.  
  
6.  Когда **миграция сервера выполнена успешно** появится сообщение, нажмите кнопку **закрыть**.  
  
 После завершения установки, вы автоматически будет выполнен вход с учетной записью администратора и пароль, который вы указали в файле ответов миграции.  
  
> [!NOTE]
>  Чтобы разблокировать рабочий стол во время установки Windows Server Essentials, используйте встроенную учетную запись администратора и оставьте пароль пустым.  
  
##  <a name="BKMK_VerifyTheHealthOfDC"></a>Убедитесь в работоспособности контроллера домена  
 Перед выполнением миграции следует убедиться, что контроллер домена и сети Windows Server Essentials находятся в работоспособном состоянии.  
  
 В следующей таблице перечислены средства, которые можно использовать для диагностики проблем на целевом сервере и сети и доменом:  
  
|Средство|Описание|  
|----------|-----------------|  
|Netdiag|Помогает изолировать сетью и проблемы с подключением. Дополнительные сведения и скачать, см. раздел [Netdiag](https://go.microsoft.com/fwlink/?LinkId=217388).|  
|Dcdiag.exe|Анализирует состояние контроллеров домена в лесу или на предприятии и сообщает о проблемах, помогая в устранении неполадок. Дополнительные сведения и скачать, см. раздел [Dcdiag](https://go.microsoft.com/fwlink/?LinkId=217389).|  
|Repadmin.exe|Содействует в диагностике проблем с репликацией между контроллерами домена. Это средство требует параметры командной строки для выполнения. Дополнительные сведения и скачать, см. раздел [Repadmin](https://go.microsoft.com/fwlink/?LinkId=217387).|  
  
 Следует устранить все проблемы, обнаруженные этими средствами перед продолжением миграции.  
  
> [!NOTE]
>  Если вы планируете миграцию электронной почты на другой локальный сервер Exchange, см. раздел [интеграции локального сервера Exchange с Windows Server Essentials](../manage/Integrate-an-On-Premises-Exchange-Server-with-Windows-Server-Essentials.md) сведения о настройке вашего локального сервера Exchange.

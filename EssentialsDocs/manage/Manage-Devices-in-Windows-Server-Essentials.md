##  <a name="BKMK_1"></a>Управление устройствами с помощью панели мониторинга  
 Windows Server Essentials позволяет выполнять стандартные административные задачи с помощью панели мониторинга Windows Server Essentials. **Устройств** страницы панели мониторинга содержит следующее:  
  
-   Список сетевых компьютеров, которая отображает:  
  
    -   Имя компьютера  
  
    -   Состояние компьютера, либо **Online** или **в автономном режиме.**  
  
    -   Описание компьютера  
  
    -   Статус резервного копирования компьютера  
  
    -   Статус обновления компьютера  
  
    -   Статус безопасности компьютера  
  
    -   Статус оповещений компьютера  
  
    -   Сведения о групповой политике для компьютера  
  
-   Область сведений с дополнительной информацией о выбранном компьютере  
  
-   Область задач, которая включает набор административных задач как просмотр свойств компьютера и оповещений, настройка резервного копирования компьютера и восстановление файлов и папок из резервной копии на устройстве  
  
#### <a name="to-view-the-status-of-network-computers"></a>Просмотр статуса сетевых компьютеров  
  
1.  Откройте панель мониторинга Windows Server Essentials.  
  
2.  На панели навигации щелкните **устройств**.  
  
3.  Просмотр состояния всех компьютеров в сети в области списка.  
  
 В следующей таблице описаны различных компьютеру и резервного копирования, доступные на панели мониторинга Windows Server Essentials. Некоторые задачи, относящиеся к компьютеру, и они видны только при выделении компьютера в списке.  
  
### <a name="computer-tasks-in-the-dashboard"></a>Задачи на компьютере на панели мониторинга  
  
|Имя задачи|Описание|  
|---------------|-----------------|  
|Просмотр свойств компьютера|Отображает общие сведения о выбранном компьютере и позволяет просматривать сведения для резервного копирования на компьютер.|  
|Настроить архивацию этого компьютера|Запускает мастер настройки резервного копирования.|  
|Настройка архивации для компьютера|Открывает свойства резервного копирования, где можно внести изменения в параметры архивации для выбранного компьютера.|  
|Запуск резервного копирования для компьютера|Запускает резервное копирование для выбранного компьютера.|  
|Остановка резервного копирования для компьютера|Останавливает резервное копирование для выбранного компьютера.|  
|Восстановление файлов или папок для компьютера|Выполняется восстановление файлов и папок мастер, который можно восстановить конкретные файлы, папки или диски.|  
|Просмотр предупреждений для компьютера|Отображает критические предупреждения и другие информационные сообщения и позволяет принимать корректирующие меры.|  
|Удаленный рабочий стол на компьютере|Открывает подключение к удаленному рабочему столу для выбранного компьютера.|  
|Удаление компьютера|Запускает мастер компьютера, отсоединяющий компьютер от панели мониторинга Windows Server Essentials для удаления.|  
|Настроить параметры истории файлов и архивации на компьютере|Открывает страницу параметров резервного копирования, из которого вы можно изменить расписание резервного копирования и параметры истории файлов для клиентских компьютеров.|  
|Как подключить компьютеры к серверу?|Открывает раздел справки, описывающий действия пошаговой инструкцией подключения компьютера к сети.|  
|Реализация групповой политики|Применяет параметры политики для компьютеров Windows 8 и Windows 7, которые присоединены к домену.|  
  
##  <a name="BKMK_2"></a>Назначение учетным записям пользователей разрешения на вход на определенных сетевых компьютерах  
 Можно назначить разрешения учетным записям пользователей, чтобы пользователи могут входить только определенные сетевые компьютеры при доступе к сети Windows Server Essentials из удаленного расположения.  
  
#### <a name="to-change-the-computer-access-for-a-user-account"></a>Чтобы изменить доступ к компьютеру для учетной записи пользователя  
  
1.  Откройте панель мониторинга Windows Server Essentials.  
  
2.  На панели навигации щелкните **пользователей**.  
  
3.  В списке учетных записей пользователей выберите учетную запись пользователя, вы хотите изменить.  
  
4.  В **задачи < пользователя Account\ >** панели, щелкните **просмотреть свойства учетной записи**. **Свойства** страница учетной записи пользователя.  
  
5.  На **доступ к компьютеру** выберите компьютер, этот пользователь удаленный доступ и нажмите кнопку **ОК**.  
  
##  <a name="BKMK_3"></a>Удаление компьютера с сервера  
 При удалении компьютера с сервера под управлением Windows Server Essentials с помощью панели мониторинга больше не осуществляется сервером. В результате сервер перестает создавать резервные копии компьютера или осуществлять мониторинг его состояния после его удаления из сети.  
  
> [!NOTE]
>  Удаление компьютера с сервера не отключает компьютер от сети. Компьютер можно по-прежнему осуществлять доступ к ресурсам в сети так же, как это может был подключен к серверу. Чтобы запретить компьютеру доступ к ресурсам сервера и отключить его от сервера, необходимо удалить компьютер из домена. Кроме того удаление компьютера с сервера автоматически не удаляет программное обеспечение Connector или панель запуска с компьютера, которое удаляется. Необходимо вручную удалить по Connector с компьютера. Дополнительные сведения см. в разделе удалить программное обеспечение Connector в [подключайтесь](../use/Get-Connected-in-Windows-Server-Essentials.md).  
  
#### <a name="to-remove-a-computer-from-the-network-by-using-the-dashboard"></a>Удаление компьютера из сети с помощью панели мониторинга  
  
1.  Откройте панель мониторинга Windows Server Essentials.  
  
2.  На панели навигации щелкните **устройств** вкладку.  
  
3.  В списке компьютеров щелкните правой кнопкой мыши компьютер, который требуется удалить из сети, а затем нажмите кнопку **удалить компьютер**.  
  
##  <a name="BKMK_5"></a>Настройка параметров групповой политики для перенаправления папок и безопасности  
 Можно настроить групповую политику и развернуть ее на компьютерах в сети Windows Server Essentials с помощью панели мониторинга Windows Server Essentials. Групповая политика в Windows Server Essentials включает параметры перенаправления папок и обеспечения безопасности, влияющие на работу центра обновления Windows, Защитник Windows и брандмауэра сети.  
  
#### <a name="to-configure-group-policy-in-windows-server-essentials"></a>Для настройки групповой политики в Windows Server Essentials  
  
1.  Откройте панель мониторинга Windows Server Essentials.  
  
2.  На панели навигации щелкните **устройств**.  
  
3.  Для Windows Server Essentials: В глобальной **задачи пользователей** панели, щелкните **реализация групповой политики**.  
  
     Для Windows Server Essentials: В глобальной **задачи устройств** панели, щелкните **реализация групповой политики**.  
  
4.  Откроется мастер реализации групповой политики.  
  
5.  На **Включение групповой политики перенаправления папок** страницы мастера можно выбрать пользовательские папки, которые необходимо перенаправить.  
  
6.  На **Включение параметров политики безопасности** страницы мастера можно выбрать включение параметров групповой политики для **центра обновления Windows**, **Защитник Windows**и **сетевой брандмауэр**.  
  
7.  Нажмите кнопку **Готово** для реализации параметров групповой политики.  
  
##  <a name="BKMK_7"></a>Подключение к сетевому компьютеру с помощью сеанса удаленного рабочего стола  
 Для удаленного доступа к своему сетевому компьютеру Windows Server Essentials, находясь вне офиса, используйте веб-браузер, войдите в систему веб-сайта организации s удаленного веб-доступа и на **компьютеры**, щелкните имя компьютера.  
  
 **Состояние** столбце показано, можно подключиться к компьютеру в сети и может содержать следующие значения:  
  
-   **Доступные**  
  
     Компьютер включен и доступен для удаленного подключения. Даже если вы видите это состояние, вы по-прежнему может не иметь возможность подключаться к этому компьютеру, если сторонний брандмауэр блокирует подключение.  
  
-   **Автономный или в спящем режиме**  
  
     Компьютер выключен или находится в режиме сна или гибернации. Если компьютер находится в автономном режиме или в спящем режиме, статус обновляется в реальном времени, чтобы знать, когда компьютер станет доступен.  
  
-   **Неподдерживаемая операционная система**  
  
     Операционной системы на компьютере не поддерживает удаленный рабочий стол. Он может занять до 6 часов на обновление на сервере, если изменение этого статуса.  
  
-   **Подключение отключено**  
  
     Подключение к компьютеру заблокировано брандмауэром, либо удаленный рабочий стол отключен на компьютере или с помощью групповой политики. Он может занять до 6 часов на обновление на сервере, если изменение этого статуса.  
  
##  <a name="BKMK_8"></a>Просмотр свойств компьютера  
 **Устройств** панели мониторинга Windows Server Essentials отображается список сетевых компьютеров. Список также содержит дополнительные сведения о каждом компьютере.  
  
#### <a name="to-view-a-list-of-computers"></a>Чтобы просмотреть список компьютеров  
  
1.  Откройте панель мониторинга Windows Server Essentials.  
  
2.  На главной панели навигации, щелкните **устройств**.  
  
3.  На панели мониторинга отображается текущий список компьютеров.  
  
#### <a name="to-view-or-change-properties-for-a-computer"></a>Просмотр или изменение свойств компьютера  
  
1.  В списке компьютеров выберите учетную запись, для которого требуется просмотреть или изменить свойства.  
  
2.  В **задачи < Computername\ >** панели, щелкните **просмотреть свойства компьютера**. **Свойства** откроется страница для компьютеров.  
  
3.  Откройте вкладку для отображения свойств для данного компьютера.  
  
4.  Чтобы сохранить изменения, внесенные в свойства компьютера, нажмите кнопку **применить**.

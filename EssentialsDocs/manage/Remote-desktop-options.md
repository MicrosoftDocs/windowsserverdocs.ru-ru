# <a name="remote-desktop-options"></a>Параметры удаленного рабочего стола
 
  
## <a name="connection-speed"></a>Скорость подключения  
 Скорость подключения к сетевому компьютеру посредством удаленного веб-доступа определяется параметрами рабочего стола, доступными на главном компьютере. В следующей таблице указаны параметры рабочего стола, доступны для скорости, который подключается к удаленному компьютеру посредством удаленного веб-доступа.  
  
||||||  
|-|-|-|-|-|  
||Медленный модем (28,8 Кбит/с)|Быстрый модем (56 Кбит/с) (по умолчанию)|Широкополосное подключение (128 Кбит/с – 1,5 Мбит/с)|Локальная сеть (1,5 Мбит/с или выше)|  
|Фон рабочего стола|Нет|Нет|Нет|Да|  
|Сглаживание шрифтов|Нет|Нет|Нет|Да|  
|Содержимое рабочего стола|Нет|Нет|Да|Да|  
|Отображение содержимого окна при перетаскивании|Нет|Нет|Да|Да|  
|Анимация меню и окон|Нет|Нет|Да|Да|  
|Темы|Нет|Да|Да|Да|  
|Кэширование графики|Да|Да|Да|Да|  
  
## <a name="screen-size"></a>Размер экрана  
 Этот параметр определяет размер окна, открываемого на локальном компьютере при подключении к удаленному компьютеру через веб-сайт удаленного доступа. Размер окна представлен в количестве пикселей.  
  
> [!NOTE]
>  При подключении к серверу открывается панель мониторинга. Можно изменять размер панели мониторинга по умолчанию — 1024 x 741  
  
-   Полноэкранный режим (несколько мониторов)  
  
-   1280 x 720  
  
-   1024 x 768 пикселей.  
  
-   800 x 600  
  
-   640 × 480  
  
## <a name="enable-the-remote-computer-to-print-to-my-local-printer"></a>Позволить удаленному компьютеру печатать на локальном принтере  
 По умолчанию включен. Этот параметр позволяет печатать на принтер, подключенный к локальному компьютеру с удаленного компьютера.  
  
## <a name="play-sounds-from-the-remote-computer"></a>Воспроизводить звуки с удаленного компьютера  
 По умолчанию включен. Этот параметр позволяет воспроизводить, например, системные звуки на локальном компьютере с удаленного компьютера.  
  
## <a name="enable-copy-and-paste-between-the-remote-computer-and-the-local-computer"></a>Включить копирование и вставку между удаленным и локальный компьютер  
 По умолчанию не включен. Этот параметр позволяет скопировать и вставить файлы между удаленного компьютера и локальный компьютер, в том же было как скопировать и вставить файлы из одного расположения в другое на локальном компьютере.  
  
## <a name="enable-the-remote-computer-to-access-drives-on-my-local-computer"></a>Позволить удаленному компьютеру доступ к дискам локального компьютера  
 По умолчанию не включен. Этот параметр позволяет получить доступ к файлам и папкам на жестких дисках, подключенных к локальному компьютеру с удаленного компьютера.

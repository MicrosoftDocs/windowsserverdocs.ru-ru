# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>Используйте приложение My Server для подключения к Windows Server Essentials

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Приложения My Server для Windows Server Essentials позволяют подключаться к ресурсам и выполнять несложные административные задачи на сервере Windows Server Essentials с ноутбука, ПК или устройства Surface. В My Server можно управлять пользователями, устройствами и оповещениями и работать с общими файлами на сервере. При работе в автономном режиме можно продолжать работать с файлами, которые недавно открывались в приложении My Server; Автономные изменения автоматически синхронизируются с сервером следующем подключении.  
  
 Если сервер работает под управлением операционной системы Windows Server Essentials, используйте исходное приложение My Server. Если сервер работает под управлением операционной системы Windows Server Essentials, необходимо использовать обновленное приложение My Server 2012 R2. Оба приложения можно скачать из [приложений для Windows](https://windows.microsoft.com/windows-8/apps).  
  
> [!TIP]

>  Получить доступ к ресурсам в Windows Server Essentials на Windows Phone, используйте приложение My Server для телефона (для Windows Server Essentials) или в приложении для телефона My Server 2012 R2 (для Windows Server Essentials). Сведения о приложении My Server для телефона см. в разделе [используйте приложение My Server для Windows Phone](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2). Дополнительные сведения о приложении My Server 2012 R2 для телефона см. в записи блога [приложения My Server 2012 R2 Windows и Windows Phone](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx).

##  <a name="BKMK_WhatsNew"></a>Какие новые возможности My Server 2012 R2 s  
 Кроме функций, доступных в My Server My Server 2012 R2 предлагает следующие новые функции для клиентов с Windows Server Essentials:  
  
-   **Доступ к сайтам SharePoint Online группы и библиотекам** -интеграцию использование Microsoft Office 365 с Windows Server Essentials для сайтов группы SharePoint Online и открывайте документы из библиотек SharePoint Online в My Server 2012 R2.

##  <a name="BKMK_OS"></a>Требования к операционной системе  
 Чтобы использовать My Server или My Server 2012 R2 на ноутбуке, ПК или устройства Surface, компьютер должен иметь одно из следующих операционных систем: Windows 8.1, Windows RT 8.1, Windows 8 или Windows RT.  
  
##  <a name="BKMK_Install"></a>Установка приложения My Server  
 Можно скачать любое нужное приложение My Server из [приложений для Windows](https://windows.microsoft.com/windows-8/apps) хранения. Если на компьютере установлена операционная система Windows 8 или Windows RT, и вы хотите подключиться к серверу в интрасети, используя имя сервера, необходимо также установить сертификат сервера.  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>Установка My Server или My Server 2012 R2 на ноутбук на базе Windows, ПК или устройство Surface  
  
1.  Установите соответствующее приложение My Server из [приложений для Windows](https://windows.microsoft.com/windows-8/apps) на основе Windows ноутбука, ПК или устройства Surface.  
  
    |Операционная система сервера|Скачать|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[My Server](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[My Server 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  Если вы хотите иметь возможность использовать имя сервера для подключения к серверу Windows Server Essentials через интрасеть, а устройство должно операционной системы Windows 8 или Windows RT, установите сертификат CAROOT.cer по умолчанию на компьютере, выполнив следующую процедуру. Установка сертификата вручную не требуется в Windows 8.1 и Windows RT 8.1.  
  
###  <a name="BKMK_InstallCert"></a>Установка сертификата для My Server на устройстве с Windows 8, Windows 8.1 или Windows RT  
  
1.  Откройте веб-браузер на компьютере и загрузить сертификат по умолчанию CAROOT.cer с сервера. Для этого введите следующую команду, указав имя своего сервера (например, **marketing.contoso.com**) для <*имя_сервера*>:  
  
     **http://<servername\>/Connect/Default.aspx?Get=caroot.cer**  
  
2.  Установка сертификата:  
  
    1.  На **запустить** странице откройте **Проводник**.  
  
    2.  Убедитесь, что скрытые элементы и расширения имен файлов отображаются: на **представление** вкладке **Показать или скрыть** группы выберите **скрытые элементы** и **расширения имен файлов** флажки.  
  
    3.  Перейдите в папку, которая содержит файл CAROOT.cer, который вы скачали.  
  
    4.  Щелкните правой кнопкой мыши файл CAROOT.cer и выберите **откройте**.  
  
    5.  В **сертификат** диалоговом нажмите кнопку **установить сертификат**.  
  
    6.  Выберите **локального компьютера** в расположение установки и нажмите кнопку **Далее**.  
  
    7.  На **хранилище сертификатов** страница мастера, выберите **поместить все сертификаты в следующее хранилище**и использовать **Обзор** Выбор **доверенные корневые центры сертификации** хранения. Нажмите кнопку **Готово**.  
  
##  <a name="BKMK_UseServer"></a>Использование My Server  
 Чтобы начать использовать приложение My Server или My Server 2012 R2, откройте приложение и кратко ознакомьтесь с его функциями.  
  
#### <a name="to-open-my-server-or-my-server-2012-r2"></a>Открытие My Server или My Server 2012 R2  
  
1.  Открытие My Server (для Windows Server Essentials) или My Server 2012 R2 (для Windows Server Essentials) из **запустить** страницы устройства с Windows.  
  
2.  Войдите на сервер Windows Server Essentials с помощью учетной записи на сервере. Для идентификации сервера:  
  
    -   В локальной среде, использовать доменное имя сервера, например, **marketing.contoso.com**.  
  
    -   Если вы подключаетесь локально по интрасети, используйте имя сервера в предыдущем примере, имя сервера — **маркетинговых**. (Если вы используете компьютер с Windows 8 или Windows RT для подключения локального, и вы хотите использовать этот метод, необходимо установить сертификат CAROOT.cer с сервера. Дополнительные сведения см. в разделе [для установки сертификата для My Server на устройстве с Windows 8, Windows 8.1 или Windows RT](#BKMK_InstallCert).)

Проверьте размер кэша в **последние** или **файлы** параметры; затем используйте **снимите** или **очистить все** очищать кэш.<br /><br /> **Совет:** чтобы узнать, какие файлы будут доступны автономно, в любой общей папке, выберите **автономные кэшированные** вместо фильтра **все**.<br /><br /> **Примечание:** по умолчанию кэше хранятся те же файл, **последние** отображаются в списке. Имейте в виду, если очистке кэша или изменении параметров кэша и **последние** не соответствует внесенным, некоторые файлы в **последние** списка могут быть недоступны автономно.|  
|Фоновая синхронизация|Ваши файлы синхронизированы с сервером каждый раз, когда вы войдете в систему на Мой сервер и при внесении изменений во время подключения к серверу. Для сохранения производительности, ресурсоемкие фоновые синхронизации не выполняются автоматически в большинстве папок. Чтобы избежать затратных передач, фоновая синхронизация не выполняется в сетях размером 3 ГБ.<br /><br /> -Разрешение конфликтов файлов, отображенных в **конфликтующих** область **состояние передачи** страницы. При выполнении передач файлов в основном окне отображаются **состояние передачи** и **оповещения** чудо-кнопки.|  
  
##  <a name="BKMK_ConnectServer"></a>Подключение к серверу из локальной сети  
 Для успешного использования приложения My Server в Essentials Windows Server для Windows Phone, Windows 8 и Windows 8.1, необходимо сначала установить сертификат сервера на вашем устройстве. Это подключит устройство к серверу под управлением Windows Server Essentials в локальной сети. Используйте следующие процедуры для подключения к серверу в локальной сети и установить сертификат сервера на компьютере под управлением Windows 8 или Windows 8.1 или на Windows Phone.  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Чтобы подключиться к серверу из вашего Windows Phone  
  
1.  На вашем Windows Phone под управлением Windows 8 или Windows 8.1 Откройте Internet Explorer.  
  
2.  В адресной строке введите **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, и нажмите клавишу ВВОД.  
  
3.  При появлении запроса на установку сертификата caroot.cer щелкните **установить**.  
  
4.  Дождитесь завершения установки сертификата.  
  
    > [!NOTE]
    >  По завершении установки сертификата вы сможете войти приложение My Server для Windows Phone, используя имя сервера и сетевые учетные данные.  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Для подключения к серверу в локальной сети на компьютере под управлением Windows 8 или Windows 8.1  
  
1.  На компьютере под управлением Windows 8 или Windows 8.1 Откройте Internet Explorer.  
  
2.  В адресной строке введите **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, и нажмите клавишу ВВОД.  
  
3.  При появлении запроса на установку сертификата caroot.cer щелкните **откройте**.  
  
4.  В **сертификат** диалоговом нажмите кнопку **установки сертификата**.  
  
5.  В мастере импорта сертификатов выберите **локального компьютера** как расположение хранилища.  
  
6.  Выберите **поместить все сертификаты в следующее хранилище**и перейдите к **доверенные корневые центры сертификации** расположение.  
  
7.  Следуйте инструкциям, чтобы завершить работу мастера.  
  
    > [!NOTE]
    >  По завершении установки сертификата вы сможете войти приложение My Server для Windows 8 или Windows 8.1, используя имя сервера и сетевые учетные данные.

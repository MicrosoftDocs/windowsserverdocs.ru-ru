# <a name="quick-start-guide-to-using-microsoft-office-365-with-windows-server-essentials"></a>Краткое руководство по использованию Microsoft Office 365 с Windows Server Essentials

>Область применения: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

 Office 365 — это простой в использовании набор веб-инструментов для доступа к электронной почте, важным документам, контакты и календарь практически отовсюду и с любого устройства. Эти средства включают Microsoft Office веб-приложения, которые являются веб-версиями настольных приложений Microsoft Office, которые вы уже знакомы, такие как Microsoft Word, Microsoft Excel и Microsoft PowerPoint. С помощью веб-приложения Office можно создавать или изменять документы на любом компьютере с веб-браузер.  
  
 Для краткого знакомства с Office 365, просмотрите [Office 365: Обзор для пользователей](https://onlinehelp.microsoft.com/office365-smallbusinesses/hh534379.aspx) видео на веб-сайте Office 365.  
  
 Если ваша организация использует веб-приложения Office 365, можно открыть приложений в Office 365 с помощью знакомых средств, таких как My Server 2012 R2 и запуска Windows Server Essentials. (Также можно, конечно, [войти Office 365 непосредственно](https://login.microsoftonline.com/login.srf?wa=wsignin1.0&rpsnv=2&ct=1384059583&rver=6.1.6206.0&wp=MBI_KEY&wreply=https:%2F%2Fwww.outlook.com%2Fowa%2F&id=260563&whr=students.tamuk.edu&CBCXT=out) с помощью идентификатора Интернет вашей организации и своего сетевого пароля.)  
  
> [!IMPORTANT]
>  В случае утери мобильного устройства, используемого для подключения к Exchange Online, свяжитесь с администратором, он может удалить конфиденциальные данные с устройства следующего время s включен.  
  
### <a name="before-you-begin"></a>Перед началом работы  
  
1.  Необходимо иметь учетную запись Microsoft Online Services (в Windows Server Essentials, он называется s *учетной записи Office 365*) вашей учетной записи пользователя. По окончании уже есть учетная запись Интернет для Office 365, обратитесь к сотруднику, обслуживающему сети, чтобы создать новую.  
  
2.  После получения учетной записи в Интернете, вход сеть и переустановите ваш пароль Windows. Вы сможете использовать тот же пароль для входа Office 365, используемый в вашей сети.  
  
    > [!NOTE]
    >  Может быть предложено изменить свой пароль Windows при входе сеть.  
  
## <a name="open-office-365-apps-from-the-launchpad"></a>Открытие приложений Office 365 из панели запуска  
 Панель запуска Windows Server Essentials предоставляет доступ для резервного копирования компьютера, общим файлам и мультимедиа и сайта удаленного веб-доступа с вашего компьютера. Если у вас есть учетная запись Microsoft online, можно также открыть веб-приложения Office 365 из панели запуска. Установка Launchpad производится автоматически при первом использовании компьютер для подключения к серверу.  
  
#### <a name="to-open-office-365-from-the-launchpad"></a>Открытие Office 365 из панели запуска  
  
1.  Откройте панель запуска, щелкните **Microsoft Office 365**, а затем нажмите кнопку **Office 365 для дома**.  
  
2.  Войдите в Office 365 с помощью полученного идентификатора сети Microsoft и вашего пароля Windows.  
  
> [!NOTE]
>  Если вы не установлена на компьютере для работы с Office 365, выполните действия, описанные в **начало** раздел на портал Office 365.  
  
## <a name="work-with-documents-in-your-sharepoint-online-libraries-from-my-server-2012-r2"></a>Работа с документами в библиотеках SharePoint Online в My Server 2012 R2  
 Если сервер имеет операционной системы Windows Server Essentials, и ваш администратор управляет Office 365 с сервера, можно работать с документами в библиотеках SharePoint Online на ноутбуке, мобильном устройстве или телефоне с помощью приложения My Server 2012 R2. Приложение предоставляется бесплатно. Необходимо сначала установить.  
  
#### <a name="to-install-my-server-2012-r2"></a>Установка My Server 2012 R2  
  
-   Загрузите и установите соответствующее приложение My Server 2012 R2 для ноутбуке, мобильном устройстве или Windows phone:  
  
    |Тип устройства|Скачать из|  
    |-----------------|-------------------|  
    |Ноутбук на базе Windows, ПК или устройство Surface|[My Server 2012 R2 (приложения для Windows)](https://apps.microsoft.com/windows/app/my-server-2012-r2/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
    | Телефон Windows 8 или Windows 8.1 телефона|[My Server 2012 R2 (магазин Windows Phone)](http://www.windowsphone.com/store/app/my-server-2012-r2/44f596b5-0477-4096-b96e-ddd6ef64ad6b)|  
  
#### <a name="to-work-with-documents-in-sharepoint-online-libraries-from-my-server-2012-r2"></a>Для работы с документами из библиотек SharePoint Online с My Server 2012 R2  
  
1.  Откройте приложение My Server 2012 R2.  
  
2.  Нажмите кнопку **SharePoint Online**.  
  
    > [!NOTE]
    >  Если t см. в разделе **SharePoint Online**, обратитесь к администратору, ли интеграции Office 365 с сервером.  
  
3.  Выберите библиотеку SharePoint Online для отображения документов в библиотеке.  
  

     Можно открыть, изменить, отправлять и загружать документы в библиотеке SharePoint Online так же, работать с документами на сервере или со SkyDrive. Дополнительные сведения см. в разделе [использования приложения My Server](Use-the-My-Server-App-to-Connect-to-Windows-Server-Essentials.md).  

     Можно открыть, изменить, отправлять и загружать документы в библиотеке SharePoint Online так же, работать с документами на сервере или со SkyDrive. Дополнительные сведения см. в разделе [использования приложения My Server](../use/Use-the-My-Server-App-to-Connect-to-Windows-Server-Essentials.md).

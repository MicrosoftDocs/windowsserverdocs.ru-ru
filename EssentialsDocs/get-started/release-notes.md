# <a name="release-documentation-for-windows-server-essentials"></a>Документация к выпуску для Windows Server Essentials

>Область применения. Windows Server2016 Essentials

Добро пожаловать в документацию для серверного по Windows Server 2016 Essentials. Этот раздел содержит сведения о проблемах, которые могут возникнуть при установке и запуске серверного по Windows 2016 Server Essentials. Прежде чем устанавливать и запускать Windows Server 2016 Essentials в вашей сети, ознакомьтесь в этом разделе.  
  
## <a name="client-computers"></a>Клиентские компьютеры  
  
### <a name="install-all-available-critical-and-security-updates-from-windows-update"></a>Установите все доступные критические обновления и обновления безопасности из Центра обновления Windows.  

Перед подключением клиентского компьютера к серверу, убедитесь, что открыть Центр обновления Windows на клиентском компьютере и установить все доступные обновления безопасности и критические.  
  
### <a name="client-computer-restore-may-not-succeed-after-migration-to-windows-10"></a>Восстановление данных клиентского компьютера может оказаться невозможны после миграции для Windows 10  
 После обновления клиентского компьютера с Windows 8.1 до Windows 10, восстановление данных клиентского компьютера может работать неправильно. Вам потребуется повторно подключите клиент к Windows Server 2016 Essentials.

# <a name="release-documentation-for-windows-server-essentials"></a>Документация по выпуску для Windows Server Essentials

>Область применения: Windows Server 2016 Essentials

Добро пожаловать в документации к версии программного обеспечения сервера Windows Server 2016 Essentials. Этот раздел содержит сведения о проблемах, которые могут возникнуть при установке и использовании программного обеспечения сервера Windows 2016 Server Essentials. В этом разделе необходимо ознакомиться, прежде чем устанавливать и запускать в вашей сети Windows Server 2016 Essentials.  
  
## <a name="client-computers"></a>Клиентские компьютеры  
  
### <a name="install-all-available-critical-and-security-updates-from-windows-update"></a>Установите все доступные критические обновления и безопасности обновления из центра обновления Windows  

Перед подключением клиентского компьютера к серверу, убедитесь, что, откройте Центр обновления Windows на клиентском компьютере и установить все доступные обновления безопасности и критические.  
  
### <a name="client-computer-restore-may-not-succeed-after-migration-to-windows-10"></a>Восстановление данных клиентского компьютера может оказаться невозможны после миграции на Windows 10  
 После обновления клиентского компьютера с Windows 8.1 до Windows 10, восстановление данных клиентского компьютера может работать неправильно. Вам потребуется повторно подключите клиент на Windows Server 2016 Essentials.

# <a name="get-started-with-windows-server-2016-essentials"></a>Начало работы с Windows Server 2016 Essentials 

>Область применения: Windows Server 2016 Essentials

**Windows Server 2016 Essentials** выпуск является подключен облака первого сервера, идеально подходит для предприятий малого бизнеса с максимум 25 пользователей и 50 устройств. Essentials является хорошим вариантом для клиентов, которые в настоящее время с помощью выпуска Foundation, который не работает с Windows Server 2016. Дополнительные сведения см. в разделе [новые возможности Windows Server Essentials](what-s-new.md) и [загрузить ознакомительную версию Windows Server Essentials](https://technet.microsoft.com/evalcenter/dn205288.aspx?wt.mc_id=TEC_144_1_7).

# Управление серверами с помощью Windows Admin Center

>Относится к: Windows Admin Center, ознакомительная версия Windows Admin Center

> [!Tip]
> Только начинаете знакомство с Windows Admin Center?
> [Узнайте дополнительные сведения о Windows Admin Center](../understand/windows-admin-center.md) или [скачайте платформу](https://aka.ms/windowsadmincenter).

## Управление компьютеров Windows Server

Вы можете добавить отдельные серверов под управлением Windows Server 2012 или более поздней версии для Windows Admin Center для управления на сервер с полным набором средств в том числе сертификаты, устройства, события, процессы, роли и функции, обновления, виртуальных машин и сведения.

![Общие сведения о экран подключения сервера](../media/manage-servers/server-overview.png)

## Добавление сервера в Windows Admin Center

Чтобы добавить сервер Windows Admin Center:

1. Выберите вариант **+ Добавить** всех подключений.
2. При необходимости добавьте **Подключения к серверу**.
3. Введите имя сервера и, если появится запрос, учетные данные для использования.
4. Нажмите кнопку **Отправить** до конца.

Сервер будет добавлен в список подключения на странице "Обзор". Щелкните его, чтобы подключиться к серверу.

> [!NOTE]
> Вы также можете добавить [отказоустойчивых кластеров](manage-failover-clusters.md) или [гиперконвергентными кластерами](manage-hyper-converged.md) как отдельное подключение в Windows Admin Center.

## Обзор

**Обзор** можно просмотреть текущее состояние ЦП, память и производительность сети, а также выполнять операции и изменять параметры на конечном компьютере или сервере.

### Возможности

В Обзор диспетчера сервера поддерживаются следующие функции:

- Просмотр сведений о сервере
- Просмотр ЦП активности
- Просмотр памяти активности
- Просмотр сетевой активности
- Перезапустите сервер
- Завершение работы сервера
- Включение метрики диска на сервере
- Изменить код компьютера на сервере
- Просмотр BMC IP-адрес с гиперссылкой (требуется BMC IPMI-совместимый).

[**Представление отзывы и предлагаемых функции Обзор сервера**](https://windowsserver.uservoice.com/forums/295071/filters/top?category_id=319162&query=%5BOverview%5D).

- Форматирование тома

- Установка управление квотами

# <a name="common-usage-scenarios"></a>Основные сценарии использования
MultiPoint Services предоставляет рабочие столы отдельного пользователя с самых важных элементов для рабочего стола Windows 10. Он также предлагает средство простого управления, используемый диспетчер MultiPoint, системные администраторы могут использовать для обнаружения и управления несколькими MultiPoint серверов и клиентов. Кроме того MultiPoint Services включает в себя MultiPoint Dashboard для обеспечения видимости в режиме реального времени. Вы можете с системой MultiPoint Services примеры ниже:  
  
- Предоставьте каждому пользователю личных вычислительной среды и личные папки без необходимости отдельный компьютер для каждого пользователя.  
- Управление нескольких систем MultiPoint в компьютерной лабораторией, аудитории, центр обучения или среде малого бизнеса.  
- Установите программу и доступ к нему с любой станции.  
- Наблюдать за каждого пользователя рабочего стола.  
- Блок экраны с настраиваемое сообщение, привлечет внимание группы.  
- Ограничьте группу только к одной или нескольких веб-сайтов.  
- Проект экрана в другие экраны для демонстрации определенной задачи.  
- Взаимодействовать с обычного пользователя, обращении за помощью в частном порядке.  
- Получить контроль над сеанса пользователя, для демонстрации задачи.  
- Сделать все элементы указанного выше для пользователя, пользующегося традиционных ПК, ноутбука или другое мобильное устройство.

# <a name="remap-selected-multipoint-systems"></a>Изменение сопоставления выбранных систем MultiPoint
Изменение сопоставления станций в MultiPoint Dashboard позволяет связывать с мониторами клавиатуры и мыши. Пока выполняется изменение сопоставления системы служб MultiPoint, работа станций локальных пользователей приостанавливается.

> [!CAUTION]
> Изменение сопоставления в основном используется в целях устранения неполадок. При изменении сопоставления параметры станции, такие как имя и данные для автоматического входа в систему, стираются.

#### <a name="to-remap-a-multipoint-services-system"></a>Изменение сопоставления системы служб MultiPoint

1.  В службах MultiPoint щелкните вкладку **Системы**.

2.  Выберите один или несколько эскизов серверов, для которых нужно изменить сопоставление, а затем на вкладке **Оборудование** выберите команду **Изменить сопоставление выбранной системы**.

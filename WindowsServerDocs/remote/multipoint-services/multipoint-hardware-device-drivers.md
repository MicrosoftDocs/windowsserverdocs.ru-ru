# <a name="collect-hardware-and-device-drivers-needed-for-the-installation"></a>Сбор драйверов оборудования и устройств, необходимых для установки
Прежде чем приступать к развертыванию системы MultiPoint Services, вам потребуется:  
  
-   **Компоненты оборудования для сервера** -установить любые дополнительные видео карты или других компонентов системы в настоящее время.  
  
-   **Компоненты оборудования для станций** — сведения о планирования станций для своей среды, см. в разделе [Выбор оборудования для операционной системы служб MultiPoint](Selecting-Hardware-for-Your-MultiPoint-services-System.md).
-   **Последние версии драйверов для вашей видеоадаптеров** -Если изготовитель вычислительной Техники или устройство не предоставил их, необходимо скачать с веб-сайт изготовителя устройства.  
  
-   **Последние версии драйверов USB zero клиента** -Если вы используете клиентские станции USB zero, необходимо установить последние версии драйверов USB zero клиента.  
  
    > [!IMPORTANT]  
    > Для установки служб MultiPoint необходимо установить 64-разрядной версии все драйверы.  
  
> [!TIP]  
> При установке MultiPoint Services на компьютере с Windows уже установлена другая версия, вы должны докопаться видеоадаптер марки и модели в диспетчере устройств, прежде чем начать установку Windows Server и убедитесь, что вы можете получить драйверы, которые являются доступно для Windows Server 2016. Откройте диспетчер устройств, откройте **Управление компьютером** из **запустить** экрана. Щелкните в дереве консоли щелкните **устройства**.

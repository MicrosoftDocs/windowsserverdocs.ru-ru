# <a name="take-control-of-a-user-session"></a>Получение контроля над сеансом пользователя
Как пользователь панели мониторинга MultiPoint, вы можете помогать другому пользователю, удаленно подключаясь к его рабочему столу с помощью функции "Взять управление".

1.  На панели мониторинга MultiPoint на вкладке **Главная** щелкните эскиз изображения рабочего стола для пользователя, которого вы хотите помогать.

2.  На вкладке **Помощь** щелкните **Взять управление**. Рабочий стол пользователя открывается на рабочем столе, а затем можно перемещаться по рабочему столу с помощью клавиатуры и мыши.

3.  По окончании работы с пользователем нажмите кнопку " **завершить**".

    > [!NOTE]
    > Может потребоваться свернуть Рабочий стол пользователя, чтобы увидеть панель мониторинга MultiPoint.

# <a name="take-control-of-a-user-session"></a>Получение контроля над сеансом пользователя
Как пользователь панели мониторинга MultiPoint, вы можете помогать другому пользователю, удаленно подключаясь к его рабочему столу с помощью функции "Взять управление".  
  
1.  На панели мониторинга MultiPoint на **Главная** щелкните эскиз рабочего стола для пользователя, которому требуется помочь.  
  
2.  На вкладке **Помощь** щелкните **Взять управление**. Рабочий стол пользователя откроется на вашем рабочем столе, и вы сможете управлять им с помощью клавиатуры и мыши.  
  
3.  Когда вы закончите, помогать пользователю, нажмите кнопку **остановить**.  
  
    > [!NOTE]  
    > Необходимо свести к минимуму рабочий стол пользователя для просмотра вашей панели мониторинга MultiPoint.

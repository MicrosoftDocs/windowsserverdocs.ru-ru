# <a name="deploy-a-new-windows-multipoint-services-system"></a>Развертывание новой системы Windows MultiPoint Services
В подразделах этого раздела объясняется, как настроить системы служб MultiPoint. Можно установить и настроить MultiPoint server; Настройка вашей станций; установить драйверы, обновления и программное обеспечение; При необходимости присоединения к домену; активировать MultiPoint Server; и добавьте клиентские лицензии (CAL) для каждой станции.  
  
> [!IMPORTANT]  
> Если вы еще не запланированы развертывания служб MultiPoint, см. в разделе [Планирование развертывания Windows MultiPoint Services](Planning-a-MultiPoint-Services-Deployment.md).

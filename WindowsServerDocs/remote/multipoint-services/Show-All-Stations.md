# <a name="show-all-stations"></a>Показать все станции
Как администратор в системе MultiPoint Services, можно просмотреть из всех станций, подключенных к системе, включая не\-активные и завершенные. Например в рамках лицензионного соглашения, может потребоваться проверить число станций, подключенных в системе MultiPoint Services. Можно просмотреть эти сведения с помощью MultiPoint Manager.  
  
1.  В обычном режиме откройте вкладку **Рабочие столы**.  
  
2.  В разделе **Общие задачи** щелкните **Показать все станции**.

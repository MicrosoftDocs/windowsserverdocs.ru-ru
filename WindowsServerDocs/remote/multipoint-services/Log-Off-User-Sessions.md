# <a name="log-off-user-sessions"></a>Завершение сеансов пользователей
Обычные пользователи, пользователи MultiPoint Dashboard и пользователи с правами администратора может войти в систему и выйдите из системы их сеансы рабочего стола, как и для любого сеанса Windows. Кроме того пользователи с правами администратора и пользователи MultiPoint Dashboard может завершить сеансы пользователей на всех наблюдаемых сеансах в системе MultiPoint Services.  
  
1.  На панели мониторинга MultiPoint щелкните **Главная** вкладки.  
  
2.  Выполните одно из следующих действий.  
  
    -   Чтобы выйти из сеанса работы одного пользователя или выбранных сеансов, выберите эскиз сеанса, который необходимо завершить и щелкните стрелку раскрывающегося меню верхнего левого. Нажмите кнопку **Завершить сеансы пользователей**, а затем — **Завершить сеансы выбранных пользователей**. Этот вариант также можно увидеть, щелкнув правой кнопкой мыши выбранные эскизы.
  
    -   Чтобы выйти все сеансы пользователей, щелкните раскрывающееся меню верхнего левого, выберите **пользователей**и нажмите кнопку **всех пользователей**.

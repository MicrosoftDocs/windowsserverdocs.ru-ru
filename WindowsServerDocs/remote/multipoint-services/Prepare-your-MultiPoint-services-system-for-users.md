# <a name="prepare-your-multipoint-services-system-for-users"></a>Подготовка системы MultiPoint Services для пользователей
После установки и настройки служб MultiPoint и выполнить дополнительную настройку и настроек оборудования, вы готовы предоставить пользователям доступ к системе. Необходимо будет для планирования и создания учетных записей пользователей. В некоторых средах необходимо также настроить станций для автоматического входа в систему и разрешает несколько сеансов для учетных записей пользователей общего. И вам нужно решить, как настроить общий доступ к файлам для пользователей. Все эти разделы рассматриваются в этом разделе.  
  
> [!NOTE]  
> После создания учетных записей пользователей и внести другие изменения конфигурации для подготовки пользователей, мы рекомендуем включить защиту диска, чтобы ни один пользователь может случайно изменить системные файлы и параметры. Дополнительные сведения см. в разделе [Настройка защиты диска](Configure-Disk-Protection-in-MultiPoint-services.md).

# <a name="view-user-connection-status"></a>Просмотр состояния подключения пользователя
Используйте вкладку **станции** для определения состояния подключения обычного или другого пользователя к станции служб MultiPoint.  
  
Варианты состояния станции  
  
-   **Выполнен выход**: сеанс пользователя на станции активен.  
  
-   **Приостановлен**: сеанс пользователя приостановлен, но по-прежнему активен. Сеанс рабочего стола пользователя сохраняется до тех пор, пока пользователь не войдет в систему снова  
  
-   **Выполнен выход**: вышедший из системы пользователь не отображается на вкладке **Станции**.  
  
Чтобы просмотреть состояние станции, откройте диспетчер MultiPoint в режиме станции и выберите пункт **станции**.

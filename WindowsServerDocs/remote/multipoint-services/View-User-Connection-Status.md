# <a name="view-user-connection-status"></a>Просмотр состояния подключения пользователя
Используйте вкладку **станции** для определения состояния подключения обычного или другого пользователя к станции служб MultiPoint.  
  
Варианты состояния станции  
  
-   **Вход выполнен**: Сеанс пользователя, активный на станции  
  
-   **Приостановлено**: Приостановленный сеанс пользователя, который по-прежнему активен на компьютере. Сеанс рабочего стола пользователя сохраняется до тех пор, пока пользователь не войдет в систему снова  
  
-   **Выполнен выход из системы**: Пользователь, который вышел из системы, не отображается на вкладке **станции**  
  
Чтобы просмотреть состояние станции, откройте диспетчер MultiPoint в режиме станции и выберите пункт **станции**.

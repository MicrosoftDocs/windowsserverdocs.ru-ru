# <a name="view-user-connection-status"></a>Просмотр состояния подключения пользователя
Используйте **станции** tab, чтобы определить состояние стандартного или подключения других пользователя с правами администратора к станции MultiPoint Services.  
  
Варианты состояния станции  
  
-   **Вошедший в систему**: Сеанс пользователя, который активен на станции  
  
-   **Приостановлено**: Сеанс пользователя, который приостанавливается, но по-прежнему активен на компьютере. Сеанс рабочего стола пользователя сохраняется до нового входа этого пользователя в систему.  
  
-   **Журнал отключить**: Пользователь, выхода из системы не отображается на **станции** вкладку  
  
Просмотр состояния станции, откройте диспетчер MultiPoint в режиме станции и нажмите кнопку **станции**.

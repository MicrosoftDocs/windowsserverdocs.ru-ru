# <a name="add-printers"></a>Добавление принтеров
Используйте процедуры в этом разделе, чтобы сделать доступными на локальном принтере для всех пользователей в системе MultiPoint Services.  
  
> [!NOTE]  
> Если вы используете учетные записи домена с системой MultiPoint Services, пользователи могут использовать любой принтер, сети из своих станциях.  
  
1.  Подключите принтер к серверу Multipoint.  
  
2.  Настройте принтер, в качестве общего принтера:  
  
    1.  Войдите на компьютер MultiPoint Server с правами администратора.  
  
    2.  На **начальном экране** откройте **Панель управления**.  
  
    3.  На панели управления выберите **оборудования**, а затем нажмите кнопку **устройства и принтеры**.  
  
    4.  В разделе **принтеры и факсы**, щелкните правой кнопкой мыши принтер и нажмите кнопку **свойства принтера**.  
  
    5.  Нажмите кнопку **доступ** вкладки.  
  
    6.  Нажмите кнопку **использование общих принтеров**, укажите имя общего принтера и нажмите кнопку **ОК**.  
  
Пользователи, вошедшие на любой станции, подключенной к компьютеру Multipoint Services будет иметь возможность видеть и использовать принтер.

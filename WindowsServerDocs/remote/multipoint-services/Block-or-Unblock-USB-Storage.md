# <a name="block-or-unblock-usb-storage"></a>Блокировка и отмена блокировки USB-накопителя
Вы можете запретить пользователям использовать запоминающие устройства USB на своих станциях.  
  
## <a name="to-block-usb-storage-for-selected-stations"></a>Блокировка USB-накопителей для выбранных станций  
1. На панели мониторинга MultiPoint щелкните станцию, которую нужно заблокировать.  
2. Щелкните **Блокировать** > **Блокировать выбранные рабочие столы**.   
  
## <a name="to-block-usb-storage-for-all-stations"></a>Блокировка USB-накопителей для всех станций  
Откройте панель мониторинга MultiPoint, а затем в раскрывающемся меню выберите **Блокировать запоминающие устройства на всех станциях**.   
  
## <a name="to-unblock-usb-storage-for-selected-stations"></a>Снятие блокировки USB-накопителей для выбранных станций  
Откройте панель мониторинга MultiPoint, выберите эскиз нужной станции и нажмите кнопку **Разблокировать**.

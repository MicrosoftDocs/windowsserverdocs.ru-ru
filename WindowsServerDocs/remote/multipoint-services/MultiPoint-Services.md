# <a name="multipoint-services"></a>Службы MultiPoint
MultiPoint Services — это решение, которое позволяет нескольким пользователям, каждый из которых предусмотрены собственные независимые и знакомых Windows опыта, одновременно используют один компьютер.

Пользовательские станции, состоящие из монитора, клавиатуры и мыши, подключаются к главному компьютеру напрямую через USB или видеокабели. Поскольку MultiPoint Services подлинность опубликованных программный продукт Майкрософт, при надлежащем лицензировании, вы имеете право на получение поддержки корпорацией Майкрософт или являетесь авторизованным партнером. Это дает все возможности Windows, доступ к все последние обновления и достоверности, которые получены требуемые возможности, которые предполагается.

Поскольку MultiPoint Services позволяет нескольким пользователям совместно использовать один компьютер, может предоставлять более низкой стоимостью по сравнению с традиционных вычислительных сценариях, где каждый пользователь имеет свои собственные компьютера. MultiPoint Services также представляет собой удобное управление инструмент для администрирования системы MultiPoint Services, известный как диспетчер MultiPoint и решение удобное управление за день\-для\-администрирования день, известный как MultiPoint Панель мониторинга.

# <a name="install-software-on-your-multipoint-services-system"></a>Установить программное обеспечение в вашей системе MultiPoint Services
При входе с правами администратора, можно установить новые программы в режиме консоли или, с рабочей станции, в режиме станции. Тем не менее мы рекомендуем установить программы в режиме консоли.  
  
Новое программное обеспечение можно установить на компьютере под управлением MultiPoint Server, чтобы все пользователи могут запускать программное обеспечение, или, чтобы только вы могли использовать программное обеспечение, в зависимости от установки и параметрах программы лицензирования.  
   
1.  Войдите на компьютер с MultiPoint Services с правами администратора.  
  
2.  Open MultiPoint Manager.  
  
3.  Нажмите кнопку **Главная** , а затем щелкните **переключиться в режим консоли**.  
  
4.  Войдите в систему как администратор и установки вашего приложения.  
  
5.  После завершения установки приложений, перевести в режим станции. Для этого на **Главная** щелкните **переключиться в режим станции**.

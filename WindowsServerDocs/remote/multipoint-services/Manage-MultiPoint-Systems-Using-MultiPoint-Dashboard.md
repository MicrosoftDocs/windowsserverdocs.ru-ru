# <a name="manage-multipoint-systems-using-multipoint-dashboard"></a>Управление системами MultiPoint с помощью MultiPoint Dashboard
На панели мониторинга MultiPoint на вкладке **системы** можно выполнить следующие действия.

- Перезапуск или завершение работы выбранных систем. Вы можете перезапустить или завершить работу выбранных систем служб MultiPoint. Дополнительные сведения см. в статье [Перезагрузка и завершение работы систем MultiPoint](Restart-or-Shut-Down-MultiPoint-Systems.md).
- Повторное сопоставление выбранных систем можно сопоставить выбранные системы служб MultiPoint. Дополнительные сведения см. в статье [Изменение сопоставления выбранных систем MultiPoint](Remap-Selected-MultiPoint-Systems.md).

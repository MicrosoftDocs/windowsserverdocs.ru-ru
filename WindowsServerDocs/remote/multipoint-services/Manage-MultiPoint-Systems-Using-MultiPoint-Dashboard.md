# <a name="manage-multipoint-systems-using-multipoint-dashboard"></a>Управление системами MultiPoint с помощью панели мониторинга MultiPoint
В MultiPoint Dashboard на **систем** вкладке, можно сделать следующее:  
  
- Перезагрузка или завершение работы выбранных систем  
Вы можете выполнить перезагрузку или завершение работы выбранных систем служб MultiPoint. Дополнительные сведения см. в статье [Перезагрузка и завершение работы систем MultiPoint](Restart-or-Shut-Down-MultiPoint-Systems.md).   
- Изменение сопоставления выбранных систем  
Вы можете изменить сопоставление выбранных систем служб MultiPoint. Дополнительные сведения см. в статье [Изменение сопоставления выбранных систем MultiPoint](Remap-Selected-MultiPoint-Systems.md).

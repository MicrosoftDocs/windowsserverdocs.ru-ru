# <a name="project-a-station-to-other-stations"></a>Проецирование станции на другие станции
Как пользователь MultiPoint Dashboard может проецировать свой рабочий стол одного пользователя станцию или станции (без прав администратора) всех пользователей. Эта возможность полезна для демонстрации задачи пользователям.  
  
## <a name="to-project-your-desktop-to-a-standard-users-station"></a>В проект рабочего стола станции обычного пользователя.  
  
1.  На панели мониторинга MultiPoint выберите эскиз рабочего стола, к которому вы хотите проецировать свою станцию.  
2.  На **Главная** щелкните **ваш рабочий стол**, а затем нажмите кнопку **проецировать рабочий стол для выбранные рабочие столы**.  
3.  Чтобы завершить проецирование, нажмите кнопку **остановить** (либо на **проекции** вкладке или в правом углу под лентой).  
  
## <a name="to-project-your-desktop-to-all-stations"></a>Проецирование рабочего стола на все станции  
  
1.  В MultiPoint Dashboard на **Главная** щелкните **ваш рабочий стол**, а затем нажмите кнопку **проецировать рабочий стол на все рабочие столы**.  
  
2.  Чтобы завершить проецирование, нажмите кнопку **остановить** (либо на **проекции** вкладке или в правом углу под лентой).  
  
## <a name="to-project-a-different-desktop-to-all-desktops"></a>Проецирование другого рабочего стола на все станции  
  
1.  Выберите эскиз рабочего стола, который необходимо спроецировать на все рабочие столы.  
  
2.  На **Главная** щелкните **выбранный рабочий стол**, а затем нажмите кнопку **выбранный рабочий стол**. Выбранный рабочий стол проецируется на все рабочие столы.  
  
3.  Чтобы завершить проецирование, нажмите кнопку **остановить** (либо на **проекции** вкладке или в правом углу под лентой).

# <a name="important-information-about-software-license-compliance"></a>Важные сведения о соответствии требованиям лицензионных соглашений
Вы должны соблюдать условия лицензии для программ, которые используются в системе MultiPoint Services. Необходимо также иметь верное количество лицензий для числа *станции* , добавляемой к системе MultiPoint Services, как описано в следующих разделах.  
  
## <a name="software-license-compliance"></a>Соответствия требованиям лицензионных соглашений  
Если вы используете другие программы корпорации Майкрософт или любые сторонние программы с программным обеспечением Microsoft или службы, условия, входящие в состав таких программ применяются к использованию их. Microsoft может для достижения любой сбой в соответствии с этими условиями, чтобы также являться нарушением лицензии от корпорации Майкрософт.  
  
## <a name="client-access-license-cal-compliance"></a>Соответствия лицензии (CAL) для доступа клиента  
Если вы хотите добавить станцию в системе MultiPoint Services, необходимо защитить, соответствующее числу *клиентских лицензий (CAL)* для всех станций подключены к службам MultiPoint.

# <a name="planning-worksheet-for-multipoint-services-migration"></a>Таблица планирования миграции MultiPoint Services

>Область применения. Windows Server 2016

Используйте приведенных ниже списках и таблиц для сбора параметров, которые потребуются во время миграции MultiPoint Services.

## <a name="source-server-settings"></a>Параметры исходного сервера

Параметры сервера можно найти на **Главная** вкладка в диспетчер MultiPoint. Установите флажок рядом с каждым параметром, используется на исходном сервере.

- Разрешить несколько сеансов для одной учетной записи.
- Разрешить удаленное управление этим компьютером.
- Разрешить наблюдения за настольными системами на этом компьютере.
- Всегда запустите в режиме консоли.
- Не показывать уведомление о конфиденциальности при первом входе пользователя.
- Назначьте уникальный IP-адрес для каждой станции.
- Разрешить обмен мгновенными Сообщениями между MultiPoint Dashboard и сеансов пользователей на этом компьютере.
- Разрешить оркестрацию администратора и сеансов пользователей MultiPoint Dashboard.
- Разрешить станциям использовать отрисовку графического Процессора.

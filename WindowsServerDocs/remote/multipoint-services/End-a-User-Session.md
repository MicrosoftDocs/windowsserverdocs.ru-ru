# <a name="end-a-user-session"></a>Завершение сеанса пользователя
Необходимо завершить сеанс пользователя, если необходимо выполнить вход пользователя из системы служб MultiPoint, чтобы вернуть параметры рабочего стола по умолчанию. Он получит предупреждение о завершении соединения. Необходимо завершить подключение пользователя, если вы хотите:  
  
-   перезапустить компьютер с системой MultiPoint Services;  
  
-   выключить компьютер с системой MultiPoint Services;  
  
-   перейти в другой режим;  
  
-   завершить сеанс пользователя, забывшего выполнить выход.  
  
Для конечных сеансов пользователей:  
  
1.  Откройте диспетчер MultiPoint в режиме станции и перейдите на вкладку **станции** .  
  
2.  Выполните одно из следующих действий.  
  
    -   Чтобы завершить один сеанс пользователя, выберите нужный сеанс в столбце **Пользователь**, а затем в разделе **Задачи** нажмите кнопку **Отключить**.  
  
    -   Чтобы завершить все сеансы пользователей, в разделе **Задачи станций** нажмите кнопку **Отключить все станции**.

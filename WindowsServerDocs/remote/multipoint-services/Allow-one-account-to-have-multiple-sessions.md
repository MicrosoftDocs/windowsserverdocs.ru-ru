# <a name="allow-one-account-to-have-multiple-sessions"></a>Разрешение запуска нескольких сеансов для одной учетной записи
Чтобы разрешить группе пользователей использовать общую учетную запись на нескольких станциях одновременно, настройте сервер MultiPoint так, чтобы одна учетная запись одновременно была зарегистрирована на нескольких станциях. По умолчанию, если пользователь входит на вторую станцию с общей учетной записью пользователя, учетная запись пользователя завершается с первой станции.  
  
1.  На **начальном** экране откройте **диспетчер MultiPoint**.  
  
2.  Перейдите на вкладку **Главная** .  
  
3.  В столбце **компьютер** щелкните имя компьютера MultiPoint Server, а затем в правой области щелкните **изменить параметры сервера**.  
  
4.  Установите флажок **Разрешить одной учетной записи использовать несколько сеансов** и нажмите кнопку **ОК**.

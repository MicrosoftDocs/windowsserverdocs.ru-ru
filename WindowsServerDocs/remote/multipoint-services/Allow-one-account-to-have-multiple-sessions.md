# <a name="allow-one-account-to-have-multiple-sessions"></a>Разрешение запуска нескольких сеансов для одной учетной записи
Чтобы включить группу пользователей использования общей учетной записи на несколько станций в то же время, настройте MultiPoint сервера, чтобы одна учетная запись входа в систему несколько станций под одновременно. По умолчанию если пользователь выполняет вход в второй станции с общей учетной записью, учетной записи пользователя выхода из системы первого станции.  
  
1.  Из **запустить** экране откройте **MultiPoint Manager**.  
  
2.  Нажмите кнопку **Главная** вкладки.  
  
3.  В **компьютера** столбца, щелкните имя сервера MultiPoint и затем в области справа щелкните **изменение параметров сервера**.  
  
4.  Выберите **разрешить несколько сеансов для одной учетной записи** флажок и нажмите кнопку **ОК**.

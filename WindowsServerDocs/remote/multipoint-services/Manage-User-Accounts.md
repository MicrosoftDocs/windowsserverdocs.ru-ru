# <a name="manage-user-accounts"></a>Управление учетными записями пользователей
В этой статье описаны различные типы учетных записей пользователей, порядок их создания и управления ими. В системе служб MultiPoint существует два типа учетных записей пользователей: *стандартные учетные записи пользователей* и *административные учетные записи пользователей*, как описано ниже.

> [!TIP]
> В разделе [Соображения по поводу учетных записей пользователей](User-Account-Considerations.md) описаны правила, которые следует учесть при создании учетных записей пользователей и управлении ими.

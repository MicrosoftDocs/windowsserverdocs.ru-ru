# <a name="set-up-a-station-for-automatic-logon"></a>Настройка автоматического входа для станции
Автоматический вход позволяет каждой станции автоматически войти в систему при запуске компьютера, на котором выполняется MultiPoint Services и отображения рабочего стола. Пользователь с правами администратора может задать эту функцию для отдельных или для всех станций.  
  
1.  Откройте диспетчер MultiPoint в режиме станции и нажмите кнопку **станции** вкладки.  
  
2.  Выберите имя станции, которой необходимо автоматически войти в систему.  
  
3.  В разделе **Задачи** щелкните **Настроить станцию**. Откроется страница **Настройка станций**.  
  
4.  Выберите **автоматического входа, используя следующие сведения**, а затем введите **учетной записи пользователя** имя.  
  
5.  Введите пароль для учетной записи пользователя, после чего повторно введите пароль для подтверждения.  
  
6.  Нажмите кнопку **ОК**. Страница будет закрыта. Имя учетной записи отображается в **автоматического входа** столбца.

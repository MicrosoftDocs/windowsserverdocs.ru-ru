# <a name="configure-stations-for-automatic-logon"></a>Настройка автоматического входа для станций
Будет доступен никому — ваши рабочие станции и пользователи не обязательно личные папки для хранения своих персональных данных или персонализированные настольных ПК, можно настроить станции на автоматический вход. Автоматический вход в систему автоматически входит в учетную запись пользователя, который был указан в параметрах автоматического входа в систему при запуске MultiPoint Services.  
  
1.  Из **запустить** экране откройте **MultiPoint Manager**.  
  
2.  Нажмите кнопку **станции** , а затем щелкните имя станции, который вы хотите настроить для автоматического входа в систему.  
  
3.  В области справа щелкните **Настройка автоматического входа**.  
  
    Откроется страница настройки автоматического входа.  
  
4.  Выберите **автоматического входа, используя следующие сведения** флажок и введите учетную запись и пароль, используемый для автоматического входа в систему. Нажмите кнопку **ОК**.  
  
    > [!NOTE]  
    > Учетной записи пользователя, который используется для автоматического входа должно иметь пароль.  
  
> [!NOTE]  
> Чтобы временно войти на станцию, которая настроена для автоматического входа в систему с учетной записью другого пользователя, наведите указатель мыши правом верхнем углу экрана, чтобы отобразить вертикальное меню, щелкните значка «параметры», щелкните этот значок, удерживайте клавишу SHIFT и щелкните **</c0Настроекотключение<spanclass="notranslate">**.</span> Удерживайте нажатой клавишу SHIFT, до появления приглашения входа в систему.

# <a name="step-1-plan-an-advanced-single-server-deployment"></a>Шаг 1. Планирование расширенного развертывания с одним сервером

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016

Первый шаг в планировании удаленного доступа с помощью проверки подлинности клиента одноразового пароля (OTP) заключается в планировании и настройке расширенного развертывания с одним сервером.  
  
## <a name="plan-a-single-server-deployment"></a>Планирование развертывания на одном сервере  
Перед развертыванием удаленного доступа с помощью OTP убедитесь, что выполнены все действия по развертыванию одного сервера удаленного доступа. См. раздел [развертывание одного сервера DirectAccess с дополнительными параметрами](https://technet.microsoft.com/windows-server-docs/networking/remote-access/directaccess/single-server-advanced/deploy-a-single-directaccess-server-with-advanced-settings).

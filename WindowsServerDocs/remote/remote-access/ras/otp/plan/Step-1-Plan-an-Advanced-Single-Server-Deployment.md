# <a name="step-1-plan-an-advanced-single-server-deployment"></a>Шаг 1 Планирование развертывания Advanced Server единый

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Для планирования и настроить развертывание дополнительных один сервер является первым шагом в планировании удаленного доступа при развертывании проверки подлинности клиента одноразовых паролей (OTP).  
  
## <a name="plan-a-single-server-deployment"></a>Планирование развертывания одиночного сервера  
Перед развертыванием удаленного доступа с OTP, убедитесь, что выполнены все этапы для развертывания одного сервера удаленного доступа. См. в разделе [развертывание одиночного сервера DirectAccess с расширенными параметрами](https://technet.microsoft.com/windows-server-docs/networking/remote-access/directaccess/single-server-advanced/deploy-a-single-directaccess-server-with-advanced-settings).

В этом разделе представлены этапы настройки, необходимые для развертывания одного сервера удаленного доступа, который можно использовать для удаленного управления клиентами DirectAccess.  
  
-   [Шаг 1. Настройте инфраструктуру удаленного доступа @ no__t-0: В этом разделе описывается, как настроить параметры сети и сервера, требования к сертификатам, параметры DNS, развертывание сервера сетевого расположения, серверы управления DirectAccess, параметры Active Directory и объекты групповая политика.  
  
-   [Шаг 2. Настройте сервер удаленного доступа @ no__t-0: В этом разделе описывается настройка клиентских компьютеров DirectAccess, параметров сервера и инфраструктуры и серверов приложений.  
  
-   [Шаг 3. Проверьте развертывание @ no__t-0: В этом разделе описывается проверка развертывания.

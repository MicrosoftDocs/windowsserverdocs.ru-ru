# <a name="deploy-remote-access-in-a-multi-forest-environment"></a>Deploy Remote Access in a Multi-Forest Environment

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Средства конфигурации удаленного доступа (консоль управления удаленным доступом и командлеты Windows PowerShell) предназначены для работы в едином лесу, содержащем один или несколько доменов. Тем не менее, при развертывании удаленного доступа в многолесовой среде администратор удаленного доступа должен выполнить некоторые настройки вручную, чтобы обеспечить успешное развертывание. В этом руководстве описаны этапы планирования и настройки для многолесового развертывания, в том числе при использовании проверки подлинности с помощью одноразового пароля (OTP).

-   [Настройка многолесового развертывания](Configure-a-Multi-Forest-Deployment.md)

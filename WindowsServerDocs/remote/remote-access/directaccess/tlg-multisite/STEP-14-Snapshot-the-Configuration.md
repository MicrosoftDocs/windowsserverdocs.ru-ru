# <a name="step-14-snapshot-the-configuration"></a>ШАГ 14 сделайте снимок конфигурации

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

На этом завершается лабораторной среде многосайтового DirectAccess. Чтобы сохранить эту конфигурацию, чтобы получить возможность быстро вернуться к работающей DirectAccess многосайтового конфигурации из которого вы сможете проверять другие модульная DirectAccess руководства по лаборатории тестирования, расширения руководство по лаборатории тестирования или для собственных экспериментов и обучения, сделайте следующее :  
  
1.  На всех физических компьютерах или виртуальных машинах лаборатории тестирования закройте все окна и выполните нормальное завершение работы.  
  
2.  Если ваша лаборатория основана на виртуальных машинах, сохраните моментальный снимок каждой виртуальной машины и назовите снимки Мультисайт DirectAccess лаборатории тестирования. Если ваша лаборатория использует физические компьютеры, создайте образы диска, чтобы сохранить конфигурацию лаборатории тестирования DirectAccess.

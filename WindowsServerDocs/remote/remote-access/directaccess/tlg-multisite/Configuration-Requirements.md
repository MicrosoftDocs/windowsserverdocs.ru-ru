# <a name="configuration-requirements"></a>Требования к конфигурации

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Следующие компоненты необходимы для настройки удаленного доступа в лаборатории тестирования:  
  
-   Диск с продуктом или файлы для Windows Server 2016, Windows Server 2012 R2 или Windows Server 2012.  
  
-   Девять компьютеров или виртуальных машин, которые удовлетворяют минимальным требованиям для Windows Server 2016, Windows Server 2012 R2 или Windows Server 2012; три из этих компьютеров имеют два сетевых адаптера.  
  
-   Диск с продуктом или файлы для Windows 10&reg; или Windows&reg; 8.  
  
-   Диск с продуктом или файлы для Windows 7 Максимальная.  
  
-   Три компьютеры или виртуальные машины, которые удовлетворяют минимальным требованиям для Windows 10&reg;, Windows 8 или Windows 7; один из этих компьютеров имеет два сетевых адаптера.

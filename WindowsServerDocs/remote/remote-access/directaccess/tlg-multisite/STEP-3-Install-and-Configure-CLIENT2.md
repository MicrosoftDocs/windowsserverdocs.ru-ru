# <a name="step-3-install-and-configure-client2"></a>Шаг 3 Установка и настройка CLIENT2

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

CLIENT2 — это в Windows 7&reg; компьютера, который используется для демонстрации обратной совместимости удаленного доступа, выполняемым на серверах Windows Server 2016.  
  
1. Для установки операционной системы на CLIENT2. Установка Windows&reg; 7 Корпоративная или Windows&reg; 7 Ultimate на CLIENT2.  
  
2. CLIENT2 присоединение к домену CORP. CLIENT2 присоединения к домену corp.contoso.com.  
  
## <a name="to-install-the-operating-system-on-client2"></a>Для установки операционной системы на CLIENT2  
  
1.  Начните установку Windows 7.  
  
2.  При появлении запроса имени пользователя введите **User1**. При появлении запроса имени компьютера введите **CLIENT2**.  
  
3.  При запросе пароля введите надежный пароль два раза.  
  
4.  Когда появится запрос для параметров защиты, нажмите кнопку **использовать рекомендуемые параметры**.  
  
5.  Когда появится запрос для текущего расположения компьютера, нажмите кнопку **Рабочая сеть**.  
  
6.  Подключение к сети, которая имеет доступ к Интернету CLIENT2 и запустите Центр обновления Windows Чтобы установить последние обновления для Windows 7, а затем отключиться от Интернета.  
  
7.  CLIENT2 подключиться к подсети Corpnet.  
  
## <a name="user-account-control"></a>Контроль учетных записей пользователей  
При настройке операционной системы Windows 7 необходимо щелкните **Продолжить** на **контроль учетных записей пользователей** диалоговое окно (UAC) для выполнения некоторых задач. Некоторые задачи по настройке требуется утверждение контроля учетных Записей. При появлении соответствующего запроса, всегда щелкнуть **Продолжить** авторизовать эти изменения.  
  
## <a name="to-join-client2-to-the-corp-domain"></a>Для присоединения к домену CORP CLIENT2  
  
1.  Щелкните **Запустить**, щелкните правой кнопкой мыши **Компьютер**, затем нажмите кнопку **Свойства**.  
  
2.  На **системы** странице **имя, домен и рабочая группа параметров компьютера** область, нажмите кнопку **изменить параметры**.  
  
3.  В диалоговом окне **"Свойства системы"** на вкладке **"Имя компьютера"** щелкните **"Изменить"** .  
  
4.  На **изменение имени компьютера или домена** диалоговом окне щелкните **домена**, тип **corp.contoso.com**, а затем нажмите кнопку **ОК**.  
  
5.  При появлении запроса имени пользователя и пароля введите имя пользователя и пароль для учетной записи User1 домена и нажмите кнопку **ОК**.  
  
6.  При появлении диалогового окна с приветствием домена corp.contoso.com нажмите кнопку **ОК**.  
  
7.  При появлении диалогового окна, предложит перезагрузить компьютер, нажмите кнопку **ОК**.  
  
8.  На **системные свойства** диалоговом окне щелкните **закрыть**, и когда появится диалоговое окно, предлагающее перезагрузите компьютер, нажмите кнопку **Перезагрузить сейчас**.  
  
9. После перезагрузки компьютера войдите учетной записью CORP\User1.

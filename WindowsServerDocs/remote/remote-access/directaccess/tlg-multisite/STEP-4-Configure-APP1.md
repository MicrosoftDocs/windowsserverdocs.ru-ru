# <a name="step-4-configure-app1"></a>Шаг 4 Настройка APP1

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Настройка статической адресации IPv6 и параметры шлюза, чтобы включить APP1 доступ к подсети 2 Corpnet.  
  
- Чтобы настроить шлюз по умолчанию и DNS-сервер. Конфигурации многосайтового развертывания использует компьютер ROUTER1 как шлюз по умолчанию. Настройка шлюза по умолчанию на APP1.  
  
## <a name="to-configure-the-default-gateway-and-dns-server"></a>Чтобы настроить шлюз по умолчанию и DNS-сервер  
  
1.  В консоли диспетчера серверов щелкните **локального сервера**, а затем в **свойства** область, рядом с полем **проводное Ethernet-подключение**, щелкните ссылку.  
  
2.  В **сетевых подключений** окно, щелкните правой кнопкой мыши **проводное Ethernet-подключение**, а затем нажмите кнопку **свойства**.  
  
3.  На **свойства проводного Ethernet-подключения** диалоговом окне щелкните **Internet Protocol версии 4 (TCP/IPv4)** , а затем нажмите кнопку **свойства**.  
  
4.  В **шлюз по умолчанию**, тип **10.0.0.254**и в **Альтернативный DNS-сервер**, тип **10.2.0.1**, а затем нажмите кнопку **ОК** .  
  
5.  На **свойства проводного Ethernet-подключения** диалоговом окне щелкните **Internet Protocol версии 6 (TCP/IPv6)** , а затем нажмите кнопку **свойства**.  
  
6.  В **шлюз по умолчанию**, тип **2001:db8:1::fe**. В **Альтернативный DNS-сервер**, тип **2001:db8:2::1**, а затем нажмите кнопку **ОК**.  
  
7.  На **свойства проводного Ethernet-подключения** диалоговом окне нажмите кнопку **закрыть**, а затем закройте **сетевых подключений** окно.

# <a name="step-9-configure-edge1"></a>ШАГ 9 Настройка EDGE1

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

На сервере EDGE1 необходимо выполнить следующие процедуры:  
  
1. Настройка DNS-серверы на EDGE1. Это необходимо настроить DNS-сервера из домена corp2.corp.contoso.com на EDGE1.  
  
2. Настройте маршрутизацию между подсетями. Настройка маршрутизации в EDGE1 для обеспечения взаимодействия между подсетями корпоративной сети и корпоративной сети. 2.  
  
## <a name="IPv6"></a>Настройка DNS-серверы на EDGE1  
  
1.  В консоли диспетчера серверов щелкните **локального сервера**, а затем в **свойства** область, рядом с полем **Corpnet**, щелкните ссылку.  
  
2.  В окне «Сетевые подключения» щелкните правой кнопкой мыши **Corpnet**, а затем нажмите кнопку **свойства**.  
  
3.  Щелкните пункт **IP версия 4 (TCP/IPv4)** и нажмите кнопку **Свойства**.  
  
4.  В **Альтернативный DNS-сервер**, тип **10.2.0.1**. а затем нажмите кнопку **ОК**.  
  
5.  Нажмите кнопку **Internet Protocol версии 6 (TCP/IPv6)** , а затем нажмите кнопку **свойства**.  
  
6.  В **Альтернативный DNS-сервер**, тип **2001:db8:2::1** и нажмите кнопку **ОК**.  
  
7.  На **свойствам корпоративной сети** диалоговом окне щелкните **закрыть**.  
  
8.  Закройте окно **Сетевые подключения** .  
  
## <a name="ConfigRouting"></a>Настройка маршрутизации между подсетями  
  
1.  На **запустить** введите**cmd.exe**, щелкните правой кнопкой мыши **cmd**, нажмите кнопку **Дополнительно**, а затем нажмите кнопку **Запуск от имени Администратор**. Если появится диалоговое окно **контроля учетных записей**, подтвердите, что отображаемое в нем действие именно то, которое требуется, и нажмите кнопку **Да**.  
  
2.  В окне командной строки введите следующие команды. После ввода каждой команды, нажмите клавишу ВВОД.  
  
    ```  
    netsh interface IPv4 add route 10.2.0.0/24 Corpnet 10.0.0.254  
    netsh interface IPv6 add route 2001:db8:2::/64 Corpnet 2001:db8:1::fe  
    ```  
  
3.  Закройте окно командной строки.

# <a name="step-9-configure-edge1"></a>Шаг 9. Настройка EDGE1

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016

На сервере EDGE1 выполняются следующие процедуры:  
  
1. Настройте DNS-серверы в EDGE1. Необходимо настроить DNS-сервер из домена corp2.corp.contoso.com в EDGE1.  
  
2. Настройте маршрутизацию между подсетями. Настройте маршрутизацию на EDGE1, чтобы обеспечить взаимодействие между корпоративной и подсетями, расположенными в корпоративной сети.  
  
## <a name="configure-the-dns-servers-on-edge1"></a><a name="IPv6"></a>Настройка DNS-серверов в EDGE1  
  
1.  В консоли диспетчер сервера щелкните **локальный сервер**, а затем в области **свойства** рядом с параметром Корпоративная **Сеть щелкните ссылку**.  
  
2.  В окне Сетевые подключения щелкните правой кнопкой **мыши элемент**Корпоративная сеть и выберите пункт **свойства**.  
  
3.  Щелкните пункт **IP версия 4 (TCP/IPv4)** и нажмите кнопку **Свойства**.  
  
4.  В **качестве альтернативного DNS-сервера**введите **10.2.0.1**. а затем нажмите кнопку **ОК**.  
  
5.  Выберите **Протокол IP версии 6 (TCP/IPv6)** , а затем **Свойства**.  
  
6.  В **альтернативном DNS-сервере**введите **2001: db8:2:: 1** и нажмите кнопку **ОК**.  
  
7.  В диалоговом окне **Свойства корпоративной** сети нажмите кнопку **Закрыть**.  
  
8.  Закройте окно **Сетевые подключения**.  
  
## <a name="configure-routing-between-subnets"></a><a name="ConfigRouting"></a>Настройка маршрутизации между подсетями  
  
1.  На **начальном** экране введите**cmd. exe**, щелкните правой кнопкой мыши **cmd**, выберите **Дополнительно**и щелкните **Запуск от имени администратора**. Если появится диалог **Контроль учетных записей**, подтвердите, что действие, которое оно отображает то, что нужно, затем щелкните **Да**.  
  
2.  В окне командной строки введите следующие команды. После ввода каждой команды нажмите клавишу ВВОД.  
  
    ```  
    netsh interface IPv4 add route 10.2.0.0/24 Corpnet 10.0.0.254  
    netsh interface IPv6 add route 2001:db8:2::/64 Corpnet 2001:db8:1::fe  
    ```  
  
3.  Закройте окно командной строки.

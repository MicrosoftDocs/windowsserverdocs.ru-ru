# <a name="directaccess-deployment-paths-in-windows-server"></a>Возможности развертывания DirectAccess в Windows Server

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Этот раздел содержит перечень документации по два основным способам развертывания удаленного доступа: **Основные** и **Advanced**.  
  
В разделе ниже позволяет получить представление о различиях между DirectAccess Basic и путей расширенного развертывания и ссылки на документацию можно использовать для поиска в руководстве по развертыванию, который лучше всего подходит для ваших целей.  
  
## <a name="deploy-basic-directaccess"></a>Развертывания базового DirectAccess  
С помощью базового развертывания DirectAccess настройки DirectAccess с параметрами по умолчанию с помощью мастера, отсутствует необходимость настраивать параметры инфраструктуры, такие как центр сертификации (ЦС) или группы безопасности Active Directory.  
  
-   [Развертывание одиночного сервера DirectAccess с помощью мастера начальной настройки](../../remote-access/directaccess/single-server-wizard/Deploy-a-Single-DirectAccess-Server-Using-the-Getting-Started-Wizard.md). В этом руководстве можно использовать для развертывания базового DirectAccess в рабочей среде.  
  
## <a name="deploy-advanced-directaccess"></a>Развертывание DirectAccess с расширенными параметрами  
С помощью расширенного развертывания DirectAccess чтобы развертывание одиночного сервера DirectAccess и настроить серверы инфраструктуры сети для поддержки DirectAccess.  
  
-   [Развертывание одиночного сервера DirectAccess с расширенными параметрами](../../remote-access/directaccess/single-server-advanced/Deploy-a-Single-DirectAccess-Server-with-Advanced-Settings.md). В этом руководстве можно использовать для развертывания DirectAccess с расширенными параметрами в рабочей среде.

# <a name="step-5-verify-otp-health-on-edge1"></a>Шаг 5 Проверка работоспособности OTP на EDGE1

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Следующие процедуры убедитесь, что OTP настроена и работает правильно в EDGE1 на наблюдение за работоспособностью сервера DirectAccess.  
  
### <a name="verify-otp-health-on-edge1-using-directaccess-server-health-monitoring"></a>Проверьте работоспособность OTP на EDGE1 с помощью наблюдение за работоспособностью сервера DirectAccess  
  
1.  Откройте на EDGE1, **управления удаленным доступом** консоли.  
  
2.  Нажмите кнопку **состояние операций**.  
  
3.  Убедитесь, что состояние OTP **работа**.

# <a name="step-5-verify-otp-health-on-edge1"></a>Шаг 5. Проверка работоспособности OTP в EDGE1

>Область применения. Windows Server (Semi-Annual Channel), Windows Server 2016

Следующие процедуры проверяют, что OTP настроен и правильно работает с помощью мониторинга работоспособности сервера DirectAccess в EDGE1.

### <a name="verify-otp-health-on-edge1-using-directaccess-server-health-monitoring"></a>Проверка работоспособности OTP в EDGE1 с помощью мониторинга работоспособности сервера DirectAccess

1.  В EDGE1 откройте консоль **управления удаленным доступом** .

2.  Щелкните **состояние операции**.

3.  Убедитесь, что состояние OTP **работает**.

# <a name="step-8-snapshot-the-directaccess-cluster-nlb-configuration"></a>ШАГ 8 сделайте снимок конфигурации кластера балансировки сетевой Нагрузки DirectAccess

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

На этом завершается лаборатория тестирования DirectAccess. Чтобы сохранить эту конфигурацию, чтобы получить возможность быстро вернуться к работающей DirectAccess с конфигурацией кластера балансировки сетевой Нагрузки, с которой вы сможете проверять другие руководства по лаборатории тестирования DirectAccess, расширения руководство по лаборатории тестирования или для собственных экспериментов и обучения следующие:  
  
1.  На всех физических компьютерах или виртуальных машинах лаборатории тестирования закройте все окна и выполните нормальное завершение работы.  
  
2.  Если ваша лаборатория основана на виртуальных машинах, сохраните моментальный снимок каждой виртуальной машины и имя кластера DirectAccess моментальные снимки и балансировки сетевой Нагрузки. Если ваша лаборатория использует физические компьютеры, создайте образы диска, чтобы сохранить конфигурацию лаборатории тестирования DirectAccess.

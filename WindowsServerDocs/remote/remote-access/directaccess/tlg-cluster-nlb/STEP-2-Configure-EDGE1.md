# <a name="step-2-configure-edge1"></a>Шаг 2 Настройка EDGE1

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Следующая процедура выполняется на сервере DirectAccess:

## <a name="to-configure-directaccess-on-edge1"></a>Чтобы настроить DirectAccess на EDGE1
  
1.  На **запустить** введите**RAMgmtUI.exe**, и нажмите клавишу ВВОД. Если появится диалоговое окно **контроля учетных записей**, подтвердите, что отображаемое в нем действие именно то, которое требуется, и нажмите кнопку **Да**.  
  
2.  В консоли управления удаленным доступом в области слева щелкните **конфигурации**.  
  
3.  В средней области консоли в **шаг 2 сервер удаленного доступа** область, нажмите кнопку **изменить**.  
  
4.  В **установки сервера удаленного доступа** мастера, нажмите кнопку **Конфигурация префикса**. На **Конфигурация префикса** страницы **префикс IPv6, назначенный клиентскому компьютеру DirectAccess**, введите **2001:db8:1:1000:: / 59**, а затем нажмите кнопку **Далее** .  
  
5.  Нажмите кнопку **Готово**.  
  
6.  В средней области консоли щелкните **Готово**.  
  
7.  На **удаленного доступа просмотрите** диалоговое окно, просмотрите параметры конфигурации и нажмите кнопку **применить**. В диалоговом окне **Применение параметров мастера настройки удаленного доступа** нажмите кнопку **Закрыть**.

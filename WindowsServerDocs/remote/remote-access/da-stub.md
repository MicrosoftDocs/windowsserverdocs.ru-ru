# <a name="directaccess"></a>DirectAccess

>Область применения. Windows Server (Semi-Annual Channel), Windows Server 2016

В Windows Server 2016 **DirectAccess и VPN** являются службой роли в **роли сервера удаленного доступа**.

DirectAccess позволяет удаленным пользователям подключаться к сетевым ресурсам Организации без необходимости в традиционных подключениях к виртуальной частной сети (VPN). 

Документация DirectAccess теперь находится в разделе [Удаленный доступ и управление серверами](../index.yml) оглавления Windows Server 2016 в разделе [Удаленный доступ](./remote-access.md). Дополнительные сведения см. в разделе [DirectAccess](directaccess/DirectAccess.md).

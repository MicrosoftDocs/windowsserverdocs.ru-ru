# <a name="remote-desktop-services---choose-how-you-pay"></a>Службы удаленных рабочих столов — Выбор способа оплаты

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Выберите в программе лицензирования зависимости от того, что имеет смысл для вашей компании. Лицензии на каждого пользователя, чтобы пользователи могли удаленным на любых устройствах в сценарии BYOD. Лицензия на устройство, если пользователи совместно используют те же устройства. Если вы являетесь поставщиком услуг (поставщик услуг Размещения или MSP) или независимого поставщика программного обеспечения выберите за каждую лицензию пользователя SALs для модели гибкие с оплатой текущих расходов.

Дополнительные сведения, ознакомьтесь с [клиентских лицензий (CAL) на лицензию развертывания служб удаленных рабочих СТОЛОВ](rds-client-access-license.md).

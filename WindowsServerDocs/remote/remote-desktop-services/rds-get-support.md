# <a name="additional-remote-desktop-resources"></a>Дополнительные материалы по удаленным рабочим столам

Помимо сведений в библиотеке Windows Server 2016, можно использовать следующие ресурсы, чтобы узнать о службах удаленных рабочих столов и получить справку по ним.

- Примите участие в общих обсуждениях служб удаленных рабочих столов на [форуме TechNet по RDS](https://aka.ms/technetforum-rds).
- Обсуждение приложений и клиентов удаленного рабочего стола для Windows, Android, iOS и Mac доступно на [форуме TechNet по клиентам удаленного рабочего стола](https://aka.ms/technetforum-rdc).
- Сведения о MultiPoint можно получить на [форуме TechNet по MultiPoint](https://aka.ms/multipoint-forum).

Если у вас есть идеи о службах удаленных рабочих столов, которыми вы хотите поделиться с нами, опубликуйте раздел на [форуме UserVoice](https://aka.ms/uservoice-rds).

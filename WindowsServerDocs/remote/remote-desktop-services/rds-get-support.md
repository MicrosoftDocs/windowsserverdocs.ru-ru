# <a name="additional-remote-desktop-resources"></a>Дополнительные ресурсы для удаленного рабочего стола

Помимо статьи в библиотеке Windows Server 2016 чтобы узнать о и получить помощь с помощью служб удаленных рабочих столов можно использовать следующие ресурсы:

- В Twitter? Так что в итоге: [@RDS4U](https://twitter.com/RDS4U)
- Принять участие в общих дискуссиях о службах удаленных рабочих столов в [форум TechNet служб удаленных рабочих СТОЛОВ](https://aka.ms/technetforum-rds).
- Обсуждение приложений/клиенты удаленного рабочего стола для Windows, Android, iOS и Mac, см. в статье [форум TechNet клиенты удаленного рабочего стола](https://aka.ms/technetforum-rdc).
- Для MultiPoint, ознакомьтесь с [форум MultiPoint TechNet](https://aka.ms/multipoint-forum).

Если у вас есть идеи о службах удаленных рабочих столов, который вы хотите поделиться с нами, опубликуйте раздела в [форуме UserVoice](https://aka.ms/uservoice-rds).

# <a name="plan-and-design-your-remote-desktop-services-environment"></a>Планирование и проектирование среды служб удаленных рабочих столов

Масштабируемое развертывание удаленного рабочего стола требует использования определенных шаблонов и рекомендаций.
Проектирование для обеспечения оптимальной производительности и масштабирования является ключом. Используйте приведенные ниже помогут вам представить, архитектуры и постоянно улучшать развертывания.

Используйте следующие сведения для планирования и разработки развертывания:

- [Сборка в любых условиях](rds-plan-build-anywhere.md)
- [Ориентироваться на различные виды пользователей](rds-plan-cater-to-users.md)
- [Доступ из любого места](rds-plan-access-from-anywhere.md)
- [Высокий уровень доступности](rds-plan-high-availability.md)
- [Многофакторная проверка подлинности](rds-plan-mfa.md)
- [Безопасное хранилище данных](rds-plan-secure-data-storage.md)
- [Включить отрисовку современных графических систем](rds-graphics-virtualization.md)
- [Подключаться с любого устройства](rds-plan-connect-from-any-device.md)
- [Выбор способа оплаты](rds-plan-choose-how-you-pay.md)

Не забудьте также просмотреть [Desktop размещение эталонной архитектуры](desktop-hosting-reference-architecture.md), которой представлен обзор архитектуры удаленного рабочего стола и помогает планировать гибридной среде служб удаленных рабочих СТОЛОВ, включающий инфраструктуры Azure.

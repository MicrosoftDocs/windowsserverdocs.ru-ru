# <a name="recommended-settings-for-vdi-desktops"></a>Рекомендуемые параметры для рабочих столов VDI

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows 10

Microsoft Desktop Virtualization автоматически обнаруживает конфигураций устройств и условия сети убедить пользователей и раньше, включив мгновенных настройки корпоративных приложений и рабочих столов, и он обеспечивает ИТ для предоставления доступа к прежней версии приложения во время перехода на Windows 10.

Несмотря на то, что очень хорошо настройке операционной системы Windows 10 без дополнительной настройки существуют возможности для уточнить их дальнейшей специально для корпоративной среды инфраструктуры виртуальных рабочих столов Майкрософт (VDI). В среде VDI многих фоновых служб и задачи будут отключены с самого начала.

В этом разделе не концептуальное представление, но вместо является руководство или отправной точки. Некоторые рекомендации могут отключить функции, которые вы хотите использовать, это следует учитывать затраты и преимущества Настройка любого конкретного параметра в вашем сценарии.

Эти инструкции и рекомендуемые параметры относятся к Windows 10 версии 1607 (версия 10.0.1393).

> [!NOTE]  
> Все параметры, не указано специально в этом разделе можно оставить их значения по умолчанию (или набор по требованиям и политикам) без даст значительного влияния на функциональные возможности VDI.

При создании образа для развертывания VDI, обязательно используйте **Current Branch**. Дополнительные сведения о текущей ветви, см. в разделе [сведения о выпуске Windows 10](https://technet.microsoft.com/windows/release-info.aspx).

## <a name="creating-the-windows-10-image"></a>Создание образа Windows 10
Первым шагом является установка эталонный образ Windows 10 версии 1607 (версия 10.0.1393) на физическом или виртуальной машины. Установка на виртуальную машину легко, позволяет сохранить версии файла виртуального жесткого диска (VHD), в случае, если требуется выполнить откат к более ранней версии.

## <a name="microsoft-store-apps"></a>Приложения Microsoft Store
Существует целый ряд приложений Microsoft Store, которые может потребоваться удалить из образа VDI; удалив их уменьшить использование ЦП и экономии дискового пространства. Хорошими кандидатами для удаления включают:

- Get Office
- Скайп (Предварительная версия)
- Начало работы (особенно при отсутствии подключения к Интернету)
- Центр отзывов
- Коллекция "Косынка" корпорации Майкрософт
- Платные Wi-Fi и сотовый

Чтобы настроить профиль пользователя по умолчанию, используемый для создания образов VDI, воспользуйтесь встроенной учетной записи администратора. Если он еще не включен, сделайте с помощью локальные пользователи и группы в Управление компьютером. Затем войдите в учетную запись администратора, выполните следующие действия.

> [!NOTE]  
> Не следует удалять системные приложения, например Store app. Трудно обеспечить их переустановки. Другие приложения, легко reinstallable из Store.

### <a name="delete-unwanted-apps-from-the-administrator-user-profile"></a>Удаление нежелательных приложений из профиля пользователя администратора
1. В Windows PowerShell выполните `Get-AppxPackage | ft PackageFamilyName` Чтобы просмотреть список установленных приложений.
2. Для каждого упаковщик приложений, которые вы хотите удалить выполните командлеты этого пример формата:

    `Get-AppxPackage *messaging* | Remove-AppxPackage`

    `Get-AppxPackage *WindowsMaps* | Remove-AppxPackage`

    `Get-AppxPackage *ZuneMusic* | Remove-AppxPackage`

### <a name="delete-the-payload-of-unwanted-store-apps"></a>Удаление нежелательных приложений Store полезных данных
Это будет препятствовать повторную установку приложения.
1. Store список приложений и другие элементы, которые подготовлены данные в хранилище с помощью этого командлета: `Get-AppxProvisionedPackage -Online`.
2. Удалить конкретный пакет с `Remove-AppxProvisionedPackage -Online -PackageName MyAppPackage`, используя соответствующие MyAppPackage возвращается из шага 1. Например, чтобы удалить пакет, связанные с Zune, можно запускать `Remove-AppxProvisionedPackage -Online -PackageName Microsoft.ZuneMusic_2019.17012.10311.0_neutral_~_8wekyb3d8bbwe`.

## <a name="removing-other-items"></a>Удаление других элементов
Можно удалить значок OneDrive и приложения, отключить системные значки и удалить загруженные обновления.

### <a name="remove-onedrive-icon-and-app"></a>Удалить значок OneDrive и приложения
1. Нажмите кнопку **запустить** и прокрутите страницу до **OneDrive** значок.
2. Щелкните правой кнопкой мыши **OneDrive** значок, укажите пункт **дополнительные**, а затем нажмите кнопку **открыть расположение файла**.
3. Щелкните правой кнопкой мыши **OneDrive** значок в ее расположение файла, а затем щелкните **удалить**.

Чтобы удалить приложение OneDrive:
1. Нажмите кнопку **запустить** и прокрутите страницу до **OneDrive** значок.
2. Щелкните правой кнопкой мыши **OneDrive** значок, а затем щелкните **удаления**. Программы и компоненты откроется.
3. Щелкните правой кнопкой мыши на программы и компоненты, **Microsoft OneDrive** и нажмите кнопку **удаления**.

### <a name="programs-and-features-from-previous-versions-of-control-panel"></a>Программы и компоненты (из предыдущих версий панели управления)
1. Push- **запустить** "Кнопка", введите *управления*, и нажмите клавишу ВВОД.
2. Коснитесь или дважды щелкните **программы и компоненты**.
3. Слева в разделе **панели управления — домашняя страница**, коснитесь или щелкните **или отключение компонентов Windows включить**. Откроется новый пользовательский интерфейс.
4. Снимите флажки для всех элементов, которые вы хочет или не нуждается в базовый образ, например: **Поддержка управления доступом файла SMB 1.0/CIFS**.

### <a name="turn-system-icons-off"></a>Отключить системные значки
1. Push-уведомлений или нажмите кнопку **запустить**, а затем нажмите кнопку **параметры** (значок шестеренки).
2. В **найти параметр** текстовое поле, тип *задач*, а затем нажмите кнопку **параметры задач**.
3. В разделе **задач** разделе, прокрутите или проведите пальцем вниз до **области уведомлений** раздел.
4. Щелкните или коснитесь **выключать системные значки**и затем включить каждый системный значок или отключить любой степенью точности для образа.

### <a name="delete-downloaded-updates"></a>Удалить загруженные обновления
1. В проводнике перейдите к **C:\Windows\Software Distribution\Download**.
2. Удалите все файлы и папки в этом каталоге.

# Начало работы с удаленного рабочего стола в Windows

>Область применения: Windows 10, Windows 8.1, Windows Server 2019 г., Windows Server 2016, Windows Server 2012 R2

Клиент удаленного рабочего стола для Windows можно использовать для работы с приложениями для Windows и настольными компьютерами удаленно на другом устройстве Windows.

Используйте следующие сведения для начала работы. Обязательно изучите [вопросы и ответы](remote-desktop-client-faq.md) , если у вас возникают вопросы.

> [!NOTE]
> - Хотите узнать, новые выпуски для клиента Windows? Ознакомьтесь с [новые возможности для удаленного рабочего стола в Windows?](windows-whatsnew.md)
> - Можно запустить клиент во всех версиях Windows 10.

## Получить клиента к удаленному рабочему Столу и начать использовать это решение

Выполните следующие действия, чтобы начать работу с помощью удаленного рабочего стола на устройстве с Windows 10.

1. Скачайте клиент удаленного рабочего стола из [Microsoft Store](https://www.microsoft.com/store/p/microsoft-remote-desktop/9wzdncrfj3ps). 
2. [Настройка компьютера на прием удаленных подключений](remote-desktop-allow-access.md).
3. Добавьте к удаленному рабочему столу или удаленного ресурса. Использовать подключение для прямого подключения к ПК с Windows и удаленного ресурса для использования программы RemoteApp, доставлять сеанса или виртуального рабочего стола, опубликованные к администратору. 
4. Закрепление элементов, таким образом можно быстро получить для удаленного рабочего стола.

### Добавьте к удаленному рабочему столу

Чтобы создать к удаленному рабочему столу:

1. В центре подключения коснитесь **+ Добавить**, а затем коснитесь **рабочего стола**.
2. Введите следующие сведения для компьютера, что вы хотите подключиться к:
  - **Имя компьютера** — имя компьютера. Это может быть имя компьютера Windows, имя домена в Интернете или IP-адреса. Сведения о портах также можно добавить к имени компьютера (например, **MyDesktop:3389** или **10.0.0.1:3389**).
  - **Учетная запись пользователя** — учетная запись пользователя для доступа к удаленному Компьютеру. Коснитесь **+** добавить новую учетную запись или выберите существующую учетную запись. Можно использовать следующие форматы для имя пользователя: *user_name*, *домен\имя_пользователя*или *user_name@domain.com*. Можно также задать на запрос имени пользователя и пароля во время подключения, выбрав **Спрашивать каждый раз**.
3. Также можно задать дополнительные параметры, нажмите **Показать дополнительные**:
  - **Отображаемое имя** — это удобного для запоминания имени на компьютере, вы подключаетесь. Можно использовать любую строку, но если не указать понятное имя, имя компьютера отображается.
  - **Группа** — укажите в группу, чтобы упростить найти подключений в дальнейшем. Можно добавить новую группу, нажав кнопку **+** или выберите его из списка.
  - **Шлюз** -шлюз удаленного рабочего стола, которое вы хотите использовать для подключения к виртуальным рабочим столам программ RemoteApp и рабочие столы на основе сеансов во внутренней корпоративной сети. Получение сведений о шлюз из к системному администратору.
  - **Подключение к сеансу администратора** - используйте этот параметр, чтобы подключиться к сеансу консоли централизованного управления Windows server.
  - **Кнопки мыши буферов** — использовать этот параметр для замены функции кнопки мыши правой кнопкой мыши. (Это особенно полезно, если удаленный компьютер настроен для левовинтовой пользователя, но используется правовинтовой мышь.)
  - **Значение Мое разрешение удаленный сеанс:** : выберите разрешение, вы хотите использовать во время сеанса. **Выберите для меня** установит разрешение на основе размера клиента.
  - **Изменение размера экрана:** — при выборе высокого разрешения статические для сеанса, вы можете создавать элементы на экране, чтобы повысить удобочитаемость. Примечание: Это касается только при подключении к Windows 8.1 или более поздней.
  - **Обновите удаленный сеанс разрешения на изменение размера** — при включении, клиент динамически обновлять разрешение сеанса, в зависимости от размера клиента. Примечание: Это касается только при подключении к Windows 8.1 или более поздней.
  - **Буфер обмена** — при включении позволяет копировать текст и изображения с удаленного компьютера.
  - **Воспроизведение звука** — выбрать устройство для использования для воспроизведения звука во время удаленного сеанса. Вы можете воспроизводить звук на локальном удаленного компьютера, или не во всех.
  - **Звукозапись** — при включении позволяет использовать локального микрофона с приложениями на удаленном Компьютере.
4. Коснитесь кнопки **Сохранить**.

Необходимо изменить эти параметры? Коснитесь меню переполнения (****...) рядом с именем компьютера, а затем коснитесь **Изменить**.

Удалить подключение? Опять же коснитесь меню переполнения (****...), а затем нажмите **Удалить**.

### Добавление удаленных ресурса
Удаленные ресурсы являются программ RemoteApp, сеанс под управлением настольных компьютеров и виртуальных рабочих столов, опубликованные ваш администратор, с помощью служб удаленных рабочих столов.

Добавление удаленных ресурса:

1. На экране Center подключения нажмите кнопку **+ Добавить**и выберите **удаленным ресурсам**. 
2. Введите **URL-адрес веб-каналов** , предоставляемые ваш администратор и коснитесь **Поиск веб-каналы**.
3. В ответ на запрос Предоставьте учетные данные, используемые для подписки на веб-канала.

Удаленными ресурсами отображается в центре подключения.


Чтобы удалить удаленным ресурсам:

1. В центре подключения выберите в меню переполнения (****...) рядом с удаленного ресурса.
2. Коснитесь **Удалить**.

### Закрепите сохраненный рабочего стола в меню "Пуск"

Чтобы закрепить подключение к меню "Пуск", нажмите кнопку в меню переполнения (****...) рядом с именем компьютера и выберите **закрепить на начальном экране**.

Теперь вы можете запустить удаленный рабочий стол непосредственно из меню "Пуск", коснувшись его.

## Подключение для доступа к ресурсам внутренней шлюза удаленных рабочих Столов

Шлюз удаленных рабочих столов (шлюз удаленных рабочих Столов) позволяет подключаться к удаленному компьютеру в корпоративной сети из любого места в Интернете. Можно создать и управлять вашей шлюзов с помощью клиента удаленного рабочего стола.

Настройка нового шлюза:

1. В центре подключения нажмите **Параметры**.
2. Рядом с шлюз, коснитесь **+** для добавления новых шлюза. Примечание: Шлюза можно также добавить при добавлении нового подключения.
3. Введите следующие сведения:
  - **Имя сервера** — имя компьютера, который необходимо использовать в качестве шлюза. Это может быть имя компьютера Windows, имя домена в Интернете или IP-адреса. Вы также можете добавить сведения о портах для имени сервера (например: **RDGateway:443** или **10.0.0.1:443**).
  - **Учетная запись пользователя** - выберите или добавьте учетную запись пользователя для использования с шлюз удаленных рабочих столов, вы подключаетесь. Также можно выбрать **использовать учетную запись рабочего стола пользователя** , чтобы использовать те же учетные данные, которые используются для подключения удаленного рабочего стола.
4. Коснитесь кнопки **Сохранить**.  

## Глобальные параметры

Можно задать следующие глобальные параметры в клиенте, коснувшись **параметров**:

УПРАВЛЯЕМЫЕ ЭЛЕМЕНТЫ
- **Учетная запись пользователя** — позволяет добавить, изменять и удалять учетные записи пользователей сохраняются в клиенте. Это хороший способ обновить пароль для учетной записи после его изменения.
- **Шлюз** — позволяет добавить, изменять и удалять серверы шлюза, сохраненные в клиенте.
- **Группа** — позволяет добавить, изменение и удаление групп, сохраненные в клиенте. Это позволяет легко группы подключения.

ПАРАМЕТРЫ СЕАНСА
- **Запустить подключения в полноэкранном режиме** - при включении всякий раз, когда запускается подключение, клиент будет использовать весь экран текущего монитора.
- **Запустите каждого подключения в новом окне** - при включении каждого подключения запускается в отдельном окне, можно поместить их в разных мониторов и переключаться между ними с помощью панели задач.
- **При изменении размера приложения:** -позволяет управлять что происходит при изменении размера окна. По умолчанию **растягивается содержимое, сохраняя пропорции**.
- **Использовать сочетания клавиш с:** -давайте вами, где используются для команд клавиатуры *ВЫИГРАТЬ* или *ALT + TAB* . По умолчанию используется только отправлять их в сеанс, когда подключение находится в полной scren.
- **Запретить экрана по истечении времени ожидания** — позволяет сохранить экрана по истечении времени ожидания при сеанс активен. Это полезно, если подключение не требует вмешательства для длительных периодов времени.

ПАРАМЕТРЫ ПРИЛОЖЕНИЯ
- **Показать эскизы рабочего стола** — позволяет просматривать Предварительный просмотр рабочего стола, в центре подключения перед подключением к нему. По умолчанию это задается **включена**.
- **Улучшить удаленного рабочего стола** — анонимные данные отправляет в корпорацию Майкрософт. Мы используем эти данные для улучшения клиента. Дополнительные сведения о том, как мы рассматриваем эти данные анонимные, закрытым, см. в [Заявлении о конфиденциальности корпорации Майкрософт](https://privacy.microsoft.com/en-us/privacystatement). **По умолчанию этот параметр включен.**

### Управление учетными записями

При подключении к рабочего стола или удаленным ресурсам, вы можете сохранить учетные записи пользователей для выбора из еще раз. Можно также определить учетных записей пользователей в клиенте, в отличие от сохранения данных пользователя при подключении к рабочему столу.

Создание новой учетной записи пользователя:

1. В центре подключения нажмите **Параметры**.
2. Рядом с учетной записью пользователя, коснитесь **+** для добавления учетной записи пользователя.
3. Введите следующие сведения:
   - **Имя пользователя** — имя пользователя для сохранения для использования с удаленного подключения. Можно ввести имя пользователя в любом из следующих форматов: user_name, домен\имя_пользователя, или user_name@domain.com.
   - **Пароль** — пароль пользователя, указанные вами. Это могут быть оставлены пустыми будет предложено ввести пароль во время подключения.
4. Коснитесь кнопки **Сохранить**.

Чтобы удалить учетную запись пользователя:

1. В центре подключения нажмите **Параметры**.
2. Выберите учетную запись, чтобы удалить из списка под учетной записью пользователя.
3. Рядом с учетной записью пользователя нажмите значок правки.
4. Коснитесь **Удалить эту учетную запись** в нижней для удаления учетной записи пользователя.
5. Также можно изменить учетную запись пользователя и нажмите кнопку **Сохранить**.

## Перейдите сеансов удаленных рабочих столов
При запуске удаленного рабочего стола, существует средств можно использовать для перехода к сеансу.

### Запустить удаленный рабочий стол

1. Коснитесь подключения удаленного рабочего стола для запуска сеанса.
2. Если вы еще не сохраненные учетные данные для подключения, вам будет предложено указать **имя пользователя** и **пароль**.
3. Если появится запрос для проверки сертификата для удаленного рабочего стола, просмотрите сведения и убедитесь, что это компьютера доверять перед нажатием **Connect**. Также можно выбрать **не запрашивайте об этом сертификате снова** всегда принимать этот сертификат.

### Панель подключения

Подключение панель обеспечивает доступ к элементам управления дополнительных навигации. По умолчанию панель подключений помещается в середине в верхней части экрана. Коснитесь и перетащите панель влево или вправо, чтобы переместить его.

- **Сдвиг управления** - сдвиг управления позволяет экрана, чтобы увеличить и перемещены. Обратите внимание, что элемент управления сдвиг доступен только на устройствах с поддержкой сенсорного ввода и с помощью режима пространств сенсорного ввода.
   - Включить / отключить сдвиг управления: нажмите значок сдвиг в строке подключения для отображения элемента управления сдвигать и масштабировать экрана. Коснитесь значка сдвиг в строке подключения еще раз, чтобы скрыть элемент управления и вернуть его исходное разрешение экрана.
   - Используйте элемент управления сдвиг - касание и пользовательского элемента управления панорамирования, а затем перетащите в направлении, нужно переместить на экране.
   - Перемещение элемента управления сдвиг - двойное касание и удерживайте сдвиг элемент управления для перемещения элемента управления на экране.
- **Дополнительные параметры** - коснитесь значка Дополнительные параметры для отображения выбора сеанса в строке и команды панели (см. ниже).
- **Клавиатура** - коснитесь значка клавиатуры для отображения или скрытия экранной клавиатуры. Элемент управления сдвиг автоматически отображается при отображении клавиатуры.

### Панель команд

Коснитесь **...** в строке подключения для отображения панели команд в правой части экрана.

- **Домашняя страница** — используйте кнопки "Домой" для возврата к центру подключения из командной строки.
  - В качестве альтернативы можно использовать кнопки "Назад" для одного действия.
  - Ваш сеанс не будет отключен.
  - Это позволяет запускать дополнительные подключения.
- **Отключить** - используйте кнопку отключения для завершения подключения.
  - Ваши приложения остается активным до тех пор, пока не завершения сеанса на удаленном Компьютере.
- **Полноэкранный** - входе или выходе полноэкранный режим.
- **Сенсорный ввод или мышь** — вы можете переключаться между режимами мыши (прямой сенсорного ввода и указатель мыши).

### Использование прямой сенсорные жесты и режима мыши в удаленном сеансе

Для взаимодействия с сеанс доступны два режима мыши.
- **Прямой сенсорный ввод**: приложение передает все сенсорные контакты в сеанс удаленно интерпретировать.
  - Используются так же, следует использовать Windows с сенсорным экраном.
- **Указатель мыши**: преобразует ваш локальный сенсорного экрана в больших сенсорной панели, что позволяет для перемещения указателя мыши в сеансе.
  - Форматируется так же, как бы вы с сенсорной панелью.

> [!NOTE]
> Взаимодействия с Windows 8 или более поздней версии собственные сенсорные жесты поддерживаются в режиме прямого сенсорного ввода. 

| Режим мыши    | Операции мыши      | Жест                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| Прямой сенсорного ввода  | Щелкайте левой кнопкой мыши           | касание пальцем 1                                                          |
| Прямой сенсорного ввода  | Щелчок правой кнопкой мыши          | 1 палец касание и удерживание                                                 |
| Указатель мыши | Щелкайте левой кнопкой мыши           | касание пальцем 1                                                          |
| Указатель мыши | Щелкните левой кнопкой мыши и перетащите  | 1 палец двойное касание и удерживание, затем перетащите                               |
| Указатель мыши | Щелчок правой кнопкой мыши          | касание пальцем 2                                                          |
| Указатель мыши | Щелкните правой кнопкой мыши и перетащите | 2 палец двойное касание и удерживание, затем перетащите                               |
| Указатель мыши | Колесика мыши          | 2 палец нажмите и удерживайте, а затем перетащите вверх или вниз                           |
| Указатель мыши | Масштаб                 | Используйте 2 пальцев и сжатие для увеличения и уменьшения перемещение пальцев друг от друга для уменьшения масштаба. |

# Начало работы с удаленного рабочего стола в Android

>Область применения: Windows 10, Windows 8.1, Windows Server 2019 г., Windows Server 2016, Windows Server 2012 R2

Клиент удаленного рабочего стола для Android можно использовать для работы с приложениями для Windows и настольными компьютерами непосредственно на устройстве Android.

Используйте следующие сведения для начала работы. Обязательно изучите [вопросы и ответы](remote-desktop-client-faq.md) , если у вас возникают вопросы.

> [!NOTE]
> - Хотите узнать, новые выпуски для Android клиента? Ознакомьтесь с [новые возможности для удаленного рабочего стола в Android?](android-whatsnew.md)
> Вы можете запустить клиент Android 4.1 и более новых устройств, а также на Chromebooks с 53 ChromeOS установки. Дополнительные сведения о Android приложений на Chrome [здесь](https://sites.google.com/a/chromium.org/dev/chromium-os/chrome-os-systems-supporting-android-apps).

## Получить клиента к удаленному рабочему Столу и начать использовать это решение

Выполните следующие действия, чтобы начать работу с помощью удаленного рабочего стола на устройстве Android.

1. Для загрузки клиента удаленного рабочего стола [Google Play](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android). 
2. [Настройка компьютера на прием удаленных подключений](remote-desktop-allow-access.md).
3. Добавьте к удаленному рабочему столу или удаленного ресурса. Используйте подключение для прямого подключения к ПК с Windows и удаленного ресурса для использования программ RemoteApp, доставлять сеанса или виртуального рабочего стола публикации в локальной. 
4. Поэтому можно получить доступ к удаленному рабочему столу быстро создайте элемент.

> [!NOTE]
> Если вам требуется для тестирования новых функций в более ранних версий рекомендуется загрузить наше приложение [Microsoft удаленного рабочего стола бета-версии](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android.beta) из хранилища Google Play. 

### Добавьте к удаленному рабочему столу

Чтобы создать к удаленному рабочему столу:

1. В центре подключения tap **+**, а затем коснитесь **рабочего стола**.
2. Введите следующие сведения для компьютера, что вы хотите подключиться к:
  - **Имя компьютера** — имя компьютера. Это может быть имя компьютера Windows, имя домена в Интернете или IP-адреса. Сведения о портах также можно добавить к имени компьютера (например, **MyDesktop:3389** или **10.0.0.1:3389**).
  - **Имя пользователя** — имя пользователя, чтобы использовать для доступа к удаленному Компьютеру. Можно использовать следующие форматы: *user_name*, *домен\имя_пользователя*или *user_name@domain.com*. Можно также задать запрашивает имя пользователя и пароль.
3. Вы также можете задать следующие дополнительные параметры:
  - **Понятное имя** — это удобного для запоминания имени на компьютере, вы подключаетесь. Можно использовать любую строку, но если не указать понятное имя, имя компьютера отображается.
  - **Шлюз** -шлюз удаленного рабочего стола, которое вы хотите использовать для подключения к виртуальным рабочим столам программ RemoteApp и рабочие столы на основе сеансов во внутренней корпоративной сети. Получение сведений о шлюз из к системному администратору.
    Необходимо настроить шлюз удаленных рабочих столов?
  - **Звук** : выберите устройства для использования для воспроизведения звука во время удаленного сеанса. Вы можете воспроизводить звук на локальном удаленного устройства, или не во всех.
  - **Настройка разрешение дисплея** — задайте пользовательские разрешения для подключения при включении этого параметра. Если отключить разрешение применяется которой были определены в глобальные параметры приложения.
  - **Кнопки мыши буферов** — использовать этот параметр для замены функции кнопки мыши правой кнопкой мыши. (Это особенно полезно, если удаленный компьютер настроен для левовинтовой пользователя, но используется правовинтовой мышь.)
  - **Подключение к сеансу администратора** - используйте этот параметр, чтобы подключиться к сеансу консоли централизованного управления Windows server.
  - **Перенаправление локального хранилища** — подключает локального хранилища как удаленной файловой системы на удаленном Компьютере.
4. Коснитесь кнопки **Сохранить**.

Необходимо изменить эти параметры? Коснитесь меню переполнения (****...) рядом с именем компьютера, а затем коснитесь **Изменить**.

Удалить подключение? Опять же коснитесь меню переполнения (****...), а затем нажмите **Удалить**.

>[!TIP]
> Если возникает ошибка 0xf07 неправильных паролей, («мы не удается подключиться к удаленному Компьютеру из-за истечения срока действия пароля, связанного с учетной записью пользователя»), изменить свой пароль и повторите попытку.

### Добавление удаленных ресурса
Удаленные ресурсы являются программ RemoteApp, настольные компьютеры на основе сеансов и публикации с помощью подключения к удаленным рабочим столам и RemoteApp виртуальных рабочих столов.

Добавление удаленных ресурса:

1. На экране Center подключения нажмите **+** и затем коснитесь **Удаленного ресурсы веб-каналов**. 
2. Введите сведения для удаленного ресурса.
   - **Электронной почты или URL-адрес** - URL-адрес сервера веб-доступа к удаленному рабочему Столу. Учетной записи корпоративной электронной почте можно также ввести в это поле — это сообщает клиенту для поиска веб-сервера к удаленному рабочему Столу, связанные с свой адрес электронной почты.
   - **Имя пользователя** - имя пользователя для веб-доступа к удаленному рабочему Столу сервер, которым вы подключаетесь.
   - **Пароль** - пароль, используемый для веб-доступа к удаленному рабочему Столу сервер, которым вы подключаетесь.
3. Коснитесь кнопки **Сохранить**.

Удаленными ресурсами отображается в центре подключения.


Чтобы удалить удаленным ресурсам:

1. В центре подключения выберите в меню переполнения (****...) рядом с удаленного ресурса.
2. Коснитесь **Удалить**.
3. Подтвердите удаление.

### Мини-приложения — PIN-код сохраненного рабочего стола на начальном экране

Удаленный рабочий стол приложения поддерживают закрепления подключения к вашей главной странице с помощью функции Android мини-приложение. Способ добавления элемент зависит от типа Android устройства, которую вы используете и от операционной системы. Вот наиболее распространенный способ добавления элемент. 

1. Коснитесь **приложения** смогут запускать меню "приложения".
2. Коснитесь **мини-приложения**.
3. Считайте через мини-приложения и найдите значок удаленного рабочего стола с описанием, «PIN-код удаленного рабочего стола.»
4. Нажмите и удерживайте, графического удаленного рабочего стола и переместить его на начальном экране.
5. При выпуске значок, вы увидите сохраненные удаленным рабочим столам. Выберите подключение, которое вы хотите сохранить на начальном экране.

Теперь вы можете запустить удаленный рабочий стол непосредственно из вашей главной странице, коснувшись его.

> [!NOTE]
> При переименовании подключения к рабочему столу в приложении удаленного рабочего стола, метки для этого закрепленных удаленного рабочего стола не будут обновляться.

## Подключение для доступа к ресурсам внутренней шлюза удаленных рабочих Столов

Шлюз удаленных рабочих столов (шлюз удаленных рабочих Столов) позволяет подключаться к удаленному компьютеру в корпоративной сети из любого места в Интернете. Можно создать и управлять вашей шлюзов с помощью клиента удаленного рабочего стола.

Настройка нового шлюза:

1. В центре подключения нажмите **Параметры > шлюзов**. Коснитесь **+** для добавления новых шлюза.
2. Введите следующие сведения:
  - **Имя сервера** — имя компьютера, который необходимо использовать в качестве шлюза. Это может быть имя компьютера Windows, имя домена в Интернете или IP-адреса. Вы также можете добавить сведения о портах для имени сервера (например: **RDGateway:443** или **10.0.0.1:443**).
  - **Имя пользователя** - имя пользователя и пароль для шлюза удаленных рабочих столов, вы подключаетесь. Также можно выбрать **использовать учетную запись рабочего стола пользователя** , чтобы использовать те же учетные данные, которые используются для подключения удаленного рабочего стола.

## Управление учетными записями

При подключении к рабочего стола или удаленным ресурсам, вы можете сохранить учетные записи пользователей для выбора из еще раз. Можно также определить учетных записей пользователей в клиенте, в отличие от сохранения данных пользователя при подключении к рабочему столу.

Создание новой учетной записи пользователя:

1. В центре подключения нажмите кнопку **Параметры**и выберите **учетных записей пользователей**.
2. Коснитесь **+** для добавления учетной записи пользователя.
3. Введите следующие сведения:
   - **Имя пользователя** - имя пользователя для сохранения для использования с удаленного подключения. Можно ввести имя пользователя в любом из следующих форматов: user_name, домен\имя_пользователя, или user_name@domain.com.
   - **Пароль** — пароль пользователя, указанные вами. Каждой учетной записи пользователя, который вы хотите сохранить для удаленных подключений необходимо иметь пароль, связанный с ним.
4. Коснитесь кнопки **Сохранить**.

Чтобы удалить учетную запись пользователя:

1. В центре подключения нажмите **Параметры > учетных записей пользователей**.
2. Нажмите и удерживайте учетной записи пользователя в списке, чтобы выбрать его. Вы можете выбрать несколько пользователей.
3. Коснитесь Корзина для удаления выбранного пользователя.

## Перейдите сеансов удаленных рабочих столов
При запуске удаленного рабочего стола, существует средств можно использовать для перехода к сеансу.

### Запустить удаленный рабочий стол

1. Коснитесь подключения удаленного рабочего стола для запуска сеанса. 
2. Если появится запрос для проверки сертификата для удаленного рабочего стола, выберите **подключиться**. Также можно выбрать **не спрашивать снова для подключения к этому компьютеру** всегда принимать сертификат.

### Управление параметрами глобальные приложения

Вы можете задать следующие глобальные параметры в Android клиента:

- **Показать эскизы рабочего стола** — позволяет просматривать Предварительный просмотр рабочего стола, в центре подключения перед подключением к нему. По умолчанию это задается **включена**.
- **Сжатие для масштабирования** - позволяет использовать жесты сжатия для масштабирования. Если вы используете через удаленный рабочий стол приложение поддерживает мультисенсорный ввод (впервые появилась в Windows 8), этот параметр Включить **параметр**.
- **Улучшить удаленного рабочего стола** — анонимные данные отправляет в корпорацию Майкрософт. Мы используем эти данные для улучшения клиента. Можно узнать больше о том, как мы рассматриваем эти данные анонимные, частный, см. в [Заявлении о конфиденциальности для удаленного рабочего стола клиента](https://www.microsoft.com/privacystatement/RemoteApp/Default.aspx). **По умолчанию этот параметр включен.**
- **Отображение** - существует две общие параметры экрана:
   - **Ориентация** - задает предпочтительный ориентации (альбомной или книжной) для сеанса. 
   >[!NOTE]
   > При подключении к Компьютеру под управлением Windows 8 или более ранней версии Windows, сеанс не будет масштабироваться правильно. Лучше отключиться от компьютера, и затем снова подключить в ориентации, которые вы хотите использовать. Еще лучшим вариантом является по крайней мере обновления компьютера для Windows 8.1.

   - **Разрешение** - задает разрешения, которые вы хотите использовать для подключения к рабочему столу глобально. Если вы уже установили пользовательские разрешения для отдельного приложения или подключения, этот параметр не цвет.
   >[!NOTE]
   >При изменении параметров дисплея, они применяются только к новые подключения с этого момента на. Чтобы увидеть изменения во время сеанса вы уже подключены к отключиться и повторите попытку подключения.

### Панель подключения

Подключение панель обеспечивает доступ к элементам управления дополнительных навигации. По умолчанию панель подключений помещается в середине в верхней части экрана. Двойное касание и перетащить строку влево или вправо, чтобы переместить его.

- **Сдвиг управления**: элемент управления сдвиг позволяет экрана, чтобы увеличить и перемещены. Обратите внимание, что сдвиг элемент управления доступен только с помощью прямой сенсорного ввода.
   - Включить / отключить сдвиг управления: нажмите значок сдвиг в строке подключения для отображения элемента управления сдвигать и масштабировать экрана. Коснитесь значка сдвиг в строке подключения еще раз, чтобы скрыть элемент управления и вернуть его исходное разрешение экрана.
   - Используйте элемент управления сдвиг: касание и удерживание сдвига элементов управления, а затем перетащите в направлении, нужно переместить на экране.
   - Перемещение элемента управления сдвиг: двойное касание и удерживание элемента управления сдвиг для перемещения элемента управления на экране.
- **Дополнительные параметры**: нажмите значок Дополнительные параметры, чтобы отображать выбора сеанса панели и командной строке (см. ниже).
- **Клавиатура**: нажмите значок клавиатуры, чтобы отобразить или скрыть клавиатуры. Элемент управления сдвиг автоматически отображается при отображении клавиатуры.
- **Переместить в строке подключения**: касание и удерживание в строке подключения и затем перетаскивания в новое место в верхней части экрана.


### Панель команд

Коснитесь в строке подключения для отображения панели команд в правой части экрана. Вы можете переключаться между режимами мыши (прямой сенсорного ввода и указатель мыши). Использование кнопки "Домой" для возврата к центру подключения из панели команд. В качестве альтернативы можно использовать кнопки "Назад" для одного действия. Ваш сеанс не будет отключен. 


### Использование прямой сенсорные жесты и режима мыши в удаленном сеансе

Клиент использует стандартные сенсорные жесты. Сенсорные жесты можно также использовать для выполнения действий с мышью на удаленный рабочий стол. Режимы мыши определяются в следующей таблице.

> [!NOTE]
> Взаимодействия с Windows 8 или более поздней версии собственные сенсорные жесты поддерживаются в режиме прямого сенсорного ввода. 

| Режим мыши    | Операции мыши      | Жест                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| Прямой сенсорного ввода  | Щелкайте левой кнопкой мыши           | касание пальцем 1                                                          |
| Прямой сенсорного ввода  | Щелчок правой кнопкой мыши          | 1 палец касание и удерживание                                                 |
| Указатель мыши | Масштаб                 | Используйте 2 пальцев и сжатие для увеличения и уменьшения перемещение пальцев друг от друга для уменьшения масштаба. |
| Указатель мыши | Щелкайте левой кнопкой мыши           | касание пальцем 1                                                          |
| Указатель мыши | Щелкните левой кнопкой мыши и перетащите  | 1 палец двойное касание и удерживание, затем перетащите                               |
| Указатель мыши | Щелчок правой кнопкой мыши          | касание пальцем 2                                                          |
| Указатель мыши | Щелкните правой кнопкой мыши и перетащите | 2 палец двойное касание и удерживание, затем перетащите                               |
| Указатель мыши | Колесика мыши          | 2 палец нажмите и удерживайте, а затем перетащите вверх или вниз                           |

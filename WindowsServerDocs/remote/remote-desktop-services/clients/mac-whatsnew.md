# <a name="whats-new-for-the-remote-desktop-client-on-macos"></a>Новые возможности для клиента удаленного рабочего стола на macOS?

Корпорация Майкрософт регулярно обновлять [клиентом удаленного рабочего стола для macOS](remote-desktop-mac.md), добавляя новые функции и устранения проблем. Ознакомьтесь с последними обновлениями ниже.

Если у вас возникнут проблемы с, всегда свяжитесь с нами посредством **обеспечения > отчет о проблеме**.

## <a name="updates-for-version-10210"></a>Обновления для версии 10.2.10
*Дата публикации: 3/30/2019 г.*

- В этом выпуске мы устранить неполадки, вызванные последние обновления macOS 10.14.4. Также мы исправили mispaints, который отображался при декодировании данных кодек AVC кодируется на сервере с NVIDIA оборудования.

## <a name="updates-for-version-1029"></a>Обновления для версии 10.2.9
*Дата публикации: 3/6/2019 г.*

- В этом выпуске мы исправили проблему подключения к удаленному рабочему Столу шлюз, которая может произойти, когда принимает Перенаправление сервера размещения.
- Мы также устраняется либо шлюз удаленных рабочих Столов, вызванное 10.2.8 обновления.

## <a name="updates-for-version-1028"></a>Обновления для версии 10.2.8
*Дата публикации: 3/1/2019 г.*

- Разрешить подключения проблемы, которые доступны при использовании шлюза удаленных рабочих Столов.
- Предупреждения об основных неверно сертификатов, отображаемые при подключении.
- Обращение к некоторых случаях, где меню и док-станции снижает скроет при запуске удаленных приложений.
- Переработана код перенаправления буфера обмена для адресов сбои и зависания, которые поражающих некоторые пользователи.
- Исправить ошибки, вызвавшем Центр подключений снижает прокрутки при запуске подключения.

## <a name="updates-for-version-1027"></a>Обновления для версии 10.2.7
*Дата публикации: 2/6/2019 г.*

- В этом выпуске мы обращение к mispaints графика (из-за ошибки кодирования сервера), который отображался при использовании режима AVC444.

## <a name="updates-for-version-1026"></a>Обновления для версии 10.2.6
*Дата публикации: 1/28/2019 г.*

- Добавлена поддержка кодек AVC (420 и 444), доступные при подключении к текущей версии Windows 10.
- В по размеру окна режиме, окно обновление теперь возникает сразу после изменения размера, чтобы убедиться, что содержимое отображается на уровне правильный интерполяции.
- Исправить ошибки макета, вызвавшем веб-каналов заголовки перекрытие для некоторых пользователей.
- Очистка настройки пользовательского интерфейса приложения.
- Тщательно проработанные добавить или изменить пользовательский Интерфейс рабочего стола.
- Внесенные множество размеру и завершить изменения представления плитки и список подключения центра для настольных компьютеров и веб-каналы.

>[!NOTE]
>Существует ошибка в папке macOS 10.14.0 и 10.14.1, которая может привести к «.com.microsoft.rdc.application-data_SUPPORT/_EXTERNAL_DATA» (вложенные глубоко внутри папки ~/Library) для использования большой объем места на диске. Чтобы решить эту проблему, удалите содержимое папки и выполнить обновление до macOS 10.14.2. Обратите внимание, что побочный эффект удаления содержимое папки, снимок изображения, присвоенные закладки будут удалены. Эти изображения будут повторно при повторном подключении к удаленному Компьютеру.

- Фиксированный прогрессивные отрисовку, некоторые пользователи испытывали проблемы с помещаются в режим окна.

- Улучшена мыши прокрутки для дельты небольшой прокрутки.

- Устранена проблема где клиенту не постоянно сбои при запуске и не запускается для некоторых пользователей.

- Обращение к ошибку, вызывающей в Excel (на котором работает в удаленном сеансе) зависает при сохранении в файл на перенаправленные папки.

# <a name="whats-new-for-the-remote-desktop-client-on-macos"></a>Новые возможности клиента удаленного рабочего стола на macOS?

Мы регулярно обновляем [клиента удаленного рабочего стола для macOS](remote-desktop-mac.md), добавляя новые компоненты и устранение проблем. Ознакомьтесь с последними обновлениями ниже.

Если возникнут проблемы, всегда с нами можно связаться через **Справка > сообщить о проблеме**.

## <a name="updates-for-version-1029"></a>Обновления для версии 10.2.9
*Дата публикации: 3/6/2019*

- В этом выпуске мы исправили проблему подключения шлюза удаленных рабочих Столов, которая может произойти, когда выполняет перенаправление сервера размещения.
- Мы также учли Регрессия шлюза удаленных рабочих Столов, вызванная 10.2.8 обновления.

## <a name="updates-for-version-1028"></a>Обновления для версии 10.2.8
*Дата публикации: 3/1/2019*

- Устранить проблемы с подключением, которые отображаются при использовании шлюз удаленных рабочих Столов.
- Исправлена неправильный сертификат предупреждения, отображаемые при подключении.
- Устранены некоторые случаи, где строка меню и закрепления ненужной скроет при запуске удаленных приложений.
- Переработаны код перенаправления буфера обмена для адреса сбоев и зависаний, которые поражающих некоторых пользователей.
- Исправлена ошибка, вызывавшая Центр подключений без необходимости прокрутки при запуске подключения.

## <a name="updates-for-version-1027"></a>Обновления для версии 10.2.7
*Дата публикации: 2/6/2019*

- В этом выпуске мы устранили mispaints графики (из-за ошибки кодирования сервера), которые были доступны при использовании режима AVC444.

## <a name="updates-for-version-1026"></a>Обновления для версии 10.2.6
*Дата публикации: 1/28/2019*

- Добавлена поддержка для кодека AVC (420 и 444), доступные при подключении к текущей версии Windows 10.
- В по размеру окна режима, окно обновления теперь возникает сразу же после изменения размера, чтобы убедиться, что содержимое отображается на уровне правильный интерполяции.
- Исправлена ошибка макета, вызвавшего веб-канала заголовки перекрывать для некоторых пользователей.
- Очистить настройки пользовательского интерфейса приложения.
- Усовершенствованный пользовательский Интерфейс рабочего стола добавить или изменить.
- Сделать много размеру и завершить настройку Центр подключений плитки и список представлений для настольных компьютеров и веб-каналы.

>[!NOTE]
>Есть ошибка в папке macOS 10.14.0 и 10.14.1, может привести к «.com.microsoft.rdc.application-data_SUPPORT/_EXTERNAL_DATA» (вложенные глубоко внутри папки ~/Library) использовать большой объем места на диске. Чтобы устранить эту проблему, удалите содержимое папки и обновление на macOS 10.14.2. Обратите внимание, что побочный эффект удаления содержимого папки удаления моментального снимка изображения, присвоенные закладки. Эти образы будут созданы заново при повторном подключении к удаленным ПК.

## <a name="updates-for-version-1024"></a>Обновления для версии 10.2.4
*Дата публикации: 12/18/2018*

- Добавлена поддержка macOS Mojave 10.14 в темно-режиме.
- Параметр для импорта из удаленного рабочего стола 8 Microsoft теперь отображается в центре подключения, если она пуста.
- Известные проблемы папку перенаправления совместимость с некоторых сторонних корпоративных приложений.
- Решенные проблемы, где пользователи получают ошибку шлюза удаленных рабочих столов 0x30000069 из-за безопасности протокола проблем со стратегией.
- Фиксированный Прогрессивная отрисовка проблемы, которые некоторые пользователи испытывали проблемы с Вписать в режиме окна.
- Исправлена ошибка, препятствующая копирование файла и вставьте Копировать последнюю версию файла.
- Улучшенная основе мыши прокрутка для небольших прокрутки «дельты».

## <a name="updates-for-version-1023"></a>Обновления для версии 10.2.3
*Дата публикации: 11/06/2018*

- Добавлена поддержка параметра файл протокола удаленного рабочего СТОЛА «remoteapplicationcmdline» для сценариев удаленного приложения.
- Заголовок окна сеанса теперь включает в себя имя RDP-файл (и имя сервера) при запуске из RDP-файл.
- Исправлены сообщаемые удаленных рабочих Столов проблемы производительности шлюза.
- Фиксированный сообщаемые шлюза удаленных рабочих Столов аварийно завершает работу.
- Исправлены ошибки, где подключение зависала при подключении через шлюз удаленных рабочих Столов.
- Более эффективная обработка удаленных приложений и полноэкранном интеллектуально скрывая строки меню и закрепления.
- Основные сценарии, где оставались скрытыми удаленные приложения после запуска.
- Устранены обновления медленно подготовки к просмотру при использовании «Помещается в окно» отключить аппаратное ускорение.
- Обрабатывать ошибки создания базы данных, вызвана неверными разрешениями при запуске клиента. 
- Исправлена проблема, когда клиенту не постоянно аварийно завершается при запуске и не запускается для некоторых пользователей.
- Исправлена сценарий, когда подключения неправильно импортировались как весь экран из удаленного рабочего стола 8.

## <a name="updates-for-version-1022"></a>Обновления для версии 10.2.2
*Дата публикации: 10/09/2018*

- Новый центр подключений, который поддерживает перетаскивание, вручную расположение настольных компьютеров, можно изменять столбцы в режиме списка, основанного на столбцах Сортировка и более простое управление группы.
- Центр подключений теперь запоминает последние active pivot (рабочие столы или веб-каналы) при закрытии приложения.
- Запрос пользовательского интерфейса и потоки учетные данные были существенно пересмотрены.
- Шлюз удаленных рабочих Столов отзывы теперь является частью связи состояние пользовательского интерфейса.
- Был улучшен Импорт параметров из клиента версии 8.
- RDP-файлы, указывающие на конечные точки RemoteApp может быть импортирована в центр подключений.
- Оптимизация отображения retina ситуаций удаленного рабочего стола одного монитора.
- Поддержка для указания уровня интерполяции графики (что влияет на размытости) без использования оптимизации Retina.
- 256-цветного поддержки, чтобы обеспечить подключение к Windows 2000.
- Исправлена обрезки справа и нижней границ экрана при подключении к Windows 7, Windows Server 2008 R2 и более ранних версий.
- Копирование в локальный файл в Outlook (работает в удаленном сеансе) теперь добавляет файл как вложение.
- Исправлена проблема, которая замедляет передачи файлов на основе монтажного стола, если бы был создан файлов из общей сетевой папки.
- Устранена ошибка, которая вызывала в Excel (под управлением в удаленном сеансе) к зависанию при сохранении в файл в перенаправленной папке.
- Исправлена проблема, которая вызывала нет свободного места, возвращаются перенаправленных папок.
- Исправлена ошибка, вызывавшая эскизов потреблять слишком много macOS 10.14 пространство на диске.
- Добавлена поддержка для применения политик перенаправления устройств шлюза удаленных рабочих Столов.
- Устранена проблема, препятствующая закрытия сеанса windows при отключении от соединения с помощью шлюза удаленных рабочих Столов.
- Если уровень проверки подлинности сети (NLA) не является обязательным для сервера, вы теперь направляются в экран входа, если истек срок действия пароля.
- Исправлены проблемы производительности, которые отображаются при передаче больших объемов данных по сети.
- Исправления перенаправления смарт-карты.
- Поддержка всех возможных значений параметра «EnableCredSspSupport» и «Уровень проверки подлинности» параметры RDP-файл, если ключ по умолчанию ClientSettings.EnforceCredSSPSupport пользователя (в домене com.microsoft.rdc.macos) имеет значение 0.
- Поддержка параметр файл протокола удаленного рабочего СТОЛА «Подсказки для учетных данных на клиент», если NLA не согласована.
- Поддержка входа на основе смарт-карт с помощью перенаправления смарт-карты в командной строке Winlogon при NLA не согласована.
- Устранена проблема, препятствующая загрузку веб-канала ресурсы, которые содержат пробелы в URL-адрес.

## <a name="updates-for-version-1021"></a>Обновления для версии 10.2.1
*Дата публикации: 08/06/2018*

- ПК, присоединенных к включено подключение к Azure Active Directory (AAD). Для подключения к ПК, присоединенном к AAD, ваше имя пользователя должно быть в одном из следующих форматов: «AzureAD\user» или "AzureAD\user@domain«.
- Исправлены некоторые ошибки, влияющие на использование смарт-карт в удаленном сеансе.

## <a name="updates-for-version-1020"></a>Обновления для версии 10.2.0
*Дата публикации: 07/24/2018*

- Соответствующие обновления для соответствия требованиям GDPR.
- MicrosoftAccount\username@domain Теперь принимается как допустимое имя пользователя.
- Совместное использование буфера обмена был переписан быстрее и поддерживать дополнительные форматы.
- Скопируйте и вставьте текст, изображения или файлы между сеансами теперь обходит буфер обмена локального компьютера.
- Теперь можно подключиться через сервер шлюза удаленных рабочих Столов с помощью недоверенного сертификата, (Если вы подтвердите установку предупреждение).
- Metal аппаратное ускорение теперь используется (если поддерживается) для ускорения визуализации и оптимизация использования батареи.
- При использовании исходного состояния системы аппаратное ускорение, мы стараемся работать чудеса, чтобы сделать рисунок сеанса более четким.
- Избавиться от некоторых экземплярах, где windows зависала после закрытия.
- Исправленные ошибки, которые Предотвращение запуска программ RemoteApp в некоторых сценариях.
- Исправлена ошибка синхронизации канала шлюза удаленных рабочих Столов, приводил к 0x204 ошибок.
- Форму курсора мыши теперь обновляет правильно при перемещении за пределы сеанса или окно RemoteApp.
- Исправлена ошибка перенаправление папки, которая вызывала потери данных при копирование и вставка папки.
- Устранена проблема перенаправление папки, вызвавшего отчетов папка размеров.
- Исправлена Регрессия, при которой входа на компьютер, присоединенный к AAD, с помощью локальной учетной записи.
- Исправленные ошибки, которые вызывали содержимое окна обрезание сеанса.
- Добавлена поддержка сертификаты конечных точек удаленных рабочих Столов, которые содержат асимметричных ключей эллиптической кривой.
- Исправлена ошибка, при которой загрузки управляемых ресурсов, в некоторых сценариях.
- Обращаться с проблемой обрезки центр закрепленных подключений.
- Исправлена флажки в окне свойств отображения для более эффективной совместной работы.
- Блокировка пропорции теперь отключена, когда изменения динамического отображения.
- Устранены проблемы совместимости с инфраструктурой F5.
- Обновление обработки пустых паролей, чтобы убедиться, что правильные сообщения отображаются во время соединения.
- Прокрутка проблемы совместимости с MapInfra Pro предопределенной мыши.
- Исправлены некоторые проблемы выравнивания в центре подключений, при работе на Mojave.

## <a name="updates-for-version-1018"></a>Обновления для версии 10.1.8
*Дата публикации: 05/04/2018*

- Добавлена поддержка изменение удаленного разрешения путем изменения размеров окна сеанса!
- Основные сценарии, где удаленный ресурс веб-канала загрузки займет слишком много времени.
- Устранена ошибка 0x207, которая может произойти при соединении с серверами, не установлено исправление обновления по исправлению, которое CredSSP шифрования oracle (CVE-2018-0886).

## <a name="updates-for-version-1017"></a>Обновления для версии 10.1.7
*Дата публикации: 04/05/2018*

- Внесены исправления безопасности для включения CredSSP шифрования oracle исправления обновлений, как описано в CVE-2018-0886.
- Улучшенная RemoteApp значок и мыши курсор визуализацию для устранения обнаруженной mispaints.
- Устранены проблемы, где применялось RemoteApp windows за Центр подключений.
- Исправлена проблема, которая произошла при редактировании локальные ресурсы после импорта из удаленного рабочего стола 8.
- Теперь вы можете соединение, нажав клавишу ВВОД на плитку рабочего стола.
- Когда вы будете в полноэкранном режиме, CMD + M теперь правильно сопоставляет WIN + M.
- Центр подключений, предпочтения и сведения о windows теперь реагировать CMD + M.
- Теперь вы можете запустить обнаружение веб-каналы, нажав клавишу ВВОД **Добавление удаленные ресурсы** страницы.
- Устранена проблема, где новый канал удаленные ресурсы показал пустой в центре подключений до, после обновления.
 
## <a name="updates-for-version-1016"></a>Обновления для версии 10.1.6
*Дата публикации: 03/26/2018*

- Устранена проблема, где RemoteApp windows будет переупорядочить сами.
- Устранена ошибка, вызывавшая некоторых окон RemoteApp для задержался после их родительского окна.
- Устранены смещения проблема указатель мыши, некоторые программы RemoteApp.
- Устранена проблема, где начиная новое соединение дал фокус к существующему сеансу, а не открывать окно нового сеанса.
- Мы исправили ошибку с сообщением об ошибке — вы увидите правильное сообщение теперь Если мы не удается найти шлюз.
- Сочетание Quit (⌘ + Q) постоянно отображается в пользовательском Интерфейсе.
- Улучшенное качество изображения при растяжении в режиме «Вписать в окно».
- Исправлена Регрессия, которая вызывала несколько экземпляров домашней папке для отображения в удаленном сеансе.
- Обновить значок по умолчанию для рабочего стола плиток.

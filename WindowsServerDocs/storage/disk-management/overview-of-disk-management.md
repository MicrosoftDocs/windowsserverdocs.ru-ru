# <a name="overview-of-disk-management"></a>Обзор средства управления дисками

> **Область применения:** Windows 10, Windows 8.1, Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Управление дисками— это системная служебная программа для управления жесткими дисками и содержащимися на них разделами и томами. Эта программа позволяет инициализировать диски, создавать тома и форматировать их с файловыми системами FAT, FAT32 или NTFS. Управление дисками позволяет выполнять большинство операций с дисками без перезагрузки системы или прерывания работы пользователей. Большинство изменений конфигурации вступают в действие немедленно.

-   [Диагностика с помощью средства управления дисками](troubleshooting-disk-management.md)

# <a name="troubleshooting-activation-error-codes"></a>Устранение ошибок активации

**Пользователи домашних ПК** Эта статья предназначена для использования агентами поддержки и ИТ-специалистами. Если вам нужны дополнительные сведения о сообщениях об ошибке активации Windows, перейдите к следующим веб-сайтам Майкрософт:

[Справка по ошибкам активации Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

При попытке использовать ключ многократной активации (MAK) или службы управления ключами (KMS) для активации корпоративных лицензий на компьютерах с Windows появляются сообщения об ошибках, которые содержат конкретные коды ошибок. В этой статье рассматриваются способы устранения этих ошибок.

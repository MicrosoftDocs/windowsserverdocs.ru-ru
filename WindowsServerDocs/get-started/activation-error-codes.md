# <a name="troubleshooting-activation-error-codes"></a>Устранение неполадок коды ошибок активации

**Пользователи домашних ПК** Эта статья предназначена для использования агентами поддержки и ИТ-специалистов. Если вам нужны дополнительные сведения о сообщениях об ошибке активации Windows, перейдите к веб-сайтах Microsoft:

[Справка: ошибки активации Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

При попытке использовать ключ многократной активации (MAK) или службы управления ключами (KMS) для выполнения активации корпоративных лицензий на один или несколько компьютеров на базе Windows, появляется сообщения об ошибках, которые содержат конкретные коды ошибок. В этой статье рассматриваются способы устранения этих ошибок.

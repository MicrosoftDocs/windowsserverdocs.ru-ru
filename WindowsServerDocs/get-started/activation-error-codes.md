# Устранение неполадок коды ошибок активации

**Домашняя пользователей** Эта статья предназначена для использования с агенты службы поддержки и ИТ-специалистов. Если вы ищете дополнительные сведения об ошибках Windows активации, перейдите на следующие веб-сайты Майкрософт:

[Справка по ошибкам активации Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

При попытке использовать несколько ключ активации (MAK) или службы управления ключами (KMS) для выполнения активации корпоративных лицензий на один или несколько компьютеров под управлением Windows, вы получите сообщения об ошибках, которые содержат коды ошибок. В этой статье рассматриваются способы устранения этих ошибок.

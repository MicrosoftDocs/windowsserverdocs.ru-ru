# <a name="avoid-mapping-one-storage-path-to-multiple-resource-pools"></a>Избегайте сопоставление один путь к хранилищу для нескольких пулов ресурсов

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>**Проблема**  
*Путь к файлу хранилища сопоставляется несколько пулов ресурсов.*  
  
## <a name="impact"></a>**Влияние**  
*Для типа указанное хранилище пула следующих родительских и дочерних пулов совместно используют один и тот же путь хранилища:*  
  
\<список пулов >  
  
## <a name="resolution"></a>**Решение**  
*Перенастройка пулов ресурсов хранения, таким образом, чтобы несколько пулов, не используйте один и тот же путь хранилища с помощью Windows PowerShell.*

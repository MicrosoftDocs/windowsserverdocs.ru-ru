# <a name="avoid-mapping-one-storage-path-to-multiple-resource-pools"></a>Избегайте сопоставления одного пути хранилища с несколькими пулами ресурсов

>Область применения: Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>**Проблема**  
*Путь к файлу хранилища сопоставляется с несколькими пулами ресурсов.*  
  
## <a name="impact"></a>**Благоприятн**  
*Для указанного типа пула носителей следующие родительские и дочерние пулы имеют один и тот же путь к хранилищу:*  
  
\<список пулов >  
  
## <a name="resolution"></a>**Решение**  
*Используйте Windows PowerShell для перенастройки пулов ресурсов хранилища, чтобы несколько пулов не использовали один и тот же путь к хранилищу.*

# <a name="a-virtual-machine-running-windows-server-2012-r2-and-configured-with-dynamic-memory-should-use-recommended-values-for-memory-settings"></a>Создание виртуальной машины под управлением Windows Server 2012 R2, настроены с помощью динамической памяти следует использовать рекомендуемые значения для параметров памяти

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>Проблемы  
*Один или несколько виртуальных машин настроены для использования динамической памяти с меньшим, чем объем памяти, рекомендуется использовать для Windows Server 2012 R2.*  
  
## <a name="impact"></a>Влияние  
*Гостевой операционной системы на следующих виртуальных машин может не запуститься или может привести запуска:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Увеличьте минимальный объем памяти менее 256 МБ, память при запуске менее 512 МБ и максимальный объем памяти по крайней мере 2 ГБ для этой виртуальной машины с помощью диспетчера Hyper-V.*  
  
#### <a name="increase-memory-using-hyper-v-manager"></a>Увеличьте объем памяти, с помощью диспетчера Hyper-V  
  
1.  Откройте диспетчер Hyper-V. (В диспетчере сервера щелкните **средства** > **диспетчера Hyper-V**.)  
  
2.  Из списка виртуальных машин, щелкните правой кнопкой мыши и нажмите щелкните **параметры**.  
  
3.  В области навигации щелкните **памяти**.  
  
4.  Изменение **ОЗУ** менее 512 МБ.  
  
5.  В разделе **динамической памяти**, изменить **минимальный объем ОЗУ** менее 256 МБ и **максимальный объем ОЗУ** до 2 ГБ.  
  
6.  Нажмите кнопку **ОК**.  
  
### <a name="increase-memory-using-windows-powershell"></a>Увеличьте объем памяти, с помощью Windows PowerShell  
  
1.  Откройте Windows PowerShell. (На рабочем столе щелкните **запустить** и начните вводить **Windows PowerShell**.)  
  
2.  Щелкните правой кнопкой мыши **Windows PowerShell** и нажмите кнопку **Запуск от имени администратора**.  
  
3.  Выполните команду, аналогичную приведенной ниже, подставив имя вашей виртуальной машины и память MyVM значения с по крайней мере значения, приведенные ниже.  
  
```  
Get-VM MyVM | Set-VMMemory -DynamicMemoryEnabled $True -MaximumBytes 2GB -MinimumBytes 256MB -StartupBytes 512MB  
```

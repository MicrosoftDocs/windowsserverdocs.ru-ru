# <a name="avoid-using-virtual-hard-disks-with-a-sector-size-less-than-the-sector-size-of-the-physical-storage-that-stores-the-virtual-hard-disk-file"></a>Старайтесь не использовать виртуальные жесткие диски с размером сектора меньше размера сектора физического хранилища, в котором хранится файл виртуального жесткого диска

>Область применения. Windows Server 2016

## <a name="issue"></a>**Проблема**  
*Один или несколько виртуальных жестких дисков имеют размер физического сектора, которая меньше, чем размер физического сектора хранилища, на котором находится файл виртуального жесткого диска.*  
  
## <a name="impact"></a>**Влияние**  
*На виртуальной машине или приложение, которое используется виртуальный жесткий диск, могут возникнуть проблемы производительности. Это влияет на следующие виртуальные машины:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>**Решение**  
*Выполните одно из следующих действий:*  
  
-   *Выполнить миграцию хранилища для перемещения виртуального жесткого диска на новую систему физических*  
  
-   *С помощью Windows PowerShell или WMI для включения виртуальный жесткий диск формата VHDX об определенных секторов размером*  
  
-   *Используйте параметр реестра, чтобы включить формат VHD виртуального жесткого диска для сообщения размером физический сектор размером 4 КБ*

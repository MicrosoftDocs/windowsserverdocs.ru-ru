# <a name="domain-membership-is-recommended-for-servers-running-hyper-v"></a>Членство в домене рекомендуется для серверов с Hyper-V

>Область применения: Windows Server 2016


  
*Дополнительные сведения о рекомендациях и проверках см. в разделе* [анализатор соответствия рекомендациям](https://go.microsoft.com/fwlink/?LinkId=122786).  
  
|Свойство|Подробности|  
|-|-|  
|**Операционная система**|Windows Server 2016|  
|**Продукт или функция**|Hyper-V|  
|**Серьезности**|Тревожное|  
|**Категория**|Конфигурация|  
  
В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>Проблема  
  
*Этот сервер является членом рабочей группы.*  
  
## <a name="impact"></a>Влияние  
  
*Для этого сервера не существует централизованного управления.*  
  
Присоединение этого компьютера к домену обеспечивает централизованное управление с помощью политик для удостоверений, безопасности и аудита.  
  
## <a name="resolution"></a>Разрешение  
  
*Если у вас есть доступная среда домена, присоедините этот сервер к этому домену.*  
  
> [!IMPORTANT]  
> Рекомендуется проверить рабочие нагрузки, выполняемые на виртуальных машинах на этом компьютере, чтобы определить, есть ли последствия присоединения этого компьютера к домену. Если какая-либо из виртуальных машин является виртуализированными контроллерами домена, см. статью [вопросы планирования для виртуализированных контроллеров домена](https://go.microsoft.com/fwlink/?LinkId=190192) (https://go.microsoft.com/fwlink/?LinkId=190192).  
  
Для присоединения компьютера к домену требуются разрешения на компьютере и в домене:   
- На компьютере потребуется учетная запись пользователя, которая является членом группы "Администраторы". Либо войдите в систему с этим типом учетной записи, либо укажите имя пользователя и пароль для учетной записи при появлении запроса.   
- В домене вам потребуется учетная запись пользователя, которая имеет право присоединить компьютер к домену. Вам будет предложено ввести имя пользователя и пароль.  
  
Инструкции см. в разделе [Присоединение компьютера к домену](https://go.microsoft.com/fwlink/?LinkId=190193) (https://go.microsoft.com/fwlink/?LinkId=190193).

# <a name="domain-membership-is-recommended-for-servers-running-hyper-v"></a>Членство в домене рекомендуется для серверов под управлением Hyper-V

>Область применения. Windows Server 2016


  
*Дополнительные сведения о рекомендациях и проверках см. в разделе* [анализатор соответствия рекомендациям](https://go.microsoft.com/fwlink/?LinkId=122786).  
  
|Свойство|Подробности|  
|-|-|  
|**Операционная система**|Windows Server 2016|  
|**Продукта или компонента**|Hyper-V|  
|**Уровень серьезности**|Предупреждение|  
|**Категория**|Конфигурация|  
  
В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>Проблемы  
  
*Этот сервер является членом рабочей группы.*  
  
## <a name="impact"></a>Влияние  
  
*Нет централизованного управления для этого сервера.*  
  
Присоединять компьютер к домену обеспечивает централизованное управление с помощью политики для идентификации, безопасности и аудита.  
  
## <a name="resolution"></a>Разрешение  
  
*При наличии доступных среде домена, присоедините сервер к этому домену.*  
  
> [!IMPORTANT]  
> Мы рекомендуем ознакомиться с рабочих нагрузок, выполняющихся на виртуальных машинах на этом компьютере, чтобы определить, имеются ли последствия для системы безопасности присоединять компьютер к домену. Если любой из виртуальных машин, виртуализованных контроллеров доменов см. в разделе [вопросы планирования для виртуализированных контроллеров домена](https://go.microsoft.com/fwlink/?LinkId=190192) (https://go.microsoft.com/fwlink/?LinkId=190192).  
  
Присоединение компьютера к домену требуются разрешения на этом компьютере и домен:   
- На компьютере вам потребуется учетную запись пользователя, которая является членом группы "Администраторы". Либо войдите в систему с этим типом учетной записи или укажите имя пользователя и пароль для учетной записи, когда появится запрос.   
- На домен вам потребуется учетную запись пользователя, которая права для присоединения компьютера к домену. Вам будет предложено имя пользователя и пароль.  
  
Инструкции см. в разделе [присоединить компьютер к домену](https://go.microsoft.com/fwlink/?LinkId=190193) (https://go.microsoft.com/fwlink/?LinkId=190193).

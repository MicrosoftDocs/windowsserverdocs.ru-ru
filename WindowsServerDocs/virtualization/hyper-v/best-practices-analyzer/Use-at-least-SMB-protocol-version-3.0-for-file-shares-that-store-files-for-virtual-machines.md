# <a name="use-at-least-smb-protocol-version-30-for-file-shares-that-store-files-for-virtual-machines"></a>Используйте по меньшей мере версия протокола SMB 3.0 для файловых ресурсов, в которых хранятся файлы для виртуальных машин.

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).  
  
|Свойство|Подробности|  
|-|-|  
|**Операционная система**|Windows Server 2016|  
|**Продукта или компонента**|Hyper-V|  
|**Уровень серьезности**|Ошибка|  
|**Категория**|Конфигурация|  
  
В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>**Проблема**  
*Файлы виртуальной машины или виртуальные жесткие диски хранятся в общей папке, которая не поддерживает по крайней мере версия протокола SMB 3.0.*  
  
## <a name="impact"></a>**Влияние**  
*Майкрософт не поддерживает эту конфигурацию. Это влияет на следующие виртуальные машины:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>**Решение**  
*Переместить файлы в общую папку, по крайней мере использует версию протокола SMB 3.0.*

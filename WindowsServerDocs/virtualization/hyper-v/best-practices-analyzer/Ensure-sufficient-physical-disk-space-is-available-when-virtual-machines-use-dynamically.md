# <a name="ensure-sufficient-physical-disk-space-is-available-when-virtual-machines-use-dynamically-expanding-virtual-hard-disks"></a>Обеспечение доступности достаточного места на физическом диске, если виртуальные машины используют динамически расширяемые виртуальные жесткие диски

>Область применения: Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>Проблема  
*Одна или несколько виртуальных машин используют динамически расширяемые виртуальные жесткие диски.*  
  
## <a name="impact"></a>Влияние  
*Для динамического расширения виртуальных жестких дисков требуется свободное место на томе размещения, чтобы можно было выделить место при записи на виртуальные жесткие диски. Если доступное место исчерпано, это может повлиять на все виртуальные машины, использующие физическое хранилище. Это влияет на следующие виртуальные машины:*  
  
\<список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Мониторинг свободного места на диске для обеспечения достаточного объема свободного места для расширения. Рекомендуется завершить работу виртуальной машины и с помощью мастера изменения дисков в диспетчере Hyper-V преобразовать каждый динамически расширяемый виртуальный жесткий диск для этой виртуальной машины в виртуальный жесткий диск фиксированного размера.*

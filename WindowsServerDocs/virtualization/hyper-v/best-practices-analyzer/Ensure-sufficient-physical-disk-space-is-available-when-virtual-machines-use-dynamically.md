# <a name="ensure-sufficient-physical-disk-space-is-available-when-virtual-machines-use-dynamically-expanding-virtual-hard-disks"></a>Убедитесь, что достаточно места на физическом диске доступен, если виртуальные машины используют динамически Расширяемые виртуальные жесткие диски

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>Проблемы  
*Один или несколько виртуальных машин при использовании динамически расширяемых виртуальных жестких дисков.*  
  
## <a name="impact"></a>Влияние  
*Динамически Расширяемые виртуальные жесткие диски необходима для доступного места на томе размещения может быть выделено места, при записи для виртуальных жестких дисков. Если доступных пространство будет исчерпано, может повлиять на любой виртуальной машины, которая зависит от физического хранилища. Это влияет на следующие виртуальные машины:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Монитор свободного места, чтобы обеспечить достаточно места для расширения. Рассмотрите возможность завершения работы виртуальной машины и использовать мастера изменения диска в диспетчере Hyper-V для преобразования каждого динамически расширяемый виртуальный жесткий диск для этой виртуальной машины в фиксированного размера виртуального жесткого диска.*

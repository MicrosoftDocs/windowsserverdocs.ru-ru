# <a name="storage-controllers-should-be-enabled-in-virtual-machines-to-provide-access-to-attached-storage"></a>Контроллеры запоминающих устройств должна быть включена в виртуальных машинах, чтобы предоставить доступ к подключенным хранилищем

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и сканировании см. в разделе [Анализатор соответствия рекомендациям](https://go.microsoft.com/fwlink/?LinkId=122786).  
  
|Свойство|Подробности|  
|-|-|  
|**Операционная система**|Windows Server 2016|  
|**Продукта или компонента**|Hyper-V|  
|**Уровень серьезности**|Предупреждение|  
|**Категория**|Конфигурация|  

В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.

## <a name="issue"></a>Проблемы  
  
*Один или несколько контроллеров запоминающих устройств может быть отключена на виртуальной машине.*  
  
## <a name="impact"></a>Влияние  
  
*Виртуальные машины нельзя использовать хранилище, подключены к контроллеру хранилища отключено. Это влияет на следующие виртуальные машины:*  
  
\<Список имен виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
  
*С помощью диспетчера устройств в гостевой операционной системы для включения всех контроллеров запоминающих устройств. Если контроллер хранилища не требуется, используйте диспетчер Hyper-V удалить его из виртуальной машины.*  
  
Инструкции о том, как использовать диспетчер устройств см. в разделе справки в гостевой операционной системы. Инструкции по удалению контроллера хранилища см. следующую процедуру.  
  
#### <a name="to-remove-a-scsi-storage-controller-from-the-virtual-machine"></a>Для удаления хранилища SCSI-контроллер из виртуальной машины  
  
1.  Откройте диспетчер Hyper-V. Нажмите кнопку **запустить**, пункты **Администрирование**, а затем нажмите кнопку **диспетчера Hyper-V**.  
  
2.  В области результатов в разделе **виртуальных машин**, выберите виртуальную машину, которую требуется настроить.  
  
3.  Если виртуальная машина работает, завершите работу виртуальной машины. Щелкните правой кнопкой мыши виртуальную машину и нажмите кнопку **завершение работы**.  
  
4.  На панели **Действия** откройте раздел **Параметры** рядом с именем виртуальной машины.  
  
5.  В левой части **параметры** диалогового **оборудования**, нажмите кнопку **SCSI-контроллер**.  
  
6.  В области справа щелкните **удалить**.

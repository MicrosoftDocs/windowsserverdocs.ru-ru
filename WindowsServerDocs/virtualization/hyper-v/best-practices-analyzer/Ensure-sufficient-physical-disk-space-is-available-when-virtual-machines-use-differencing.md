# <a name="ensure-sufficient-physical-disk-space-is-available-when-virtual-machines-use-differencing-virtual-hard-disks"></a>Убедитесь, что достаточно места на физическом диске доступен, если виртуальные машины используют разностных виртуальных жестких дисков

>Область применения. Windows Server 2016

## <a name="issue"></a>Проблемы  
*Один или несколько виртуальных машин при использовании разностных виртуальных жестких дисков.*  
  
## <a name="impact"></a>Влияние  
*Разностных виртуальных жестких дисков необходима для доступного места на томе размещения может быть выделено места, при записи для виртуальных жестких дисков. Если доступных пространство будет исчерпано, может повлиять на любой виртуальной машины, которая зависит от физического хранилища. Это влияет на следующие виртуальные машины:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Монитор свободного места, чтобы обеспечить достаточно места для расширения виртуального жесткого диска. Рассмотрите возможность слияния разностных виртуальных жестких дисков в родительских. В диспетчере Hyper-V изучите разностный диск, чтобы определить родительский виртуальный жесткий диск. Если разностный диск на родительском диске, который является общим для других разностные диски, это действие приведет к повреждению связь между другими разностные диски и на родительском диске, что делает их невозможно использовать. Убедившись, что родительский виртуальный жесткий диск не является общим, можно использовать мастера изменения диска для слияния разностный диск для родительского виртуального жесткого диска.*

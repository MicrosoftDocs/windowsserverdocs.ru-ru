# <a name="display-adapters-should-be-enabled-in-virtual-machines-to-provide-video-capabilities"></a>Для предоставления возможностей видео на виртуальных машинах должны быть включены видеоадаптеры.

>Область применения. Windows Server 2016

## <a name="issue"></a>Проблема

*На виртуальной машине может быть отключено видео-устройство шины виртуальной машины (Майкрософт).*

Видеоустройство шины виртуальной машины (Майкрософт) — это виртуальный видеоадаптер, оптимизированный для использования с виртуальными машинами Hyper-V. Если виртуальная машина не настроена для использования видеоустройства шины виртуальной машины Майкрософт, используется устаревший видеоадаптер. Видеоустройство шины виртуальной машины Майкрософт работает лучше, чем устаревший видеоадаптер.

## <a name="impact"></a>Влияние

*Производительность видео для следующих виртуальных машин будет снижена:*

\<list of virtual machine names>

## <a name="resolution"></a>Решение

*Используйте Device Manager в гостевой операционной системе, чтобы включить видеоустройство шины виртуальной машины Майкрософт.*

Действия, необходимые для использования Device Manager, зависят от операционной системы. Инструкции см. в справке операционной системы на виртуальной машине.

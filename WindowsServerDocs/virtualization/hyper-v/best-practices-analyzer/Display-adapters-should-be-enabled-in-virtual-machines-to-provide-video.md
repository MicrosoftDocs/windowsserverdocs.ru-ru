# <a name="display-adapters-should-be-enabled-in-virtual-machines-to-provide-video-capabilities"></a>Видеоадаптеры следует включить в виртуальных машинах для обеспечения возможностей видео

>Область применения. Windows Server 2016


  
*Дополнительные сведения о рекомендациях и проверках см. в разделе* [анализатор соответствия рекомендациям](https://go.microsoft.com/fwlink/?LinkId=122786).

|**Продукта или компонента**|Hyper-V|  
|**Уровень серьезности**|Предупреждение|

В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>Проблемы  
  
*Устройство видео шины виртуальной машины Microsoft может быть отключена на виртуальной машине.*  
  
Устройство видео шины виртуальной машины Microsoft является виртуальный адаптер видео, оптимизирован для использования с виртуальными машинами Hyper-V. При виртуальной машины не настроен для использования устройства видео шины виртуальной машины Microsoft, используются устаревшие видеоадаптера. Устройство видео шины виртуальной машины Microsoft работает лучше, чем устаревший видеоадаптера.  
  
## <a name="impact"></a>Влияние  
  
*Приведет к снижению производительности видео для следующих виртуальных машин:*  
  
\<Список имен виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
  
*Используйте диспетчер устройств в гостевой операционной системы, чтобы включить устройство видео шины виртуальной машины Microsoft.*  
  
Шаги, необходимые для использования диспетчера устройств зависят от операционной системы. Инструкции см. в разделе справки в гостевой операционной системы.

# <a name="one-or-more-network-adapters-should-be-configured-as-the-source-for-port-mirroring"></a>Один или несколько сетевых адаптеров должны быть настроены как источник для зеркального отображения портов

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).  
  
|Свойство|Подробности|

|**Продукта или компонента**|Hyper-V|  
|**Уровень серьезности**|Предупреждение|  
|**Категория**|Конфигурация|  
  
В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>**Проблема**  
*Один или несколько виртуальных машин имеют сетевой адаптер, настроена в качестве назначения для зеркального отображения портов, но отсутствует соответствующий источник на виртуальном коммутаторе.*  
  
## <a name="impact"></a>**Влияние**  
*Зеркальное отображение портов будут завершаться ошибкой для следующих виртуальных коммутаторов и виртуальных машин:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>**Решение**  
*Для выполнения или исправьте конфигурацию зеркального отображения портов, используйте Windows PowerShell или диспетчера Hyper-V.*

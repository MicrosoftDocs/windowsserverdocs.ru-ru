# <a name="avoid-storing-smart-paging-files-on-a-system-disk"></a>Избегайте хранения файлов Smart paging на системном диске

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и сканировании см. в разделе [Запуск сканирования анализатором соответствия рекомендациям и управление результатами сканирования](https://go.microsoft.com/fwlink/p/?LinkID=223177).

|**Продукт или компонент**|Hyper-V|
|**Уровень серьезности**|Предупреждение|
|**Категория**|Операции|

## <a name="issue"></a>Проблема
*При настройке памяти для одной или нескольких виртуальных машин может потребоваться интеллектуальное разбиение на страницы, если виртуальная машина перезагружается, а в указанном расположении для файла Smart подкачки — системный диск сервера, на котором выполняется Hyper-V.*

## <a name="impact"></a>Влияние
*Использование системного диска для Smart paging может привести к возникновению проблем на сервере, на котором работает Hyper-V. Это влияет на следующие виртуальные машины:*

\<list of virtual machines>

## <a name="resolution"></a>Решение
*Перенастройте виртуальные машины для хранения файлов Smart paging на несистемном диске.*

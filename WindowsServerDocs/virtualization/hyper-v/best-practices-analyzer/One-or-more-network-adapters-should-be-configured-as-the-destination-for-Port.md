# <a name="one-or-more-network-adapters-should-be-configured-as-the-destination-for-port-mirroring"></a>Один или несколько сетевых адаптеров должны быть настроены в качестве места назначения для зеркального отображения портов.

>Область применения: Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

|**Продукт или функция**|Hyper-V|  
|**Серьезности**|Warning|  
|**Категория**|Настройка|

## <a name="issue"></a>**Проблема**  
*Для одной или нескольких виртуальных машин настроен сетевой адаптер в качестве источника зеркального отображения портов, но в виртуальном коммутаторе нет соответствующего назначения.*  
  
## <a name="impact"></a>**Благоприятн**  
*Зеркальное отображение портов не будет правильно работать для следующих виртуальных коммутаторов и виртуальных машин:*  
  
\<список виртуальных машин >  
  
## <a name="resolution"></a>**Решение**  
*Используйте Windows PowerShell или диспетчер Hyper-V для завершения или исправления конфигурации зеркального отображения портов.*

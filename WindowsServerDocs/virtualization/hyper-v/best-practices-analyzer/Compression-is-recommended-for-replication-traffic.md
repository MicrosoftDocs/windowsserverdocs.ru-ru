# <a name="compression-is-recommended-for-replication-traffic"></a>Рекомендуется использовать сжатие для трафика репликации

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>Проблемы  
*Трафик репликации, отправляемых по сети с сервера-источника на сервер-реплику без сжатия.*  
  
## <a name="impact"></a>Влияние  
*Трафик репликации будет использовать больше пропускной способности, чем необходимо. Это влияет на следующие виртуальные машины:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Настройка реплики Hyper-V для сжатия данных, передаваемых по сети в параметрах виртуальной машины в диспетчере Hyper-V. Средства вне Hyper-V также можно выполнять сжатие.*

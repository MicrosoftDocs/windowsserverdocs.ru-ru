# <a name="replication-is-paused-for-one-or-more-virtual-machines-on-this-server"></a>Репликация для одной или нескольких виртуальных машин на этом сервере приостановлена

## <a name="issue"></a>Проблема  
*Репликация приостановлена для одной или нескольких виртуальных машин. Пока основная виртуальная машина приостановлена, все произошедшие изменения будут накапливаться и будут отправлены на виртуальную машину реплики после возобновления репликации.*  
  
## <a name="impact"></a>Влияние  
*Пока репликация приостановлена, накопленные изменения, происходящие на основной виртуальной машине, будут потреблять доступное дисковое пространство на сервере-источнике. После возобновления репликации может возникнуть большой объем сетевого трафика на сервере реплики. Это влияет на следующие виртуальные машины:*  
  
\<список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Убедитесь, что приостановка репликации предполагалась. Если репликация была приостановлена для устранения нехватки места на диске или подключения к сети, возобновите репликацию сразу после устранения этих проблем.*

# <a name="vmq-should-be-enabled-on-vmq-capable-physical-network-adapters-bound-to-an-external-virtual-switch"></a>VMQ, которые должны быть включены на физических сетевых адаптеров с VMQ привязан к внешнему виртуальному коммутатору

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>**Проблема**  
*Следующие сетевые адаптеры поддерживают очередь виртуальной машины (VMQ), но эта возможность отключена.*  
  
## <a name="impact"></a>**Влияние**  
*Windows не сможет воспользоваться всеми преимуществами доступных аппаратную разгрузку на сетевых адаптерах следующие:*  
  
\<Список сетевых адаптеров >  
  
## <a name="resolution"></a>**Решение**  
*Включите очередь виртуальных МАШИН с помощью командлета Enable-NetAdapterVmq Windows PowerShell или дополнительные свойства пользовательского интерфейса для сетевого адаптера.*

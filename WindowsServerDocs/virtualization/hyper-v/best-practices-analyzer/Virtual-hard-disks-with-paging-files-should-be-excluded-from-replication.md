# <a name="virtual-hard-disks-with-paging-files-should-be-excluded-from-replication"></a>Виртуальные жесткие диски с файлами подкачки следует исключить из репликации

>Область применения. Windows Server 2016

## <a name="issue"></a>Проблемы  
*Файлы подкачки следует исключить из участия в репликации, но диски не были исключены.*  
  
## <a name="impact"></a>Влияние  
файлы *Paging имеют большой объем действий ввода-вывода, что необязательно требует гораздо большего количества ресурсов для участия в репликации. Это влияет на следующие виртуальные машины:* .  
  
@no__t 0list виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*If, вы еще не сделали этого, создайте отдельный виртуальный жесткий диск для файла подкачки Windows. Если начальная репликация уже выполнена, используйте диспетчер Hyper-V для удаления репликации. Затем снова настройте репликацию и исключите виртуальный жесткий диск с файлом подкачки из репликации.*

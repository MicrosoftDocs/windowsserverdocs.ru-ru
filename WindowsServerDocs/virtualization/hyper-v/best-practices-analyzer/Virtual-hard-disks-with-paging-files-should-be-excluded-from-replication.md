# <a name="virtual-hard-disks-with-paging-files-should-be-excluded-from-replication"></a>Виртуальные жесткие диски с файлами подкачки, которые должны быть исключены из репликации

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>Проблемы  
*Разбиение на страницы файлы должны быть исключены из репликации, но диски не были исключены.*  
  
## <a name="impact"></a>Влияние  
*Файлы подкачки столкнуться с большим объемом операций ввода вывода, без необходимости требовалось гораздо больше ресурсов, участвующих в репликации. Это влияет на следующие виртуальные машины:*  
  
\<Список виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
*Если вы еще не сделано, создайте отдельный виртуальный жесткий диск для файла подкачки Windows. Если начальная репликация уже была завершена, удалите репликацию с помощью диспетчера Hyper-V. Затем снова настроить репликацию и исключить виртуальный жесткий диск с файлом подкачки из репликации.*

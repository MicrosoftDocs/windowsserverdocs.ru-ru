# <a name="virtual-hard-disks-with-paging-files-should-be-excluded-from-replication"></a>Виртуальные жесткие диски с файлами подкачки следует исключить из репликации

>Область применения. Windows Server 2016

## <a name="issue"></a>Проблема
*Файлы подкачки следует исключить из участия в репликации, но диски не были исключены.*

## <a name="impact"></a>Влияние
*Файлы подкачки имеют большой объем действий ввода-вывода, что необязательно требует гораздо большего количества ресурсов для участия в репликации. Это влияет на следующие виртуальные машины:*

\<list of virtual machines>

## <a name="resolution"></a>Решение
*Создайте отдельный виртуальный жесткий диск для файла подкачки Windows, если вы еще этого не сделали. Если начальная репликация уже выполнена, используйте диспетчер Hyper-V для удаления репликации. Затем снова настройте репликацию и исключите виртуальный жесткий диск с файлом подкачки из репликации.*

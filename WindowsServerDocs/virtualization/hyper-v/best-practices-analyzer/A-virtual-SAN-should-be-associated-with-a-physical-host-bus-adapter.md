# <a name="a-virtual-san-should-be-associated-with-a-physical-host-bus-adapter"></a>Виртуальная сеть SAN должна быть связана с физического адаптера шины

>Область применения. Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

## <a name="issue"></a>**Проблема**  
*Связь с адаптер шины (HBA) была настроена виртуальная сеть хранения (данных SAN).*  
  
## <a name="impact"></a>**Влияние**  
*Виртуальная машина не запускается, если она настроена с помощью виртуального адаптера Fibre Channel, подключенных к неправильной настройкой виртуальной сети SAN. Это влияет на следующих виртуальных сетей SAN:*  
  
  
\<Список виртуальных сетей хранения данных >  
  
  
## <a name="resolution"></a>**Решение**  
*Перенастройте виртуальную сеть SAN путем подключения его к адаптер шины.*

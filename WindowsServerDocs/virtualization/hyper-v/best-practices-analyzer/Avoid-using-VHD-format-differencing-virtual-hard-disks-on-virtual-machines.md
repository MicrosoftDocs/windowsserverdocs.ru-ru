## <a name="issue"></a>**Проблема**  
*Одна или несколько виртуальных машин используют разностные виртуальные жесткие диски в формате VHD.*  
  
## <a name="impact"></a>**Благоприятн**  
При возникновении сбоя питания *VHD-Format разностные виртуальные жесткие диски могут столкнуться с проблемами согласованности. Проблемы согласованности могут возникать, если физический диск выполняет неполное или неправильное обновление сектора в VHD-файле, который изменяется при сбое питания. Это влияет на следующие виртуальные машины:*  
  
@no__t 0list виртуальных машин >  
  
## <a name="resolution"></a>**Решение**  
@no__t 0Shut виртуальную машину и преобразуйте цепочку разностных виртуальных жестких дисков в формате VHD в формат VHDX или объедините цепочку с фиксированным виртуальным жестким диском. (Формат VHDX имеет механизмы надежности, которые помогают защитить диск от повреждений из-за сбоев питания.) Однако не преобразуйте виртуальный жесткий диск, если он, вероятно, будет подключен к более ранней версии Windows в некоторый момент. Версии Windows, предшествующие Windows Server 2012, не поддерживают формат VHDX. *

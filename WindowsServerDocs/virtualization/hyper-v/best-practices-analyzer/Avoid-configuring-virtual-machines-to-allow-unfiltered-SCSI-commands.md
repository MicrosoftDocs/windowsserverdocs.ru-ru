# <a name="avoid-configuring-virtual-machines-to-allow-unfiltered-scsi-commands"></a>Не настраивать виртуальные машины, чтобы разрешить нефильтрованные команды SCSI

>Область применения. Windows Server 2016

## <a name="issue"></a>Проблемы  
  
*Виртуальная машина настроена возможность нефильтрованные командам SCSI.*  
  
## <a name="impact"></a>Влияние  
  
*Обход команду SCSI, фильтрация создает угрозу безопасности. Эта конфигурация должна быть включена только в том случае, если это необходимо для обеспечения совместимости с хранения приложений, работающих в гостевой операционной системы. Следующие виртуальные машины настроены для разрешения нефильтрованные команд SCSI:*  
  
\<Список имен виртуальных машин >  
  
## <a name="resolution"></a>Разрешение  
  
*Обратитесь к поставщику хранилища, чтобы определить, если эта конфигурация не требуется. Кроме того Если управляющая операционная система или другие гостевые операционные системы будут скомпрометированы, необычное поведение различных перенастройте виртуальную машину, чтобы блокировать команды.*

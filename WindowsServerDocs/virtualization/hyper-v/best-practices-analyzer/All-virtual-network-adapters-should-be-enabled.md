# <a name="all-virtual-network-adapters-should-be-enabled"></a>Должен быть включен всех виртуальных сетевых адаптеров

>Область применения. Windows Server 2016


  
|Свойство|Подробности|  
|-|-|  
|**Операционная система**|Windows Server 2016|  
|**Продукта или компонента**|Hyper-V|  
|**Уровень серьезности**|Предупреждение|  
|**Категория**|Конфигурация|  
  
В следующих разделах курсив указывает текст пользовательского Интерфейса, который отображается в анализатор соответствия рекомендациям для этой проблемы.  
  
## <a name="issue"></a>Проблемы  
  
*Один или несколько виртуальных сетевых адаптеров, связанных с физическим сетевым адаптером отключены в управляющей операционной системе.*  
  
## <a name="impact"></a>Влияние  
  
*Конфигурация этого сервера не является оптимальным.*  
  
Управляющей операционной системе не удается подключиться к физической сети (внешний), с помощью одного из физических сетевых адаптеров на этом компьютере, так как оно связано с отключенный виртуальный сетевой адаптер.  
  
## <a name="resolution"></a>Разрешение  
  
*Используйте параметры сети и Интернета, чтобы включить виртуальный сетевой адаптер. Также можно использовать диспетчер виртуальных коммутаторов, перенастройка внешний виртуальный коммутатор, таким образом, чтобы он не используется совместно с управляющей операционной системе.*

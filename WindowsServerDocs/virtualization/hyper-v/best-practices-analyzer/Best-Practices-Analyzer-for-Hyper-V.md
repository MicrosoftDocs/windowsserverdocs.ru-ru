# <a name="best-practices-analyzer-for-hyper-v"></a>Анализатор соответствия рекомендациям для Hyper-V

>Область применения. Windows Server 2016
  
*Рекомендации* по управлению Windows — это руководства, которые эксперты считают идеальным способом настройки сервера в обычных условиях. Нарушение рекомендаций, даже очень важных, необязательно может вызывают проблемы. Но они могут указывают конфигурации сервера, которые могут привести к снижению производительности, надежности, непредвиденным конфликтам, повышению рисков для безопасности или другим потенциальным неполадкам.  
  
Анализатор соответствия рекомендациям проверяет компьютер, с помощью правил на основе этих рекомендаций и сообщает о результатах. Каждого правила рекомендаций включает в себя сведения о том, как в соответствии с правилом. В данном разделе описываются эти правила, чтобы помочь вам понять рекомендации, которые применяются к Hyper-V, а также Интерпретация результатов при проверки находит условия, которые не соответствуют рекомендациям.  
  
Дополнительные сведения о анализатор соответствия рекомендациям и проверках см. в разделе [анализатор соответствия рекомендациям](https://go.microsoft.com/fwlink/?LinkId=122786).  
  
## <a name="about-hyper-v"></a>Сведения о Hyper-V  
Hyper-V позволяет запускать несколько операционных систем одновременно на одном физическом компьютере, выполнив каждый из них в виртуальной машине. Виртуальные машины помогут вам использовать вычислительные ресурсы, гибко и более эффективно. Дополнительные сведения о Hyper-V, см. в разделе [Hyper-V в Windows Server 2016](../Hyper-V-on-Windows-Server.md).

# <a name="configure-a-virtual-machine-with-a-scsi-controller-to-be-able-to-hot-plug-and-hot-unplug-storage"></a>Настройка виртуальной машины с контроллером SCSI, чтобы иметь возможность горячей замены и "Горячий", отключите хранилище

## <a name="issue"></a>Проблемы  
  
*Найдено виртуальной машины, не настроен контроллер SCSI.*  
  
## <a name="impact"></a>Влияние  
  
*Вы не сможете горячей замены или "Горячий", отключите хранилище для следующих виртуальных машин:*  
  
\<Список имен виртуальных машин >  
  
Возможность горячей замены или "Горячий", отключите хранилище упрощает управление потребностей в хранении виртуальной машины без необходимости простоев. Виртуальные машины без контроллеров SCSI необходимо завершить работу, прежде чем можно добавлять или удалять хранилища.  
  
## <a name="resolution"></a>Разрешение  
  
*Если вы не обязательно должны горячей замены или "Горячий", отключите хранилище для этой виртуальной машины, никаких действий не требуется. В противном случае завершите работу виртуальной машины и добавьте в конфигурацию SCSI-контроллер.*  
  
SCSI-контроллер для горячей замены и "Горячий", отключите хранилище, гостевая операционная система выполняется текущая версия служб integration services.

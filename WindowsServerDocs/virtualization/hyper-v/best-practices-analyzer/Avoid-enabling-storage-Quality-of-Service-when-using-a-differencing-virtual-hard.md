# <a name="avoid-enabling-storage-quality-of-service-when-using-a-differencing-virtual-hard-disk-when-the-parent-and-child-virtual-hard-disks-are-on-different-volumes"></a>Старайтесь не включать качество обслуживания хранилища при использовании разностного виртуального жесткого диска, если родительские и дочерние виртуальные жесткие диски находятся на разных томах.

>Область применения: Windows Server 2016

Дополнительные сведения о рекомендациях и проверках см. в разделе [запуска наиболее проверок анализатором соответствия рекомендациям и Управление результатами проверок](https://go.microsoft.com/fwlink/p/?LinkID=223177).

|**Продукт или функция**|Hyper-V|  
|**Серьезности**|Warning|  
|**Категория**|Настройка|

## <a name="issue"></a>**Проблема**  
*Для разностного виртуального жесткого диска с родительским и дочерним виртуальными жесткими дисками на разных томах включено качество обслуживания хранилища.*  
  
## <a name="impact"></a>**Благоприятн**  
*Такая конфигурация может привести к непредвиденному поведению качества службы хранилища для разностного виртуального жесткого диска, а также к другим виртуальным жестким дискам на родительском и дочернем томах. Это влияет на следующие виртуальные жесткие диски:*  
  
\<список виртуальных жестких дисков >  
  
## <a name="resolution"></a>**Решение**  
*Отключите качество обслуживания хранилища на указанных виртуальных жестких дисках или выполните миграцию хранилища, чтобы переместить родительский и дочерний виртуальный жесткий диск в один и тот же том.*

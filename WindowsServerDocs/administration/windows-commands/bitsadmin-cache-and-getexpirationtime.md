# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin cache и getexpirationtime

Получает время истечения срока действия кэша.

## <a name="syntax"></a>Синтаксис

```
bitsadmin /cache /getexpirationtime
```

## <a name="examples"></a>Примеры

Чтобы получить время истечения срока действия кэша, сделайте следующее:

```
bitsadmin /cache /getexpirationtime
```

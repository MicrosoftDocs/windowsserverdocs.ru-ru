# <a name="scwcmd"></a>Scwcmd

> Область применения. Windows Server 2012 R2, Windows Server 2012

Средство командной строки Scwcmd.exe, включенные с помощью мастера настройки безопасности (SCW) можно использовать для выполнения следующих задач:
-   Настройте один или несколько серверов с политикой сформированные SCW.
-   Анализ один или несколько серверов с политикой сформированные SCW.
-   Просмотр результатов анализа в формате HTML.
-   Выполнить откат политики SCW.
-   Преобразуйте политику сформированные SCW в файлы, которые поддерживаются с помощью групповой политики.
-   Зарегистрируйте расширение базы данных настройки безопасности с помощью этого мастера.

При использовании **scwcmd** для настройки, анализ или откатить политику на удаленном сервере, SCW необходимо установить на удаленном сервере.

## <a name="syntax"></a>Синтаксис

```
scwcmd <command> [<subcommand>]
```

## <a name="parameters"></a>Параметры

|Подкоманда|Описание|
|----------|-----------|
|/ analyze|Определяет, является ли компьютер соответствующих требованиям политики.</br>См. в разделе [Scwcmd: анализ](scwcmd-analyze.md) синтаксис и параметры.|
|/ configure|Применяет политику безопасности, сформированные SCW на компьютере.</br>См. в разделе [Scwcmd: Настройка](scwcmd-configure.md) синтаксис и параметры.|
|/ Register|Расширяет или настраивает базы данных конфигурации безопасности SCW, зарегистрировав файл базы данных настройки безопасности, содержащий роли, задачи, службы или определения порта.</br>См. в разделе [Scwcmd: регистрация](scwcmd-register.md) синтаксис и параметры.|
|/ROLLBACK|Применяет самые последние доступные политику отката, а затем удаляет эту политику отката.</br>См. в разделе [Scwcmd: отката](scwcmd-rollback.md) синтаксис и параметры.|
|/ Transform|Преобразует файл политики безопасности, созданные с помощью мастера настройки безопасности в новый объект групповой политики (GPO) в доменных службах Active Directory.</br>См. в разделе [Scwcmd: преобразование](scwcmd-transform.md) синтаксис и параметры.|
|/ View|Отображает XML-файл с помощью указанного .xsl преобразования.</br>См. в разделе [Scwcmd: представление](scwcmd-view.md) синтаксис и параметры.|
|/?|Отображение справки в командной строке.|

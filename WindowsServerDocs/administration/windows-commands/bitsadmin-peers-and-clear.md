# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers и clear

Очищает список одноранговых узлов.

## <a name="syntax"></a>Синтаксис

```
bitsadmin /peers /clear
```

## <a name="examples"></a>Примеры

Для очистки однорангового списка.

```
bitsadmin /peers /clear
```

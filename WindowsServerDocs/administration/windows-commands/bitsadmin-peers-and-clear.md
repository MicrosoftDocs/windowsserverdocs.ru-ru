# <a name="bitsadmin-peers-and-clear"></a>одноранговые узлы bitsadmin и произведите его очистку



Очищает список однорангового узла.

## <a name="syntax"></a>Синтаксис

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>Примеры

В следующем примере удаляется список одноранговых узлов.
```
C:\>bitsadmin /Peers /Clear
```

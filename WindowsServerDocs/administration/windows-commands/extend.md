# <a name="extend"></a>extend

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Расширение тома или раздела с концентрацией и его файловой системе, на бесплатный \(нераспределенного\) места на диске.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
extend [size=<n>] [disk=<n>] [noerr]  
extend filesystem [noerr]  
```  
  
## <a name="parameters"></a>Параметры  
  
| Параметр  |                                                                                             Описание                                                                                              |
|------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Размер\=<n>  |      Задает объем пространства в мегабайтах \(МБ\) добавляемый текущего тома или раздела. Если размер не задан, используется вся смежных свободного места на диске.       |
| диск\=<n>  |                          Задает диск, на котором расширяется тома или раздела. Если диск не указан, в раздел или том расширяется на текущем диске.                          |
| файловая система |                                   расширяет файловой системы тома с фокусом. Для использования только на дисках, где не была расширена в файловой системе с томом.                                    |
|   успешного    | для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки. |
  
## <a name="remarks"></a>Примечания  
  
-   На базовом диске свободного пространства должен быть на том же диске тома или раздела, имеющего фокус. Оно должно также следовать сразу за тома или раздела, имеющего фокус \(то есть оно должно начинаться в следующего смещения сектора\).  
  
-   На динамических дисках с простой или составные тома тома можно расширить в любое свободное пространство на любом динамическом диске. С помощью следующей команды, вы можете преобразовать простой динамический том в составной динамический том. В зеркальном отображении, RAID\-5 и чередующиеся тома не могут быть расширены.  
  
-   Если раздел ранее был отформатирован с файловой системой NTFS, файловая система автоматически расширяется для заполнения большего размера секции, и произойдет без потери данных.  
  
-   Если раздел ранее был отформатирован с файловой системой, отличной от NTFS, команда завершается ошибкой без ущерба для секции.  
  
-   Если раздел не был ранее отформатирован с файловой системой, будут по-прежнему расширены секции.  
  
-   Секция должна иметь связанные тома, прежде чем ее можно расширить.  
  
## <a name="BKMK_examples"></a>Примеры  
Чтобы расширить том или раздела, имеющего фокус, 500 мегабайт, на диске 3, введите:  
  
```  
extend size=500 disk=3  
```  
  
Чтобы расширить файловой системы тома после его была расширена, введите следующую команду:  
  
```  
extend filesystem  
```

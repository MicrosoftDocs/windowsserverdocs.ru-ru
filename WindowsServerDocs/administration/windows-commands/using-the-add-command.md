# <a name="using-the-add-command"></a>С помощью команды add

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

## <a name="subcommands"></a>Подкоманды
|Подкоманда|Описание|
|-------|--------|
|[С помощью команды add устройства](using-the-add-device-command.md)|Готовит компьютера в active directory.|
|[С помощью команды add образа](using-the-add-image-command.md)|Добавляет образы загрузки или установки.|
|[С помощью команды add-ImageGroup](using-the-add-imagegroup-command.md)|Добавляет группу образов.|
|[С помощью команды add-DriverGroupPackage](using-the-add-drivergrouppackage-command.md)|Добавляет группу драйверов пакет драйверов.|
|[С помощью команды add-DriverGroupPackages](using-the-add-drivergrouppackages-command.md)|Добавляет группу драйверов пакеты драйверов.|
|[С помощью команды add-DriverPackage](using-the-add-driverpackage-command.md)|Добавляет пакет драйверов на сервер.|
|[С помощью команды add-ImageDriverPackage](using-the-add-imagedriverpackage-command.md)|Добавляет пакет драйверов, который находится в хранилище драйверов на существующий образ загрузки на сервере. Версия образа должна быть Windows 7 или Windows Server 2008 R2 или более поздней версии.|
|[С помощью команды add-ImageDriverPackages](using-the-add-imagedriverpackages-command.md)|Добавляет пакеты драйверов из хранилища драйверов в загрузочный образ на сервере. Версия образа должна быть Windows 7 или Windows Server 2008 R2 или более поздней версии.|
|[С помощью подкоманды add-AllDriverPackages](using-the-add-alldriverpackages-subcommand.md)|добавления пакетов драйверов из папки на сервере.|
|[С помощью команды add-DriverGroup](using-the-add-drivergroup-command.md)|Добавляет группу драйверов на сервер.|
|[С помощью команды add-DriverGroupFilter](using-the-add-drivergroupfilter-command.md)|Добавление фильтра к группе драйверов на сервере.|

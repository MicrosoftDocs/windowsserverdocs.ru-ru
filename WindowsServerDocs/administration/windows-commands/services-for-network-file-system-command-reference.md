# <a name="services-for-network-file-system-command-reference"></a>Справочник по командам службы для NFS



Службы для сетевой файловой системы (NFS) предоставляют решение, которое обеспечивает передачу файлов между компьютерами под управлением Windows Server 2008 и операционные системы UNIX, используя протокол NFS к файлам.

Ниже приведен список инструментов командной строки для NFS.

|Command|Описание|
|-------|-----------|
|[mapadmin](mapadmin.md)|Управление отображения имен пользователей для служб Майкрософт для сетевой файловой системы.|
|[подключить](mount.md)|Сетевые папки сетевой файловой системы (NFS) подключения.|
|[NFSAdmin](nfsadmin.md)|Управление сервером для NFS и клиента для NFS.|
|[nfsshare](nfsshare.md)|Управлять общие папки сетевой файловой системы (NFS).|
|[Nfsstat](nfsstat.md)|Отображение или сброс количество вызовов на сервер для NFS.|
|[rpcinfo](rpcinfo.md)|Список программ на удаленных компьютерах.|
|[showmount](showmount.md)|Отображение подключенных каталогов.|

# <a name="gettype"></a>gettype

Команда **GetType** устарела и не гарантируется, что она будет поддерживаться в будущих выпусках Windows.

Это средство входит в Windows Server 2003. Дополнительные сведения см. в разделе [GetType](https://docs.microsoft.com/previous-versions/orphan-topics/ws.10/cc773104(v=ws.10)).

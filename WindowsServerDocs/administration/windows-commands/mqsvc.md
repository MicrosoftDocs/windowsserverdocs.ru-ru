# <a name="mqsvc"></a>mqsvc



Технологии служба очереди сообщений позволяет приложениям, выполняющимся в разное время, взаимодействовать через неоднородные сети и системы, которые могут быть временно отключены. Очередь сообщений обеспечивает гарантированную доставку сообщений, эффективную маршрутизацию, безопасность и обмена сообщениями на основе приоритета. Его можно использовать для реализации решения для асинхронные и синхронные сценарии обмена сообщениями. Дополнительные сведения об этой команде см. в разделе [Message Queuing (MSMQ)](https://go.microsoft.com/fwlink/?LinkId=248723) на сайте MSDN.

## <a name="syntax"></a>Синтаксис

```
Mqsvc.exe
```

### <a name="parameters"></a>Параметры

Нет

# <a name="pause"></a>pause



Приостанавливает выполнение пакетного файла и отображает следующее сообщение:
```
Press any key to continue . . .
```
В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
pause
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

- При запуске **приостановить** команду, появляется следующее сообщение:  
  ```
  Press any key to continue . . .
  ```  
- Если нажать клавишу CTRL + C, чтобы остановить пакетная программа, появится следующее сообщение:  
  ```
  Terminate batch job (Y/N)?
  ```  
  Если нажать клавишу Y (Да) в ответ на это сообщение, программа будет закрыта и элемент управления возвращает операционную систему.
- Вы можете вставить **приостановить** перед разделом пакетный файл, который может не потребоваться обработать команду. Когда **приостановить** приостанавливает обработки пакетного можно нажать CTRL + C и нажмите клавишу Y для остановки программы.

## <a name="BKMK_examples"></a>Примеры

Чтобы создать пакетный файл, пользователю будет предложено заменить диск в одном из этих дисков, введите:
```
@echo off 
:Begin 
copy a:*.* 
echo Put a new disk into drive A 
pause 
goto begin
```
В этом примере все файлы на диск в дисководе являются скопирован в текущий каталог. После сообщения запрашивает, нужно поместить новый диск в дисководе A, **приостановить** команда приостанавливает обработку, то есть можно заменить диск и нажмите любую клавишу для продолжения процесса. Эта программа пакетной службы выполняется в бесконечном цикле — **начать goto** команда отправляет интерпретатор команд на начало пакетного файла. Чтобы остановить этот пакетная программа, нажмите клавиши CTRL + C и нажмите клавишу Y.

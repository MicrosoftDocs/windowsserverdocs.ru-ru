# <a name="repair"></a>восстановление.

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Восстанавливает том RAID-5 с фокусом, заменяя неисправной регион диска на указанный динамический диск.

Для выполнения этой операции необходимо выбрать том в массиве RAID-5. Используйте команду **выбрать том** , чтобы выбрать том и переместить фокус на него.

## <a name="syntax"></a>Синтаксис

```
repair disk=<n> [align=<n>] [noerr]
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
|--|--|
| диск =`<n>` | Указывает динамический диск, который заменит неисправный регион диска. Где *n* должно иметь свободное пространство больше или равно общему размеру неисправного региона диска в томе RAID-5. |
| Выровняйте =`<n>` | Выравнивает все экстенты томов или секций до ближайшей границы выравнивания. Где *n* — это количество килобайтов (КБ) от начала диска до ближайшей границы выравнивания. |
| Noerr | только для сценариев. При возникновении ошибки DiskPart продолжит обрабатывать команды, как если бы ошибка не возникала. Без этого параметра ошибка приводит к выходу из программы DiskPart с кодом ошибки. |

### <a name="examples"></a>Примеры

Чтобы заменить том фокусом, заменив его динамическим диском 4, введите:

```
repair disk=4
```

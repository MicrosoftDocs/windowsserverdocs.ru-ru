# <a name="repair"></a>чинить

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Восстанавливает том RAID\-5 с фокусом, заменяя неисправной регион диска на указанный динамический диск.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
repair disk=<n> [align=<n>] [noerr]  
```  
  
### <a name="parameters"></a>Параметры  
  
| Параметр  |                                                                                             Описание                                                                                              |
|------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \=диска <n>  |                                                                 Указывает динамический диск, который заменит неисправный регион диска.                                                                 |
| Выровняйте\=<n> |          Выравнивает все экстенты томов или секций до ближайшей границы выравнивания. *n* — это количество килобайтов \(КБ\) от начала диска до ближайшей границы выравнивания.           |
|   Noerr    | Только для сценариев. При возникновении ошибки DiskPart продолжит обрабатывать команды, как если бы ошибка не возникала. Без этого параметра ошибка приводит к выходу из программы DiskPart с кодом ошибки. |
  
## <a name="remarks"></a>Примечания  
  
-   На указанном динамическом диске должно быть свободное пространство больше или равно общему размеру неисправного региона диска в томе RAID\-5.  
  
-   Для выполнения этой операции необходимо выбрать том в массиве RAID\-5. Используйте команду **выбрать том** , чтобы выбрать том и переместить фокус на него.  
  
## <a name="examples"></a><a name=BKMK_examples></a>Примеров  
Чтобы заменить том фокусом, заменив его динамическим диском 4, введите:  
  
```  
repair disk=4  
```

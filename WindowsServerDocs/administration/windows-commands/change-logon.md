# <a name="change-logon"></a>change logon
Включает или отключает вход сеансов или отображает текущее состояние входа в систему.
Эта служебная программа используется для обслуживания системы.
Примеры для использования этой команды, см. в разделе [примеры](#BKMK_examples).
> [!NOTE]
> В Windows Server 2008 R2 службы терминалов были переименованы на службы удаленных рабочих столов. Чтобы найти новые возможности в последней версии, см. в разделе [какие возможности служб удаленных рабочих столов в Windows Server 2012](https://technet.microsoft.com/library/hh831527) в технической библиотеке Windows Server.
> ## <a name="syntax"></a>Синтаксис
> ```
> change logon {/query | /enable | /disable | /drain | /drainuntilrestart}
> ```
> ## <a name="parameters"></a>Параметры
> 
> |     Параметр      |                                                       Описание                                                        |
> |--------------------|--------------------------------------------------------------------------------------------------------------------------|
> |       / Query       |                             Отображает текущее состояние входа в систему, включена или отключена.                              |
> |      / Enable       |                              Включение входа пользователя из сеанса, а не из консоли.                              |
> |      / Disable      |  Отключение входа пользователя из сеанса, а не из консоли. Не влияет на настоящий момент в системе пользователей.   |
> |       /drain       |                 Запрещает вход новых сеансов, но разрешает повторное подключение к существующим сеансам.                 |
> | /drainuntilrestart | Отключает вход нового сеанса, пока компьютер перезапускается, но разрешает повторное подключение к существующим сеансам. |
> |         /?         |                                           Отображение справки в командной строке.                                           |
> 
> ## <a name="remarks"></a>Примечания
> - Только администраторы могут использовать **изменить входа** команды.
> - Вход в систему не будут повторно включены после перезагрузки системы. Если вы подключены к серверу узла сеансов удаленных рабочих столов (rd узла сеансов) из сеанса клиента и отключить вход в систему и выйдите из системы, прежде чем повторно включить вход в систему, вы не сможете повторно подключиться к сеансу. Чтобы снова включить вход сеансов, войдите в консоль.
>   ## <a name="BKMK_examples"></a>Примеры
> - Для отображения текущего состояния входа, введите:
>   ```
>   change logon /query
>   ```
> - Чтобы включить вход сеансов, введите следующую команду:
>   ```
>   change logon /enable
>   ```
> - Чтобы отключить вход в систему клиента, введите:
>   ```
>   change logon /disable
>   ```

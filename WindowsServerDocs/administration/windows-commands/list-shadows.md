# <a name="list-shadows"></a>перечислить тени



Список постоянных и существующих непостоянных теневых копий, имеющихся в системе.

## <a name="syntax"></a>Синтаксис

```
list shadows {all | set <SetID> | id <ShadowID>}
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|all|Список всех теневых копий.|
|задать \<Сетид >|Выводит список теневых копий, принадлежащих указанному ИДЕНТИФИКАТОРу набора теневых копий.|
|ID \<Шадовид >|Отображает все теневые копии с указанным ИДЕНТИФИКАТОРом теневой копии.|

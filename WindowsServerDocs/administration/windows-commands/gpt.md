# <a name="gpt"></a>таблицы

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

На базовых дисках с таблицей разделов GUID (GPT) назначает атрибут GPT секции с фокусом.  Атрибуты разделов GPT предоставляют дополнительные сведения об использовании секции. Некоторые атрибуты относятся к идентификатору GUID типа секции.

> [!CAUTION]
> Изменение атрибутов GPT может привести к тому, что основным томам данных не удастся присвоить буквы дисков или предотвратить подключение файловой системы. Не следует изменять атрибуты GPT, если вы не являетесь изготовителем оборудования (OEM) или ИТ-специалистом, имеющим опыт работы с дисками GPT.

## <a name="syntax"></a>Синтаксис

```
gpt attributes=<n>
```

### <a name="parameters"></a>Параметры

|   Параметр    |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| атрибуты =<n> | Задает значение атрибута, который необходимо применить к секции с фокусом. Поле атрибута GPT является 64-битовым полем, содержащим два подполя. Более высокое поле интерпретируется только в контексте идентификатора секции, а нижнее поле является общим для всех идентификаторов секций. Допустимые значения:<p>-   **0x0000000000000001**. Указывает, что для правильной работы компьютера требуется секция.<br />-   **0x8000000000000000**. Указывает, что при перемещении диска на другой компьютер по умолчанию раздел не будет иметь букву диска или если диск будет впервые виден компьютером.<br />-   **0x4000000000000000**. Скрывает том раздела. Это значит, что этот раздел не будет обнаружен диспетчером подключения.<br />-   **0x2000000000000000**. Указывает, что Секция является теневой копией другой секции.<br />-   **0x1000000000000000**. Указывает, что Секция доступна только для чтения. Этот атрибут предотвращает запись тома в.<p>Дополнительные сведения об этих атрибутах см. в разделе Attributes статьи [Create_PARTITION_PARAMETERS Structure](https://go.microsoft.com/fwlink/?LinkId=203812). |

## <a name="remarks"></a>Примечания

- Системный раздел EFI содержит только те двоичные файлы, которые необходимы для запуска операционной системы. Это упрощает размещение двоичных файлов OEM или двоичных файлов, относящихся к операционной системе, в другие разделы.
- Для выполнения этой операции необходимо выбрать базовый раздел GPT. Используйте команду **Выбор секции** , чтобы выбрать базовый раздел GPT и переместить фокус на него.

## <a name="examples"></a>Примеры

  Если вы перемещаете диск GPT на новый компьютер и хотите запретить этому компьютеру автоматически назначать букву диска разделу с фокусом, введите:
  ```
  gpt attributes=0x8000000000000000
  ```

# <a name="simulate-restore"></a>Имитировать восстановление

Проверяет успешность операции записи в сеансах восстановления на компьютере без выдачи событий предварительного **восстановления** или **восстановления** в модули записи.

> [!NOTE]
> Для успешности команды **имитировать восстановление** необходимо выбрать файл метаданных DiskShadow. Используйте [команду загрузить метаданные](load-metadata.md) , чтобы загрузить выбранные модули записи и компоненты для восстановления.

## <a name="syntax"></a>Синтаксис

```
simulate restore
```

# <a name="goto"></a>goto



Направляет cmd.exe в строку с меткой в пакетном файле. В пакетном **goto** передает строку, которая идентифицируется метку управление. При обнаружении метки, обработка продолжается, начиная с помощью команд, которые начинаются на следующей строке.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
goto <Label> 
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Метка >|Указывает строку текста, используемый в качестве метки в программе пакетной службы.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Работа с расширениями команды

    Если команда расширения включен (по умолчанию), и использовать **goto** с меткой целевой **: EOF**, передачи управления в конец текущего пакетного файла и выйти из пакетного файла скрипта без определения метки. При использовании **goto** с **: EOF** метки, необходимо вставить двоеточие перед меткой. Пример:  
    ```
    goto:EOF
    ```  
-   С помощью допустимым *метка* значения

    Можно использовать пробелы в *метка* параметр, но не может включать другие разделители (например, запятой или знак равенства).
-   Сопоставление *метка* с меткой в программе пакетной службы

    *Метка* значение должно соответствовать метку в программе пакетной службы. Метки в рамках программы пакетной службы должно начинаться с двоеточия (:). Если строка начинается с двоеточия, он рассматривается как метку и игнорируются все команды в этой строке. Если пакетном файле не содержит метку, указываемое в *метка*, программа останавливается и отображается следующее сообщение:  
    ```
    Label not found
    ```  
-   С помощью **goto** для обработки условий

    Можно использовать **goto** с другими командами для выполнения условных операций. Дополнительные сведения об использовании **goto** условных операций, см. в разделе [Если](if.md) описанием команды.

## <a name="BKMK_examples"></a>Примеры

Следующая пакетная программа форматирует диск в дисководе A как системный диск. Если операция выполнена успешно, **goto** команда передает управление на **: end** метки:
```
echo off
format a: /s
if not errorlevel 1 goto end
echo An error occurred during formatting.
:end
echo End of batch program. 
```

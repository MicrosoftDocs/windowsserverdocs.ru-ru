# <a name="wbadmin-get-status"></a>wbadmin get status

Сообщает состояние операции резервного копирования или восстановления, которая выполняется в данный момент.

Чтобы получить состояние выполняемой в данный момент операции резервного копирования или восстановления с помощью этой команды, необходимо быть членом группы " **Операторы архива** " или " **Администраторы** ", либо вам должны быть делегированы соответствующие разрешения. Кроме того, необходимо запустить программу **Wbadmin** из командной строки с повышенными привилегиями, щелкнув правой кнопкой мыши **командную строку** и выбрав команду **Запуск от имени администратора**.

> [!IMPORTANT]
> Эта команда не останавливается до завершения операции резервного копирования или восстановления. Команда продолжит работу, даже если закрыть окно командной строки. Чтобы прерывать текущую операцию резервного копирования или восстановления, выполните команду [Wbadmin останавливают Job](wbadmin-stop-job.md) .

## <a name="syntax"></a>Синтаксис

```
wbadmin get status
```

# <a name="unexpose"></a>unexpose

Унекспосес теневую копию, которая была предоставлена с помощью [команды предоставления](expose.md). Предоставленная теневая копия может быть задана с помощью идентификатора теневой копии, буквы диска, общего ресурса или точки подключения.

## <a name="syntax"></a>Синтаксис

```
unexpose {<shadowID> | <drive:> | <share> | <mountpoint>}
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
|--|--|
| `<shadowID>` | Отображает теневую копию, заданную по заданному ИДЕНТИФИКАТОРу тени. Вместо можно использовать существующий псевдоним или переменную среды `<shadowID>` . Используйте [команду Добавить](add.md) без параметров, чтобы просмотреть все существующие псевдонимы. |
| `<drive:>` | Отображает теневую копию, связанную с заданной буквой диска (например, диск P). |
| `<share>` | Отображает теневую копию, связанную с указанной общей папкой (например, `\\MachineName` ). |
| `<mountpoint>` | Отображает теневую копию, связанную с указанной точкой подключения (например, `C:\shadowcopy\` ). |
| add | При использовании без параметров будут показаны существующие псевдонимы. |

## <a name="examples"></a>Примеры

Чтобы Unexpose теневую копию, связанную с * диском P: \* , введите:

```
unexpose P:
```

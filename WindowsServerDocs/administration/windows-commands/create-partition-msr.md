# <a name="create-partition-msr"></a>Создание раздела msr

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает зарезервированный Microsoft \(MSR\) секции в таблицу разделов \(gpt\) диска.  
  
> [!CAUTION]  
> Будьте очень внимательны при использовании этой команды. Так как диски gpt требуется макет определенной секции, создавая секции зарезервированные может привести к диска становятся недоступными для чтения.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
create partition msr [size=<n>] [offset=<n>] [noerr]  
```  
  
## <a name="parameters"></a>Параметры  
  
|Параметр|Описание|  
|-------|--------|  
|Размер\=<n>|Размер раздела в мегабайтах \(МБ\). Раздел имеет по крайней мере, если в байтах, числа, указанного по <n>. Если размер не указан, раздел занимает имеется свободное пространство в текущей области.|  
|Смещение\=<n>|Задает смещение в килобайтах \(КБ\), в которой создается секция. Смещение выполняет округление до полного заполнения используется размер сектора. Если смещение не указано, раздел помещается в первое пространство на диске, достаточное для их хранения.|  
|успешного|для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.|  
  
## <a name="remarks"></a>Примечания  
  
-   На GPT-дисках, которые используются для загрузки операционной системы Windows, Extensible Firmware Interface \(EFI\) системный раздел является первым разделом на диске, следуют такие. на дисках, которые используются только для хранения данных не имеют системный раздел EFI, в котором регистр такие является первым разделом.  
  
-   Windows не подключить резервный раздел Майкрософт. Не удается сохранить данные на них и их нельзя удалить.  
  
-   Такие секции требуется на каждом диске gpt. Размер этой секции зависит от общего объема диска gpt. Размер диска gpt должен быть по крайней мере 32 МБ для создания раздела зарезервированные.  
  
-   Для успешного выполнения этой операции необходимо выбрать базовый gpt-диск. Используйте **выберите диск** команду, чтобы выбрать базовый gpt-диск и перетянуть внимание к нему.  
  
## <a name="BKMK_examples"></a>Примеры  
Чтобы создать раздел зарезервированные 1000 МБ, размер, введите:  
  
```  
create partition msr size=1000  
```

# <a name="logman-create-trace"></a>Logman создать трассировку

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создайте сборщиком данных трассировки событий.  

## <a name="syntax"></a>Синтаксис  
```  
logman create trace <[-n] <name>> [options]  
```

## <a name="remarks"></a>Примечания  
Там, где [-] присутствует в списке, отключает параметр.  
## <a name="BKMK_examples"></a>Примеры  
Следующий пример создает сборщик данных трассировки событий вызывается с помощью буферы не меньше 16 и не более 256, каждый буфер 64 КБ в размер trace_log и выводит результат в c:\logfile расположение.  
```  
logman create trace trace_log -nb 16 256 -bs 64 -o c:\logfile  
```

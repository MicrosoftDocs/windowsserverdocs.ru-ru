# <a name="add-drivergroupfilter"></a>Add-Дриверграупфилтер

Добавляет фильтр в группу драйверов на сервере.

## <a name="syntax"></a>Синтаксис

```
WDSUTIL /Add-DriverGroupFilter /DriverGroup:<Group Name> [/Server:<Server name>] /FilterType:<Filter Type> /Policy:{Include | Exclude} /Value:<Value> [/Value:<Value> ...]
```

| /Дриверграуп: имя группы\<> |                                                                                                                                                                                                                                                                                                                                                                                                                                                              Указывает имя группы драйверов.                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

Если имя сервера не указано, используется локальный сервер.                                                                                                                                                                                                                                                                                                                                                                                                               |
| /Филтертипе:\<FilterType >  |                                                                                                                                                                                                   Указывает тип фильтра, добавляемого в группу. ## <a name="examples"></a><a name=BKMK_examples></a>Примеров

Чтобы добавить фильтр в группу драйверов, введите одно из следующих действий:
```
WDSUTIL /Add-DriverGroupFilter /DriverGroup:PrinterDrivers /FilterType:Manufacturer /Policy:Include /Value:Name1 /Value:Name2
```
```
WDSUTIL /Add-DriverGroupFilter /DriverGroup:PrinterDrivers /FilterType:Manufacturer /Policy:Include /Value:Name1 /FilterType:ChassisType /Policy:Exclude /Value:Tower /Value:MiniTower
```

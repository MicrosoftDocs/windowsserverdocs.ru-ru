# <a name="manage-bde-protectors"></a>готов: предохранители

>Область применения. Windows Server (полугодовой канал), Windows Server 2016

Управляет методами защиты, используемый для ключа шифрования BitLocker. Примеры использования этой команды см. в разделе [примеры](#BKMK_Examples).
## <a name="syntax"></a>Синтаксис
```
manage-bde -protectors [{-get|-add|-delete|-disable|-enable|-adbackup|-aadbackup}] <Drive> [-computername <Name>] [{-?|/?}] [{-help|-h}]
```
### <a name="parameters"></a>Параметры

|   Параметр   |                                                                                                                                                                                           Описание                                                                                                                                                                                            |
|---------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     -get      |                                                                                                                                            Отображаются все методы защиты ключа, на диске включено и предоставляет их тип и идентификатор (ID).                                                                                                                                             |
|     -Добавление      |                                                                                                                                   добавляет методы защиты ключей, заданные с помощью дополнительных [добавить параметры](manage-bde-protectors.md#BKMK_addprotectors).                                                                                                                                    |
|    — удалить    | Удаляет методы защиты ключа, используемого функцией BitLocker. Будут удалены все предохранители ключа с диска, если необязательный [-удалить параметры](manage-bde-protectors.md#BKMK_deleteprotectors) можно определять, какие именно из компонентов для удаления. При удалении последнего предохранителя на диске, чтобы убедиться, что доступ к данным не потеряно случайно отключена защита BitLocker диска. |
|   -Отключение    |                      Отключает защиту, которые позволят любому человеку для доступа к зашифрованным данным путем предоставления ключа шифрования незащищенным на диске. Предохранители ключа не удаляются. Защита будет возобновлена при очередном Windows загружается в том случае, если необязательный [-отключить параметры](manage-bde-protectors.md#BKMK_disableprot) используются для указания числа перезагрузки.                       |
|    -включить    |                                                                                                                             Включает защиту, удаляя незащищенный ключ шифрования с диска. Применяются все настроенные предохранители ключа на диске.                                                                                                                             |
|   -adbackup   |                                                                          Резервное копирование всех сведений о восстановлении для диске, указанном в доменных службах Active Directory (AD DS). Чтобы выполнить резервное копирование только одно восстановление ключа в AD DS, добавьте **-идентификатор** параметр и укажите идентификатор ключа восстановления, определенных для резервного копирования.                                                                           |
|  -aadbackup   |                                                                            Резервное копирование всех сведений о восстановлении для диске, указанном в Azure Active Directory (Azure Ad). Чтобы выполнить резервное копирование только одно восстановление ключа в Azure AD, добавьте **-идентификатор** параметр и укажите идентификатор ключа восстановления, определенных для резервного копирования.                                                                             |
|    <Drive>    |                                                                                                                                                                          Представляет букву диска, за которым следует двоеточие.                                                                                                                                                                          |
| -computername |                                                                                                              Указывает, что управление bde.exe будет использовать для изменения защиты BitLocker на другом компьютере. Можно также использовать **- cn** как сокращенную версию этой команды.                                                                                                              |
|    <Name>     |                                                                                                                 Представляет имя компьютера, на которой требуется изменить защиту BitLocker. Допустимые значения включают имя NetBIOS компьютера и IP-адрес компьютера.                                                                                                                  |
|   -? или /?    |                                                                                                                                                                            Отображение кратких справки в командной строке.                                                                                                                                                                            |
|  -help или -h  |                                                                                                                                                                          Отображает полную справку в командной строке.                                                                                                                                                                           |

### <a name="BKMK_addprotectors"></a>-Добавьте синтаксиса и параметров
```
manage-bde  protectors  add [<Drive>] [-forceupgrade] [-recoverypassword <NumericalPassword>] [-recoverykey <pathToExternalKeydirectory>]
[-startupkey <pathToExternalKeydirectory>] [-certificate {-cf <pathToCertificateFile>|-ct <CertificateThumbprint>}] [-tpm] [-tpmandpin] 
[-tpmandstartupkey <pathToExternalKeydirectory>] [-tpmandpinandstartupkey <pathToExternalKeydirectory>] [-password][-adaccountorgroup <securityidentifier> [-computername <Name>] 
[{-?|/?}] [{-help|-h}]
```

|          Параметр           |                                                                                                                                                                                   Описание                                                                                                                                                                                   |
|------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|           <Drive>            |                                                                                                                                                                 Представляет букву диска, за которым следует двоеточие.                                                                                                                                                                  |
|      -recoverypassword       |                                                                                                                                    Добавляет предохранитель числового пароля. Можно также использовать **- rp** как сокращенную версию этой команды.                                                                                                                                     |
|     <NumericalPassword>      |                                                                                                                                                                        Представляет пароль восстановления.                                                                                                                                                                        |
|         -recoverykey         |                                                                                                                                Добавляет предохранителя внешнего ключа для восстановления. Можно также использовать **- ринтерах** как сокращенную версию этой команды.                                                                                                                                 |
| <pathToExternalKeydirectory> |                                                                                                                                                               Представляет путь к каталогу для ключа восстановления.                                                                                                                                                                |
|         -startupkey          |                                                                                                                                 Добавляет предохранителя внешнего ключа для запуска. Можно также использовать **-sk** как сокращенную версию этой команды.                                                                                                                                 |
| <pathToExternalKeydirectory> |                                                                                                                                                                Представляет путь к каталогу для ключа запуска.                                                                                                                                                                |
|         -сертификата         |                                                                                                                               Добавляет общедоступный предохранитель для диска с данными. Можно также использовать **-cert** как сокращенную версию этой команды.                                                                                                                               |
|             -cf              |                                                                                                                                              Указывает, что файл сертификата будет использоваться для предоставления сертификата открытого ключа.                                                                                                                                              |
|   <pathToCertificateFile>    |                                                                                                                                                             Представляет путь к файлу сертификата.                                                                                                                                                              |
|             -ct              |                                                                                                                                           Указывает, что отпечаток сертификата будет использоваться для идентификации сертификата открытого ключа                                                                                                                                           |
|   <CertificateThumbprint>    |                                                       Задает значение свойства отпечаток сертификата, который вы хотите использовать. Например, значение отпечатка сертификата из «a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d 42 77 a3 2a 7b» должны быть указаны в виде «a909502dd82ae41433e6f83886b00d4277a32a7b».                                                        |
|          -tpmandpin          |                                                                                           Добавление доверенного платформенного модуля (TPM) и персональный идентификационный номер (ПИН-код) предохранитель для диска операционной системы. Можно также использовать **- tp** как сокращенную версию этой команды.                                                                                           |
|      -tpmandstartupkey       |                                                                                                                    Добавление предохранителя ключа TPM и запуска для диска операционной системы. Можно также использовать **-tsk** как сокращенную версию этой команды.                                                                                                                    |
|   -tpmandpinandstartupkey    |                                                                                                                Добавление доверенного платформенного МОДУЛЯ, ПИН-код и предохранитель ключа запуска для диска операционной системы. Можно также использовать **- tpsk** как сокращенную версию этой команды.                                                                                                                 |
|          -пароль           |                                                                                                                              Добавляет предохранитель ключа пароля для диска с данными. Можно также использовать **- pw** как сокращенную версию этой команды.                                                                                                                              |
|      -adaccountorgroup       | Добавляет идентификатор безопасности (SID) — на основе предохранителя удостоверений для тома.  Можно также использовать **-sid** как сокращенную версию этой команды. **ВАЖНО:** По умолчанию невозможно добавить предохранитель ADAccountOrGroup удаленно с помощью WMI или готов.  Если для развертывания требуется возможность добавлять этот предохранитель удаленно, необходимо включить ограниченного делегирования. |

|        <Drive>         |                                                             Представляет букву диска, за которым следует двоеточие.                                                             |
|         -Тип          |                               Идентифицирует предохранитель ключа для удаления. Можно также использовать **-t** как сокращенную версию этой команды.                               |
|    recoverypassword    |                                                 Указывает, что любой ключа предохранители паролей восстановления должен быть удален.                                                 |

|          доверенный платформенный модуль           |                                        Указывает, что любой доступный только для доверенного платформенного МОДУЛЯ предохранители ключа, связанного с устройством должен быть удален.                                         |
|    tpmandstartupkey    |                                Указывает, что любой доверенный платформенный модуль и запуска на основе ключа предохранители ключа сопоставленный диск должен быть удален.                                |
|       tpmandpin        |                                    Указывает, что любой доверенный платформенный модуль и ПИН-код на основе предохранители ключа, связанного с устройством должен быть удален.                                    |
| tpmandpinandstartupkey |                             Указывает, что любой доверенного платформенного МОДУЛЯ, ПИН-код, после чего на основе ключа предохранители ключа сопоставленный диск должен быть удален.                             |
|        пароль        |                                        Указывает, что любой пароль предохранители ключа, связанного с устройством должен быть удален.                                         |
|        для проверки подлинности        |                                        Указывает, что любой предохранители ключа удостоверения, связанного с устройством должен быть удален.                                         |
|          -Идентификатор           |                Идентифицирует предохранитель ключа, чтобы удалить с помощью идентификатора ключа. Этот параметр является альтернативным вариантом для **-тип** параметр.                 |

### <a name="BKMK_disableprot"></a>-отключить синтаксиса и параметров
```
manage-bde  protectors  disable <Drive> [-RebootCount <integer 0 - 15>] [-computername <Name>] [{-?|/?}] [{-help|-h}]
```

|   Параметр   |                                                                                                                                                                                                                   Описание                                                                                                                                                                                                                    |
|---------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|    <Drive>    |                                                                                                                                                                                                  Представляет букву диска, за которым следует двоеточие.                                                                                                                                                                                                  |
|  RebootCount  | Начиная с Windows 8, указывает количество времени, указанного в параметре RebootCount, после перезапуска, защиты тома операционной системы было приостановлено и будет возобновлена после Windows. Укажите 0, чтобы приостановить защиты неопределенное время. Если этот параметр не задан, защита BitLocker будет автоматически возобновляется после перезапуска Windows. Можно также использовать **-rc** как сокращенную версию этой команды. |
| -computername |                                                                                                                                      Указывает, что управление bde.exe будет использовать для изменения защиты BitLocker на другом компьютере. Можно также использовать **- cn** как сокращенную версию этой команды.                                                                                                                                      |
|    <Name>     |                                                                                                                                         Представляет имя компьютера, на которой требуется изменить защиту BitLocker. Допустимые значения включают имя NetBIOS компьютера и IP-адрес компьютера.                                                                                                                                          |
|   -? или /?    |                                                                                                                                                                                                    Отображение кратких справки в командной строке.                                                                                                                                                                                                    |
|  -help или -h  |                                                                                                                                                                                                  Отображает полную справку в командной строке.                                                                                                                                                                                                   |

## <a name="BKMK_Examples"></a>Примеры
В следующем примере демонстрируется использование **-предохранители** команду, чтобы добавить предохранитель ключа сертификата, определяется файл сертификата на диск E.
```
manage-bde  protectors  add E: -certificate  cf "c:\File Folder\Filename.cer"
```
В следующем примере демонстрируется использование **-предохранители** команду, чтобы добавить **adaccountorgroup** предохранитель ключа, идентифицируемое по его имени домена и пользователя на диск E.
```
manage-bde  protectors  add E: -sid DOMAIN\user
```
В следующем примере демонстрируется использование **предохранители** команду, чтобы отключить защиту, пока компьютер был перезагружен 3 раза.
```
manage-bde  protectors  disable C: -rc 3
```
В следующем примере демонстрируется использование **-предохранители** команду, чтобы удалить все TPM и ключ запуска на основе предохранители ключа на диске C.
```
manage-bde  protectors  delete C: -type tpmandstartupkey
```
В следующем примере демонстрируется использование **-предохранители** команду, чтобы архивировать все данные восстановления для диска C в AD DS.
```
manage-bde  protectors  adbackup C:
```

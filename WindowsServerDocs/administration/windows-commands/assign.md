# <a name="assign"></a>assign

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

назначает диска букву или точка подключения тома, имеющего фокус.

## <a name="syntax"></a>Синтаксис
```
assign [{letter=<d> | mount=<path>}] [noerr]
```
## <a name="parameters"></a>Параметры

|  Параметр   |                                                                                                                                 Описание                                                                                                                                 |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  буква =<d>  |                                                                                                             Буква диска, который вы хотите назначить тому.                                                                                                              |
| Подключите =<path> | Путь к точке подключения, который вы хотите назначить тому.<br /><br />инструкции о том, как использовать эту команду, см. в разделе [назначить путь к папке точки подключения для диска](https://go.microsoft.com/fwlink/?LinkId=207059) (<https://go.microsoft.com/fwlink/?LinkId=207059>). |
|    успешного     |                                    для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.                                     |

## <a name="remarks"></a>Примечания
- Если не буква диска или монтирования точка указан, назначается следующая доступная буква диска. Если буква диска или монтирования точка уже используется, формируется ошибка.
- С помощью команды назначения, можно изменить букву диска, связанный с съемный накопитель.
- Буквы дисков невозможно назначить системных томов, загрузочных томов или тома, содержащие в файл подкачки. Кроме того нельзя назначить букву диска разделу изготовителя оборудования (OEM) или любого раздела таблица разделов GUID (gpt), отличный от основного раздела диска.
- Для успешного выполнения этой операции необходимо выбрать том. Используйте **выберите том** команду, чтобы выбрать том и перетянуть внимание к нему.
  ## <a name="BKMK_examples"></a>Примеры
  Чтобы назначить тома в фокусе буквы E, введите следующую команду:
  ```
  assign letter=e
  ```

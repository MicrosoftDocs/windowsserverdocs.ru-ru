# <a name="whoami"></a>whoami



Отображает сведения пользователя, группы и привилегии для пользователя, вошедшего в систему в локальной системе. При использовании без параметров, **whoami** текущее имя домена и пользователя.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
whoami [/upn | /fqdn | /logonid]
whoami {[/user] [/groups] [/priv]} [/fo <Format>] [/nh]
whoami /all [/fo <Format>] [/nh]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/upn|Отображает имя пользователя в формате имени участника (UPN) пользователя.|
|/FQDN|Отображает имя пользователя в формате полного доменного имени (FQDN).|
|/ logonID|Отображает идентификатор входа текущего пользователя.|
|/ User|Отображает имя текущего домена и пользователя и идентификатор безопасности (SID).|
|/ Groups|Отображает группы пользователей, к которым принадлежит текущий пользователь.|
|/ Priv|Привилегии безопасности текущего пользователя.|
|/FO \<Формат >|Задает формат выходных данных. Допустимые значения:</br>**Таблица** отображает выводимые данные в таблицу. Это значение по умолчанию.</br>**список** отображает выходные данные в списке.</br>**CSV** отображает выходные данные в формате с разделителями-запятыми (CSV).|
|/ all|Отображает все данные в текущий маркер доступа, включая действующие имя пользователя, идентификаторы безопасности (SID), привилегии и групп, к которым принадлежит текущий пользователь.|
|/NH|Указывает, что заголовок столбца не должен отображаться в выходных данных. Это допустимо только для таблицы и формат CSV.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_examples"></a>Примеры

Чтобы отобразить домен и имя пользователя, вошедшего в систему на этом компьютере, введите следующую команду:
```
whoami
```
Появится результат, аналогичный приведенному ниже:
```
DOMAIN1\administrator
```
Чтобы отобразить все данные в текущий маркер доступа, введите следующую команду:
```
whoami /all
```

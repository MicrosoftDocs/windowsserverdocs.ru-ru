# <a name="shift"></a>shift



Изменение положения пакетных параметров в пакетном файле.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
shift [/n <N>]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/n \<N >|Указывает начинать сдвиг с *N*-го аргумента, где *N* — любое значение от 0 до 8. Требует расширения команд, которые включены по умолчанию.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   **Shift** команда изменяет значения параметров пакета **%0** через **%9** путем копирования каждого параметра в предыдущий — значение **%1** будет скопирована **%0**, значение **%2** копируется **%1**, и т. д. Это полезно для написания пакетный файл, который выполняет ту же операцию на любое количество параметров.
-   Если включены расширения команд, **shift** команда поддерживает **/n** параметр командной строки. **/N** указывает начинать сдвиг с n-й аргумент, где **N** — любое значение от 0 до 8. Например **SHIFT /2** сместится **%3** для **%2**, **%4** для **%3**, и т. д. и оставить **%0** и **%1** не затрагиваются. Расширения команд включены по умолчанию.
-   Можно использовать **shift** команду, чтобы создать пакетный файл, который может принимать более чем 10 параметров. Если указать более чем 10 параметров в командной строке, они, отображаться после десятого (**%9**) будет сдвинут поочередно в **%9**.
-   **Shift** команда не оказывает влияния **% \*** пакета параметра.
-   Нет, не обратной **shift** команды. После реализации **shift** команды, вы не сможете восстановить параметра пакетного (**%0**), существовавшие до смены.

## <a name="BKMK_examples"></a>Примеры

Приведенный ниже пример пакетного файла вызывается Mycopy.bat демонстрируется использование **shift** с любым количеством параметров пакета. В этом примере Mycopy.bat копирует список файлов в определенный каталог. Параметры пакета, представлены по аргументам имени каталогов и файлов.
```
@echo off 
rem MYCOPY.BAT copies any number of files
rem to a directory.
rem The command uses the following syntax:
rem mycopy dir file1 file2 ... 
set todir=%1
:getfile
shift
if "%1"=="" goto end
copy %1 %todir%
goto getfile
:end
set todir=
echo All done
```

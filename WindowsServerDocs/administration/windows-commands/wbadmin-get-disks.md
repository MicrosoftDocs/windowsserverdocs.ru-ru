# <a name="wbadmin-get-disks"></a>wbadmin get disks

Список внутренних и внешних дисков, находящихся в сети для локального компьютера.

Чтобы получить список сетевых дисков с помощью этой команды, необходимо быть членом группы " **Операторы архива** " или " **Администраторы** ", либо вам должны быть делегированы соответствующие разрешения. Кроме того, необходимо запустить программу **Wbadmin** из командной строки с повышенными привилегиями, щелкнув правой кнопкой мыши **командную строку**и выбрав команду **Запуск от имени администратора**.

## <a name="syntax"></a>Синтаксис

```
wbadmin get disks
```

# <a name="sysocmgr"></a>sysocmgr

Функция sysocmgr устарела и не гарантируется, что она будет поддерживаться в будущих выпусках Windows.

Это средство входит в Windows Server 2003. Дополнительные сведения см. в разделе [sysocmgr](https://technet.microsoft.com/library/cc773290(v=ws.10).aspx).

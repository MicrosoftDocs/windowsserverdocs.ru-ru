# <a name="begin-backup"></a>Начать резервное копирование



Запускает сеанс полной архивации.

## <a name="syntax"></a>Синтаксис

```
begin backup
```

## <a name="remarks"></a>Примечания

-   Этот параметр переопределяет параметр резервного копирования по умолчанию.

# <a name="ftp-glob_1"></a>FTP: glob_1

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Переключает расширение имен локальных файлов с использованием подстановочных знаков.   
## <a name="syntax"></a>Синтаксис  
```  
glob  
```  
#### <a name="parameters"></a>Параметры  
none  
## <a name="remarks"></a>Примечания  
- По умолчанию глобализации имеет значение ON.  
- Глобализации позволяет использовать звездочку (*) и вопросительный знак (?) в качестве подстановочных знаков в именах локальных файлов или путей.  
  ## <a name="examples"></a>Примеры  
  Переключить расширение имен локальных файлов с использованием подстановочных знаков.  
  ```  
  glob  
  ```

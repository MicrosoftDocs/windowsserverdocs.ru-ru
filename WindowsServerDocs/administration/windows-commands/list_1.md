# <a name="list"></a>list



Список дисков, разделов на диске, томов на диске или виртуальных жестких дисков (VHD).

## <a name="syntax"></a>Синтаксис

```
list { disk | partition | volume | vdisk }
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|диск|Список дисков и сведения о них, такие как размер, объем доступного свободного места, находится ли диск базового или динамического диска и использует ли диск основной загрузочной записью (MBR) или стилем разделов (GPT) таблицы разделов GUID.|
|раздел|Отображает разделы, перечисленные в таблице разделов текущего диска.|
|том|Отображает список базовых и динамических томов на всех дисках.|
|виртуальный диск|Отображает список виртуальных жестких дисков, подключенных и/или выбираются. Эта команда выводит список отсоединенных виртуальных жестких дисков, если они выбраны в данный момент; Тем не менее тип диска имеет значение Unknown, пока не будет подключен виртуальный жесткий ДИСК. Виртуальный жесткий ДИСК, отмеченные звездочкой (*) имеет фокус.</br>Примечание. Эта команда доступна только для Windows 7 и Windows Server 2008 R2.|

## <a name="remarks"></a>Примечания

-   При перечислении секций на динамическом диске, секций может не соответствовать динамические тома на диске. Это несоответствие возникает, поскольку динамические диски содержат записи в таблице разделов для системного или загрузочного тома (при наличии на диске). Здесь также содержится раздел, занимающий оставшуюся часть диска, чтобы зарезервировать пространство для использования динамическими томами.
-   Объект, отмеченные звездочкой (*) имеет фокус.
-   При перечислении дисков, если диск отсутствует, указывается его номер диска M. Например, первый диск отсутствует нумеруется M0.

## <a name="BKMK_examples"></a>Примеры

```
list disk
list partition
list volume
list vdisk
```

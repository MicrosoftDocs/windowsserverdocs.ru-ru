# <a name="attrib"></a>attrib



Отображает, задает или удаляет атрибуты, назначенные файлы или каталоги. При использовании без параметров, **attrib** отображает атрибуты всех файлов в текущем каталоге.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
attrib [{+|-}r] [{+|-}a] [{+|-}s] [{+|-}h] [{+|-}i] [<Drive>:][<Path>][<FileName>] [/s [/d] [/l]]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|{+\|-}r|Наборы (**+**) или очищает (**-**) атрибут только для чтения файла.|
|{+\|-}|Наборы (**+**) или очищает (**-**) атрибутов.|
|{+\|-}s|Наборы (**+**) или очищает (**-**) атрибут системного файла.|
|{+\|-} h|Наборы (**+**) или очищает (**-**) атрибут Hidden.|
|{+\|-}i|Наборы (**+**) или очищает (**-**) не индексируются содержимого атрибут файла.|
|[\<Диска >:] [<Path>] [<FileName>]|Указывает расположение и имя каталога, файл или группу файлов, для которых требуется отобразить или изменить атрибуты. Можно использовать **?** и **&#42;** подстановочные знаки в *FileName* параметр для отображения или изменения атрибутов для группы файлов.|
|/s|Применяет **attrib** и параметры командной строки для соответствующих файлов в текущем каталоге и всех его подкаталогах.|
|/d|Применяет **attrib** и параметры командной строки для каталогов.|
|/l|Применяет **attrib** и параметры командной строки символьную ссылку, а не целевого объекта символьную ссылку.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Можно использовать подстановочные знаки (**?** и **&#42;**) с *FileName* параметр для отображения или изменения атрибутов для группы файлов.
-   Если для файла система (**s**) или Hidden (**h**) набор атрибутов, атрибут необходимо очистить, чтобы изменить все другие атрибуты для этого файла.
-   Атрибут "архивный" (****) помечает файлы, которые были изменены с момента последнего их резервного копирования. Обратите внимание, что **xcopy** команда использует архивировать атрибуты.

## <a name="BKMK_examples"></a>Примеры

Чтобы вывести атрибуты файла News86, расположенный в текущем каталоге, введите следующую команду:
```
attrib news86 
```
Чтобы назначить атрибут только для чтения к файлу Report.txt, введите следующую команду:
```
attrib +r report.txt 
```
Чтобы удалить атрибут только для чтения из файлов в общем каталоге и его подкаталогах на диск в дисководе B, введите следующую команду:
```
attrib -r b:\public\*.* /s 
```
Чтобы установить этот атрибут для всех файлов на диск A, а затем снимите атрибут "архивный" для файлов с расширением BAK, введите:
```
attrib +a a:*.* & attrib -a a:*.bak 
```

# <a name="tcmsetup"></a>tcmsetup



Устанавливает или отключает клиент TAPI.

## <a name="syntax"></a>Синтаксис

```
tcmsetup [/q] [/x] /c <Server1> [<Server2> …] 
tcmsetup  [/q] /c /d
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/q|Блокирует отображение окон сообщений.|
|/x|Указывает, что обратные вызовы, ориентированного на подключение будет использоваться для сетей с высокой нагрузкой где высокий уровень потери пакетов. Если этот параметр указан, используются обратные вызовы без установления соединения.|
|/c|Обязательный. Указывает параметры клиента.|
|\<Server1 >|Обязательный. Указывает имя удаленного сервера с TAPI поставщики услуг, клиент будет использовать. Клиент будет использовать поставщиков услуг линии и телефоны. Клиент должен быть в том же домене, что и сервер или в домене, имеющем двустороннее отношение доверия с доменом, который содержит сервер.|
|\<Server2 >...|Указывает дополнительный сервер или серверы, которые будут доступны клиенту. Если указан список серверов используется, используйте пробел для разделения имен серверов.|
|/d|Очищает список удаленных серверов. Отключает клиент TAPI, предотвращая использование поставщиков услуг TAPI, которые находятся на удаленных серверах.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Для выполнения данной процедуры необходимо входить в группу "Администраторы" на локальном компьютере или получить соответствующие полномочия путем делегирования. Если компьютер присоединен к домену, эту процедуру могут выполнять члены группы "Администраторы домена". В качестве лучшего решения с точки зрения безопасности для выполнения этой процедуры рекомендуется использовать функцию **Запуск от имени**.
-   Чтобы TAPI для правильной работы, необходимо запустить **tcmsetup** для указания удаленных серверов, которые будут использоваться клиентами TAPI.
-   Прежде чем пользователь клиентского компьютера можно использовать телефон или строки на сервере TAPI, администратор сервера телефонии необходимо назначить пользователю телефон или строки.
-   Список серверов, созданный этой командой, заменяет существующий список серверов, доступных клиенту. Чтобы добавить в существующий список нельзя использовать эту команду.

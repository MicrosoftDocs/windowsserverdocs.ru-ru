# <a name="wdsutil-new"></a>WDSUTIL New

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает записи и обнаружения образов, многоадресных передач и пространств имен.

## <a name="subcommands"></a>Подкоманды
|Подкоманда|Описание|
|-------|--------|
|[Команда WDSUTIL New-каптуреимаже](wdsutil-new-captureimage.md)|создает новый образ записи из существующего образа загрузки.|
|[Команда WDSUTIL New-дисковеримаже](wdsutil-new-discoverimage.md)|создает новый образ обнаружения из существующего образа загрузки.|
|[Команда WDSUTIL New-мултикасттрансмиссион](wdsutil-new-multicasttransmission.md)|создает новую многоадресную передачу.|
|[Команда WDSUTIL New-Namespace](wdsutil-new-namespace.md)|Создает новое пространство имен.|

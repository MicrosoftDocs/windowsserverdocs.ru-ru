# <a name="bitsadmin-util-and-help"></a>bitsadmin util и справки



Отображает использование командной строки для **Util** коммутаторов.

## <a name="syntax"></a>Синтаксис

```
Bitsadmin /Util /Help 
```

## <a name="BKMK_examples"></a>Примеры

Следующая команда отображает справку командной строки для **Util** коммутаторов.
```
Bitsadmin /Util /Help
```

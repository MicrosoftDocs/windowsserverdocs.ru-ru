# <a name="detail-volume"></a>detail volume

Отображает диски, на которых находится текущий том. Прежде чем начать, необходимо выбрать том для выполнения этой операции. Используйте команду [выбрать том](select-volume.md) , чтобы выбрать том и переместить фокус на него. Сведения о томе не применимы к томам только для чтения, например DVD-дискам или дисководам компакт-дисков.

## <a name="syntax"></a>Синтаксис

```
detail volume
```

## <a name="examples"></a>Примеры

Чтобы просмотреть все диски, в которых находится текущий том, введите:

```
detail volume
```

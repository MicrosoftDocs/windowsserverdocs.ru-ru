# <a name="online-disk"></a>диска в оперативном режиме



Переводит дисков, которые будут в автономном режиме, подключающееся к сети.

> [!IMPORTANT]
> Эта команда не доступна в любом выпуске Windows Vista.

> [!IMPORTANT]
> Эта команда завершится ошибкой, если он используется на диске только для чтения.

Инструкции о том, как использовать эту команду, см. в разделе [повторная активация отсутствует или автономном режиме динамического диска](https://go.microsoft.com/fwlink/?LinkId=207046) (https://go.microsoft.com/fwlink/?LinkId=207046).

## <a name="syntax"></a>Синтаксис

```
online disk [noerr]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|успешного|для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.|

## <a name="remarks"></a>Примечания

-   При использовании без параметров в Windows Vista эта команда работает в группе дисков. Для базовых дисков никогда не имеется более одного диска на каждую группу. Для динамических дисков группа включает всех неинородному динамических дисков.
-   Для базовых дисков эта команда попытается перевести в оперативный режим выбранного диска и все тома на этом диске.
-   Для динамических дисков эта команда попытается перевести все диски, которые не помечены как внешние на локальном компьютере. Он также попытаться перевести все тома на наборе динамических дисков.
-   Если динамический диск в группе диск будет подключен к сети, и это только диск в группе, исходная группа создается заново, и диск перемещается в эту группу. Если нет других дисков в группе, и они находятся в сети, диска просто добавляется обратно в группу.
-   Если группа выбранного диска содержит зеркальные тома RAID-5, эта команда также синхронизирует эти тома.
-   Для успешного выполнения этой команды необходимо выбрать диск. Используйте **выберите диск** команду, чтобы выбрать диск и перетянуть внимание к нему.

## <a name="BKMK_examples"></a>Примеры

Чтобы подключить диск с фокусом в Интернете, введите следующую команду:
```
online disk
```

# <a name="ftp-hash"></a>ftp hash

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Переключает печать знака решетки (#) для каждого переданного блока данных. По умолчанию команда hash отключена. Размер блока данных составляет 2048 байт.

## <a name="syntax"></a>Синтаксис

```
hash
```

### <a name="examples"></a>Примеры

Чтобы переключить печать знака решетки (#) для каждого переданного блока данных, введите:

```
hash
```

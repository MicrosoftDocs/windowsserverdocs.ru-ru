# <a name="bitsadmin-peers-and-help"></a>bitsadmin коллег и справки



Отображает использование командной строки для **одноранговые узлы** коммутаторов.

## <a name="syntax"></a>Синтаксис

```
Bitsadmin /Peers /Help 
```

## <a name="BKMK_examples"></a>Примеры

Следующая команда отображает справку командной строки для **одноранговые узлы** коммутаторов.
```
Bitsadmin /Peers /Help
```

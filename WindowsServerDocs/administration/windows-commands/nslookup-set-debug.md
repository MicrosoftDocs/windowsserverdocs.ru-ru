# <a name="nslookup-set-debug"></a>nslookup set debug



Включает режим отладки, включить или отключить.

## <a name="syntax"></a>Синтаксис

```
set [no]debug
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|**NODEBUG**|Отключает режим отладки. По умолчанию используется синтаксис **nodebug**.|
|**debug**|Включает режим отладки.|
|** {справки | ?}**|Отображает краткое описание **nslookup** подкоманды.|

## <a name="remarks"></a>Примечания

-   Режим отладки, включен будут выводиться Дополнительные сведения о пакетах, отправленных на сервер и полученный ответ.

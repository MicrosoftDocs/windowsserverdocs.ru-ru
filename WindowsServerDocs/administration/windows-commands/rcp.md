# <a name="rcp"></a>rcp

>[!IMPORTANT]
> Эта команда устарела.

Вы можете установить подсистему для приложений на основе UNIX с помощью **мастера добавления компонентов**. Дополнительные сведения и файлы для загрузки см. в разделе [Служебные программы и пакеты SDK для подсистемы приложений на базе UNIX в Microsoft Windows 7 и Windows Server 2008 R2](https://www.microsoft.com/download/details.aspx?id=2391).

После установки можно открыть командное окно C Shell (КШ или ТКШ) или Корншелл и выполнить команду **rcp**. Для получения дополнительных сведений введите **Man RCP** в командной строке C Shell или корншелл.

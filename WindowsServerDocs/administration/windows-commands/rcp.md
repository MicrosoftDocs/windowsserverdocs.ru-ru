# <a name="rcp"></a>rcp



Копирует файлы между компьютерами. Эта команда устарела. Вы можете установить подсистему для приложений на основе UNIX с помощью мастера добавления компонентов. Дополнительные сведения см. в разделе [Компоненты взаимодействия Unix Windows Server 2008](https://go.microsoft.com/fwlink/?LinkId=191835) на веб-сайте Майкрософт. После установки можно открыть командную окно оболочки C или Korn-оболочки и выполнить команду **rcp**. Для получения дополнительных сведений введите **Man RCP** в командной строке C Shell или Korn shell.

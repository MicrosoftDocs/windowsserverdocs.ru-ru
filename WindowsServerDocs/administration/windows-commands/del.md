# <a name="del"></a>del



Удаляет один или несколько файлов. Эта команда аналогична команде **Erase** .

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
del [/p] [/f] [/s] [/q] [/a[:]<Attributes>] <Names>
erase [/p] [/f] [/s] [/q] [/a[:]<Attributes>] <Names>
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|Имена \<>|Указывает список из одного или нескольких файлов или каталогов. Для удаления нескольких файлов можно использовать подстановочные знаки. Если указан каталог, все файлы в каталоге будут удалены.|
|/p|Запрашивает подтверждение перед удалением указанного файла.|
|/f|Принудительное удаление файлов, которые доступны только для чтения.|
|/s|Удаляет указанные файлы из текущего каталога и всех подкаталогов. Отображает имена файлов при их удалении.|
|/q|Задает тихий режим. Подтверждение удаления не запрашивается.|
|/a[:]\<атрибуты >|Удаляет файлы на основе следующих атрибутов файлов:</br>файлы **r** только для чтения</br>**h** скрытые файлы</br>**я** не проиндексированные файлы содержимого</br>**системные файлы**</br>готовые **к** архивированию</br>**l** точки повторного анализа</br>— Значение префикса "not"|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Замечания

> [!CAUTION]
> Если удалить файл с диска с помощью **Del** , вы не сможете получить его.
> -   При использовании параметра **/p**команда **Del** отображает имя файла и отправляет следующее сообщение:

    `FileName, Delete (Y/N)?`

    To confirm the deletion, press Y. To cancel the deletion and display the next file name (that is, if you specified a group of files), press N. To stop the **del** command, press CTRL+C.
- Если вы отключаете расширения команд, параметр **/s** отображает имена всех файлов, которые не были найдены, вместо того, чтобы отображать имена удаляемых файлов (то есть поведение изменяется на противоположное).
- Если указать в *именах*папку, все файлы в папке будут удалены. Например, следующая команда удаляет все файлы в папке \Ворк:  
  ```
  del \work
  ```  
- Можно использовать подстановочные знаки ( **&#42;** и **?** ) для удаления нескольких файлов за раз. Однако, чтобы избежать непреднамеренного удаления файлов, следует соблюдать подстановочные знаки с помощью команды **Del** . Например, если ввести следующую команду:  
  ```
  del *.*
  ```  
  Команда **Del** выводит следующее сообщение:

  `Are you sure (Y/N)?`

  Чтобы удалить все файлы в текущем каталоге, нажмите клавишу Y и нажмите клавишу ВВОД. Чтобы отменить удаление, нажмите клавишу N и нажмите клавишу ВВОД.

> [!NOTE]
> Прежде чем использовать подстановочные знаки с командой **Del** , используйте те же подстановочные знаки с командой **dir** , чтобы получить список всех файлов, которые будут удалены.
> -   Команда **Del** с различными параметрами доступна в консоли восстановления.

## <a name="BKMK_examples"></a>Примеров

Чтобы удалить все файлы в папке с именем Test на диске C, введите следующую команду:
```
del c:\test
del c:\test\*.*
```
Чтобы удалить все файлы с расширением bat из текущего каталога, введите:
```
del *.bat
```
Чтобы удалить все файлы только для чтения в текущем каталоге, введите:
```
del /a:r *.*
```

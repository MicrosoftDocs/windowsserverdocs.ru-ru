# <a name="ftp-bye"></a>ftp bye

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Завершает сеанс FTP на удаленном компьютере, а затем завершает работу.

> [!NOTE]
> Эта команда аналогична [команде FTP Quit](ftp-quit.md).

## <a name="syntax"></a>Синтаксис

```
bye
```

### <a name="examples"></a>Примеры

Чтобы завершить сеанс FTP с удаленным компьютером и выйти из программы, введите:

```
bye
```

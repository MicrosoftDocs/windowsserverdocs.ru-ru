# <a name="ftp-bye"></a>FTP: Bye

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Завершает сеанс FTP с удаленным компьютером и завершает работу.   
## <a name="syntax"></a>Синтаксис  
```  
bye  
```  
#### <a name="parameters"></a>Параметры  
none  
## <a name="remarks"></a>Примечания  
**Bye** и **завершите** сеанс FTP с удаленным компьютером и завершит работу.  
## <a name="examples"></a>Примеры  
Завершите сеанс FTP с удаленным компьютером и завершите работу.  
```  
bye  
```

# <a name="ftp-bye"></a>FTP: Bye

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Завершает сеанс FTP с удаленным компьютером и завершает работу.   
## <a name="syntax"></a>Синтаксис  
```  
bye  
```  
#### <a name="parameters"></a>Параметры  
нет  
## <a name="remarks"></a>Примечания  
**Bye** и **завершите** сеанс FTP с удаленным компьютером и завершит работу.  
## <a name="examples"></a><a name=BKMK_Examples></a>Примеров  
Завершите сеанс FTP с удаленным компьютером и завершите работу.  
```  
bye  
```

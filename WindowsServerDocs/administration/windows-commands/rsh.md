# <a name="rsh"></a>rsh



Выполнение команд на удаленных компьютерах, где RSH службы или управляющей программы. Эта команда является устаревшим. Можно установить подсистемы для UNIX-приложений с помощью мастера добавления компонентов. Дополнительные сведения см. в разделе [компоненты взаимодействия с Windows Server 2008 UNIX](https://go.microsoft.com/fwlink/?LinkId=191835) на веб-сайте Майкрософт. После установки можно затем откройте командное окно оболочки C или Korn-оболочки и запустите **rsh**. Дополнительные сведения, введите **man rsh** в командной строке оболочки C или Korn-оболочки.

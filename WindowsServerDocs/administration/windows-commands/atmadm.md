# <a name="atmadm"></a>atmadm

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Наблюдает за подключениями и адресами, зарегистрированными диспетчером вызовов atM в сети с асинхронным режимом пересылки (atM). С помощью **atmadm** можно отображать статистику для входящих и исходящих вызовов atM-адаптеров. При использовании без параметров **atmadm** отображает статистику для мониторинга состояния активных atM-подключений.

## <a name="syntax"></a>Синтаксис

```
atmadm [/c][/a][/s]
```

#### <a name="parameters"></a>Параметры

| Параметр | Описание |
| ------- | -------- |
| /C | Отображает сведения о вызовах для всех текущих подключений к сетевому адаптеру atM, установленному на этом компьютере. |
| /a | Отображает зарегистрированный адрес адресной службы сети atM (NSAP) для каждого адаптера, установленного на этом компьютере. |
| /s | Отображает статистику для мониторинга состояния активных atM-подключений. |
| /? | Отображение справки в командной строке. |

## <a name="examples"></a>Примеры

Чтобы отобразить сведения о вызовах для всех текущих подключений к сетевому адаптеру atM, установленному на этом компьютере, введите:

```
atmadm /c
```

Чтобы отобразить зарегистрированный адрес NSAP для каждого адаптера, установленного на этом компьютере, введите следующую команду:

```
atmadm /a
```

Чтобы отобразить статистику для наблюдения за состоянием активных atM-подключений, введите:

```
atmadm /s
```

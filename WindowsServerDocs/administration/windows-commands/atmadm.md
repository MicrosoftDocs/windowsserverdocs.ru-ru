# <a name="atmadm"></a>atmadm

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Контроль подключений и адресов, зарегистрированные с atM вызов Manager асинхронная передача сетях с режимом (atM). Можно использовать **atmadm** для отображения статистики входящих и исходящих вызовов atM-адаптеров. При использовании без параметров, **atmadm** отображает статистику для мониторинга состояния active atM-подключений. 

## <a name="syntax"></a>Синтаксис

```
atmadm [/c][/a][/s]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|-------|--------|
|/c|Вывод сведений о вызовах для всех текущих подключений к сетевому адаптеру atM установлен на этом компьютере.|
|/a|Отображает доступ сетевой службы зарегистрированные atM точек адрес (NSAP) для каждого адаптера, установленного на компьютере.|
|/s|Отображает статистику для мониторинга состояния active atM-подключений.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_Examples"></a>Примеры

Для отображения сведений о вызовах для всех текущих подключений к сетевому адаптеру atM установлен на этом компьютере, введите:

```
atmadm /c
```

Для отображения зарегистрированных atM сети службы адреса точки доступа (NSAP) для каждого адаптера, установленного на компьютере, введите:

```
atmadm /a
```

Чтобы отобразить статистические данные для мониторинга состояния active atM-подключений, введите следующую команду:

```
atmadm /s
```

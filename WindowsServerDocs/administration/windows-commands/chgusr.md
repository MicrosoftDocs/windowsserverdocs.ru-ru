# <a name="chgusr"></a>chgusr

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Изменяет режим установки для сервера узла сеансов удаленный рабочий стол.

> [!NOTE]
> Эта команда была заменена **командой изменения пользователя**. Дополнительные сведения, включая сведения о синтаксисе и параметрах, см. в разделе [Изменение пользовательской команды](change-user.md).

# <a name="wdsutil-disable"></a>отключить WDSUTIL

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Отключает все службы для служб развертывания Windows.

## <a name="subcommands"></a>Подкоманды
|Подкоманда|Описание|
|-------|--------|
|[Команда WDSUTIL Disable-Server](wdsutil-disable-server.md)|Отключает все службы развертывания Windows на указанном сервере (сервере развертывания).|
|[Команда WDSUTIL Disable-транспортсервер](wdsutil-disable-transportserver.md)|Отключает все службы развертывания Windows на указанном транспортном сервере.|

# <a name="chglogon"></a>chglogon

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Включает или отключает вход из сеансов клиента на удаленный рабочий стол сервере узла сеансов или отображает текущее состояние входа в систему.

> [!NOTE]
> Эта команда была заменена **командой журнала изменений**. Дополнительные сведения, включая сведения о синтаксисе и параметрах, см. в разделе [изменение имени входа](change-logon.md).

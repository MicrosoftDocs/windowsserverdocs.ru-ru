# <a name="tsdiscon"></a>tsdiscon

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Отключает сеанс от сервера узла сеансов удаленный рабочий стол. Если не указать идентификатор сеанса или имя сеанса, эта команда отключает текущий сеанс.

> [!IMPORTANT]
> Для отключения другого пользователя от сеанса необходимо иметь разрешение на **полный доступ** или **Отключить специальное** разрешение на доступ.

> [!NOTE]
> Чтобы узнать о новых возможностях последней версии, см. статью [новые возможности службы удаленных рабочих столов в Windows Server](/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/dn283323(v=ws.11)).

## <a name="syntax"></a>Синтаксис

```
tsdiscon [<sessionID> | <sessionname>] [/server:<servername>] [/v]
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
|--|--|
| `<sessionID>` | Указывает идентификатор сеанса для отключения. |
| `<sessionname>` | Указывает имя сеанса для отключения. |
| /server:`<servername>` | Указывает сервер терминалов, содержащий сеанс, который необходимо отключить. В противном случае используется текущий сервер узла сеансов удаленный рабочий стол. Этот параметр является обязательным только при выполнении команды **тсдискон** с удаленного сервера. |
| /v | Отображает сведения о выполняемых действиях. |
| /? | Отображение справки в командной строке. |

#### <a name="remarks"></a>Комментарии

- Все приложения, выполняющиеся при отключении сеанса, автоматически выполняются при повторном подключении к этому сеансу без потери данных. [Команда сбросить сеанс](reset-session.md) можно использовать для завершения работающих приложений отключенного сеанса, но это может привести к потере данных в сеансе.

- Не удается отключить сеанс консоли.

## <a name="examples"></a>Примеры

Чтобы отключить текущий сеанс, введите:

```
tsdiscon
```

Чтобы отключить *сеанс 10*, введите:

```
tsdiscon 10
```

Чтобы отключить сеанс с именем *TERM04*, введите:

```
tsdiscon TERM04
```

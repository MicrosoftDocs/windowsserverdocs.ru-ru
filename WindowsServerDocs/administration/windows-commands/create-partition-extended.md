# <a name="create-partition-extended"></a>Создание

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает дополнительный раздел на диске с фокусом. Эта команда используется только на основная загрузочная запись \(MBR\) дисков.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
create partition extended [size=<n>] [offset=<n>] [align=<n>] [noerr]  
```  
  
## <a name="parameters"></a>Параметры  
  
|  Параметр  |                                                                                                                             Описание                                                                                                                              |
|-------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  Размер\=<n>  |                                                  Указывает размер раздела в мегабайтах \(МБ\). Если размер не указан, раздел занимает на диске имеется свободное пространство.                                                  |
| Смещение\=<n> |                     Задает смещение в килобайтах \(КБ\), в которой создается секция. Если смещение не указано, раздел начинается в начале свободного места на диске, достаточное для хранения новой секции.                      |
| Выравнивание\=<n>  | Выравнивает все экстенты секции до ближайшей границы выравнивания. Обычно используется с оборудованием логический номер устройства RAID \(LUN\) массивы для повышения производительности. <n> Число килобайт \(КБ\) с самого начала диска до ближайшей границы выравнивания. |
|    успешного    |                                 для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.                                 |
  
## <a name="remarks"></a>Примечания  
  
-   После создания раздела фокус автоматически переносится на этот раздел.  
  
-   Можно создать только один дополнительный раздел на диск.  
  
-   Эта команда не выполняется, если вы пытаетесь создать дополнительный раздел в другом дополнительном разделе.  
  
-   Перед созданием логических дисков, необходимо создать дополнительный раздел.  
  
-   Для успешного выполнения этой операции необходимо выбрать базовый диск MBR. Используйте **выберите диск** команду, чтобы выбрать диск и перетянуть внимание к нему.  
  
## <a name="BKMK_examples"></a>Примеры  
Чтобы создать дополнительный раздел 1000 МБ, размер, введите следующую команду:  
  
```  
create partition extended size=1000  
```

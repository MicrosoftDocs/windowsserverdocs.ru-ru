# <a name="nslookup-finger-command"></a>nslookup finger Command



Подключается к серверу finger на текущем компьютере.

## <a name="syntax"></a>Синтаксис

```
finger [<UserName>] [{[>] <FileName>|[>>] <FileName>}]
```

## <a name="parameters"></a>Параметры

|  Параметр  |                                                                               Описание                                                                               |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \<Имя пользователя > |                                                               Указывает имя пользователя для поиска.                                                                |
| \<Имя файла > | Задает имя файла, в котором следует сохранить выходные данные. Можно использовать больше (>) и двойные больше (>>) символы для перенаправления выходных данных обычным способом. |
|    {справки    |                                                                                   ?}                                                                                    |

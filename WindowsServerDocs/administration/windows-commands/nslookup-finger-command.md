# <a name="nslookup-finger-command"></a>nslookup finger Command



Подключается к серверу finger на текущем компьютере.

## <a name="syntax"></a>Синтаксис

```
finger [<UserName>] [{[>] <FileName>|[>>] <FileName>}]
```

### <a name="parameters"></a>Параметры

|  Параметр  |                                                                               Описание                                                                               |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \<имя пользователя > |                                                               Указывает имя пользователя для поиска.                                                                |
| \<имя файла > | Указывает имя файла, в котором следует сохранить выходные данные. Для перенаправления выходных данных в обычном режиме можно использовать символы "больше" (>) и "больше" (> >). |
|    {Справка    |                                                                                   ?}                                                                                    |

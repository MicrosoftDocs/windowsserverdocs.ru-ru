# <a name="bitsadmin-cache-and-help"></a>кэш bitsadmin и справки



Отображает использование командной строки для **кэша** коммутаторов.

## <a name="syntax"></a>Синтаксис

```
Bitsadmin /Cache /Help 
```

## <a name="BKMK_examples"></a>Примеры

Следующая команда отображает справку командной строки для **кэша** коммутаторов.
```
Bitsadmin /Cache /Help
```

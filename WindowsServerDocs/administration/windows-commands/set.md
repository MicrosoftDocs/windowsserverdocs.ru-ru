# <a name="set-commands-shadow-copy-creation"></a>Задание команд (создание теневой копии)

Задает контекст, параметры, подробный режим и файл метаданных для создания теневой копии. Если используется без параметров, **установите** список всех текущих параметров.

## <a name="syntax"></a>Синтаксис

```
set
set context
set option
set verbose
set metadata
```

### <a name="parameters"></a>Параметры

| Параметры | Описание |
|--|--|
| [set context](set-context.md) | Задает контекст для создания теневой копии. |
| [set metadata](set-metadata.md) | Задает имя и расположение файла метаданных для создания теневой копии. |
| [set option](set-option.md) | Задает параметры для создания теневой копии. |
| [set verbose](set-verbose.md) | Включает или выключает режим подробных выходных данных. |
| /? | Отображение справки в командной строке. |

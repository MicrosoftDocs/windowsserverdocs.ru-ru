# <a name="nslookup-set-d2"></a>nslookup set d2



Включает углубленный режим отладки, включить или отключить. Выводятся все поля каждого пакета.

## <a name="syntax"></a>Синтаксис

```
set [no]d2
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|**nod2**|Отключает углубленный режим отладки. По умолчанию используется синтаксис **nod2**.|
|**d2**|Включает углубленный режим отладки.|
|{справки | ?}|Отображает краткое описание **nslookup** подкоманды.|

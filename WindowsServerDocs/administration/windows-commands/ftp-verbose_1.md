# <a name="ftp-verbose"></a>ftp verbose

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Переключает режим подробного вывода. По умолчанию включен режим подробного вывода. Если режим подробного протоколирования включен, отображаются все ответы на команды **FTP** . После завершения перемещения файла также отображаются статистические данные, касающиеся эффективности перемещения.

## <a name="syntax"></a>Синтаксис

```
verbose
```

### <a name="examples"></a>Примеры

Чтобы включить или отключить режим подробного вывода, введите:

```
verbose
```

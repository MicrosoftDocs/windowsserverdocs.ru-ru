# <a name="ftp-debug"></a>ftp debug

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Переключает режим отладки. По умолчанию режим отладки отключен. Если включен режим отладки, вы увидите каждую команду, отправленную на удаленный компьютер, буква `>` символом.

## <a name="syntax"></a>Синтаксис

```
debug
```

### <a name="examples"></a>Примеры

Чтобы включить или отключить режим отладки, введите:

```
debug
```

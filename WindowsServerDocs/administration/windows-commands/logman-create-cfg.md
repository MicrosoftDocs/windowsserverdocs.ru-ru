# <a name="logman-create-cfg"></a>Logman Create cfg

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создание сборщика данных конфигурации.  

## <a name="syntax"></a>Синтаксис  
```  
logman create cfg <[-n] <name>> [options]  
```

## <a name="remarks"></a>Примечания  
Где [-] присутствует, дополнительный — инвертирует параметр.  
## <a name="examples"></a>Примеры  
Следующая команда создает сборщик данных конфигурации с именем cfg_log, используя раздел реестра HKEY_LOCAL_MACHINE \Софтваре\микрософт\виндовс Нт\куррентверион\\.  
```  
logman create cfg cfg_log -reg HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\Currentverion\\  
```  
Следующая команда создает сборщик данных конфигурации с именем cfg_log, который записывает все объекты WMI из рут\вми в столбце базы данных MSNdis_Vendordriverversion.  
```  
logman create cfg cfg_log -mgt root\wmi:select * FROM MSNdis_Vendordriverversion  
```

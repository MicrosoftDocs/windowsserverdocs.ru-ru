# <a name="graftabl"></a>graftabl



Позволяет операционных систем Windows для отображения национальных алфавитов в графическом режиме. При использовании без параметров, **graftabl** отображает предыдущей и текущей кодовой странице.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
graftabl <CodePage>
graftabl /status
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Кодовая страница >|Указывает кодовую страницу для определения внешнего вида символов национального алфавита в графическом режиме.</br>Идентификационные номера допустимый код страницы являются:</br>437: США</br>850: Многоязыковой (Latin I)</br>852: Восточная и Центральная Европа (латиница II)</br>855: Кириллица (русская)</br>857: Турецкий</br>860: Португальский</br>861: Исландский</br>863: Французский (Канада)</br>865: Nordic</br>866: Русский</br>869: Греческая современная|
|/status|Отображает текущее кодовая страница, **graftabl** использует.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   **Graftabl** влияет только на отображение монитора символов национального алфавита кодовой страницы, указанной вами. Это не приводит к изменению консольному кодовой страницы входных данных. Чтобы изменить входной кодовую страницу консоли, используйте **режим** или **chcp** команды.
-   Ниже перечислены коды завершения и краткое описание его.  
    |Код завершения|Описание|
    |---------|-----------|
    |0|Набор символов успешно загружен. Предыдущий кодовая страница не была загружена.|
    |1|Указан неверный параметр. Никаких действий предпринято не было.|
    |2|Ошибка файла.|
-   Можно использовать переменную среды ERRORLEVEL в пакетном коды выхода процесса, возвращаемых **graftabl**.

## <a name="BKMK_examples"></a>Примеры

Чтобы просмотреть текущую кодовую страницу, используемые **graftabl**, тип:
```
graftabl /status
```
Чтобы загрузить графических символов для кодовую страницу 437 (США) в память, введите следующую команду:
```
graftabl 437
```
Чтобы загрузить графических символов кодовая страница 850 (многоязыкового) в памяти, введите следующую команду:
```
graftabl 850
```

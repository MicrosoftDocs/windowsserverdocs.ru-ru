# <a name="nslookup-exit-command"></a>nslookup exit Command



завершает работу **nslookup**.

## <a name="syntax"></a>Синтаксис

```
Nslookup /exit
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|{справки | ?}|Отображает краткое описание **nslookup** подкоманды.|

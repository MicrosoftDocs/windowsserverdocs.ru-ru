# <a name="nslookup-set-all"></a>nslookup set all



Выводит текущие значения параметров конфигурации.

## <a name="syntax"></a>Синтаксис

```
set all 
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|{справки | ?}|Отображает краткое описание **nslookup** подкоманды.|

## <a name="remarks"></a>Примечания

-   **Установить все** также выводит сведения о сервере по умолчанию и компьютер (узел).

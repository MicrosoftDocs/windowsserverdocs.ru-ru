# <a name="schtasks-commands"></a>команды SchTasks

Планирует выполнение команд и программ периодически или в определенное время, добавляет и удаляет задачи из расписания, запускает и останавливает задачи по требованию, а также отображает и изменяет запланированные задачи.

> [!NOTE]
> Средство **schtasks.exe** выполняет те же операции, что и **запланированные задачи** на **панели управления**. Эти инструменты можно использовать совместно и взаимозаменяемы.

## <a name="required-permissions"></a>Необходимые разрешения

- Чтобы запланировать, просмотреть и изменить все задачи на локальном компьютере, необходимо быть членом группы "Администраторы".

- Чтобы запланировать, просмотреть и изменить все задачи на удаленном компьютере, необходимо быть членом группы "Администраторы" на удаленном компьютере или использовать параметр **/u** для предоставления учетных данных администратора удаленного компьютера.

- Параметр **/u** в операции **/CREATE** или **/Change** можно использовать, если локальный и удаленный компьютеры находятся в одном домене или если локальный компьютер находится в домене, которому доверяет домен удаленного компьютера. В противном случае удаленный компьютер не может проверить подлинность указанной учетной записи пользователя и не сможет проверить, является ли эта учетная запись членом группы "Администраторы".

- Задача, которую планируется запустить, должна иметь соответствующее разрешение; Эти разрешения зависят от задачи. По умолчанию задачи выполняются с разрешениями текущего пользователя локального компьютера или с разрешениями пользователя, заданного параметром **/u** , если он включен. o запустите задачу с разрешениями другой учетной записи пользователя или с системными разрешениями, используя параметр **/ru** .

## <a name="syntax"></a>Синтаксис

```
schtasks change
schtasks create
schtasks delete
schtasks end
schtasks query
schtasks run
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
|--|--|
| [команда Schtasks Change](schtasks-change.md) | Изменяет одно или несколько из следующих свойств задачи:<ul><li>Программа, выполняемая задачей (/tr)</li><li>Учетная запись пользователя, под которой выполняется задача (/ru)</li><li>Пароль для учетной записи пользователя (/RP)</li><li>Добавляет интерактивное свойство в задачу (/IT)</li></ul> |
| [команда Schtasks CREATE](schtasks-create.md) | Планирует новую задачу.|
| [команда Schtasks Delete](schtasks-delete.md) | Удаляет запланированную задачу. |
| [команда Schtasks End](schtasks-end.md) | Останавливает программу, запущенную задачей. |
| [запрос SchTasks](schtasks-query.md) | Отображает задачи, запланированные для запуска на компьютере. |
| [команда Schtasks Run](schtasks-run.md) | Немедленно запускает запланированную задачу. Операция **запуска** игнорирует расписание, но использует расположение файла программы, учетную запись пользователя и пароль, сохраненные в задаче для немедленного выполнения задачи. |

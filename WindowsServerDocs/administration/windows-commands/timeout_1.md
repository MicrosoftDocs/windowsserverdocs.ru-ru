# <a name="timeout"></a>timeout



Приостанавливает обработку команды для указанного числа секунд.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
timeout /t <TimeoutInSeconds> [/nobreak] 
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/t \<TimeoutInSeconds>|Указывает десятичное число секунд (от -1 и 99999) ожидания перед продолжением обработки команд. Значение -1 заставляет компьютер для неограниченного времени ожидания нажатия клавиши.|
|/NOBREAK|Задает необходимость игнорировать нажатий клавиш пользователем.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   **Время ожидания** команда обычно используется в пакетных файлах.
-   Нажатие клавиши пользователем возобновляет выполнение процессора команда немедленно, даже если истечения времени ожидания не истек.
-   При использовании в сочетании с **спящего режима** команде **время ожидания** аналогичен **приостановить** команды.

## <a name="BKMK_examples"></a>Примеры

Для остановки обработки команд на десять секунд, введите следующую команду:
```
timeout /t 10
```
Чтобы приостановить интерпретатор командной строки для 100 секунд и Игнорировать нажатия клавиш, введите следующую команду:
```
timeout /t 100 /nobreak
```
Для остановки обработки команд, бесконечно, пока не нажата клавиша, введите следующую команду:
```
timeout /t -1
```

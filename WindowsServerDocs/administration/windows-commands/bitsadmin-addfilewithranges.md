# <a name="bitsadmin-addfilewithranges"></a>bitsadmin addfilewithranges

Добавляет файл в указанное задание. Служба BITS загружает указанные диапазоны из удаленного файла. Этот параметр допустим только для заданий скачивания.

## <a name="syntax"></a>Синтаксис

```
bitsadmin /AddFileWithRanges <Job> <RemoteURL> <LocalName> <RangeList>
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|Job|Отображаемое имя задания или идентификатор GUID|
|ремотеурл|*Ремотеурл* — это URL-адрес файла на сервере.|
|Локально|*LocalName* — это имя файла на локальном компьютере. *LocalName* должно содержать абсолютный путь к файлу.|
|ранжелист|*Ранжелист* — это разделенный запятыми список пар "смещение: длина". Используйте двоеточие, чтобы разделить значение смещения от значения длины. Например, значение `0:100,2000:100,5000:eof` указывает, что биты пересылают 100 байт со смещения 0, 100 байт из смещения 2000, а оставшиеся байт от смещения 5000 до конца файла.|

## <a name="more-information"></a>Дополнительные сведения

-   Токен **EOF** — это допустимое значение длины в пределах пар offset и length в *\<RangeList >* . Он указывает службе выполнить чтение до конца указанного файла.
-   Обратите внимание, что Аддфилевисранжес завершится с кодом ошибки 0x8020002c, когда диапазон нулевой длины указан вместе с другим диапазоном с тем же смещением, например: К:\битс > битсадмин/аддфилевисранжес J2 http://bitsdc/dload/1k.zip c:\1k.zip 100:0100:5

    Сообщение об ошибке: Не удалось добавить файл в задание — 0x8020002c. Список диапазонов байтов содержит несколько перекрывающихся диапазонов, которые не поддерживаются.

    Обходное решение. сначала не указывайте диапазон нулевой длины. Например: битсадмин/аддфилевисранжес J2 http://bitsdc/dload/1k.zip c:\1k.zip 100:5100:0.

## <a name="examples"></a>Примеры

В следующем примере показано, как биты пересылают 100 байт со смещения 0, 100 байт из смещения 2000, а оставшиеся байт от смещения 5000 до конца файла.

```
C:\>bitsadmin /addfilewithranges http://downloadsrv/10mb.zip c:\10mb.zip "0:100,2000:100,5000:eof"
```

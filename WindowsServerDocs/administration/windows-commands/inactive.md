# <a name="inactive"></a>Неактивные



На дисках с основной загрузочной записью (MBR) помечает системный раздел и загрузочный раздел как неактивный.

## <a name="syntax"></a>Синтаксис

```
inactive
```

## <a name="remarks"></a>Примечания

> [!CAUTION]
> Компьютер может не запуститься без активного раздела. Не следует помечать системный или загрузочный раздел неактивным, если вы являетесь опытным пользователем исчерпывающие сведения о в операционных системах семейства Windows.</br>> Если вы не сможете запустить на компьютере после того, системный или загрузочный раздел неактивным, вставить Компакт-диска Windows в дисковод компакт-дисков, перезагрузите компьютер и затем восстановить секции с помощью **fixmbr** и **fixboot** команды в консоли восстановления.
> -   Если пометить системный раздел и загрузочный раздел неактивным, компьютер загружается с следующего параметра, указанного в BIOS, например компакт-диска или предзагрузочной среды выполнения (PXE).
> -   Для успешного выполнения этой операции необходимо выбрать активный системный или загрузочный раздел. Используйте **выберите секцию** команду, чтобы выбрать активный раздел и перетянуть внимание к нему.

## <a name="BKMK_examples"></a>Примеры

```
inactive
```

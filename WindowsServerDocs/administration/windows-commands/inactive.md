# <a name="inactive"></a>неактивно



На дисках с основной загрузочной записью (MBR) помечает системный раздел или загрузочный раздел на фокус как неактивный.

## <a name="syntax"></a>Синтаксис

```
inactive
```

## <a name="remarks"></a>Примечания

> [!CAUTION]
> Компьютер может не запуститься без активного раздела. Не помечайте системный или загрузочный раздел как неактивный, если вы не являетесь опытным пользователем, хорошо знакомым с семейством операционных систем Windows.</br>> если не удается запустить компьютер после отметки системного или загрузочного раздела как неактивного, вставьте программа установки Windows компакт-диск в дисковод компакт-дисков, перезагрузите компьютер и восстановите раздел с помощью команд **Fixmbr** и **Fixboot** в консоли восстановления.
> -   После того как системный раздел или загрузочный раздел будет помечен как неактивный, компьютер будет запускаться с помощью следующего параметра, указанного в BIOS, например устройства чтения компакт-дисков или среды выполнения предварительной загрузки (PXE).
> -   Для выполнения этой операции необходимо выбрать активный системный или загрузочный раздел. Используйте команду **Выбор секции** , чтобы выбрать активный раздел и переместить фокус на него.

## <a name="examples"></a>Примеры

```
inactive
```

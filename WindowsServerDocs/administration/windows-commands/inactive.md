# <a name="inactive"></a>неактивно

Помечает системный раздел или загрузочный раздел как неактивный на базовых дисках с основной загрузочной записью (MBR).

Для выполнения этой операции необходимо выбрать активный системный или загрузочный раздел. Используйте команду [Выбор команды Partition](select-partition.md) , чтобы выбрать активный раздел и переместить фокус на него.

> [!CAUTION]
> Компьютер может не запуститься без активного раздела. Не помечайте системный или загрузочный раздел как неактивный, если вы не являетесь опытным пользователем, хорошо знакомым с семейством операционных систем Windows.<p>Если не удается запустить компьютер после отметки системного или загрузочного раздела как неактивного, вставьте компакт-диск программа установки Windows в дисковод для чтения компакт-дисков, перезагрузите компьютер, а затем восстановите раздел с помощью команд **Fixmbr** и **Fixboot** в консоли восстановления.
>
> После того как системный раздел или загрузочный раздел будет помечен как неактивный, компьютер будет запускаться с помощью следующего параметра, указанного в BIOS, например устройства чтения компакт-дисков или среды выполнения предварительной загрузки (PXE).

## <a name="syntax"></a>Синтаксис

```
inactive
```

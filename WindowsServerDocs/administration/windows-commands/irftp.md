# <a name="irftp"></a>irftp

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Передача файлов по инфракрасной связи.    
## <a name="syntax"></a>Синтаксис  
```  
irftp [<Drive>:\] [[<path>] <FileName>] [/h][/s]  
```  

### <a name="parameters"></a>Параметры  
|Параметр|Описание|  
|-------|--------|  
|Диск:\|диск, который содержит файлы, которые вы хотите отправить по инфракрасной связи.|  
|[путь] Имя файла|Указывает расположение и имя файла или набора файлов, которые вы хотите отправить по инфракрасной связи. Если указан набор файлов, необходимо указать полный путь для каждого файла.|  
|/h|Скрытый режим. При использовании скрытого режима отправки файлов без отображения диалогового окна инфракрасной связи.|  
|/s|Откроется окно инфракрасной связи, так что можно выбрать файл или набор файлов, которые будут отправляться без использования командной строки, чтобы указать диск, путь и имена файлов.|  

## <a name="remarks"></a>Примечания  
-   Прежде чем использовать эту команду, убедитесь, что устройства, которые вы хотите обмениваться данными по инфракрасной связи функциональные возможности включены и работают правильно, и что инфракрасную связь устанавливается между устройствами.  
-   Без параметров или использовании **/s**, **irftp** открывает **инфракрасной связи** диалоговое окно, в котором можно выбрать файлы, которые будут отправляться без использования командной строки.  

## <a name="BKMK_Examples"></a>Примеры  
Отправьте Example.txt инфракрасной связи.  
```  
irftp c:\example.txt  
```

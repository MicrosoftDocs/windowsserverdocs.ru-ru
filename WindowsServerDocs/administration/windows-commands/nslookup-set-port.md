# <a name="nslookup-set-port"></a>nslookup set port

Изменяет порт сервера доменных имен (DNS) по умолчанию TCP/UDP на указанное значение.

## <a name="syntax"></a>Синтаксис

```
set port=<port>
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
| ---------- | ---------- |
| `<port>` | Указывает новое значение для порта сервера DNS-имен TCP/UDP по умолчанию. Порт по умолчанию — **53**. |
| /? | Отображение справки в командной строке. |
| /help | Отображение справки в командной строке. |

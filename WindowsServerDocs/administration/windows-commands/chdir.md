# <a name="chdir"></a>chdir

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Отображает имя текущего каталога или изменяет текущий. Если используется только с буквой диска (например, `chdir C:` ), **chdir** отображает имена текущего каталога на указанном диске. Если используется без параметров, в **chdir** отображается текущий диск и каталог.

> > [!NOTE]
> Эта команда была заменена **командой CD**. Дополнительные сведения, включая сведения о синтаксисе и параметрах, см. в разделе [команда CD](cd.md).

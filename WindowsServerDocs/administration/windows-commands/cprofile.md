# <a name="cprofile"></a>cprofile

Кпрофиле является устаревшим и не гарантируется поддержка в будущих выпусках Windows.

Кпрофиле. exe: чистый профиль. Службы терминалов были обновлены для службы удаленных рабочих столов. Дополнительные сведения см. [в статье Добро пожаловать в службы удаленных рабочих столов](https://docs.microsoft.com/windows-server/remote/remote-desktop-services/welcome-to-rds).

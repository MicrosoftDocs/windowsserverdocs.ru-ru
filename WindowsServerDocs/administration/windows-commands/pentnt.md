# <a name="pentnt"></a>pentnt

> Область применения: Windows Server 2003, Windows Server 2003 R2, Windows Server 2003 с пакетом обновления 1 (SP1), Windows Server 2003 с пакетом обновления 2 (SP2)

>[!IMPORTANT]
> Эта команда устарела.

Обнаруживает ошибку деления с плавающей запятой (при наличии) в микросхеме Pentium, отключает оборудование с плавающей запятой и включает эмуляцию плавающей запятой. Описание и сведения об использовании см. в разделе [пентнт](/previous-versions/orphan-topics/ws.10/cc755868(v=ws.10)).

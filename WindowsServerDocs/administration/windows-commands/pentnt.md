Пентнт является устаревшим и не гарантируется поддержка в будущих выпусках Windows.

Это средство входит в Windows Server 2003. Дополнительные сведения см. в разделе [пентнт](https://technet.microsoft.com/library/cc755868(v=ws.10).aspx).

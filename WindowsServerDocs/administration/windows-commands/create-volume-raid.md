# <a name="create-volume-raid"></a>Создание тома raid

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает RAID\-5 тома, с помощью трех или более указанные динамические диски.  
  
> [!IMPORTANT]  
> Эта команда DiskPart не доступна в любом выпуске Windows Vista.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
create volume raid [size=<n>] disk=<n>,<n>,<n>[,<n>,...] [align=<n>] [noerr]  
```  
  
## <a name="parameters"></a>Параметры  
  
|           Параметр           |                                                                                                                                                                                                                                              Описание                                                                                                                                                                                                                                              |
|-------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|           Размер\=<n>           | Объем места на диске в мегабайтах \(МБ\), том будет занимать на каждом диске. Если размер не задан, наибольшее возможных RAID\-будет создан том 5. Размер определяется диском с наименьшей смежных свободное место для RAID\-5 тома и такой же объем пространства выделяется из каждого диска. Фактический объем доступного дискового пространства в RAID\-5 тома меньше, чем общий объем места на диске, так как часть места на диске необходим для четности. |
| диск\=<n>,<n>,<n>\[,<n>,...\] |                                                                                                                                               Динамические диски, на котором требуется создать RAID\-том RAID5. Необходимо по крайней мере три динамических дисков для создания RAID\-том RAID5. Объем пространства, равным **размер\= <n>**  выделяется на каждом диске.                                                                                                                                                |
|          Выравнивание\=<n>           |                                                                                                                   Выравнивает все области тома до ближайшей границы выравнивания. Обычно используется с оборудованием логический номер устройства RAID \(LUN\) массивы для повышения производительности. *n* число килобайт \(КБ\) с самого начала диска до ближайшей границы выравнивания.                                                                                                                   |
|             успешного             |                                                                                                                                                 для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.                                                                                                                                                  |
  
## <a name="remarks"></a>Примечания  
  
-   После создания тома фокус автоматически переносится на новый том.  
  
## <a name="BKMK_examples"></a>Примеры  
Создание RAID\-5 объем 1000 МБ, с помощью дисков, 1, 2 и 3, type:  
  
```  
create volume raid size=1000 disk=1,2,3  
```

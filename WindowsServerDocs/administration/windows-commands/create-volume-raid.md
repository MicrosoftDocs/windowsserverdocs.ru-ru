# <a name="create-volume-raid"></a>Создание тома RAID

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает том RAID-5, используя три или более указанных динамических диска.  

> [!IMPORTANT]  
> Эта команда DiskPart недоступна ни в одном из выпусков Windows Vista.

## <a name="syntax"></a>Синтаксис  
  
```  
create volume raid [size=<n>] disk=<n>,<n>,<n>[,<n>,...] [align=<n>] [noerr]  
```  
  
### <a name="parameters"></a>Параметры  
  
|           Параметр           |                                                                                                                                                                                                                                              Описание                                                                                                                                                                                                                                              |
|-------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|           Размер\=<n>           | Объем дискового пространства в мегабайтах \(МБ\), который будет занимать том на каждом диске. Если размер не указан, будет создан самый крупный том RAID\-5. Диск с наименьшим доступным непрерывным свободным пространством определяет размер тома RAID\-5 и тот же объем дискового пространства, выделенный для каждого диска. Фактический объем свободного дискового пространства на томе RAID\-5 меньше, чем общий объем дискового пространства, так как для четности требуется некоторое дисковое пространство. |
| \=диска <n>,<n>,<n>\[,<n>,...\] |                                                                                                                                               Динамические диски, на которых создается том RAID\-5. Для создания тома RAID\-5 требуется по крайней мере три динамических диска. Объем пространства, равный **размеру\=<n>** выделено на каждом диске.                                                                                                                                                |
|          Выровняйте\=<n>           |                                                                                                                   Выравнивает все экстенты томов по ближайшей границе выравнивания. Обычно используется с аппаратным номером логического устройства RAID \(LUN\) массивы для повышения производительности. *n* — это количество килобайтов \(КБ\) от начала диска до ближайшей границы выравнивания.                                                                                                                   |
|             Noerr             |                                                                                                                                                 Только для сценариев. При возникновении ошибки DiskPart продолжит обрабатывать команды, как если бы ошибка не возникала. Без этого параметра ошибка приводит к выходу из программы DiskPart с кодом ошибки.                                                                                                                                                  |
  
## <a name="remarks"></a>Примечания  
  
-   После создания тома фокус автоматически переместится на новый том.  
  
## <a name="examples"></a><a name=BKMK_examples></a>Примеров  
Чтобы создать том RAID\-5 размером 1000 МБ, используя диски 1, 2 и 3, введите:  
  
```  
create volume raid size=1000 disk=1,2,3  
```

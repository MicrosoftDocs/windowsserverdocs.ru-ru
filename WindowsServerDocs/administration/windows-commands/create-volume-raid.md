# <a name="create-volume-raid"></a>Создание тома RAID

>Область применения. Windows Server (половина ежегодного канала), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает том RAID @ no__t-05, используя три или более указанных динамических диска.  
  
> [!IMPORTANT]  
> Эта команда DiskPart недоступна ни в одном из выпусков Windows Vista.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
create volume raid [size=<n>] disk=<n>,<n>,<n>[,<n>,...] [align=<n>] [noerr]  
```  
  
## <a name="parameters"></a>Параметры  
  
|           Параметр           |                                                                                                                                                                                                                                              Описание                                                                                                                                                                                                                                              |
|-------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|           size @ no__t-0 @ no__t-1           | Объем дискового пространства в мегабайтах \(MB @ no__t-1, который будет занимать том на каждом диске. Если размер не указан, будет создан самый крупный том RAID @ no__t-05. Диск с наименьшим доступным непрерывным свободным пространством определяет размер тома RAID @ no__t-05 и тот же объем дискового пространства, выделенный для каждого диска. Фактический объем свободного дискового пространства на томе RAID @ no__t-05 меньше, чем общий объем дискового пространства, так как для четности требуется определенное дисковое пространство. |
| Disk @ no__t-0 @ no__t-1, <n>, <n> @ no__t-4, <n>,... \] |                                                                                                                                               Динамические диски, на которых создается том RAID @ no__t-05. Для создания тома RAID @ no__t-05 необходимо по крайней мере три динамических диска. Объем пространства, равный **size @ no__t-1 @ no__t-2** , выделяется на каждом диске.                                                                                                                                                |
|          ALIGNED @ no__t-0 @ no__t-1           |                                                                                                                   Выравнивает все экстенты томов по ближайшей границе выравнивания. Обычно используется с аппаратными массивами RAID с номерами устройств \(LUN @ no__t-1 для повышения производительности. *n* — это количество килобайтов \( КБ @ no__t-2 от начала диска до ближайшей границы выравнивания.                                                                                                                   |
|             Noerr             |                                                                                                                                                 только для сценариев. При возникновении ошибки DiskPart продолжит обрабатывать команды, как если бы ошибка не возникала. Без этого параметра ошибка приводит к выходу из программы DiskPart с кодом ошибки.                                                                                                                                                  |
  
## <a name="remarks"></a>Примечания  
  
-   После создания тома фокус автоматически переместится на новый том.  
  
## <a name="BKMK_examples"></a>Примеров  
Чтобы создать том RAID @ no__t-05 размером 1000 МБ, используя диски 1, 2 и 3, введите:  
  
```  
create volume raid size=1000 disk=1,2,3  
```

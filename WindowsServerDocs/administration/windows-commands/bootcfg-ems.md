# <a name="bootcfg-ems"></a>bootcfg ems

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Позволяет пользователю добавлять или изменять параметры перенаправления консоли служб аварийного управления на удаленный компьютер. Включив службы аварийного управления, добавляет `redirect=Port#` строку в раздел [boot loader] файла Boot. ini вместе с параметром/Redirect в указанную строку ввода операционной системы. Компонент "службы аварийного управления" включен только на серверах.

## <a name="syntax"></a>Синтаксис

```
bootcfg /ems {on | off | edit}[/s <computer> [/u <domain>\<user> /p <password>]] [/port {COM1 | COM2 | COM3 | COM4 | BIOSSET}] [/baud {9600 | 19200 | 38400 | 57600 | 115200}] [/id <osentrylinenum>]
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
| --------- | ----------- |
| `{on | off | edit}` | Указывает значение для перенаправления служб аварийного управления, включая:<ul><li>**on.** Включает удаленный выход для указанного `<osentrylinenum>`. Также добавляет параметр/redirect к указанному <osentrylinenum> и `redirect=com<X>` параметру в раздел [boot loader]. Значение `com<X>` задается параметром **/Port** .</li><li>**автоном.** Отключает вывод на удаленный компьютер. Также удаляет параметр/Redirect для указанного <osentrylinenum> и `redirect=com<X>` параметра из раздела [boot loader].</li><li>**Редактор.** Позволяет изменять параметры портов, изменяя `redirect=com<X>` параметр в разделе [boot loader]. Значение `com<X>` задается параметром **/Port** .</li></ul> |
| `/s <computer>` | Указывает имя или IP-адрес удаленного компьютера (не используйте обратную косую черту). По умолчанию используется локальный компьютер. |
| `/u <domain>\<user>`  | Выполняет команду с разрешениями учетной записи пользователя, указанного `<user>` в `<domain>\<user>`параметре или. По умолчанию заданы разрешения текущего вошедшего в систему пользователя на компьютере, выполняющем команду. |
| `/p <password>` | Указывает пароль учетной записи пользователя, указанной в параметре **/u** . |
| `/port {COM1 | COM2 | COM3 | COM4 | BIOSSET}` |  Указывает COM-порт, используемый для перенаправления. Параметр BIOS направляет службы аварийного управления на получение параметров BIOS, чтобы определить, какой порт следует использовать для перенаправления. Не используйте этот параметр, если отключено удаленное администрирование выходных данных. |
| `/baud {9600 | 19200 | 38400 | 57600 | 115200}` | Указывает скорость передачи, используемую для перенаправления. Не используйте этот параметр, если отключено удаленное администрирование выходных данных. |
| `/id <osentrylinenum>` | Указывает номер строки записи операционной системы, к которой добавляется параметр службы аварийного управления в разделе [Operating Systems] файла Boot. ini. Первая строка после заголовка раздела [Operating Systems] — 1. Этот параметр является обязательным, если значение служб аварийного управления установлено в **On** или **Off**. |
| /? | Отображение справки в командной строке. |

## <a name="examples"></a>Примеры

Чтобы использовать команду **bootcfg/EMS** :

```
bootcfg /ems on /port com1 /baud 19200 /id 2
bootcfg /ems on /port biosset /id 3
bootcfg /s srvmain /ems off /id 2
bootcfg /ems edit /port com2 /baud 115200
bootcfg /s srvmain /u maindom\hiropln /p p@ssW23 /ems off /id 2
```

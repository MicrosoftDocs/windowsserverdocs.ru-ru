# <a name="create-partition-logical"></a>Создать логический раздел

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает логический раздел в существующий дополнительный раздел. Эту команду можно использовать только на основной загрузочной записью \(MBR\) дисков.  
  
  
  
## <a name="syntax"></a>Синтаксис  
  
```  
create partition logical [size=<n>] [offset=<n>] [align=<n>] [noerr]  
```  
  
## <a name="parameters"></a>Параметры  
  
|  Параметр  |                                                                                                                                                                                                                       Описание                                                                                                                                                                                                                        |
|-------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  Размер\=<n>  |                                                                                                              Указывает размер логической секции в мегабайтах \(МБ\), который должен быть меньше, чем дополнительный раздел. Если размер не указан, раздел занимает на диске имеется свободное пространство.                                                                                                               |
| Смещение\=<n> | Задает смещение в килобайтах \(КБ\), в которой создается секция. Смещение выполняет округление до полного заполнения используется размер цилиндра. Если смещение не указано, раздел помещается в первое пространство на диске, достаточное для их хранения. Раздел имеет по крайней мере, если в байтах, числа, указанного по **размер\=<n>** . Если указать размер логической секции, должен быть меньше, чем дополнительный раздел. |
| Выравнивание\=<n>  |                                                                                     Выравнивает все экстенты тома или раздела до ближайшей границы выравнивания. Обычно используется с оборудованием логический номер устройства RAID \(LUN\) массивы для повышения производительности.  <n> Число килобайт \(КБ\) с самого начала диска до ближайшей границы выравнивания.                                                                                      |
|    успешного    |                                                                                                                           для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.                                                                                                                           |
  
## <a name="remarks"></a>Примечания  
  
-   Если **размер** и **смещение** параметры не указаны, в наибольшее область диска, доступных в дополнительном разделе создается логический раздел.  
  
-   После создания раздела фокус автоматически переносится на новый логический раздел.  
  
-   Для успешного выполнения этой операции необходимо выбрать базовый диск MBR. Используйте **выберите диск** команду, чтобы выбрать диск и перетянуть внимание к нему.  
  
## <a name="BKMK_examples"></a>Примеры  
Чтобы создать логический раздел 1000 МБ размер дополнительного раздела выбранного диска, введите:  
  
```  
create partition logical size=1000  
```

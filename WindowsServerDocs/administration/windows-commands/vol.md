# <a name="vol"></a>vol



Отображает метку тома диска и серийного номера, если они существуют.  При использовании без параметров, **vol** отображает сведения для текущего диска.

## <a name="syntax"></a>Синтаксис

```
vol [<Drive>:]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Диск >:|Диск, содержащую данный диск, для которого будет отображаться метка тома и серийный номер.|
|/?|Отображение справки в командной строке.|

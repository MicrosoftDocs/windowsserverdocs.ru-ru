# <a name="vol"></a>vol



Отображает метку тома диска и серийный номер, если они существуют.  Если используется без параметров, **Vol** отображает сведения о текущем диске.

## <a name="syntax"></a>Синтаксис

```
vol [<Drive>:]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|> \<диска:|Указывает диск, содержащий диск, для которого требуется отобразить метку тома и серийный номер.|
|/?|Отображает справку в командной строке.|

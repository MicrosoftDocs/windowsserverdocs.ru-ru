# <a name="vol"></a>vol

Отображает метку тома диска и серийный номер, если они существуют.  Если используется без параметров, **Vol** отображает сведения о текущем диске.

## <a name="syntax"></a>Синтаксис

```
vol [<drive>:]
```

### <a name="parameters"></a>Параметры

| Параметр | Описание |
|--|--|
| `<drive>:` | Указывает диск, содержащий диск, для которого требуется отобразить метку тома и серийный номер. |
| /? | Отображение справки в командной строке. |

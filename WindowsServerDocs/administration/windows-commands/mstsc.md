# <a name="mstsc"></a>mstsc

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Создает подключения к серверам узла сеансов удаленных рабочих столов (rd узла сеансов) или других удаленных компьютерах, изменяет существующий файл конфигурации удаленного рабочего стола (RDP-) и перенос устаревших файлов подключений, которые были созданы с помощью клиентских подключений Чтобы новые файлы подключения RDP.
Примеры для использования этой команды, см. в разделе [примеры](#BKMK_examples).
> [!NOTE]
> В Windows Server 2008 R2 службы терминалов были переименованы на службы удаленных рабочих столов. Чтобы найти новые возможности в последней версии, см. в разделе [какие возможности служб удаленных рабочих столов в Windows Server 2012](https://technet.microsoft.com/library/hh831527) в технической библиотеке Windows Server.

## <a name="syntax"></a>Синтаксис
```
mstsc.exe [<Connection File>] [/v:<Server>[:<Port>]] [/admin] [/f] [/w:<Width> /h:<Height>] [/public] [/span]
mstsc.exe /edit <Connection File>
mstsc.exe /migrate
```

## <a name="parameters"></a>Параметры
|Параметр|Описание|
|-------|--------|
|<Connection File>|Указывает имя для подключения RDP-файл.|
|/ v: < сервера [:<Port>]|Указывает удаленный компьютер и, при необходимости, номер порта, к которому необходимо подключиться.|
|/ Admin.|Подключение к сеансу для администрирования сервера.|
|/f|запускает подключение удаленного рабочего стола в полноэкранном режиме.|
|/w:<Width>|Указывает ширину окна удаленного рабочего стола.|
|/h:<Height>|Указывает высоту окна удаленного рабочего стола.|
|/ public|Запускает удаленного рабочего стола в общедоступном режиме. В общедоступном режиме пароли и растровые изображения не кэшируются.|
|/ span|Совпадает с удаленного рабочего стола ширину и высоту с локальным виртуальным рабочим столом, охватывающее несколько мониторов, при необходимости.|
|/edit <Connection File>|Открывает указанный файл .rdp для редактирования.|
|/ Перенос|Перенос устаревших файлов подключений, созданных диспетчером подключения клиента, чтобы новые файлы подключения RDP.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания
-   Default.RDP сохраняется для каждого пользователя как скрытый файл в папке документов пользователя. Пользователь, созданный RDP-файлы сохраняются в папке документов пользователя по умолчанию, но можно сохранить в любом расположении.
-   Охватывающая мониторов, мониторы должны использовать одинаковое разрешение и должны быть выровнены горизонтально (то есть, бок о бок). В настоящее время не поддерживается для охвата нескольких мониторов по вертикали на клиентской системе.

## <a name="BKMK_examples"></a>Примеры
-   Чтобы подключиться к сеансу в полноэкранном режиме, введите следующую команду:
    ```
    mstsc /f
    ```
-   Чтобы открыть файл имя_файла.RDP для редактирования, введите следующую команду:
    ```
    mstsc /edit filename.rdp
    ```

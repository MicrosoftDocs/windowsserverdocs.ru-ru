# <a name="rundll32-printuidllprintuientry"></a>rundll32 PRINTUI. dll, Принтуиентри

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Автоматизирует многие задачи по настройке принтера. PrintUI. dll — это исполняемый файл, содержащий функции, используемые в диалоговых окнах настройки принтера. Эти функции также могут вызываться из скрипта или из пакетного файла командной строки, а также могут выполняться в интерактивном режиме в командной строке. Примеры использования этой команды см. в разделе [примеры](#BKMK_Examples).  
## <a name="syntax"></a>Синтаксис  
```  
rundll32 printui.dll PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
Можно также использовать следующие альтернативные синтаксисы, хотя в примерах в этом разделе используется предыдущий синтаксис:  
```  
rundll32 printui.dll,PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
```  
rundll32 printui PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
```  
rundll32 printui,PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
### <a name="parameters"></a>Параметры  
Существует два типа параметров: основные параметры и параметры изменения. Базовые параметры укажите функцию, которую должна выполнить команда. В данной командной строке могут присутствовать только один из этих параметров. Затем можно изменить базовый параметр, используя один или несколько параметров изменения, если они применимы к базовому параметру (не все параметры изменения поддерживаются всеми базовыми параметрами).

## <a name="remarks"></a>Примечания  
- Ключевое слово **принтуиентри** чувствительно к регистру, и необходимо ввести синтаксис для этой команды с учетом точной капитализации, приведенной в примерах этого раздела.  
- Синтаксис некоторых распространенных задач см. в разделе [примеры](#BKMK_Examples) в этом документе. Для получения дополнительных примеров введите в командной строке: **rundll32 PRINTUI. dll, принтуиентри/?**  
  ## <a name="examples"></a><a name=BKMK_Examples></a>Примеров  
  Чтобы добавить новый удаленный принтер, Printer1, для компьютера КЛИЕНТ1, который отображается для учетной записи пользователя, в которой выполняется эта команда, введите:  
  ```  
  rundll32 printui.dll PrintUIEntry /in /n\\client1\printer1  
  ```  
  Чтобы добавить принтер с помощью мастера добавления принтера и использовать INF-файл Инффиле. INF, расположенный на диске c: в Инфпас, введите:  
  ```  
  rundll32 printui.dll PrintUIEntry /ii /f c:\Infpath\InfFile.inf  
  ```  
  Чтобы удалить существующий принтер Printer1, на компьютере КЛИЕНТ1 введите:  
  ```  
  rundll32 printui.dll PrintUIEntry /dn /n\\client1\printer1  
  ```  
  Чтобы добавить подключение к принтеру на компьютере, Printer2 для всех пользователей компьютера, Клиент2 Type (подключение будет применяться при входе пользователя в систему):  
  ```  
  rundll32 printui.dll PrintUIEntry /ga /n\\client2\printer2  
  ```  
  Чтобы удалить подключение к принтеру на компьютере, Printer2 для всех пользователей компьютера, Клиент2, введите (подключение будет удалено при входе пользователя в систему):  
  ```  
  rundll32 printui.dll PrintUIEntry /gd /n\\client2\printer2  
  ```  
  Чтобы просмотреть свойства сервера печати, printServer1, введите:  
  ```  
  rundll32 printui.dll PrintUIEntry /s /t1 /c\\printserver1  
  ```  
  Чтобы просмотреть свойства принтера printer3, введите:  
  ```  
  rundll32 printui.dll PrintUIEntry /p /n\\printer3  
  ```

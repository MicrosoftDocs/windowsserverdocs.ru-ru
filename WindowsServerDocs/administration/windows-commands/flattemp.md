# <a name="flattemp"></a>flattemp

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Включает или отключает плоские временные папки.
Примеры для использования этой команды, см. в разделе [примеры](#BKMK_examples).

> [!NOTE]
> В Windows Server 2008 R2 службы терминалов были переименованы на службы удаленных рабочих столов. Чтобы найти новые возможности в последней версии, см. в разделе [какие возможности служб удаленных рабочих столов в Windows Server 2012](https://technet.microsoft.com/library/hh831527) в технической библиотеке Windows Server.

## <a name="syntax"></a>Синтаксис
```
flattemp {/query | /enable | /disable}
```

## <a name="parameters"></a>Параметры
|Параметр|Описание|
|-------|--------|
|/ Query|Запрашивает текущий параметр.|
|/ Enable|Включает плоские временные папки. Пользователи будут совместно использовать временную папку, если временной папке находится в домашнюю папку пользователя s.|
|/ Disable|Отключает плоские временные папки. Каждый временной папке пользователя s будет находиться в отдельной папке (определяется пользователем s идентификатор сеанса).|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания
-   **Flattemp** команда доступна только в случае, если вы установили службу роли сервера терминалов на компьютер под управлением Windows Server 2008 или службу роли узла сеансов удаленных рабочих столов на компьютере под управлением Windows Server 2008 R2.
-   Необходимо иметь учетные данные администратора для запуска **flattemp**.
-   Если каждый пользователь имеет отдельную временную папку, используйте **/Enable flattemp** для включения плоские временные папки.
-   Метод по умолчанию для создания временных папок для нескольких пользователей (обычно указываемых переменными среды TEMP и TMP) является создание вложенных папок в **\Temp** папки, используя данный идентификатор в качестве имени подпапки. Например если переменная среды TEMP указывает C:\Temp, папке назначен данный идентификатор пользователя 4 — C:\Temp\4. С помощью **flattemp**, можно указывать непосредственно в папке \Temp и запретить использование подпапок. Это полезно в том случае, если нужно, чтобы временные папки пользователей, которые должны содержаться в домашние папки на локальном диске сервера узла сеансов удаленных рабочих столов или на общий сетевой диск. Следует использовать **/Enable flattemp** команды только в том случае, если каждый пользователь имеет отдельную временную папку.
-   Могут возникнуть ошибки приложения, если временную папку пользователя на сетевом диске. Это происходит, когда общий сетевой диск становится временно недоступным в сети. Так как временные файлы приложения являются либо недоступен, либо из синхронизации, он будет считать, что диск был остановлен. Перемещение во временную папку на сетевой диск не рекомендуется. Значение по умолчанию — сохранить временные папки на локальном жестком диске. При возникновении непредвиденного поведения или ошибки повреждения диска в определенных приложений, стабилизация сети или переместить во временных каталогах обратно на локальный жесткий диск.
-   Если отключить использование отдельных временные папки для каждого сеанса, **flattemp** параметры игнорируются. Этот параметр задается в средстве настройки служб удаленных рабочих столов.

## <a name="BKMK_examples"></a>Примеры
-   Чтобы отобразить текущее значение параметра плоская временные папки, введите следующую команду:
    ```
    flattemp /query
    ```
-   Чтобы включить плоских временных папок, введите следующую команду:
    ```
    flattemp /enable
    ```
-   Чтобы отключить плоских временных папок, введите следующую команду:
    ```
    flattemp /disable
    ```

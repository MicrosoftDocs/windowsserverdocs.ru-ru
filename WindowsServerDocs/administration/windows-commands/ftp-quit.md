# <a name="ftp-quit"></a>ftp quit

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Завершает сеанс FTP с удаленным компьютером, а затем завершает работу.

> [!NOTE]
> Эта команда совпадает с [командой FTP Bye](ftp-bye.md).

## <a name="syntax"></a>Синтаксис

```
quit
```

### <a name="examples"></a>Примеры

Чтобы завершить сеанс FTP с удаленным компьютером и вернуться в командную строку операционной системы, введите следующую команду:

```
quit
```

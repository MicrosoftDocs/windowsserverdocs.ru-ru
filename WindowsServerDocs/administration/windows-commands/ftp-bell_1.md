# <a name="ftp-bell_1"></a>FTP: bell_1

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Переключает звуковой сигнал на выполнение после завершения каждой команды по переносу файлов.   
## <a name="syntax"></a>Синтаксис  
```  
bell  
```  
#### <a name="parameters"></a>Параметры  
нет  
## <a name="remarks"></a>Примечания  
По умолчанию Звонок отключен.  
## <a name="examples"></a><a name=BKMK_Examples></a>Примеров  
Переключите звуковой сигнал после завершения каждой команды по переносу файлов.  
```  
bell  
```

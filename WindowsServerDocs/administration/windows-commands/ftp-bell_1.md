# <a name="ftp-bell1"></a>ftp: bell_1

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Переключает звуковой сигнал возникает после завершения каждой команды передачи файлов.   
## <a name="syntax"></a>Синтаксис  
```  
bell  
```  
### <a name="parameters"></a>Параметры  
none  
## <a name="remarks"></a>Примечания  
По умолчанию сигнал отключен.  
## <a name="BKMK_Examples"></a>Примеры  
Переключите звуковой сигнал возникает после завершения каждой команды передачи файлов.  
```  
bell  
```

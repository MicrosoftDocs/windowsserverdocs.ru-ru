# <a name="attributes"></a>атрибуты



Отображает, задает или удаляет атрибуты диска или тома.

## <a name="syntax"></a>Синтаксис

```
Attributes disk
Attributes volume
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|[атрибуты диска](attributes-disk.md)|Отображает, задает или удаляет атрибуты диска.|
|[атрибуты тома](attributes-volume.md)|Отображает, задает или удаляет атрибуты тома.|

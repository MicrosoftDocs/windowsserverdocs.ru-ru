# <a name="attributes"></a>атрибуты



Отображает, устанавливает или очищает атрибуты диска или тома.

## <a name="syntax"></a>Синтаксис

```
Attributes disk
Attributes volume
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|[Диск с атрибутами](attributes-disk.md)|Отображает, устанавливает или очищает атрибуты диска.|
|[Атрибуты тома](attributes-volume.md)|Отображает, устанавливает или очищает атрибуты тома.|

# <a name="time"></a>time



Отображает или устанавливает системное время. При использовании без параметров, **время** отображает текущее системное время и предложит ввести новое значение времени.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
time [/t | [<HH>[:<MM>[:<SS>]] [am|pm]]]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<ЧЧ > [:\<мм > [:\<SS > [.\< NN >]]] [меня\|pm]|Задает новое время указано, где системное время *HH* в часах (обязательно), *мм* в минутах, и *SS* задается в секундах. *NN* может использоваться для указания сотые доли секунды. Если **меня** или **pm** не указан, **время** по умолчанию используется 24-часовом формате.|
|/t|Отображает текущее время, не запрашивая новое значение времени.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Чтобы изменить текущее время, требуются права администратора.
-   В списке необходимо разделять значения для *HH*, *мм*, и *SS* с двоеточия (:). *SS* и *NN* должны быть разделены точкой (.).
-   Допустимые *HH* значения от 0 до 24.
-   Допустимые *мм* и *SS* значения от 0 до 59.

## <a name="BKMK_examples"></a>Примеры

Если включены расширения команд, чтобы отобразить текущее системное время, введите:
```
time /t
```
Чтобы изменить текущее системное время до 17:30:00, введите одно из следующих:
```
time 17:30:00
time 5:30 pm
```
Для отображения текущего системного времени, следуют приглашение для ввода нового, введите:
```
The current time is: 17:33:31.35
Enter the new time:
```
Чтобы сохранить текущее время и вернуться к командной строке, нажмите клавишу ВВОД. Чтобы изменить текущее время, введите новое время и нажмите клавишу ВВОД.

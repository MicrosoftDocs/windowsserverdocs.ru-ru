# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers и list

Список всех узлов.

## <a name="syntax"></a>Синтаксис

```
bitsadmin /peers /list
```

## <a name="examples"></a>Примеры

Чтобы получить список всех узлов, выполните следующие действия.

```
bitsadmin /peers /list
```

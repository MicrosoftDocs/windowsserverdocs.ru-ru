# <a name="print"></a>print



Отправляет в текстовый файл на принтер.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
Print [/d:<PrinterName>] [<Drive>:][<Path>]<FileName>[ ...]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/ d:\<Имя_принтера >|Задает для задания печати принтера. Для печати на принтере локально подключенных, укажите порт на компьютере, где он подключен.</br>— Допустимые значения для параллельных портов: LPT1, LPT2 и LPT3.</br>— Допустимые значения для последовательных портов: COM1, COM2, COM3 и COM4.</br>Можно также указать сетевой принтер с помощью его имени очереди (\\\\*ServerName*\*Имя_принтера *). Если принтер не указан, задание печати отправляется LPT1 по умолчанию.|
|\<Диск >:|Указывает логическое или физическое диске, где находится файл, который требуется напечатать. Этот параметр не является обязательным, если файл, который требуется напечатать находится на текущем диске.|
|\<Путь >|Указывает расположение файла, который требуется напечатать. Этот параметр не является обязательным, если файл для печати находится в текущем каталоге.|
|\<Имя файла > [...]|Обязательный. Указывает файл, который требуется напечатать. Можно включить несколько файлов в одной команде.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Файл можно напечатать в фоновом режиме, если вы отправляете в принтера, подключенного к последовательному или параллельному порту на локальном компьютере.
-   Можно выполнять многие задачи конфигурации из командной строки с помощью **режим** команды.

    См. в разделе [режим](mode.md) Дополнительные сведения о:  
    -   Настройка принтер подключен к параллельному порту
    -   Настройке принтера, подключенных к последовательному порту
    -   Отображение состояния принтера
    -   Подготовка к принтеру для кодовых страниц

## <a name="BKMK_examples"></a>Примеры

Чтобы отправить файл Report.txt в текущем каталоге на принтер, подключенный к порту LPT2 на локальном компьютере, введите:
```
print /d:lpt2 report.txt
```
Отправить файл в каталоге c:\Accounting Report.txt Printer1 очереди печати в \\ \\тип CopyRoom сервера:
```
print /d:\\copyroom\printer1 c:\accounting\report.txt 
```

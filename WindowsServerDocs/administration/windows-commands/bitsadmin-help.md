# <a name="bitsadmin-help"></a>bitsadmin help



Отображает использование командной строки.

## <a name="syntax"></a>Синтаксис

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Примеры

В следующем примере извлекается Справка по командной строке.
```
C:\>bitsadmin /help
```

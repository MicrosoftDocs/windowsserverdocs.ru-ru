# <a name="tree"></a>tree



Графически отображает структуру каталогов, пути или на диске.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
tree [<Drive>:][<Path>] [/f] [/a]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Диск >:|Диск, содержащую данный диск, для которого будет отображаться в списке каталогов.|
|\<Путь >|Указывает каталог, для которого будет отображаться в списке каталогов.|
|/f|Отображает имена файлов в каждом каталоге.|
|/a|Указывает, что **дерева** заключается в использовании текстовых символов вместо графическими символами для отображения строк, которые связывают подкаталоги.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

Структура, отображаемого элементом **дерева** зависит от параметров, указываемые в командной строке. Если не указать диск или путь, **дерева** отображает структуру дерева, начиная с текущего каталога на текущем диске.

## <a name="BKMK_examples"></a>Примеры

Чтобы отобразить имена всех подкаталогов на текущем диске диске, введите:
```
tree \
```
Чтобы отобразить один экран за раз, файлы во всех каталогах на диске C, введите:
```
tree c:\ /f | more 
```
Для вывода списка всех каталогов на диске C, введите следующую команду:
```
tree c:\ /f  prn 
```

# <a name="bootcfg"></a>bootcfg

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Настройка, запрос или изменение параметров файла Boot.ini.  
## <a name="syntax"></a>Синтаксис  
```  
bootcfg <parameter> [arguments...]  
```  
## <a name="parameters"></a>Параметры  
|Параметр|Описание|  
|-------|--------|  
|[bootcfg addsw](bootcfg-addsw.md)|Добавление параметров загрузки операционной системы для записи определенной операционной системы.|  
|[bootcfg копирования](bootcfg-copy.md)|Копирует существующий элемент, к которому можно добавить параметры командной строки.|  
|[bootcfg dbg1394](bootcfg-dbg1394.md)|Настройка отладки порта 1394 для определенной записи операционной системы.|  
|[bootcfg отладки](bootcfg-debug.md)|Добавляет или изменяет параметры отладки для определенной записи операционной системы.|  
|[по умолчанию bootcfg](bootcfg-default.md)|Указывает запись операционной системы, используемой по умолчанию.|  
|[bootcfg delete](bootcfg-delete.md)|Удаляет запись операционной системы в **[операционные системы]** раздел файла Boot.ini.|  
|[bootcfg ems](bootcfg-ems.md)|Позволяет пользователю добавить или изменить параметры для перенаправления консоли служб аварийного управления к удаленному компьютеру.|  
|[bootcfg запроса](bootcfg-query.md)|Запрос и отображение [загрузчик] и **[операционные системы]** разделе записей из файла Boot.ini.|  
|[необработанные bootcfg](bootcfg-raw.md)|Добавление параметров загрузки операционной системы, указанное в виде строки для записи операционной системы в **[операционные системы]** раздел файла Boot.ini.|  
|[bootcfg rmsw](bootcfg-rmsw.md)|Удаляет параметры загрузки операционной системы для записи определенной операционной системы.|  
|[время ожидания bootcfg](bootcfg-timeout.md)|изменяет значение времени ожидания операционной системы.|

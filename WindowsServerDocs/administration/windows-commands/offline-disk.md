# <a name="offline-disk"></a>автономный диск



Принимает диска в оперативном режиме с фокусом в автономный режим.

> [!IMPORTANT]
> Эта команда DiskPart не доступна в любом выпуске Windows Vista.

## <a name="syntax"></a>Синтаксис

```
offline disk [noerr]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|успешного|для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.|

## <a name="remarks"></a>Примечания

-   Эта команда работает на дисках, которые находятся в режиме в сети SAN. Он изменяет свой режим сети хранения данных в автономный режим.
-   Если переводится в автономный режим динамического диска в группе диск, диск приобретает значение **отсутствует** и группе показывает диска, который находится в автономном режиме. Недопустимая группа перемещается недостающий диск. Если динамический диск последнего диска в группе, то состояние диска изменится на **автономных**, и будет удален пустую группу.
-   Диск должен быть выбран **автономный диск** для успешного выполнения команды. Используйте **выберите диск** команду, чтобы выбрать диск и перетянуть внимание к нему.

## <a name="BKMK_examples"></a>Примеры

Чтобы отключить диск, имеющий фокус, введите следующую команду:
```
offline disk
```

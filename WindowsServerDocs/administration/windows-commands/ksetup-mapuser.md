# <a name="ksetupmapuser"></a>ksetup:mapuser



Сопоставляет имя участника Kerberos учетной записи. Примеры использования этой команды см. в разделе [примеры](#BKMK_Examples).

## <a name="syntax"></a>Синтаксис

```
ksetup /mapuser <Principal> <Account>
```

### <a name="parameters"></a>Параметры

|  Параметр   |                                                   Описание                                                   |
|--------------|-----------------------------------------------------------------------------------------------------------------|
| \<Основной > |              Полное доменное имя любому участнику; например mike@corp.CONTOSO.COM.              |
|  \<Учетная запись >  | Любой учетной записи или имя группы безопасности, существует на этом компьютере, например гостя, пользователей домена или администратора. |

## <a name="remarks"></a>Примечания

Учетную запись можно специально идентифицировать, скажем гости домена. Или можно использовать подстановочный знак (*) для включения всех учетных записей.

Если имя учетной записи указан, сопоставление удаляется для указанного участника.

Компьютер будет только проверка подлинности участников заданной области, если они представляют действующие билеты Kerberos.

Используйте **ksetup** без параметров и аргументов, чтобы просмотреть текущий сопоставлены параметры и область по умолчанию.

Каждый раз, когда для внешнего центра распространения ключей (KDC) и конфигурации области вносятся изменения, требуется перезагрузка компьютера, где был изменен параметр.

## <a name="BKMK_Examples"></a>Примеры

Сопоставьте учетной записи Майка Данселио в пределах области Kerberos CONTOSO гостевая учетная запись на этом компьютере, предоставив ему все привилегии членом встроенной учетной записи гостя не проходят проверку подлинности на этом компьютере:
```
ksetup /mapuser mike@corp.CONTOSO.COM guest
```
Удалите сопоставление учетной записи Майка Данселио, чтобы учетная запись гостя на этом компьютере, чтобы не дать ему выполнить проверку подлинности к этому компьютеру, используя свои учетные данные от компании CONTOSO:
```
ksetup /mapuser mike@corp.CONTOSO.COM 
```
Сопоставление учетной записи Майка Данселио в пределах области CONTOSO Kerberos любой существующей учетной записи на этом компьютере. (если только для обычных пользователей и гостевые учетные записи активны на этом компьютере, привилегии Майка устанавливается в те):
```
ksetup /mapuser mike@corp.CONTOSO.COM *
```
Сопоставьте все учетные записи в области CONTOSO Kerberos любой существующей учетной записи с таким же именем на этом компьютере:
```
ksetup /mapuser * *
```

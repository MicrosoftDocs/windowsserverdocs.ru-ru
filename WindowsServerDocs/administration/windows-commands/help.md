# <a name="help"></a>справка



Online сведения о системе команд (то есть не сетевых команд). При использовании без параметров, **помочь** список и краткое описание всех системных команд.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
help [<Command>] 
[<Command>] /?
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Команда >|Задает имя команды, которая вам нужны сведения о.|

## <a name="BKMK_examples"></a>Примеры

Чтобы просмотреть сведения о **robocopy** команды, введите одно из следующих:
```
help robocopy
robocopy /? 
```

# <a name="assoc"></a>assoc



Отображение или изменение сопоставления типов файлов. При использовании без параметров, **assoc** отображает список всех текущее имя ассоциации расширений файлов.

> [!NOTE]
> Эта команда поддерживается только в пределах CMD. EXE-файла и не доступен из PowerShell.
>

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
assoc [<.ext>[=[<FileType>]]]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|<.ext>|Указывает расширение имени файла.|
|\<Тип файла >|Указывает тип файла должен быть сопоставлен указанного расширения имени файла.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Чтобы удалить сопоставление типов файлов для расширения имени файла, добавьте пробел после знака равенства, нажав клавишу ПРОБЕЛ.
-   Чтобы просмотреть текущие типы файлов, для которых определены команды Открыть, используйте **ftype** команды.
-   Для перенаправления выходных данных **assoc** в текстовый файл, используйте **>** оператор перенаправления.

## <a name="BKMK_examples"></a>Примеры

Чтобы просмотреть текущее сопоставление типов файлов для расширению имени файла .txt, введите:
```
assoc .txt
```
Чтобы удалить сопоставление типов файлов для .bak расширение имени файла, введите следующую команду:
```
assoc .bak= 
```

> [!NOTE]
> Не забудьте добавить пробел после знака равенства.

Чтобы просмотреть выходные данные **assoc** один экран за раз, тип:
```
assoc | more
```
Для отправки выходных данных **assoc** чтобы assoc.txt файл, введите:
```
assoc>assoc.txt
```

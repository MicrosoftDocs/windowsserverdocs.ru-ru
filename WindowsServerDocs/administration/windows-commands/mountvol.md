# <a name="mountvol"></a>mountvol



Создает, удаляет и перечисляет точек подключения тома.

Примеры использования этой команды см. в [примеры](#BKMK_examples).

## <a name="syntax"></a>Синтаксис

```
mountvol [<Drive>:]<Path VolumeName>
mountvol [<Drive>:]<Path> /d
mountvol [<Drive>:]<Path> /l
mountvol [<Drive>:]<Path> /p
mountvol /r
mountvol [/n | /e]
mountvol <Drive>: /s
```

## <a name="parameters"></a>Параметры

|     Параметр     |                                                                                                                           Описание                                                                                                                            |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [\<Диска >:]<Path> |                                                                                             Указывает существующий каталог NTFS, где находится точка подключения.                                                                                             |
|   \<Имя тома >   |                     Указывает имя тома, который является целевым объектом точки подключения. Имя тома использует следующий синтаксис, где *GUID* — это глобально уникальный идентификатор:</br>`\\\\?\Volume\{GUID}\`</br>Необходимы скобки {}.                      |
|        /d         |                                                                                                    Удаляет точку подключения из указанной папки.                                                                                                     |
|        /l         |                                                                                                     Выводит список имен подключенный том для указанной папки.                                                                                                      |
|        /p         | Удаляет точку подключения из указанного каталога, отключает основной том и принимает базовый том в автономный режим, исключая возможность его подключения. Если другие процессы используют том, **mountvol** закрывает все открытые дескрипторы перед отключением тома. |
|        /r         |             Удаляет каталоги точек подключения томов и параметры реестра для томов, которые больше не находятся в системе, предотвращая их автоматическое подключение и учитывая их бывший тома точек подключения для присутствовавших при добавлении в систему.              |
|        /n         |                                                                      Отключает автоматическое подключение новых базовых томов. Новые тома не подключаются автоматически при добавлении в систему.                                                                       |
|        /e         |                                                                                                       Заново включает автоматическое подключение новых базовых томов.                                                                                                        |
|        /s         |                                                                                Подключает системного раздела EFI на указанном диске. Доступно при использовании только компьютеры, на базе Itanium.                                                                                |
|        /?         |                                                                                                               Отображение справки в командной строке.                                                                                                               |

## <a name="remarks"></a>Примечания

-   **Mountvol** позволяет связывать тома без необходимости букву диска.
-   Тома, которые отключены с помощью **/p** указаны в списке томов, как «Создается не ПОДКЛЮЧЕННЫЙ UNTIL A точка подключения тома». Если том имеет более одного подключения точки, используйте **/d** для удаления дополнительных точек подключения перед использованием **/p**. Можно сделать базовый том подключить снова назначив точек подключения тома.
-   Если необходимо расширить дисковое пространство без переформатирования или замены жесткого диска, можно добавить путь подключения на другой том. Преимущества использования одного тома с несколькими путями подключения является, что доступны все локальные тома с помощью одну букву диска (например, `C:`). Необходимо помнить, какой том соответствует какая буква диска, несмотря на то, что вы все равно можете присоединить локальные тома и назначить их буквы дисков.

## <a name="BKMK_examples"></a>Примеры

Чтобы создать точку подключения, введите следующую команду:
```
mountvol \sysmount \\?\Volume\{2eca078d-5cbc-43d3-aff8-7e8511f60d0e}\
```

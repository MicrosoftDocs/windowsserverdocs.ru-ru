# <a name="perfmon"></a>perfmon

Запустите монитор надежности и производительности Windows в определенном автономном режиме.

## <a name="syntax"></a>Синтаксис

```
perfmon </res|report|rel|sys>
```

### <a name="parameters"></a>Параметры

| Параметр | Описание: |
|--|--|
| /RES | Запускает представление ресурсов. |
| /Report | Запускает набор сборщиков данных диагностики системы и отображает отчет о результатах. |
| /рел | Запускает монитор надежности. |
| /sys | Запускает системный монитор. |

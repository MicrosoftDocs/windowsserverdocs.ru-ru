# <a name="auditpol-restore"></a>Восстановление средства Auditpol

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Восстанавливает параметры политики аудита системы, параметры политики аудита на пользователя для всех пользователей и все параметры аудита из файла, который согласуется с форматом файла значений с разделителями запятыми (CSV), используемые / Backup синтаксически параметр.

## <a name="syntax"></a>Синтаксис
```
auditpol /restore /file:<filename>
```
## <a name="parameters"></a>Параметры
|Параметр|Описание|
|-------|--------|
|/ File|Указывает файл, из которой нужно восстановить политику аудита. Файл должен быть создан с помощью / Backup параметр или должны соответствовать синтаксически формат CSV-файла, используемые / Backup параметр.|
|/?|Отображение справки в командной строке.|
## <a name="remarks"></a>Примечания
для операции восстановления для каждого пользователя политики и системную политику необходимо написать или набор разрешений полного доступа для этого объекта в дескрипторе безопасности. Можно также выполнить операции восстановления, исходя из **Управление аудитом и журналом безопасности** право пользователя (SeSecurityPrivilege). Права доступа SeSecurityPrivilege полезно при восстановлении дескриптор безопасности в случае непреднамеренного ошибки или вредоносной атаки.
## <a name="BKMK_examples"></a>Примеры
Чтобы восстановить параметры политики аудита системы, параметры политики аудита на пользователя для всех пользователей и все параметры аудита из файла с именем auditpolicy.csv, который был создан с помощью / Backup команды, введите:
```
auditpol /restore /file:c:\auditpolicy.csv
```

# <a name="nbtstat"></a>nbtstat

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Отображает NetBIOS через TCP/IP (NetBT) статистики протокола, таблицы имя NetBIOS для локального компьютера и удаленный компьютеры и NetBIOS имя кэша. **nbtstat** позволяет обновить кэш имен NetBIOS и имен, зарегистрированных с имя службы Windows Internet (WINS). При использовании без параметров, **nbtstat** отображает справку. 

## <a name="syntax"></a>Синтаксис

```
nbtstat [/a <remoteName>] [/A <IPaddress>] [/c] [/n] [/r] [/R] [/RR] [/s] [/S] [<Interval>]
```

### <a name="parameters"></a>Параметры

|    Параметр    |                                                                                                                         Описание                                                                                                                         |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| /a <remoteName> |    Отображает таблицу имен NetBIOS удаленного компьютера, где *Удаленноеимя* — имя компьютера NetBIOS удаленного компьютера. Таблицу имен NetBIOS — список имен NetBIOS, соответствующее NetBIOS приложений, работающих на этом компьютере.     |
| /A <IPaddress>  |                                                           Отображает таблицу имен NetBIOS удаленного компьютера, определяемое IP-адрес (в точечно-десятичной нотации) удаленного компьютера.                                                            |
|       /c        |                                                                        Отображает содержимое NetBIOS имя кэша, таблицу имен NetBIOS и их разрешенных IP-адресов.                                                                         |
|       /n        |                                            Отображает таблицу имен NetBIOS локального компьютера. Состояние **зарегистрирован** указывает, что имя зарегистрировано с помощью широковещательной рассылки или WINS-сервер.                                             |
|       /r        |      Отображение статистики разрешения имен NetBIOS. На компьютере под управлением Windows XP или Windows Server 2003, который настроен для использования WINS, этот параметр Возвращает количество имен, которые были устранены, и зарегистрированных широковещательной рассылки или WINS.       |
|       /R        |                                                                      Удаляет содержимое кэша имен NetBIOS и затем перезагружает # предварительно tagged записей из **Lmhosts** файл.                                                                      |
|       /RR       |                                                                           Освобождает и затем обновляет имена NetBIOS для локального компьютера, который регистрируется с помощью WINS-серверов.                                                                            |
|       /s        |                                                                          Отображает клиентских и серверных сеансов NetBIOS, попытка преобразовать имя IP-адрес назначения.                                                                           |
|       /S        |                                                                          Отображает NetBIOS сеансов клиента и сервера, список удаленных компьютеров по конечному IP-адресу.                                                                          |
|   <Interval>    | Отображение выбранной статистики, приостановка указанного количества секунд в *интервал* вывод. Нажмите клавиши CTRL + C, чтобы остановить обновление статистики. Если этот параметр опущен, **nbtstat** выводит сведения о текущей конфигурации только один раз. |
|       /?        |                                                                                                            Отображение справки в командной строке.                                                                                                             |

## <a name="remarks"></a>Примечания

-   **nbtstat** командной строки параметры зависят от регистра.

-   В следующей таблице описаны заголовки столбцов, создаваемых **nbtstat**:

    |Направление|Описание|
    |------|--------|
    |Ввод|Число полученных байтов.|
    |Вывод|Число отправленных байтов.|
    |В/Из|Является ли подключение с компьютера (исходящий трафик) или с другого компьютера, на локальном компьютере (входящий трафик).|
    |Жизнь|Оставшееся время, которая будет жить запись имя таблицы перед данные удаляются.|
    |Локальное имя|Локальное имя NetBIOS, связанный с подключением.|
    |удаленный узел|Имя или IP-адрес, связанный с удаленным компьютером.|
    |<03>|Последний байт имени NetBIOS, преобразовать в Шестнадцатеричный вид. Каждое имя NetBIOS — 16 символов. Последний байт часто имеет специальное значение, так как имя, совпадающее может присутствовать несколько раз на одном компьютере, различаясь только последний байт. Например, < 20 > — это пространство, в коде ASCII.|
    |type|Тип имени. Имя может быть уникальное имя или имя группы.|
    |Состояние|Следует ли служба NetBIOS на удаленном компьютере работает (зарегистрированного) или имя компьютера зарегистрировал одной и той же службе (конфликт).|
    |Состояние|Состояние подключения NetBIOS.|

-   В следующей таблице описаны возможные состояния подключения NetBIOS:

    |Состояние|Описание|
    |-----|--------|
    |Подключен|Сеанс установлено.|
    |Связанные|Конечная точка подключения создан и связанные с IP-адресом.|
    |прослушивание|Эта конечная точка доступна для входящих подключений.|
    |Idle|Эта конечная точка был открыт, но не может принимать подключения.|
    |Подключение|Сеанс находится в состоянии подключения и определено сопоставление имя IP-адресов назначения.|
    |Принятие|Сеанс принят, подключение будет установлено.|
    |Повторное подключение|Повторная попытка установки подключения (не удалось подключиться с первой попытки).|
    |Исходящий|Сеанс находится в состоянии подключения, и сейчас создается подключение TCP.|
    |Входящий|Сеанс находится в состоянии подключения.|
    |Отключение|Сеанс находится в процессе отключения.|
    |Disconnected (Отключено).|Локальный компьютер отправил запрос на отключение и ожидает подтверждения от удаленной системе.|

-   Эта команда доступна только в том случае, если протокол Интернета (TCP/IP) устанавливается как компонент в свойствах сетевого адаптера в сетевых подключений.

## <a name="BKMK_Examples"></a>Примеры
Чтобы просмотреть таблицу имен NetBIOS удаленного компьютера с помощью NetBIOS-имя компьютера CORP07, введите:

```
nbtstat /a CORP07
```

Чтобы просмотреть таблицу имен NetBIOS удаленного компьютера, IP-адрес 10.0.0.99, введите:

```
nbtstat /A 10.0.0.99
```

Чтобы просмотреть таблицу имен NetBIOS локального компьютера, введите:

```
nbtstat /n
```

Чтобы просмотреть содержимое кэша имен NetBIOS локального компьютера, введите:

```
nbtstat /c
```

Чтобы очистить кэш имен NetBIOS и перезагрузить # записи tagged из локального файла Lmhosts, введите:

```
nbtstat /R
```

Чтобы освободить имена NetBIOS, зарегистрированные на сервере WINS и повторно зарегистрировать их, введите:

```
nbtstat /RR
```

Чтобы просмотреть статистику сеанса NetBIOS, IP-адрес каждые пять секунд, введите:

```
nbtstat /S 5
```

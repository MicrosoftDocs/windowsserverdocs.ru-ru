# <a name="rwinsta"></a>rwinsta

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Позволяет сбросить (Удалить) сеанс на сервере узла сеансов удаленных рабочих столов (rd узла сеансов).

> [!NOTE]
> В Windows Server 2008 R2 службы терминалов были переименованы на службы удаленных рабочих столов. Чтобы найти новые возможности в последней версии, см. в разделе [новые возможности служб удаленных рабочих столов в Windows Server 2012](https://technet.microsoft.com/library/hh831527) в технической библиотеке Windows Server.

## <a name="remarks"></a>Примечания
Эта команда совпадает со значением **сбросить настройки сеанса** команды.

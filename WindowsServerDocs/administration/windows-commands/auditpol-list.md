# <a name="auditpol-list"></a>Список Auditpol

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

списки аудита категории политик и/или подкатегорий или списки пользователей, для которых определен политику аудита пользователя.

## <a name="syntax"></a>Синтаксис
```
auditpol /list
[/user|/category|subcategory[:<categoryname>|<{guid}>|*]]
[/v] [/r]
```
## <a name="parameters"></a>Параметры
|Параметр|Описание|
|-------|--------|
|/ User|Извлекает все пользователи, для которых был определен политика аудита пользователя. При использовании с параметром /v также отображается идентификатор безопасности (SID) пользователя.|
|/ Category|Отображает имена категорий, понятый системой. При использовании с параметром /v также отображается категория глобально уникальный идентификатор (GUID).|
|/SubCategory|Отображает имена подкатегорий и связанными с ними GUID.|
|/v|Отображает GUID с категорию или подкатегорию, или при использовании с/User, отображается идентификатор безопасности каждого пользователя.|
|/r|Отображает выходные данные в виде отчета в формате с разделителями-запятыми (CSV).|
|/?|Отображение справки в командной строке.|
## <a name="remarks"></a>Примечания
все операции list для политики пользователя необходимо наличие разрешения Read на этот объект, задайте в дескрипторе безопасности. Можно также выполнять операции списка, исходя из **Управление аудитом и журналом безопасности** право пользователя (SeSecurityPrivilege). Тем не менее это право позволяет дополнительные права доступа, которые не являются необходимыми для выполнения операции вывода списка.
## <a name="BKMK_examples"></a>Примеры
Чтобы получить список всех пользователей, имеющих политику определенных аудита, введите следующую команду:
```
auditpol /list /user
```
Чтобы получить список всех пользователей, имеющих политику аудита, определенных и их соответствующий идентификатор безопасности SID, введите следующую команду:
```
auditpol /list /user /v
```
Чтобы получить список всех категорий и подкатегорий в формате отчета, введите следующую команду:
```
auditpol /list /subcategory:* /r
```
Чтобы получить список подкатегорий подробное отслеживание и доступ к службе Каталогов категорий, введите следующую команду:
```
auditpol /list /subcategory:"detailed Tracking","DS Access"
```

# <a name="date"></a>date



Отображает или задает системную дату. При использовании без параметров, **даты** отображает текущей системной даты и предложит ввести новую дату.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
date [/t | <Month-Day-Year>]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Месяц, день, год >|Задает дату указано, где *месяц* -месяц со значением (одной или двумя цифрами), *день* — это день (одной или двумя цифрами), и *год* год (две или четыре цифры).|
|/t|Отображает текущую дату, не запрашивая новую дату.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   Чтобы изменить текущую дату, требуются права администратора.
-   В списке необходимо разделять значения для *месяц*, *день*, и *год* с точки (.), дефисы (-) или косой черты помечает (/).
-   Допустимые *месяц* значения: от 1 до 12.
-   Допустимые *день* значения: от 1 до 31.
-   Допустимые *год* значения: 00 до 99 или от 1980 до 2099. При использовании двух цифр, значения 80 до 99 лет от 1980 до 1999 соответствуют.

## <a name="BKMK_examples"></a>Примеры

Если включены расширения команд, чтобы отобразить текущую системную дату, введите:
```
date /t
```
Чтобы изменить текущую системную дату на 3 августа 2007 г., можно ввести любое из следующих:
```
date 08.03.2007
date 08-03-07
date 8/3/07
```
Чтобы отобразить текущую системную дату, следуют из строки, чтобы ввести новую дату, введите:
```
The current date is: Mon 04/02/2007
Enter the new date: (mm-dd-yy)
```
Чтобы сохранить текущую дату и вернуться к командной строке, нажмите клавишу ВВОД. Чтобы изменить текущую дату, введите новую дату и нажмите клавишу ВВОД.

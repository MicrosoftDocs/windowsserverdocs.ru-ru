# <a name="wdsutil-enable-commands"></a>команды WDSUTIL Enable

> Область применения: Windows Server (половина ежегодного канала), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Включает все службы для служб развертывания Windows.

## <a name="parameters"></a>Параметры

| Параметр | Описание |
|--|--|
| [Команда WDSUTIL Enable-Server](wdsutil-enable-server.md) | Включает все службы на указанном сервере служб развертывания Windows (сервер развертывания). |
| [Команда WDSUTIL Enable-транспортсервер](wdsutil-enable-transportserver.md) | Включает все службы на указанном транспортном сервере. |

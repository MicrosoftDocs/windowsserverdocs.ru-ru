# <a name="ntbackup"></a>ntbackup

Резервное копирование и восстановление компьютера и файлов из командной строки. Эта команда была заменена [командой Wbadmin](wbadmin.md).

> [!IMPORTANT]
> Команда **Wbadmin** не может восстановить резервные копии, созданные с помощью команды **Ntbackup** . Тем не менее версия **Ntbackup** доступна для загрузки из [служебной программы Windows NT Backup-Restore](https://www.microsoft.com/download/details.aspx?id=4220). Эта загружаемая версия помогает выполнять восстановление устаревших резервных копий, но не может создавать новые резервные копии.

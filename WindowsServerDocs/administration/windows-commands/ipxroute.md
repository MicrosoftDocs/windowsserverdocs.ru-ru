# <a name="ipxroute"></a>ipxroute

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Отображение и изменение сведений о таблицах маршрутизации используется протокол IPX. При использовании без параметров, **ipxroute** параметры по умолчанию для пакетов, отправляемых на адреса неизвестно, вещания и многоадресной рассылки.   
## <a name="syntax"></a>Синтаксис  
```  
ipxroute servers [/type=X]  
ipxroute ripout <Network>  
ipxroute resolve {guid | name} {GUID | <AdapterName>}  
ipxroute board= N [def] [gbr] [mbr] [remove=xxxxxxxxxxxx]  
ipxroute config  
```  
### <a name="parameters"></a>Параметры  
|Параметр|Описание|  
|-------|--------|  
|servers[ /type=X]|Выводит таблицу для указанного сервера тип точки доступа службы (SAP).  **X** должно быть целым числом. Например **/type = 4** отображает все файловые серверы. Если вы не укажете **/type**, **серверы ipxroute** отображает всех типов серверов, указав их по имени сервера.|  
|ripout сети|Обнаруживает Если *сети* путем consulting стека IPX таблицы маршрутов и отправить запрос зарезервированного IP-адреса при необходимости.  *Сети* — номер сегмента сети IPX.|  
|Разрешить {GUID&#124; имя} {GUID&#124; AdapterName}|Разрешает имя идентификатора GUID в его понятное имя или понятное имя для его идентификатора GUID.|  
|плата = *N*|Указывает сетевой адаптер, для которого нужно задать параметры или запрашивать.|  
|DEF|Отправляет пакеты на МНОГОАДРЕСНУЮ рассылку. Если пакет передается по уникальному адресу карточки доступа носителя (MAC), не в исходной таблице маршрутизации, **ipxroute** отправляет пакет ОДНОАДРЕСНАЯ рассылка по умолчанию.|  
|GBR|Отправляет пакеты на МНОГОАДРЕСНУЮ рассылку. Если пакет посылается широковещательный адрес (FFFFFFFFFFFF), **ipxroute** отправляет пакет ОДНОАДРЕСНАЯ рассылка по умолчанию.|  
|MBR|Отправляет пакеты на МНОГОАДРЕСНУЮ рассылку. Если пакет посылается на адрес многоадресной рассылки (C000xxxxxxxx), **ipxroute** отправляет пакет ОДНОАДРЕСНАЯ рассылка по умолчанию.|  
|remove= *xxxxxxxxxxxx*|Удаляет указанный адрес узла из исходной таблицы маршрутизации.|  
|конфигурации|Отображает сведения обо всех привязок, для которых настроен протокол IPX.|  
|/?|Отображение справки в командной строке.|  
## <a name="BKMK_Examples"></a>Примеры  
Чтобы отобразить сегментами сети, к которым подключен рабочей станции, адрес узла рабочей станции и используемый тип кадра, введите следующую команду:  
```  
ipxroute config  
```

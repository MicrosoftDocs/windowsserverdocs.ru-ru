# <a name="cmstp"></a>cmstp

>Область применения. Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Устанавливает или удаляет профиль службы диспетчера подключений. Без необязательных параметров **cmstp** устанавливает профиль службы с параметрами по умолчанию, соответствующую операционную систему и разрешения пользователя. 
## <a name="syntax"></a>Синтаксис
Синтаксис 1:
```
ServiceProfileFileName .exe /q:a /c:"cmstp.exe ServiceProfileFileName .inf [/nf] [/ni] [/ns] [/s] [/su] [/u]"
```
Синтаксис 2:
```
cmstp.exe [/nf] [/ni] [/ns] [/s] [/su] [/u]  [Drive:][path]ServiceProfileFileName.inf"
```
### <a name="parameters"></a>Параметры
|Параметр|Описание|
|-------|--------|
|< ИмяФайлаПрофиляСлужбы > .exe|Задает имя, пакет установки, который содержит профиль, который вы хотите установить.<br /><br />Требуется синтаксис 1, но не является допустимым для синтаксиса 2.|
|/q:a|Указывает, что профиль должен устанавливаться без запроса пользователя. По-прежнему будет отображаться в проверочном сообщении, что установка прошла успешно.<br /><br />Требуется синтаксис 1, но не является допустимым для синтаксиса 2.|
|[Диск:] [путь] <ServiceProfileFileName>.inf|Обязательный. Указывает имя файла конфигурации, определяет, каким образом должна устанавливаться профиль.<br /><br />[Диск:] [путь] не является допустимым для синтаксиса 1.|
|/ NF|Указывает, что необходимые файлы поддержки не должно быть установлено.|
|/Ni|Указывает, что не следует создавать значка на рабочем столе. Этот параметр допустим только для компьютеров под управлением Windows 95, Windows 98, Windows NT 4.0 или Windows Millennium edition.|
|/NS|Указывает, что не следует создавать ярлык на рабочем столе. Этот параметр допустим только для компьютеров, работающих под управлением семейства Windows Server 2003, Windows 2000 или Windows XP.|
|/s|Указывает, что установлен профиль службы, или они будут удалены в автоматическом режиме (без отображения запросов для ответа от пользователя и проверочное сообщение).|
|/su|Указывает, что профиль службы должно быть установлено для одного пользователя, а не для всех пользователей. Этот параметр допустим только для компьютеров под управлением Windows Server 2003, Windows 2000 или Windows XP.|
|/AU|Указывает, что профиль службы должно быть установлено для всех пользователей. Этот параметр допустим только для компьютеров под управлением Windows Server 2003, Windows 2000 или Windows XP.|
|/u|Указывает, что нужно удалить этот профиль.|
|/?|Отображение справки в командной строке.|
## <a name="remarks"></a>Примечания
**/s** является единственным параметром, который можно использовать в сочетании с **/u**.
Синтаксис 1 — это типичный синтаксис, используемый в приложении выборочной установки. Чтобы использовать этот синтаксис, необходимо запустить **cmstp** из каталога, содержащего <ServiceProfileFileName>файл .exe.
## <a name="BKMK_Examples"></a>Примеры
Чтобы установить профиль службы художественной без поддержки файлов, введите:
```
fiction.exe /c:"cmstp.exe fiction.inf /nf"
```
Автоматическая установка профиля службы художественной для одного пользователя, введите следующую команду:
```
fiction.exe /c:"cmstp.exe fiction.inf /s /su"
```
Автоматическое удаление профиля службы фантастикой, введите следующую команду:
```
fiction.exe /c:"cmstp.exe fiction.inf /s /u"
```

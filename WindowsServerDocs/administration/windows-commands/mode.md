# <a name="mode"></a>mode



Отображает состояние системы, изменения параметров системы или перенастройки портов или устройств. При использовании без параметров, **режим** отображает все изменяемые атрибуты консоли и доступные COM-устройства.

Можно использовать **режим** для выполнения следующих задач, каждая задача использует различный синтаксис:
-   [Настройка последовательного порта](#BKMK_1)
-   [Чтобы отобразить состояние всех устройств или из одного устройства](#BKMK_2)
-   [Перенаправить выходные данные порта для последовательного порта](#BKMK_3)
-   [Чтобы выбрать, обновление или Вывод номера кодовых страниц для консоли](#BKMK_4)
-   [Чтобы изменить размер буфера экрана командной строки](#BKMK_5)
-   [Чтобы задать частоту повторений клавиатуры](#BKMK_6)

## <a name="BKMK_1"></a>Настройка последовательного порта

### <a name="syntax"></a>Синтаксис

```
mode com<M>[:] [baud=<B>] [parity=<P>] [data=<D>] [stop=<S>] [to={on|off}] [xon={on|off}] [odsr={on|off}] [octs={on|off}] [dtr={on|off|hs}] [rts={on|off|hs|tg}] [idsr={on|off}]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|COM\<M > [:]|Указывает число асинхронных Prncnfg.vbshronous COM-портом.|
|скорость передачи в бодах =\<B >|Скорость передачи в битах в секунду. В следующей таблице перечислены допустимые значения для *B* и их связанные курсы.</br>-   **11** = 110 бод</br>-   **15** = 150 бод</br>-   **30** = 300 бод</br>-   **60** = 600 бод</br>-   **12** = 1200 бод</br>-   **24** = 2400 бод</br>-   **48** = 4800 бод</br>-   **96** = 9600 бод</br>-   **19** = 19200 бод|
|четность =\<P >|Указывает, как система использует бит четности для проверки ошибок передачи. В следующей таблице перечислены допустимые значения для *P*. Значение по умолчанию — **e**. Не все компьютеры поддерживают значения **m** и **s**.</br>-   **n** = нет</br>-   **e** даже =</br>-   **o** = нечетным</br>-   **m** = пометить</br>-   **s** = space|
|данные =\<D >|Указывает количество битов в символе. Допустимые значения для **d** находятся в диапазоне от 5 до 8. Значение по умолчанию — 7. Не все компьютеры поддерживают значения 5 и 6.|
|Остановить =\<S >|Указывает число стоповых битов, определяющих конец символа: 1, 1.5 или 2. Если скорость в бит / 110, значение по умолчанию — 2. В противном случае — значение по умолчанию — 1. Не все компьютеры поддерживают значение 1,5.|
|= {on | OFF}|Указывает, является ли неограниченное время ожидания обработки или отключить. Значение по умолчанию — off.|
|XON = {on | OFF}|Разрешение или отключение протокола xon / xoff для управления потоком данных.|
|odsr={on | OFF}|Указывает, является ли выходов с использованием сигнала готовности данных (DSR) или отключить.|
|octs = {on | OFF}|Указывает, является ли выходов с использованием Clear To Send (CTS) или отключить.|
|DTR = {on | отключить | hs}|Определяет, включена или выключена или значение подтверждения готовности терминала (DTR) канала.|
|RTS = {on | отключить | hs | tg}|Указывает ли канал запроса для передачи (RTS) имеет значение on, off, сигнала.|
|idsr={on | OFF}|Указывает, является ли DSR-чувствительность канала или отключить.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_2"></a>Чтобы отобразить состояние всех устройств или из одного устройства

### <a name="syntax"></a>Синтаксис

```
mode [<Device>] [/status]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Устройства >|Указывает имя устройства, для которого будет отображаться состояние.|
|/status|Запрашивает состояние перенаправленного параллельного принтера. Вы можете сократить **/Status** параметр командной строки как **/STA**.|
|/?|Отображение справки в командной строке.|

### <a name="remarks"></a>Примечания

При использовании без параметров, **режим** отображает состояние всех устройств, которые установлены в вашей системе.

## <a name="BKMK_3"></a>Перенаправить выходные данные порта для последовательного порта

### <a name="syntax"></a>Синтаксис

```
mode lpt<N>[:]=com<M>[:]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|LPT\<N > [:]|Обязательный. Указывает параллельный порт. Допустимые значения для *N* находятся в диапазоне от 1 до 3.|
|COM\<M > [:]|Обязательный. Указывает последовательный порт. Допустимые значения для *M* находятся в диапазоне от 1 до 4.|
|/?|Отображение справки в командной строке.|

### <a name="remarks"></a>Примечания

Необходимо быть членом группы "Администраторы", для перенаправления печати.

### <a name="examples"></a>Примеры

Чтобы настроить систему таким образом, чтобы он отправляет вывод параллельного принтера на последовательный, необходимо использовать **режим** команду дважды. В первый раз, используйте **режим** для конфигурирует последовательный порт. Во второй раз, используйте **режим** перенаправляет вывод параллельного принтера на последовательный порт, указанный в первом **режим** команды.

Например если последовательный принтер работает со скоростью 4800 бит / с четности, и он подключен к порту COM1 (первое последовательное устройство на компьютере), введите:
```
mode com1 48,e,,,b
mode lpt1=com1
```
Если перенаправление параллельных печати из LPT1 к порту COM1, но затем вы решите, что вы хотите напечатать файл, используя LPT1, введите следующую команду, перед печатью файла:
```
mode lpt1
```
Эта команда запрещает перенаправление файла из LPT1 к порту COM1.

## <a name="BKMK_4"></a>Чтобы выбрать, обновление или Вывод номера кодовых страниц для консоли

### <a name="syntax"></a>Синтаксис

```
mode <Device> codepage select=<YYY>
mode <Device> codepage [/status]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|\<Устройства >|Обязательный. Указывает устройство, для которого вы хотите выбрать кодовую страницу. CON является единственным допустимым именем устройства.|
|Выберите кодовую страницу =|Обязательный. Указывает нужную кодовую страницу для использования с заданным устройством. Вы можете сократить **кодовая страница** **выберите** как **cp** **sel**.|
|\<YYY &GT;|Обязательный. Указывает номер кодовой страницы для выбора. В следующем списке показан кодовые страницы, которая поддерживается и его страны или региона или языка.</br>437: США</br>850: Многоязыковой (Latin I)</br>852: Восточная и Центральная Европа (латиница II)</br>855: Кириллица (русская)</br>857: Турецкий</br>860: Португальский</br>861: Исландский</br>863: Французский (Канада)</br>865: Nordic</br>866: Русский</br>869: Греческая современная|
|Кодовая страница|Обязательный. Отображает номера код страницы (если таковые имеются), выбранные для заданного устройства.|
|/status|Отображает номера кодовых страниц, выбранных для заданного устройства. Вы можете сократить **/Status** для **/STA**. Ли вы указать **/Status**, **кодовая страница режим** отображает номера кодовых страниц, которые выбираются для заданного устройства.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_5"></a>Чтобы изменить размер буфера экрана командной строки

### <a name="syntax"></a>Синтаксис

```
mode con[:] [cols=<C>] [lines=<N>]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|CON [:]|Обязательный. Указывает, что изменение применяется окно командной строки.|
|cols=\<C>|Указывает количество столбцов в буфере экрана командной строки.|
|строки =\<N >|Указывает число строк в буфере экрана командной строки.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_6"></a>Чтобы задать частоту повторений клавиатуры

### <a name="syntax"></a>Синтаксис

```
mode con[:] [rate=<R> delay=<D>]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|CON [:]|Обязательный. Ссылается на клавиатуре.|
|скорость =\<R >|Указывает частоту повторений символ на экране при удержании нажатой клавиши.|
|delay=\<D>|Указывает количество времени, которое должно пройти после нажмите и удерживайте нажатой клавишу перед вывода символов.|
|/?|Отображение справки в командной строке.|

### <a name="remarks"></a>Примечания

-   Частота повторений — скорость повтора символов при удержании нажатой клавиши. Частота повторений имеет два компонента, частота и задержка. Некоторые клавиатуры не распознают эту команду.
-   С помощью **скорость = *** R*

    Допустимые значения находятся в диапазоне от 1 до 32. Эти значения являются равно примерно 2 до 30 символов в секунду. Значение по умолчанию — 20 для IBM AT-совместимых клавиатур и 21 клавиатур IBM PS/2-совместимыми. Если задать частоту, необходимо также задать задержку.
-   С помощью **задержка**=*D*

    Допустимые значения для *D* : 1, 2, 3 и 4 (представляющие 0,25, 0.50, 0,75 и 1 секунду). Значение по умолчанию — 2. Если задать значение задержки, необходимо также задать скорость.

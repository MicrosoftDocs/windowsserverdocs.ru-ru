# <a name="mode"></a>mode



Отображает состояние системы, изменяет параметры системы или перестраивает порты или устройства. Если используется без параметров, в **режиме** отображаются все управляемые атрибуты консоли и доступные устройства com.

**Режим** можно использовать для выполнения следующих задач — каждая задача использует другой синтаксис:
-   [Настройка порта последовательного подключения](#BKMK_1)
-   [Отображение состояния всех устройств или одного устройства](#BKMK_2)
-   [Перенаправление вывода с параллельного порта на последовательный порт связи](#BKMK_3)
-   [Выбор, обновление или отображение номеров кодовых страниц для консоли](#BKMK_4)
-   [Изменение размера буфера экрана командной строки](#BKMK_5)
-   [Задание скорости ввода с клавиатуры](#BKMK_6)

## <a name="BKMK_2"></a>Отображение состояния всех устройств или одного устройства

### <a name="syntax"></a>Синтаксис

```
mode [<Device>] [/status]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|@no__t 0Device >|Указывает имя устройства, для которого необходимо отобразить состояние.|
|/status|Запрашивает состояние всех перенаправленных параллельных принтеров. Можно сократить параметр командной строки **/Status** в качестве параметра **/STA**.|
|/?|Отображение справки в командной строке.|

### <a name="remarks"></a>Примечания

Если используется без параметров, в поле **режим** отображается состояние всех устройств, установленных в системе.

## <a name="BKMK_3"></a>Перенаправление вывода с параллельного порта на последовательный порт связи

### <a name="syntax"></a>Синтаксис

```
mode lpt<N>[:]=com<M>[:]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|LPT @ no__t-0N > [:]|Обязательный. Указывает параллельный порт. Допустимые значения для *N* находятся в диапазоне от 1 до 3.|
|com @ no__t-0 млн > [:]|Обязательный. Указывает последовательный порт. Допустимые значения для *M* находятся в диапазоне от 1 до 4.|
|/?|Отображение справки в командной строке.|

### <a name="remarks"></a>Примечания

Для перенаправления печати необходимо быть членом группы администраторов.

### <a name="examples"></a>Примеры

Чтобы настроить систему таким образом, чтобы она отправляла параллельный вывод принтера на последовательный принтер, необходимо дважды использовать команду **mode** . В первый раз используйте **режим** для настройки последовательного порта. Во второй раз используйте **режим** для перенаправления вывода параллельного принтера на последовательный порт, указанный в команде первого **режима** .

Например, если последовательный принтер работает на скорости 4800 бод с четностью и подключен к порту COM1 (первое последовательное подключение на компьютере), введите:
```
mode com1 48,e,,,b
mode lpt1=com1
```
Если вы перенаправляли выходные данные параллельного принтера с порта LPT1 на порт COM1, но решили напечатать файл с помощью порта LPT1, перед печатью файла введите следующую команду:
```
mode lpt1
```
Эта команда предотвращает перенаправление файла с порта LPT1 на порт COM1.

## <a name="BKMK_4"></a>Выбор, обновление или отображение номеров кодовых страниц для консоли

### <a name="syntax"></a>Синтаксис

```
mode <Device> codepage select=<YYY>
mode <Device> codepage [/status]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|@no__t 0Device >|Обязательный. Указывает устройство, для которого необходимо выбрать кодовую страницу. CON — это единственное допустимое имя устройства.|
|выбор кодовой страницы =|Обязательный. Указывает кодовую страницу, используемую с указанным устройством. Можно сократить выбор **кодовой страницы** в качестве **CP** **SEL**.|
|@NO__T 0YYY >|Обязательный. Указывает номер кодовой страницы для выбора. В следующем списке показана каждая поддерживаемая кодовая страница и ее страна, регион или язык.</br>437: США</br>850: Многоязычная (латиница I)</br>852: Славянские (латиница II)</br>855: Кириллица (Русский)</br>857: Турецкий</br>860: Португальский</br>861: Исландский</br>863: Канада — французский</br>865: Скандинавская</br>866: Русский</br>869: Современный греческий|
|страница|Обязательный. Отображает номера кодовых страниц (если таковые имеются), выбранных для указанного устройства.|
|/status|Отображает номера текущей кодовой страницы, выбранной для указанного устройства. Можно сократить значение **/Status** до **/STA**. Независимо от того, задано ли значение **/Status**, **кодовая страница режима** отображает номера кодовых страниц, выбранных для указанного устройства.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_5"></a>Изменение размера буфера экрана командной строки

### <a name="syntax"></a>Синтаксис

```

```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|Con [:]|Обязательный. Указывает, что изменение применяется к окну командной строки.|
|Колс = @no__t — 0C >|Указывает число столбцов в буфере экрана командной строки.|
|строки = \<N >|Указывает число строк в буфере экрана командной строки.|
|/?|Отображение справки в командной строке.|

## <a name="BKMK_6"></a>Задание скорости ввода с клавиатуры

### <a name="syntax"></a>Синтаксис

```
mode con[:] [rate=<R> delay=<D>]
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|Con [:]|Обязательный. Ссылается на клавиатуру.|
|Rate = \<R >|Указывает скорость, с которой символ повторяется на экране при удержании клавиши.|
|Delay = \<D >|Указывает время, которое будет продолжено после нажатия и удерживания ключа перед повторным выводом символа.|
|/?|Отображение справки в командной строке.|

### <a name="remarks"></a>Примечания

- Скорость повторения — это скорость, с которой символ повторяется при удержании ключа для этого символа. Скорость повторения имеет два компонента: скорость и задержку. Некоторые клавиатуры не распознают эту команду.
- **Коэффициент использования =** <em>R</em>

  Допустимые значения находятся в диапазоне от 1 до 32. Эти значения равны приблизительно от 2 до 30 символов в секунду. Значение по умолчанию — 20 для клавиатуры, совместимых с IBM, и 21 для IBM PS/2-совместимых клавиатур. Если задать частоту, необходимо также задать задержку.
- Использование **delay**=*D*

  Допустимыми значениями для *D* являются 1, 2, 3 и 4 (представляющие 0,25, 0,50, 0,75 и 1 секунда). Значение по умолчанию — 2. Если задана задержка, необходимо также задать скорость.

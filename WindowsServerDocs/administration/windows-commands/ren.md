# <a name="ren"></a>ren

Переименовывает файлы или каталоги. Эта команда аналогична команде **Rename** .



## <a name="syntax"></a>Синтаксис

```
ren [<Drive>:][<Path>]<FileName1> <FileName2>
rename [<Drive>:][<Path>]<FileName1> <FileName2>
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|[\<Диск>:] [\<Путь>] \<Имя_файла1>|Указывает расположение и имя файла или набора файлов, которые требуется переименовать. *Имя_файла1* может содержать подстановочные знаки (**&#42;** и **?**).|
|\<Имя_файла2>|Указывает новое имя для файла. Для указания новых имен нескольких файлов можно использовать подстановочные знаки.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

- При переименовании файлов нельзя указывать новый диск или путь.
- Нельзя использовать команду **ren** для переименования файлов на дисках или для перемещения файлов в другой каталог.
- Можно использовать подстановочные знаки (**&#42;** и **?**) в любом из параметров *filename* . Символы, представленные подстановочными знаками в *имя_файла2* , будут совпадать с соответствующими символами в *имя_файла1*.
- *Имя_файла2* должно быть уникальным именем файла. Если параметр *имя_файла2* совпадает с существующим именем файла, то **ren** отображает следующее сообщение:  
  ```
  Duplicate file name or file not found
  ```

## <a name="examples"></a><a name="BKMK_examples"></a>Примеры

Чтобы изменить все расширения имени файла txt в текущем каталоге на расширения DOC, введите:
```
ren *.txt *.doc 
```
Чтобы изменить имя каталога с Chap10 на Part10, введите:
```
ren chap10 part10 
```

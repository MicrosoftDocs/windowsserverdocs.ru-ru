# <a name="pwlauncher"></a>пвлаунчер



Включает или отключает параметры запуска Windows To Go (пвлаунчер). Программа командной строки **пвлаунчер** позволяет настроить компьютер для автоматической загрузки в рабочую область Windows To Go (при условии, что она есть), не требуя ввода встроенного по или изменения параметров запуска.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
Pwlauncher {/enable | /disable}
```

### <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|разрешение|Включает параметры запуска Windows To Go, чтобы компьютер автоматически загружался с USB-устройства при его наличии.|
|/Disable|Отключает параметры запуска Windows To Go, чтобы компьютер не загружался с USB-устройства, если он не настроен вручную в микропрограмме.|
|/?|Отображает справку в командной строке.|

## <a name="remarks"></a>Примечания

Крупнейшим препятствием для пользователя, желающего использовать Windows To Go, является загрузка компьютера с помощью USB. Обычно это делается путем ввода встроенного по и выбора других параметров конфигурации до тех пор, пока компьютер не будет настроен правильно. Это не просто для большинства пользователей и чрезвычайно рискованно, поскольку встроенное по содержит параметры, которые могут привести к невозможности использования системы при неправильном использовании. Чтобы помочь устранить эту проблему, Windows 8and более поздние операционные системы включают в себя функцию с именем запуска Windows To Go, которая позволяет пользователю настроить компьютер для загрузки с USB-устройств из Windows, не вводя встроенное по, при условии, что их встроенное по поддерживает загрузку с USB. При включении системы для всегда загружаться с USB сначала следует учитывать последствия. Например, USB-устройство, включающее вредоносную программу, может быть непреднамеренно загрузилось, чтобы нарушить безопасность системы или подключить к сети несколько USB-накопителей, вызывающих конфликт загрузки. По этой причине параметры запуска Windows To Go по умолчанию отключены. Кроме того, для настройки параметров запуска Windows To Go требуются права администратора. Если включить параметры запуска Windows To Go с помощью программы командной строки пвлаунчер или приложения " **изменить параметры запуска Windows To Go** ", компьютер будет пытаться загрузиться с любого USB-устройства, вставленного на компьютер перед его запуском.

## <a name="examples"></a><a name=BKMK_examples></a>Примеров

В следующем примере показано, как можно использовать команду **пвлаунчер** для включения загрузки с USB:
```
Pwlauncher /enable
```

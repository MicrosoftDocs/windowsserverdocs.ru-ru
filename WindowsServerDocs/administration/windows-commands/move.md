# <a name="move"></a>move



Перемещает один или несколько файлов из одного каталога в другой каталог.

В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

## <a name="syntax"></a>Синтаксис

```
move [{/y | /-y}] [<Source>] [<Target>]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|/y|Запрет на выдачу запроса, чтобы убедиться, что вы хотите перезаписать существующий целевой файл.|
|/-y|Выдача запроса подтверждения, следует ли перезаписать существующий целевой файл.|
|\<Источник >|Указывает путь и имя файла или файлов для перемещения. Если вы хотите переместить или переименовать каталог, *источника* должен быть текущий путь к каталогу и имя.|
|\<Целевой объект >|Указывает путь и имя, чтобы переместить файлы. Если вы хотите переместить или переименовать каталог, *целевой* должен быть указан путь к нужному каталогу и имя.|
|/?|Отображение справки в командной строке.|

## <a name="remarks"></a>Примечания

-   **/Y** параметр командной строки, которые могут быть предустановлены в перекрываться. Его можно переопределить с помощью **/-y** в командной строке. По умолчанию используется для запроса на перезапись файлов, если не **копирования** команда запускается из в пакетном файле.
-   Перемещение зашифрованных файлов с томом, который не поддерживает результатов шифрованной файловой системы (EFS) в ошибку. Сначала расшифровать файлы и не перемещайте файлы в томе, который поддерживает файловой системы EFS.

## <a name="BKMK_examples"></a>Примеры

Чтобы переместить все файлы с расширением .xls из каталога \Data \Second_Q\Reports каталог, введите следующую команду:
```
move \data\*.xls \second_q\reports\ 
```

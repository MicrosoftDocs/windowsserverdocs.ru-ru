# <a name="nslookup-set-recurse"></a>nslookup set recurse



Указывает, что сервер имя системы доменных имен (DNS) для запроса других серверов, если он не содержит данные.

## <a name="syntax"></a>Синтаксис

```
set [no]recurse
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|**norecurse**|Останавливает имя сервера доменных имен (DNS) запрашивать другие серверы, если он не содержит данные.|
|**Recurse**|Указывает, что сервер имя системы доменных имен (DNS) для запроса других серверов, если он не содержит данные. По умолчанию используется синтаксис **recurse**.|
|{справки | ?}|Отображает краткое описание **nslookup** подкоманды.|

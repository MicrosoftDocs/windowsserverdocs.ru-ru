# <a name="reg-add"></a>Добавление reg


Добавляет новый подраздел или запись в реестр.

## <a name="syntax"></a>Синтаксис

```
reg add <KeyName> [{/v ValueName | /ve}] [/t DataType] [/s Separator] [/d Data] [/f]
```
В разделе [Примеры](#BKMK_examples) показан принцип использования этой команды.

### <a name="parameters"></a>Параметры

|      Параметр      |                                                                                                                                                                                                                                                                   Описание                                                                                                                                                                                                                                                                   |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \<KeyName<em>></em> | Задает полный путь к добавляемому подразделу или записи. Чтобы указать удаленный компьютер, укажите имя компьютера (в формате \\\\\<ComputerName >\) в составе *keyName*. Пропуск \\\\ComputerName \ приводит к тому, что по умолчанию операция выполняется на локальном компьютере. *KeyName* должен содержать допустимый корневой ключ. Допустимые корневые ключи для локального компьютера: HKLM, HKCU, HKCR, HKU и ХККК. Если указан удаленный компьютер, допустимые корневые ключи: HKLM и HKU. Если имя раздела реестра содержит пробел, заключите имя ключа в кавычки. |
|   /v \<ValueName >   |                                                                                                                                                                                                                                Задает имя записи реестра, добавляемой в указанный подраздел.                                                                                                                                                                                                                                 |
|         /ве         |                                                                                                                                                                                                                                Указывает, что запись реестра, добавляемая в реестр, имеет значение null.                                                                                                                                                                                                                                |
|     /t \<типа >      |                                                                                                                                          Указывает тип для записи реестра. *Тип* должен быть одним из следующих:</br>REG_SZ</br>REG_MULTI_SZ</br>REG_DWORD_BIG_ENDIAN</br>REG_DWORD</br>REG_BINARY</br>REG_DWORD_LITTLE_ENDIAN</br>REG_LINK</br>REG_FULL_RESOURCE_DESCRIPTOR</br>REG_EXPAND_SZ                                                                                                                                          |
|   /s \<разделитель >   |                                                                                                                                                              Задает символ, используемый для разделения нескольких экземпляров данных при указании типа данных REG_MULTI_SZ и необходимости в перечислении нескольких записей. Если значение не указано, разделителем по умолчанию является **\ 0**.                                                                                                                                                              |
|     /d \<> данных      |                                                                                                                                                                                                                                                 Задает данные для новой записи реестра.                                                                                                                                                                                                                                                  |
|         /f          |                                                                                                                                                                                                                                           Добавляет запись реестра без запроса подтверждения.                                                                                                                                                                                                                                           |
|         /?          |                                                                                                                                                                                                                                              Отображает справку по **reg Add** в командной строке.                                                                                                                                                                                                                                               |

## <a name="remarks"></a>Примечания

-   Невозможно добавить поддеревья с этой операцией. Эта версия **reg** не запрашивает подтверждение при добавлении подраздела.
-   В следующей таблице перечислены возвращаемые значения для операции **reg Add** .

| Значение | Описание |
|-------|-------------|
|   0   |   Выполнено   |
|   1   |   Отказ   |

-   Для REG_EXPAND_SZ типа ключа используйте символ крышки ( **^** ) с **%** в параметре/d.

## <a name="examples"></a><a name=BKMK_examples></a>Примеров

Чтобы добавить ключ Хклм\софтваре\мико на удаленном компьютере ABC, введите:
```
REG ADD \\ABC\HKLM\Software\MyCo
```
Чтобы добавить запись реестра в Хклм\софтваре\мико со значением с именем **Data** типа REG_BINARY и данными **fe340ead**, введите:
```
REG ADD HKLM\Software\MyCo /v Data /t REG_BINARY /d fe340ead
```
Чтобы добавить многозначную запись реестра в Хклм\софтваре\мико с именем значения **MRU** типа REG_MULTI_SZ и данными **fax\0mail\0\0**, введите:
```
REG ADD HKLM\Software\MyCo /v MRU /t REG_MULTI_SZ /d fax\0mail\0\0
```
Чтобы добавить развернутую запись реестра в Хклм\софтваре\мико с именем параметра **path** типа REG_EXPAND_SZ и данными **% systemroot%** , введите:
```
REG ADD HKLM\Software\MyCo /v Path /t REG_EXPAND_SZ /d ^%systemroot^%
```

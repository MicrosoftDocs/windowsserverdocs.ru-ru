# <a name="recover"></a>восстановить



Обновляет состояние всех дисков в группе дисков, восстановления дисков в группе недопустимый диск и повторно синхронизирует зеркальные тома и тома RAID-5, которые есть устаревшие данные.

> [!IMPORTANT]
> Эта команда DiskPart не доступна в любом выпуске Windows Vista.

## <a name="syntax"></a>Синтаксис

```
recover [noerr]
```

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|---------|-----------|
|успешного|для использования в сценариях. При обнаружении ошибки, DiskPart продолжает обрабатывать команды, как если бы ошибки не было. Без этого параметра по ошибке будет возобновлено DiskPart завершить работу с кодом ошибки.|

## <a name="remarks"></a>Примечания

-   Эта команда работает в группе дисков.
-   Эта команда применяется только к группам динамических дисков. Если эта команда используется на группу с собой базовый диск, он не вернет ошибку, но будет выполнено никаких действий.
-   Эта команда обрабатывает диски, для которых выполнена или регистрации сбоя. Он также работает на томах, для которых выполнена, произошел сбой, или в состоянии сбоя избыточности.
-   Для успешного выполнения этой команды необходимо выбрать диск, который является частью группы дисков. Используйте **выберите диск** команду, чтобы выбрать диск и перетянуть внимание к нему.

## <a name="BKMK_examples"></a>Примеры

Чтобы выполнить восстановление группы дисков, содержащую данный диск с фокусом, введите следующую команду:
```
recover
```

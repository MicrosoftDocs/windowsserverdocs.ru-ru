# <a name="rexec"></a>rexec

Выполняет указанную команду на удаленном узле. Для подключения rexec к удаленному узлу должна быть запущена служба рексекд (или управляющая программа).

> [!IMPORTANT]
> Команда **rexec** устарела и не гарантируется, что она будет поддерживаться в Windows.

# <a name="roles-role-services-and-features-included-in-windows-server---server-core"></a>Роли, службы ролей и компонентов, содержащихся в Windows Server — основных серверных компонентов

> Применимо к: Windows Server (Channel точками годовая) и Windows Server 2016

Мы обсудим обычно [what's *не* в основных серверных компонентов](server-core-removed-roles.md) — теперь мы будем используется другой подход и сообщает, что *включены* и является ли что-то *установленного по умолчанию*. Роли, службы ролей и компонентов, *в* вариант установки основных серверных компонентов Windows Server. Используйте эти сведения помогут понять, если параметр Server Core работает для вашей среды. Так как это больших списков, рассмотрите возможность поиска для конкретных ролей или компонентов, которые вы хотите — Если, что поиск не возвращает, что вы ищете, он не включен в основных серверных компонентов.

Например при поиске «Хост» — не найдете его на этой странице. Вот так как узла сеансов удаленных рабочих Столов не включен в образ основных серверных компонентов.

Помните, что можно [просмотреть](server-core-removed-roles.md) на то, что *не* включено. Это лишь другой способ откроете вещей.

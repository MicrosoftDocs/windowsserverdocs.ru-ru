# <a name="add-host-information-to-the-hgs-configuration"></a>Добавьте сведения об узле в конфигурацию HGS

>Относится к: Windows Server 2019 г., Windows Server (полугодовой канал), Windows Server 2016

Получить сведения об узлах, у администратора структуры и добавьте его к HGS. Этот шаг зависит от режима аттестации:

- [Добавьте сведения об узле для аттестацию с доверием администратора](guarded-fabric-add-host-information-for-admin-trusted-attestation.md)
- [Добавьте сведения об узле для аттестации доверенного платформенного МОДУЛЯ](guarded-fabric-add-host-information-for-tpm-trusted-attestation.md)

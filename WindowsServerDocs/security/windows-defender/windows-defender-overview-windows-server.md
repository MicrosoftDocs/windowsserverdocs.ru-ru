# <a name="windows-defender-antivirus-for-windows-server"></a>Антивирусная программа Защитника Windows для Windows Server

>Область применения: Windows Server 2016

Windows Server 2016 теперь предоставляет антивирусной программы Защитником Windows. AV Защитник Windows — защиты от вредоносных программ, немедленно и активно защищает операционную систему Windows Server 2016 от известных вредоносных программ и защиты от вредоносных программ определения через Центр обновления Windows регулярно обновляются.

В разделе [антивирусной программы Защитником Windows в Windows 10](https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-in-windows-10) библиотеки документации Дополнительные сведения.


Функциональные возможности, настройки и управления является в значительной степени одинаково для AV Защитника Windows на Windows 10 или Windows Server 2016, есть несколько ключевых отличия:

- В Windows Server 2016 [автоматические исключения](https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/configure-server-exclusions-windows-defender-antivirus) применяются в зависимости от вашей определенные роли сервера.
- В Windows Server 2016 AV Защитник Windows не отключается при использовании другой антивирусное программное обеспечение.

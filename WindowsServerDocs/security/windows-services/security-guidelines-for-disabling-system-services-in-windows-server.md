## <a name="guidance-on-disabling-system-services-on-windows-server-2016-with-desktop-experience"></a>Инструкции по отключению системных служб в Windows Server 2016 с возможностями рабочего стола

Применяется к: Windows Server 2016

Операционная система Windows включает в себя много системные службы, которые предоставляют важные функции. Разные службы будет включать политики запуска по умолчанию: некоторые запускаются по умолчанию (автоматически), некоторые при необходимости (вручную), а также по умолчанию отключены и необходимо явно включить до их выполнения. Эти настройки по умолчанию были выбраны тщательно для каждой службы для балансировки сетевой безопасности, производительности и функциональные возможности для типичных пользователей.

Однако некоторые корпоративные клиенты могут предпочесть баланс, более ориентированные на безопасность для своих ПК под управлением Windows и серверов, снижает минимально их поверхность для атаки, а поэтому может потребоваться полностью отключить все службы, которые не требуются в конкретных средах. Для этих пользователей Microsoft® предоставляет соответствующей рекомендации о том, какие службы можно безопасно отключить для этой цели.

Рекомендации — для Windows Server 2016 с возможностями рабочего стола (если не использовать в качестве замены рабочего стола для конечных пользователей). Каждой службы в системе классифицируется следующим образом:

-   **Следует отключить:** предприятия, ориентированные на безопасность будет скорее предпочитаете отключить эту службу и не функции (см. ниже приведены дополнительные сведения).
- **ОК, чтобы отключить:** эта служба предоставляет функциональные возможности для некоторых, но не все предприятий и безопасности в организациях, не используйте его можно отключить его.
- **Не отключайте:** отключение этой службы будет влияет основных функций или препятствовать правильной работе или иных ролей или компонентов. Поэтому его не следует отключать.
-  **(Не рекомендации):** последствия отключения эти службы не был произведен полностью. Таким образом не должно изменяться в конфигурации по умолчанию эти службы.


Клиенты могут настроить компьютеры под управлением Windows и серверы Отключение выбранных служб с помощью шаблонов безопасности в их групповых политик или с помощью PowerShell автоматизации. В некоторых случаях руководство включает определенные параметры групповой политики, которые напрямую, отключение функции службы в качестве альтернативы при отключении самой службы.

Корпорация Майкрософт рекомендует, что клиенты отключить следующие службы и своих запланированных задач в Windows Server 2016 с возможностями рабочего стола:

2. Xbox Live Save игры

### <a name="disabling-services-not-installed-by-default"></a>Отключение служб, не устанавливается по умолчанию

Корпорация Майкрософт не рекомендует применение политик для отключения службы, которые не устанавливаются по умолчанию.
-  Служба обычно требуется, если он установлен. Установка служба или компонент требуются права администратора. Запрещаете установку компонента, а не запуска службы.
-  Блокировка службы Microsoft Windows не остановить администратора (или без прав администратора в некоторых случаях) установку аналогичные эквивалент сторонних производителей, возможно по одному с высокий риск для безопасности.
-  Базовые или тестирования, который отключает службу Windows не по умолчанию (например, W3SVC) обеспечит некоторые аудиторы ошибочный впечатление, что технологии (например, IIS) по своей сути небезопасен и никогда не будет использоваться.
-  Если компонент (и службы) не устанавливается на компьютере, это просто добавляет ненужные массового для базовых показателей и проверки работы.

# <a name="deploy-a-software-defined-network-infrastructure"></a>Развертывание инфраструктуры программно-конфигурируемой сети

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Ниже описаны методы, которые можно использовать для развертывания инфраструктуры программно-конфигурируемые сети (SDN) корпорации Майкрософт.   
  
Эти развертывания приложений включают все технологии, необходимые для полнофункциональной инфраструктуры, включая виртуализации сети Hyper-V (HNV), контроллеры сети, программного обеспечения подсистемы балансировки нагрузки (SLB/MUX) и шлюзы.  
  
    
  
-   [Настройка инфраструктуры для программного обеспечения конфигурируемой сети (SDN) в структуре VMM](https://technet.microsoft.com/en-us/system-center-docs/vmm/scenario/sdn-overview)  
  
    Используйте этот метод, если вы хотите включить System Center виртуальной машины диспетчере (VMM) для управления инфраструктурой SDN.  
  
-   [Развертывание инфраструктуры программно-конфигурируемой сети с помощью сценариев](../../sdn/deploy/Deploy-a-Software-Defined-Network-infrastructure-using-scripts.md)  
  
    Используйте этот метод, если вы не хотите использовать для управления инфраструктурой SDN VMM, или если у вас есть другой способ управления.  
  
 Если вы хотите развернуть отдельные технологии SDN вместо всей инфраструктуры, см.:  
[Развертывание технологий программно-конфигурируемой сети с помощью Windows PowerShell](Deploy-Software-Defined-Network-Technologies-using-Windows-PowerShell.md).

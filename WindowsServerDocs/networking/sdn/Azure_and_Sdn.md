# <a name="microsoft-azure-and-software-defined-networking"></a>Microsoft Azure и программно-конфигурируемой сети

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Microsoft Azure — это облачная платформа корпорации Майкрософт: растущий набор интегрированных служб - вычислений, хранения, данных, сети и приложение -, позволяющие перемещать быстрее, выполнять дополнительные действия и экономить средства.  
  
Подход корпорации Майкрософт к программно-конфигурируемая сеть включает проектирования, построения и функционирования сетей глобальных ЦОД для таких служб, как Microsoft Azure. В глобальных ЦОД Microsoft Azure выполняются десятки тысяч изменений сети каждый день, что возможно только из-за программно-конфигурируемая сеть.  
  
Microsoft Azure работает на платформах Windows Server и Hyper-V, которые включены в Windows Server. Windows Server и System Center содержат усовершенствования и реализации рекомендаций из практического опыта Microsoft по в функционирования глобальных сетей ЦОД как Microsoft Azure для вас, что позволяет развертывать те же технологии для обеспечения гибкости, автоматизации и управления с помощью программного обеспечения разработанных сетевых технологий.  
  
Дополнительные сведения см. в разделе [что такое Microsoft Azure? ](https://azure.microsoft.com/overview/what-is-azure/?WT.mc_id=azurebg_us_sem_bing_br_nontest_whatisazure_whatisazure&WT.srch=1).

# <a name="microsoft-azure-and-software-defined-networking"></a>Microsoft Azure и программно-конфигурируемая сеть

>Область применения: Windows Server (Semi-Annual Channel), Windows Server 2016

Microsoft Azure облачной платформой Майкрософт: растущей набор интегрированных служб — вычислений, хранилищ, данных, сетей и приложений, которые помогают быстрее перемещаться, выполнять больше и экономить деньги.  
  
Подход корпорации Майкрософт к программно-определяемой сети включает проектирование, создание и эксплуатацию сетей центров обработки данных с глобальными масштабами для таких служб, как Microsoft Azure. В глобальных ЦОД Microsoft Azure ежедневно выполняются десятки тысяч изменений сети, что возможно только в среде программно-конфигурируемых сетей.  
  
Microsoft Azure работает на платформах Windows Server и Hyper-V, входящих в состав Windows Server. Windows Server и System Center содержат усовершенствования и реализации рекомендаций, разработанных на основе опыта Майкрософт в сфере управления глобальными сетями ЦОД, такими как Microsoft Azure, что позволяет вам развертывать те же самые программные сетевые технологии для обеспечения гибкости, автоматизации и управляемости.  
  
Дополнительные сведения см. в разделе [что такое Microsoft Azure?](https://azure.microsoft.com/overview/what-is-azure/?WT.mc_id=azurebg_us_sem_bing_br_nontest_whatisazure_whatisazure&WT.srch=1).

# <a name="microsoft-azure-and-software-defined-networking"></a>Microsoft Azure и программно-конфигурируемая сеть

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Microsoft Azure — это облачная платформа Майкрософт: постоянно растущая коллекция интегрированных служб - вычислений, хранения, данных, сети и приложения — которые помогут вам быстродействие, повышает производительность и сэкономить деньги.  
  
Подход корпорации Майкрософт к программно-конфигурируемая сеть включает в себя проектирование, создание и использование сети центра обработки данных в глобальном масштабе для служб, таких как Microsoft Azure. В глобальных ЦОД Microsoft Azure ежедневно выполняются десятки тысяч изменений сети, что возможно только в среде программно-конфигурируемых сетей.  
  
Microsoft Azure работает на платформах Windows Server и Hyper-V, входящих в состав Windows Server. Windows Server и System Center содержат усовершенствования и рекомендации из опыта корпорации Майкрософт на операционных глобальных сетей ЦОД как Microsoft Azure вам, что можно развертывать технологии для гибкости, автоматизации и управляемости При использовании программное обеспечение, разработанное сетевых технологий.  
  
Дополнительные сведения см. в разделе [что такое Microsoft Azure?](https://azure.microsoft.com/overview/what-is-azure/?WT.mc_id=azurebg_us_sem_bing_br_nontest_whatisazure_whatisazure&WT.srch=1).

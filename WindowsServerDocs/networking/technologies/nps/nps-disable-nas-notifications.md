# <a name="disable-nas-notification-forwarding-in-nps"></a>Отключите пересылку уведомлений сервера на сервере политики сети

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Эта процедура позволяет отключить перенаправление запуска и отмены сообщения серверы доступа к сети (NAS) к членам группы удаленных серверов RADIUS настроен на сервере политики сети.

Если у вас есть настроенные группы внешних RADIUS-серверов и на сервере политики сети **политики запросов на подключение**, снять **пересылать запросы учета для этой группы удаленных серверов RADIUS** флажок, эти группы представлены по-прежнему отправленных NAS, запускать и останавливать сообщений уведомления. 

При этом создается лишний сетевой трафик. Чтобы исключить этот трафик, отключите NAS пересылку для отдельных серверов в каждой группе удаленных RADIUS-сервера.

Для выполнения этой процедуры необходимо быть членом группы **Администраторы**.

### <a name="to-disable-nas-notification-forwarding"></a>Чтобы отключить пересылку уведомлений NAS

1. В диспетчере серверов щелкните **средства**, а затем нажмите кнопку **сервера политики сети**. Откроется консоль NPS.

2. В консоли сервера политики сети, дважды щелкните **клиенты и серверы RADIUS**, нажмите кнопку **удаленных групп серверов RADIUS**, а затем дважды щелкните группу удаленных RADIUS-серверов, которую требуется настроить. Группе удаленных RADIUS-серверов **свойства** откроется диалоговое окно.

3. Дважды щелкните член группы, который вы хотите настроить, а затем нажмите кнопку **проверки подлинности и учетные данные** вкладки.

4. В **учет**снимите **вперед запуск сервера доступа к сети и прекратить отправку уведомлений на этот сервер** флажок и нажмите кнопку **ОК**.

5. Повторите шаги 3 и 4 для всех членов группы, которые вы хотите настроить.

Дополнительные сведения об управлении NPS см. в разделе [управление сервером политики сети](nps-manage-top.md).

Дополнительные сведения о сервере политики сети, см. в разделе [сервера политики сети (NPS)](nps-top.md).

# <a name="view-dns-resource-records-for-a-specific-ip-address"></a>Просмотр записей ресурсов DNS для определенного IP-адреса

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

В этом разделе можно использовать для просмотра записей ресурсов DNS, связанные с IP-адрес, которую вы выберете.  
  
Членство в группе **Администраторы**, или в эквивалентной минимальным требованием для выполнения этой процедуры.  
  
### <a name="to-view-resource-records-for-an-ip-address"></a>Чтобы просмотреть записи ресурсов для IP-адрес  
  
1.  В диспетчере серверов щелкните **IPAM**. Откроется консоль IPAM-клиента.  
  
2.  В области навигации в **ПРОСТРАНСТВО IP-адресов**, нажмите кнопку **перечень IP-адресов**. В нижней области навигации щелкните **IPv4** или **IPv6**. Перечень IP-адресов отображается в представлении поиска на панели отображения. Найдите и выберите IP-адрес которого записей ресурсов DNS, которую требуется просмотреть.  
  
    ![Просмотр перечень IP-адресов](../../media/View-DNS-Resource-Records-for-a-Specific-IP-Address/ipam_IPInventory_01.jpg)  
  
3.  На панели отображения **Просмотр подробностей**, нажмите кнопку **записей ресурсов DNS**. Отображаются записи ресурсов, которые связаны с выбранной IP-адрес.  
  
    ![Просмотр записей ресурсов DNS](../../media/View-DNS-Resource-Records-for-a-Specific-IP-Address/ipam_IPInventory_02.jpg)

# <a name="purge-utilization-data"></a>Очистка данных об использовании

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

В этом разделе можно использовать, чтобы узнать, как удалить данные об использовании из базы данных IPAM.  

Необходимо быть членом **Администраторы IPAM**, локальный компьютер **Администраторы** или аналогичной группы, для выполнения этой процедуры.

## <a name="to-purge-the-ipam-database"></a>Чтобы удалить базу данных IPAM  
1. Откройте диспетчер серверов и перейдите к интерфейс клиента IPAM.
2. Перейдите в одно из следующих расположений: **блоки IP-адресов**, **перечень IP-адресов**, или **группы диапазонов IP-адресов**.  
3. Нажмите кнопку **ЗАДАЧИ**, а затем нажмите кнопку **Очистка данных об использовании**. **Очистка данных об использовании** откроется диалоговое окно.
4. В **очистить все об использовании данных или до**, нажмите кнопку **выберите дату**.
5. Выберите дату, для которого требуется удалить все записи базы данных, так и до этой даты.
6. Нажмите кнопку **ОК**. IPAM удаляет все записи, которые вы указали.

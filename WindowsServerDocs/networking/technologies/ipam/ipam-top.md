# <a name="ip-address-management-ipam"></a>Управление IP-адресами (IPAM)

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Управление IP-адресами (IPAM) является встроенный набор инструментов для начала до конца планирования, развертывания, администрирования и отслеживания инфраструктуры IP-адресов в многофункциональном пользовательском интерфейсе. IPAM автоматически определяет серверы инфраструктуры IP-адресов и доменных имен (DNS) серверы в сети и позволяет управлять ими из центрального интерфейса.

# <a name="edit-a-dns-zone"></a>Изменение зоны DNS

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

В этом разделе можно использовать, чтобы изменить зону DNS в консоли ipam-клиента.  
  
Для выполнения этой процедуры необходимо быть членом группы **Администраторы** или пользователем с аналогичными правами.  
  
### <a name="to-edit-a-dns-zone"></a>Чтобы изменить зону DNS  
  
1.  В диспетчере серверов щелкните **IPAM**. Откроется консоль ipam-клиента.  
  
2.  В области навигации в **НАБЛЮДЕНИЕ и УПРАВЛЕНИЕ**, нажмите кнопку **зон DNS**. На панели навигации делит на верхней панели навигации и нижней панели навигации.  
  
3.  В нижней области навигации внесите одно из следующих выбранных элементов.  
  
    -   Прямого просмотра  
  
    -   Обратный просмотр IPv4  
  
    -   Обратный просмотр IPv6  
  
4.  Например выберите обратный просмотр IPv4.  
  
    ![Выберите тип зоны](../../media/Edit-a-DNS-Zone/ipam_EditZone_01.jpg)  
  
5.  В панели «дисплей» щелкните правой кнопкой мыши зону, который требуется изменить, а затем нажмите кнопку **зоны DNS редактирования**.  
  
    ![Изменение зоны DNS](../../media/Edit-a-DNS-Zone/ipam_EditZone_02.jpg)  
  
6.  **Зоны DNS редактирования** откроется диалоговое окно с **Общие** выбрана страница. При необходимости измените свойства общие зоны: **DNS-сервер**, **категории зоны**, и **тип зоны**, а затем нажмите кнопку **применить** или, если внесения изменений, **ОК**.  
  
    ![Измените свойства зоны и сохраните](../../media/Edit-a-DNS-Zone/ipam_EditZone_03a.jpg)  
  
7.  В **зоны DNS редактирования** диалоговом окне щелкните **Дополнительно**. **Дополнительно** откроется страница свойств зоны. При необходимости измените свойства, которые вы хотите изменить и нажмите кнопку **применить** или, если внесения изменений, **ОК**.  
  
    ![Изменить свойства зоны](../../media/Edit-a-DNS-Zone/ipam_EditZone_04a.jpg)  
  
8.  При необходимости выберите имена страниц свойств дополнительную зону (серверы имен, SOA, передач зоны), внесите изменения и нажмите кнопку **применить** или **ОК**. Чтобы просмотреть все правки зоны, щелкните **Сводка**, а затем нажмите кнопку **ОК**.

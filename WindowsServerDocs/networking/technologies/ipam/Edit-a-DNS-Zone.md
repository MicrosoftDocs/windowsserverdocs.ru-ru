# <a name="edit-a-dns-zone"></a>Изменение зоны DNS

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

В этом разделе можно использовать для изменения зоны DNS в консоли IPAM-клиента.  
  
Членство в группе **Администраторы**, или в эквивалентной минимальным требованием для выполнения этой процедуры.  
  
### <a name="to-edit-a-dns-zone"></a>Изменение зоны DNS  
  
1.  В диспетчере серверов щелкните **IPAM**. Откроется консоль IPAM-клиента.  
  
2.  В области навигации в **НАБЛЮДЕНИЕ и УПРАВЛЕНИЕ**, нажмите кнопку **зон DNS**. На панели навигации разделяет на верхней панели навигации и нижней панели навигации.  
  
3.  В нижней области навигации можно создайте следующие варианты:  
  
    -   Прямого просмотра  
  
    -   Обратный просмотр IPv4  
  
    -   Обратный просмотр IPv6  
  
4.  Например выберите обратный просмотр IPv4.  
  
    ![Выберите тип зоны](../../media/Edit-a-DNS-Zone/ipam_EditZone_01.jpg)  
  
5.  На панели отображения щелкните правой кнопкой мыши зону, который требуется изменить, а затем нажмите кнопку **изменение зоны DNS**.  
  
    ![Изменение зоны DNS](../../media/Edit-a-DNS-Zone/ipam_EditZone_02.jpg)  
  
6.  **Изменение зоны DNS** откроется диалоговое окно с **Общие** выбранные страницы. При необходимости измените свойства зоны общие: **DNS-сервер**, **категории зоны**, и **типа зоны**и нажмите кнопку **применить** или, если изменения не будут завершены, **ОК**.  
  
    ![Измените свойства зоны и сохраните](../../media/Edit-a-DNS-Zone/ipam_EditZone_03a.jpg)  
  
7.  В **изменение зоны DNS** диалоговом нажмите кнопку **Дополнительно**. **Дополнительно** откроется страница свойств зоны. При необходимости изменить свойства, которые вы хотите изменить и нажмите кнопку **применить** или, если изменения не будут завершены, **ОК**.  
  
    ![Измените свойства зоны](../../media/Edit-a-DNS-Zone/ipam_EditZone_04a.jpg)  
  
8.  При необходимости выберите дополнительные зоны имена страницы свойств (серверы имен, SOA, передач зон), внести необходимые изменения и нажмите кнопку **применить** или **ОК**. Чтобы просмотреть все внесенные изменения зоны, нажмите кнопку **Сводка**, а затем нажмите кнопку **ОК**.

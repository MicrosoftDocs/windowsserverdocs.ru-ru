# <a name="branchcache-network-shell-and-windows-powershell-commands"></a>BranchCache Network Shell и команд Windows PowerShell

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

В Windows Server 2016 можно настроить и управлять BranchCache с помощью Windows PowerShell или команды сетевой оболочки (Netsh) для BranchCache.  
  
В будущих версиях Windows, корпорация Майкрософт может удалить функции netsh для BranchCache. Если вы сейчас используете netsh для настройки и управления BranchCache и других сетевых технологий, Майкрософт рекомендует перейти на Windows PowerShell.  
  
Ссылки на команды Windows PowerShell и netsh представлены в следующих местах. Несмотря на то, что обе ссылки на команды были опубликованы для операционных систем более ранних, чем Windows Server 2016, эти ссылки правильности для этой операционной системы.  
  
-   [Команды Netsh для BranchCache в Windows Server 2008 R2](https://technet.microsoft.com/library/dd979561(v=ws.10))  
  
-   [Командлеты BranchCache в Windows PowerShell](https://technet.microsoft.com/library/hh848392.aspx) для Windows Server 2012.  
  
> [!TIP]  
> Чтобы просмотреть список команд Windows PowerShell для BranchCache в командной строке Windows PowerShell, введите `Get-Command -Module BranchCache`в командной строке Windows PowerShell и нажмите клавишу ВВОД.

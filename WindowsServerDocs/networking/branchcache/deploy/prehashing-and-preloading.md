# <a name="prehashing-and-preloading-content-on-hosted-cache-servers-optional"></a>Предварительное хэширование и предварительная загрузка содержимого на серверах размещенного кэша (необязательно)

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Эту процедуру можно использовать для принудительного создания сведений о содержимом - также называется хэши - на включена служба BranchCache файла и веб-серверах. Вы можете также собрать данные в файл и веб-серверами в пакеты, которые можно перенести на серверах удаленного размещенного кэша.  Это дает возможность предварительная загрузка содержимого на серверах удаленного размещенного кэша, чтобы данные были доступны для первого клиентского доступа.  
  
Необходимо быть членом **Администраторы**, или аналогичные им для выполнения этой процедуры.  
  
### <a name="to-prehash-content-and-preload-the-content-on-hosted-cache-servers"></a>Чтобы содержимое Предварительное хэширование и предварительная загрузка содержимого на серверах размещенного кэша  
  
1.  Войдите в файл или веб-сервер, содержащий данные, которые необходимо предварительно загрузить и установить папок и файлов, которые требуется загрузить на один или несколько серверов удаленного размещенного кэша.  
  
2.  Запустите Windows PowerShell с правами администратора. Для каждой папки и файл, запустите `Publish-BCFileContent`команды или `Publish-BCWebContent`команду, в зависимости от типа содержимого сервера, для добавления данных в пакет данных и активировать создания хэша.  
  
3.  После добавления всех данных пакет данных, экспортировать его с помощью `Export-BCCachePackage`команду, чтобы создать файл пакета данных.  
  
4.  Переместите файл пакета данных на серверах удаленного размещенного кэша с помощью своего технологии передачи файлов.  FTP, SMB, HTTP, DVD-диска и переносных дисков являются все работало транспортировки.  
  
5.  Импортируйте файл пакета данных на серверах удаленного размещенного кэша с помощью `Import-BCCachePackage`команды.

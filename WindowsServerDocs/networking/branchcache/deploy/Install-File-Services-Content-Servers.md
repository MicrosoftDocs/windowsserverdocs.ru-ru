# <a name="install-file-services-content-servers"></a>Установка серверов содержимого файловых служб

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Развертывание серверов содержимого, работающих под управлением роли сервера файловых служб, необходимо установить BranchCache для сетевых файлов службы роли для роли сервера файловых служб. Кроме того необходимо включить BranchCache на общих файловых ресурсах, в соответствии с требованиями.  
  
Во время настройки сервера содержимого вы можете разрешить BranchCache публикации содержимого для всех общих папок, или можно выбрать подмножество общих папок для публикации.  
  
> [!NOTE]  
> При развертывании BranchCache включена файлового сервера или веб-сервер в качестве сервера содержимого, сведения о содержимом теперь рассчитываются в автономном режиме до того, как клиент BranchCache запросит файл. Из-за этого улучшения вам не обязательно должны настроить публикация хэша для серверов содержимого, как описано в предыдущей версии BranchCache.  
>   
> Это поколение автоматического хэш обеспечивает повышенную производительность и дополнительно экономя пропускную способность, так как сведения о содержимом готов для первого клиента, который запрашивает содержимое, а расчеты уже завершены.

# <a name="install-file-services-content-servers"></a>Установка серверов содержимого файловых служб

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Развертывание серверов содержимого, на которых выполняется роль сервера файловых служб, необходимо установить компонент BranchCache для сетевых файлов службы роли для роли сервера файловых служб. Кроме того необходимо включить BranchCache на общие файловые ресурсы в соответствии с требованиями.  
  
Во время настройки сервер содержимого вы можете разрешить публикацию BranchCache содержимого для всех общих папок, или можно выбрать подмножество файловых ресурсов общего доступа к публикации.  
  
> [!NOTE]  
> При развертывании BranchCache включена файлового сервера или веб-сервер в качестве сервера содержимого сведения о содержимом теперь рассчитываются в автономном режиме до, как клиент BranchCache запросит файл. Из-за это улучшение не необходимо настроить публикация хэша для серверов содержимого, как описано в предыдущей версии BranchCache.  
>   
> Это поколение автоматического хэш обеспечивает повышение производительности и дополнительно экономя пропускную способность, так как сведения о содержимом готовы к моменту первого запроса содержимого, а расчеты уже завершены.

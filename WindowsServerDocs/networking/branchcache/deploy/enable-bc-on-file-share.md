# <a name="enable-branchcache-on-a-file-share-optional"></a>Включение BranchCache в общем файловом ресурсе (необязательно)

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Эта процедура позволяет включить BranchCache на файловом ресурсе.  
  
> [!IMPORTANT]  
> Необходимо выполнить эту процедуру, если вы настраиваете параметр публикации хэш со значением **разрешить публикацию хэшей для всех общих папок**.  
  
Членство в группе **Администраторы**, или эквивалентной является минимальным требованием для выполнения этой процедуры.  
  
### <a name="to-enable-branchcache-on-a-file-share"></a>Чтобы включить BranchCache на файловом ресурсе  
  
1.  Откройте Windows PowerShell, введите **mmc**и затем нажмите клавишу "ВВОД". Отобразится консоль управления (MMC).  
  
2.  В MMC на **файл** меню, щелкните **Add/Remove Snap-in**. **Добавление или удаление оснасток** откроется диалоговое окно.  
  
3.  В **Добавление или удаление оснасток**в **Доступные оснастки**, дважды щелкните **общих папок**. Выбрав объект с локального компьютера, откроется мастер общих папок. Настроить представление, которое вы хотите использовать, щелкните **Готово**, а затем нажмите кнопку **ОК**.  
  
4.  Дважды щелкните **общих папок (локальный)**, а затем нажмите кнопку **общих папок**.  
  
5.  В области сведений щелкните правой кнопкой мыши общую папку затем **свойства**. Дистрибутивного общего ресурса **свойства** откроется диалоговое окно.  
  
6.  В **свойства** диалоговом окне **Общие** щелкните **Настройка автономных**. **Настройка автономных** откроется диалоговое окно.  
  
7.  Убедитесь, что **файлы и программы, которые пользователи указывают вне сети доступны только** выбран, а затем нажмите кнопку **включить BranchCache**.  
  
8.  Щелкните дважды **ОК** .

# <a name="enable-branchcache-on-a-file-share-optional"></a>Включение BranchCache в общем файловом ресурсе (необязательно)

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Эта процедура позволяет включить BranchCache для общей папки.  
  
> [!IMPORTANT]  
> Необходимо выполнить эту процедуру, если настроить параметр публикации хэша со значением **разрешить публикация хэша для всех общих папок**.  
  
Членство в группе **Администраторы**, или эквивалентной группе минимальным требованием для выполнения этой процедуры.  
  
### <a name="to-enable-branchcache-on-a-file-share"></a>Включить BranchCache для общей папки  
  
1.  Откройте Windows PowerShell, введите **mmc**, и нажмите клавишу ВВОД. Открытие Microsoft Management Console (MMC).  
  
2.  В консоли MMC на **файл** меню, нажмите кнопку **добавить или удалить оснастку**. **Добавить или удалить оснастку** откроется диалоговое окно.  
  
3.  В **добавить или удалить оснастку**в **Доступные оснастки**, дважды щелкните **общих папок**. Объект локальный компьютер открывает мастера общих папок. Настройте представление, которое вы предпочитаете, нажмите кнопку **Готово**, а затем нажмите кнопку **ОК**.  
  
4.  Дважды щелкните **общие папки (локальная)**, а затем нажмите кнопку **общих ресурсов**.  
  
5.  В области сведений щелкните правой кнопкой мыши общую папку и нажмите кнопку **свойства**. Общий ресурс **свойства** откроется диалоговое окно.  
  
6.  В **свойства** диалогового **Общие** щелкните **параметров автономного**. **Параметров автономного** откроется диалоговое окно.  
  
7.  Убедитесь, что **файлы и программы, которые определяют пользователей вне сети доступны только** установлен, а затем нажмите кнопку **включить BranchCache**.  
  
8.  Нажмите кнопку **ОК** дважды.

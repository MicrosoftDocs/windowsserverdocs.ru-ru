# <a name="install-content-servers-that-use-the-branchcache-feature"></a>Установка серверов содержимого, использующих компонент BranchCache

>Применяется к: Windows Server (Semi-Annual Channel), Windows Server 2016

Для развертывания серверов содержимого, использующих веб-серверы с протоколом HTTPS, веб-серверы на основе протокола HTTP и фоновой интеллектуальной службы передачи (BITS), такие как Windows Server Update Services (WSUS) и серверы системы сайта распространения филиала Configuration Manager, необходимо установить функцию BranchCache, запустить службу BranchCache и (только для серверов WSUS) выполнить дополнительные действия по настройке.

Сведения о развертывании серверов содержимого см. в следующих разделах.

-   [Установка функции BranchCache](Install-the-BranchCache-Feature.md)

-   [Настройка Windows Server Update Services &#40;WSUS&#41; серверах содержимого](configure-wsus-content-servers.md)

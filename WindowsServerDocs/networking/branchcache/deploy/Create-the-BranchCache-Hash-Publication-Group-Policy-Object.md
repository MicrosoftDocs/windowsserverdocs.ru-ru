# <a name="create-the-branchcache-hash-publication-group-policy-object"></a>Создание объекта политики группа публикации хэша BranchCache

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Эту процедуру можно использовать для создания публикации хэша BranchCache объекта групповой политики (GPO).  
  
Членство в группе **"Администраторы домена"**, или эквивалентной группе минимальным требованием для выполнения этой процедуры.  
  
> [!NOTE]  
> Перед выполнением этой процедуры, необходимо создать BranchCache подразделения файловых серверов и перемещение файловых серверов в Подразделение. Дополнительные сведения см. в разделе [включить публикация хэша для файловых серверов домена](../../branchcache/deploy/Enable-Hash-Publication-for-Domain-Member-File-Servers.md).  
  
### <a name="to-create-the-branchcache-hash-publication-group-policy-object"></a>Чтобы создать объект групповой политики публикации хэша BranchCache  
  
1.  Откройте Windows PowerShell, введите **mmc**, и нажмите клавишу ВВОД. Открытие Microsoft Management Console (MMC).  
  
2.  В консоли MMC на **файл** меню, нажмите кнопку **добавить или удалить оснастку**. **Добавить или удалить оснастку** откроется диалоговое окно.  
  
3.  В **добавить или удалить оснастку**в **Доступные оснастки**, дважды щелкните **Управление групповой политикой**, а затем нажмите кнопку **ОК**.  
  
4.  В окне Консоли управления групповой политики разверните путь к файловым серверам BranchCache Подразделение, которое вы создали ранее. Например, если лес с именем example.com, ваш домен называется example1.com и оно называется файловых серверов BranchCache, разверните следующий путь: **Управление групповой политикой**, **леса: example.com**, **домены**, **example1.com**, **файловых серверов BranchCache**.  
  
5.  Щелкните правой кнопкой мыши **файловых серверов BranchCache**, а затем нажмите кнопку **создать объект групповой Политики в этом домене и связать его**. **Новый объект групповой Политики** откроется диалоговое окно. В **имя**, введите имя нового объекта групповой Политики. Например, если вы хотите указать объект публикации хэша BranchCache, введите **публикации хэша BranchCache**. Нажмите кнопку **ОК**.

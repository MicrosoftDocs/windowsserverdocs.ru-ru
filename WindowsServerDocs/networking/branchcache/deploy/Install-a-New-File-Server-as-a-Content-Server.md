# <a name="install-a-new-file-server-as-a-content-server"></a>Установка нового файлового сервера в качестве сервера содержимого

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Эту процедуру можно использовать для установки роли сервера файловых служб и **BranchCache для сетевых файлов** службы на компьютере под управлением Windows Server 2016.  
  
Членство в группе **Администраторы**, или эквивалентной является минимальным требованием для выполнения этой процедуры.  
  
> [!NOTE]  
> Для выполнения этой процедуры с помощью Windows PowerShell, запустите Windows PowerShell от имени администратора, введите следующие команды в командной строке Windows PowerShell и нажмите клавишу ВВОД.  
>   
> `Install-WindowsFeature FS-BranchCache -IncludeManagementTools`  
>   
> `Restart-Computer`  
>   
> Чтобы установить службу роли дедупликации данных, введите следующую команду и нажмите клавишу ВВОД.  
>   
> `Install-WindowsFeature FS-Data-Deduplication -IncludeManagementTools`  
  
### <a name="to-install-file-services-and-the-branchcache-for-network-files-role-service"></a>Чтобы установить файловые службы и BranchCache для сетевых файлов службы роли  
  
1.  Откройте диспетчер серверов, щелкните **Управление**, а затем нажмите кнопку **Добавить роли и компоненты**. Откроется мастер добавления ролей и компонентов. На странице **Прежде чем приступить к работе** нажмите кнопку **Далее**.  
  
2.  В **Выбор типа установки**, убедитесь, что **Установка ролей или компонентов** выбран, а затем нажмите кнопку **Далее**.  
  
3.  В **Выбор целевого сервера**, убедитесь, что выбран правильный сервер и нажмите кнопку **Далее**.  
  
4.  В **Выбор ролей сервера**в **ролей**, обратите внимание, что **службами хранения файлов и** роль уже установлена, щелкните стрелку слева от имя роли, чтобы развернуть Выбор служб ролей, а затем щелкните стрелку слева от **файл и iSCSI служб**.  
  
5.  Установите флажки для **файлового сервера** и **BranchCache для сетевых файлов**.  
  
    > [!TIP]  
    > Рекомендуется установить флажок для **дедупликации данных**.
  
    Нажмите кнопку **Далее**.  
  
6.  В **Выбор компонентов**, нажмите кнопку **Далее**.  
  
7.  В **подтверждение выбранных параметров установки**, просмотрите выбранные параметры и нажмите кнопку **установить**. **Ход выполнения установки** панель отображается во время установки. По завершении установки нажмите кнопку **закрыть**.

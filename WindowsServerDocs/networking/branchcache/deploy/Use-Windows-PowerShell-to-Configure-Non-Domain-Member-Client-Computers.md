# <a name="use-windows-powershell-to-configure-non-domain-member-client-computers"></a>Использование Windows PowerShell для настройки не входящих в домен клиентских компьютеров

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Эту процедуру можно использовать, чтобы вручную настроить клиентский компьютер для режима распределенного кэша BranchCache или режима размещенного кэша.  
  
> [!NOTE]  
> Если вы настроили клиентских компьютеров в BranchCache с помощью групповой политики, параметры групповой политики переопределяют любые ручные настройки клиентских компьютеров, к которым применяются политики.  
  
Членство в группе **Администраторы**, или эквивалентной группе минимальным требованием для выполнения этой процедуры.  
  
### <a name="to-enable-branchcache-distributed-or-hosted-cache-mode"></a>Чтобы включить BranchCache в режиме распределенного и размещенного кэша  
  
1.  На клиентском компьютере BranchCache, который вы хотите настроить запустите Windows PowerShell от имени администратора, а затем выполните одно из следующих.  
  
    -   Чтобы настроить клиентский компьютер для использования режима распределенного кэша BranchCache, введите следующую команду и нажмите клавишу ВВОД.  
  
        `Enable-BCDistributed`  
  
    -   Чтобы настроить клиентский компьютер на наличие режим размещенного кэша BranchCache, введите следующую команду и нажмите клавишу ВВОД.  
  
        `Enable-BCHostedClient`  
  
        > [!TIP]  
        > Если вы хотите указать серверов доступны размещенного кэша, используйте `-ServerNames` параметр с разделителями запятыми список серверов размещенного кэша как значение параметра. Например при наличии двух серверов размещенного кэша, с именем HCS1 и HCS2 Настройка клиентского компьютера в режиме размещенного кэша с помощью следующей команды.  
        >   
        > `Enable-BCHostedClient -ServerNames HCS1,HCS2`

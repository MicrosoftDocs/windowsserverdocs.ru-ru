# <a name="configure-the-file-services-server-role"></a>Настройка роли сервера файловых служб

>Применяется к: Windows Server (Semi-Annual Channel), Windows Server 2016

Серверы содержимого на базе файлового сервера BranchCache можно развернуть на компьютерах под управлением Windows Server 2016 и роли сервера файловых служб с установленной службой роли **BranchCache для сетевых файлов** .

-   Сведения об установке сервера содержимого BranchCache на компьютере, на котором еще не установлены файловые службы, см. в разделе [Установка нового файлового сервера в качестве сервера содержимого](../../branchcache/deploy/Install-a-New-File-Server-as-a-Content-Server.md).

-   Сведения об установке сервера содержимого BranchCache на компьютере, на котором уже настроена роль сервера файловых служб, см. в разделе [Настройка существующего файлового сервера в качестве сервера содержимого](../../branchcache/deploy/Configure-an-Existing-File-Server-as-a-Content-Server.md).

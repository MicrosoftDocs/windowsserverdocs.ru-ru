# <a name="configure-the-file-services-server-role"></a>Настройка роли сервера файловых служб

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Можно развернуть BranchCache файл серверных серверов содержимого на компьютерах под управлением Windows Server 2016 и роль сервера файловых служб с **BranchCache для сетевых файлов** установлена служба роли.  
  
-   Чтобы установить сервер содержимого BranchCache на компьютере, который еще нет файловых служб установлены, см. в разделе [установите новый файловый сервер в качестве сервера содержимого](../../branchcache/deploy/Install-a-New-File-Server-as-a-Content-Server.md).  
  
-   Чтобы установить сервер содержимого BranchCache на компьютере, который уже настроен с ролью сервера файловых служб, см. в разделе [настройки существующего файлового сервера в качестве сервера содержимого](../../branchcache/deploy/Configure-an-Existing-File-Server-as-a-Content-Server.md).

# <a name="configure-the-file-services-server-role"></a>Настройка роли сервера файловых служб

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Можно развернуть файл серверных серверов содержимого BranchCache на компьютерах под управлением Windows Server 2016 и роли сервера файловых служб с **BranchCache для сетевых файлов** установлена служба роли.  
  
-   Чтобы установить сервер содержимого BranchCache на компьютере, который не имеет файловых служб установлены, см. [Установка нового файлового сервера в качестве сервера содержимого](../../branchcache/deploy/Install-a-New-File-Server-as-a-Content-Server.md).  
  
-   Чтобы установить сервер содержимого BranchCache на компьютере, который уже настроен с ролью сервера файловых служб, в разделе [Настройка имеющегося файлового сервера в качестве сервера содержимого](../../branchcache/deploy/Configure-an-Existing-File-Server-as-a-Content-Server.md).

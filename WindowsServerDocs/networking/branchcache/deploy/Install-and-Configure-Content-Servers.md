# <a name="install-and-configure-content-servers"></a>Установка и настройка серверов содержимого

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Когда вы развертываете BranchCache в режиме распределенного кэша или в режиме размещенного кэша, необходимо развернуть один или несколько серверов содержимого в главном офисе или в облаке. Серверы содержимого веб-серверы или серверы приложений, использующих компонент BranchCache. Серверы содержимого, которые представляют собой файловые серверы используют BranchCache для сетевых файлов службы роли для роли сервера файловых служб в Windows Server 2016.

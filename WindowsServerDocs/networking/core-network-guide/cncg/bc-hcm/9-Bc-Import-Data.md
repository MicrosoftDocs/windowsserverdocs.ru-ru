# <a name="import-data-packages-on-the-hosted-cache-server-optional"></a>Импорт пакетов данных на размещенного кэша \(Optional\) сервера

>Область применения: Windows Server (канал точками годовой), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Эту процедуру можно использовать для импорта пакетов данных и предварительная загрузка содержимого на серверах размещенного кэша.

Эта процедура является обязательным, потому что вы не должны prehash и предварительная загрузка содержимого на серверах размещенного кэша.

В противном случае не предварительно загрузить содержимое данных автоматически добавляется в размещенного кэша, клиенты загружают через подключения по глобальной сети.

Необходимо быть членом группы "Администраторы" для выполнения этой процедуры.

## <a name="to-import-data-packages-on-the-hosted-cache-server"></a>Импорт пакетов данных на сервер размещенного кэша  

1. На сервере откройте Windows PowerShell с правами администратора.

2. Введите следующую команду, заменив значение параметра-Path с расположение папки, где хранятся данные пакетов и нажмите клавишу ВВОД.

    ```  
    Import-BCCachePackage –Path D:\temp\PeerDistPackage.zip
    ```  

3. При наличии более одного сервера размещенного кэша, где необходимо предварительно загрузить содержимое, выполните эту процедуру на каждом сервере размещенного кэша.

Чтобы продолжить работу в этом руководстве, см. [настроить автоматическое размещенного кэша обнаружения клиентом с использованием точки подключения службы](10-Bc-Client-By-Scp.md).

# <a name="import-data-packages-on-the-hosted-cache-server-optional"></a>Импорт пакетов данных на сервере размещенного кэша \(необязательно\)

>Относится к: Windows Server (полугодовой канал), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Эта процедура позволяет импортировать пакеты данных и выполнению предварительной загрузки содержимого на серверах размещенного кэша.

Эта процедура необязательна, так как вам не требуется prehash и предварительной загрузки содержимого на серверах размещенного кэша.

В этом случае не pre\-загрузки содержимого, данные для размещенного кэша автоматически добавляется как клиенты загружают список через ГЛОБАЛЬНУЮ сеть.

Необходимо быть членом группы администраторов для выполнения этой процедуры.

## <a name="to-import-data-packages-on-the-hosted-cache-server"></a>Для импорта пакетов данных на сервере размещенного кэша  

1. На компьютере сервера откройте Windows PowerShell с правами администратора.

2. Введите следующую команду, заменив значение для параметра-Path с расположение папки, где хранятся пакеты данных или нажмите клавишу ВВОД.

    ```  
    Import-BCCachePackage –Path D:\temp\PeerDistPackage.zip
    ```  

3. При наличии более одного сервера размещенного кэша, где необходимо предварительно загрузить содержимое, выполните эту процедуру на каждом сервере размещенного кэша.

Чтобы продолжить работу в этом руководстве, см. в разделе [настроить клиент автоматического размещенного кэша обнаружения с использованием точки подключения службы](10-Bc-Client-By-Scp.md).

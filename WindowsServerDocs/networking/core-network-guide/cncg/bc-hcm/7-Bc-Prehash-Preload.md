# <a name="prehash-and-preload-content-on-the-hosted-cache-server-optional"></a>Предварительное хэширование и предварительная загрузка содержимого на размещенного кэша \(Optional\) сервера

>Область применения: Windows Server (канал точками годовой), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Предварительное хэширование содержимого на серверах содержимого, добавлять содержимое в пакеты данных и затем предварительной загрузке содержимого на серверах размещенного кэша, можно использовать процедуры, описанные в этом разделе. 

Эти процедуры являются необязательными, так как вы не должны prehash и предварительная загрузка содержимого на серверах размещенного кэша. 

Если не предварительно загрузить содержимое, данные автоматически добавляется в размещенного кэша, клиенты загружают через подключения по глобальной сети.

>[!IMPORTANT]
>Несмотря на то, что данные процедуры необязательны вместе, если вы решите prehash и предварительная загрузка содержимого на серверах размещенного кэша не требуется выполнять обе процедуры.

- [Создание пакетов данных сервера содержимого для Web и содержимое файла & #40; необязательно & #41;](8-Bc-Data-Packages.md)
  
- [Импорт пакетов данных на сервер размещенного кэша & #40; необязательно & #41;](9-Bc-Import-Data.md)

Чтобы продолжить работу в этом руководстве, см. [создания содержимого пакетов данных сервера для Web и содержимое файла & #40; необязательно & #41; ](8-Bc-Data-Packages.md).

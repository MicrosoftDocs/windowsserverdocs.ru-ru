# <a name="dns-policy-scenario-guide"></a>Руководство по сценарию политики DNS

>Относится к: Windows Server (полугодовой канал), Windows Server 2016

Это руководство предназначено для использования администраторами DNS, сети и системы.  
  
Политики DNS — это новая функция для DNS в Windows Server&reg; 2016. Можно использовать в этом руководстве вы научитесь использовать политику DNS для управления как DNS-сервер обрабатывает запросы разрешения имен, на основе различных параметров, определенных вами политик.   
  
Это руководство содержит общие сведения политики DNS, а также конкретных вариантов использования политик DNS, которые представлены инструкции по настройке поведения DNS-сервера для достижения ваших целей, в том числе управление трафиком на основе географического расположения для основного и вторичные DNS-серверы, высокий уровень доступности приложения, разделенная система DNS и многое другое.

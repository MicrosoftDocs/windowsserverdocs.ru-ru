# <a name="dns-policy-scenario-guide"></a>Руководство по сценарию политики DNS

>Область применения: Windows Server (канал точками годовой), Windows Server 2016

Это руководство предназначено для использования с DNS-сервера, сетевые и системные администраторы.  
  
Политики DNS — это новая функция для DNS-сервера в Windows Server&reg; 2016. Чтобы узнать, как использование политики DNS для управления, как DNS-сервер обрабатывает запросы разрешения имен, на основании различных параметров, определяющих в политиках, можно использовать в этом руководстве.   
  
Это руководство содержит общие сведения политики DNS, а также конкретных сценариев политики DNS, которые предоставляют инструкции по настройке поведения DNS-сервера для достижения ваших целей, включая управление трафиком на основе географического расположения для основного и дополнительного DNS-серверы, приложения высокой доступности, разделенная система DNS и более.

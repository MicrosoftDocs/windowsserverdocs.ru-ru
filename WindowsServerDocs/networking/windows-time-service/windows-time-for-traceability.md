# <a name="windows-time-for-traceability"></a>Время Windows для отслеживания
>Относится к: Windows Server 2016 версии 1709 или более поздней версии и Windows 10 версии 1703 или более поздней версии


Законы множества секторы требуют систем, чтобы четко прослеживаться в формат UTC.  Это означает, что можно аттестовать системы смещение относительно UTC.  Чтобы включить сценарии соответствия нормативным требованиям, Windows 10 (версия 1703 или более поздней версии) и Windows Server 2016 (версия 1709 или более поздней версии) предоставляет новые журналы событий для получения картины с точки зрения операционной системы, чтобы сформировать представление о действиях, выполненных системные часы.  Эти журналы событий создаются постоянно для службы времени Windows и можно проверить или в архив для последующего анализа.

Эти новые события позволяют отвечать на следующие вопросы:

* Был ли изменен системных часов
* От его тактовой частоты изменения
* Изменена конфигурация службы времени Windows

## <a name="availability"></a>Доступность

Эти усовершенствования включены в Windows 10 версии 1703 или более поздней версии и Windows Server 2016 версии 1709 или более поздней версии.

## <a name="configuration"></a>Конфигурация

Чтобы реализовать эту функцию не требуется никакой настройки.  Эти журналы событий включены по умолчанию и можно найти в средстве просмотра в **приложений и служб Log\Microsoft\Windows\Time-Service\Operational** канала.

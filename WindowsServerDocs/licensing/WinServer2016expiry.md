# <a name="windows-server-technical-preview-expiration"></a>Окончание срока действия Windows Server Technical Preview
Все выпуски Windows Server 2016 Technical Preview содержит время\-конфиденциальных программного обеспечения. После даты истечения срока действия вам будет отказано в использовании данной версии.  

## <a name="license-expiration-dates-for-technical-previews"></a>Даты окончания срока действия лицензии для Technical Preview  
В следующей таблице представлены даты окончания срока действия лицензии для всех выпусков Windows Server 2016 Technical Preview.  

|Выпуск|Дата выпуска|Дата окончания срока действия лицензии|  
|-----------|----------------|---------------------------|  
|Windows Server «2016 Technical Preview 3|Август 2015 г.|1 августа 2016 г.|  
|Windows Server «2016 Technical Preview 4|Ноябрь 2015 г.|15 октября 2016 г.|  
|Windows Server «2016 Technical Preview 5|Апрель 2016 г.|28 февраля 2017 г.|

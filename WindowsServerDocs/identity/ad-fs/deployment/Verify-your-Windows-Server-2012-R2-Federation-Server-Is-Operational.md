# <a name="verify-your-windows-server-2012-r2-federation-server-is-operational"></a>Убедитесь, что на сервере федерации Windows Server 2012 R2 Operational



Чтобы проверить работоспособность сервера федерации, выполните указанные ниже процедуры. Работоспособный сервер доступен всем клиентам, находящимся в одной с ним сети.  
  
Минимальным требованием для выполнения этой процедуры является членство в одной из следующих групп: **Пользователи**, **Операторы архива**, **Опытные пользователи**, **Администраторы**, — или эквивалентной группе на локальном компьютере.  Просмотрите сведения об использовании соответствующих учетных записей и членства в группах в [локальные и доменные группы по умолчанию](https://go.microsoft.com/fwlink/?LinkId=83477).   
  
### <a name="procedure-1-to-verify-that-a-federation-server-is-operational"></a>Процедура 1. Проверка работоспособности сервера федерации  
  
1.  Чтобы убедиться, что в службах IIS \(IIS\) правильно настроен на сервере федерации, войдите на клиентский компьютер, расположенный в том же лесу, что и сервер федерации.  
  
2.  Откройте окно браузера, в адресной строке введите имя узла DNS сервера федерации и затем добавьте \/adfs\/fs\/названием federationserverservice.asmx – это к нему для нового сервера федерации, например:  
  
    **https:\/\/fs1.fabrikam.com\/adfs\/fs\/federationserverservice.asmx**  
  
3.  Нажмите клавишу ВВОД и выполните указанную ниже процедуру на компьютере сервера федерации. Если отображается сообщение **Ошибка в сертификате безопасности этого веб-узла**, нажмите кнопку **Продолжить открытие этого веб-узла**.  
  
    В результате должна отобразиться XML-страница с описанием службы. Если она отображается, службы IIS на сервере федерации доступны и успешно обрабатывают страницы.  
  
Для выполнения этой процедуры минимальным требованием является членство в группе **Администраторы** или эквивалентные права на локальном компьютере.  Просмотрите сведения об использовании соответствующих учетных записей и членства в группах в [локальные и доменные группы по умолчанию](https://go.microsoft.com/fwlink/?LinkId=83477).   
  
### <a name="procedure-2-to-verify-that-a-federation-server-is-operational"></a>Процедура 2. Проверка работоспособности сервера федерации  
  
1.  Войдите на новый сервер федерации с правами администратора.  
  
2.  На **запустить** введите**средство просмотра событий**, и нажмите клавишу ВВОД.  
  
3.  В области сведений дважды\-щелкните **журналы приложений и служб**, double\-щелкните **события AD FS**, а затем нажмите кнопку **администратора**.  
  
4.  В **событие с Идентификатором** столбца, найдите событие с Идентификатором 100. Если сервер федерации настроен правильно, вы увидите новое событие, в журнале приложений, средства просмотра событий — с кодом 100. Это событие говорит, что серверу федерации не может успешно взаимодействовать со службой федерации.

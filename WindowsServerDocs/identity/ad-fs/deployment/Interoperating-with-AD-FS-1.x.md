# <a name="interoperating-with-ad-fs-1x"></a>Взаимодействие с AD FS 1.x

>Область применения. Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Для обеспечения взаимодействия между службами федерации Active Directory \(AD FS\) в Windows Server® 2012 и AD FS 1. *x*, выполнить одну или несколько из следующих задач, в зависимости от потребностей вашей организации:  
  
-   Планирование для обеспечения взаимодействия между AD FS в Windows Server 2012 и предыдущие версии AD FS и узнайте больше о идентификатор имени тип утверждения. Дополнительные сведения см. в разделе [планирование взаимодействия с AD FS 1.x](https://technet.microsoft.com/library/ff678040.aspx).  
  
-   Если вы будете отправлять утверждения из службы федерации AD FS в Windows Server 2012, которые могут использоваться службой AD FS 1. *x* службы федерации, см. в разделе [контрольный список: Настройка служб AD FS для отправки утверждений в службе федерации AD FS 1.x](Checklist--Configuring-AD-FS-to-Send-Claims-to-an-AD-FS-1.x-Federation-Service.md).  
  
-   Если вы будете отправлять утверждения от службой федерации AD FS в Windows Server 2012, которые могут использоваться приложением, которое размещается в веб-сервера AD FS 1. *x* утверждений\-виду веб-агент, см. в разделе [контрольный список: Настройка служб AD FS для отправки утверждений для агента AD FS 1.x Claims-Aware Web](Checklist--Configuring-AD-FS-to-Send-Claims-to-an-AD-FS-1.x-Claims-Aware-Web-Agent.md).  
  
-   Если вы будете отправлять утверждения от AD FS 1. *x* службы федерации, которые должны обрабатываться службой федерации AD FS в Windows Server 2012, см. в разделе [контрольный список: Настройка AD FS использовать утверждения из AD FS 1.x](Checklist--Configuring-AD-FS--to-Consume-Claims-from-AD-FS-1.x.md).  
  
## <a name="differences-between-federation-service-settings"></a>Различия между параметрами службы федерации  
Несмотря на то что большинство AD FS 1. *x* службы федерации параметры работают так же, как службы федерации AD FS в параметрах Windows Server 2012, изменились имена некоторых параметров. В следующей таблице перечислены имена параметров для AD FS 1. *x* службы федерации и именам эквивалентных для службы федерации AD FS в Windows Server 2012.  
  
|AD FS 1.x Federation Service параметр|Эквивалентное службой федерации AD FS в Windows Server 2012 параметр  
|----------------------------------------|---------------------------------------------------------------------------------------------------------- 
|Партнер по учетным записям|Отношения доверия с поставщиком утверждений  
|Партнер по ресурсам|Отношения доверия с проверяющей стороной 
|Приложение|Отношения доверия с проверяющей стороной  
|Свойства приложения|Свойства стороны отношения доверия с проверяющей стороной  
|URL-адрес приложения|Доверяющая сторона идентификатор и WS\-федерации конечной точки URL-адрес пассивного  
|URI службы федерации|Идентификатор службы федерации  
|URL-адрес конечной точки службы федерации|WS\-URL-адрес конечной точки пассивной федерации

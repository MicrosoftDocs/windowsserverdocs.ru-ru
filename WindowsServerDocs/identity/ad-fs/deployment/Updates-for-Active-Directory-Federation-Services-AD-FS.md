# <a name="required-updates-for-active-directory-federation-services-ad-fs-and-web-application-proxy-wap"></a>Обновления, необходимые для служб федерации Active Directory (AD FS) и прокси веб-приложения (WAP)

>Область применения. Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2 SP1

Начиная с октября 2016 года все обновления для всех компонентов Windows Server освобождаются только через Windows Update (WU).  Есть, без дополнительных исправлений или отдельные файлы для загрузки.
Это относится к Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 и Windows Server 2008 R2 SP1.

На этой странице перечислены накопительных пакетов особый интерес для AD FS и WAP, а также за предыдущие периоды список исправлений, рекомендуется использовать для AD FS и WAP.

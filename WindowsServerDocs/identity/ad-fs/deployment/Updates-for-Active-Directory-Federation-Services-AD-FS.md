# <a name="required-updates-for-active-directory-federation-services-ad-fs-and-web-application-proxy-wap"></a>Необходимые обновления для служб федерации Active Directory (AD FS) и прокси веб-приложения (WAP)

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2 SP1

Начиная с октября 2016 г. все обновления для всех компонентов Windows Server выпускаются только через обновления Windows (WU).  Отсутствуют новые исправления не дополнительные или отдельные файлы для загрузки.
Это относится к Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 и Windows Server 2008 R2 SP1.

На этой странице перечислены накопительных пакетов особый интерес для Служб федерации Active Directory и WAP, а также исторические списка исправлений, рекомендуемые для Служб федерации Active Directory и WAP.

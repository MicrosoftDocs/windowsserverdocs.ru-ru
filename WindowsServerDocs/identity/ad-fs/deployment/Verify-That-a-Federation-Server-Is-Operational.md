# <a name="verify-that-a-federation-server-is-operational"></a>Проверка работоспособности сервера федерации

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Можно использовать следующие процедуры, чтобы проверка работоспособности сервера федерации; то есть что новый сервер федерации может связаться с любого клиента в той же сети.  
  
Членство в группе **пользователей**, **операторы архива**, **Опытные пользователи**, **Администраторы** или эквивалентной группе на локальном компьютере минимальным требованием для выполнения этой процедуры.  Подробные сведения об использовании соответствующих учетных записей и членстве в группах в [локальные и доменные группы по умолчанию ](https://go.microsoft.com/fwlink/?LinkId=83477).   
  
### <a name="procedure-1-to-verify-that-a-federation-server-is-operational"></a>Процедура 1: Чтобы проверка работоспособности сервера федерации  
  
1.  Чтобы проверить правильность настройки \(IIS\) службы IIS на сервере федерации, войдите на клиентский компьютер, который находится в том же лесу, что и сервер федерации.  
  
2.  Откройте окно браузера, в строке типа адреса федерации имя узла DNS-сервера и добавьте /adfs/fs/federationserverservice.asmx к нему для нового сервера федерации, например:  
  
    **https://FS1.Fabrikam.com/ADFS/FS/federationserverservice.asmx**  
  
3.  Нажмите клавишу ВВОД и затем выполните следующую процедуру на компьютере сервера федерации. Если вы видите сообщение **существует проблема с сертификатом безопасности этого веб-сайта**, нажмите кнопку **Продолжить открытие этого веб-сайта**.  
  
    В результате должна отобразиться XML-ФАЙЛ с описанием службы. Если эта страница отображается, службы IIS на сервере федерации успешно — рабочей и обрабатывают страницы.  
  
Членство в группе **Администраторы**, или эквивалентной группе на локальном компьютере минимальным требованием для выполнения этой процедуры.  Подробные сведения об использовании соответствующих учетных записей и членстве в группах в [локальные и доменные группы по умолчанию ](https://go.microsoft.com/fwlink/?LinkId=83477).   
  
### <a name="procedure-2-to-verify-that-a-federation-server-is-operational"></a>Процедура 2: Чтобы проверка работоспособности сервера федерации  
  
1.  Войдите на новый сервер федерации от имени администратора.  
  
2.  На **запустить** введите **средство просмотра событий**, и нажмите клавишу ВВОД.  
  
3.  В области сведений щелкните двойном щелчке **журналы приложений и служб**, двойном щелчке **события AD FS**, а затем нажмите кнопку **администратора**.  
  
4.  В **событие с Идентификатором** столбец, найдите событие с кодом 100. Если сервер федерации настроен правильно, можно увидеть новое событие, в журнале приложений средства просмотра событий — с кодом 100. Это событие свидетельствует, что серверу федерации не удалось связаться со службой федерации.

# <a name="join-a-computer-to-a-domain"></a>Присоединение компьютера к домену

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Для служб федерации Active Directory \(AD FS\) функции каждый компьютер, выступающий в качестве сервера федерации должен быть присоединен к домену. прокси-серверы федерации могут быть присоединены к домену, но это не является обязательным.  
  
Необходимо присоединить к домену веб-сервера, если веб-сервере размещается только claims\ совместимых приложений.  
  
Членство в группе **Администраторы**, или эквивалентной группе на локальном компьютере минимальным требованием для выполнения этой процедуры.  Подробные сведения об использовании соответствующих учетных записей и членстве в группах в [локальные и доменные группы по умолчанию ](https://go.microsoft.com/fwlink/?LinkId=83477).   
  
### <a name="to-join-a-computer-to-a-domain"></a>Чтобы присоединить компьютер к домену  
  
1.  На **запустить** введите**панели управления**, и нажмите клавишу ВВОД.  
  
2.  Перейдите к **система и безопасность**, а затем нажмите кнопку **системы**.  
  
3.  В разделе **параметры имя домена и рабочей группы компьютеров**, нажмите кнопку **изменить параметры**.  
  
4.  На **имя компьютера** щелкните **изменение**.  
  
5.  В разделе **членом**, нажмите кнопку **домена**, введите имя домена, этот компьютер будет присоединиться и нажмите кнопку **ОК**.  
  
6.  Нажмите кнопку **ОК**, а затем перезагрузить компьютер.

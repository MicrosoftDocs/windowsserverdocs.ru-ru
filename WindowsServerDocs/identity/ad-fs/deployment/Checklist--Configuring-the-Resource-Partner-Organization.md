# <a name="checklist-configuring-the-resource-partner-organization"></a>Контрольный список: Настройка организации партнера по ресурсам

Организация партнера по ресурсам содержит веб-серверы, на которых размещены приложения Web @ no__t-0based, к которым будут обращаться пользователи в партнере по учетным записям. Администраторы в этой организации должны использовать оснастку управления AD FS @ no__t-0in для создания отношений доверия поставщиков утверждений для представления отношений доверия с организациями партнера по учетным записям. Администратор партнера по учетным записям должен создать отношения доверия с проверяющей стороной для каждой организации партнера по учетным записям, которой они хотят доверять.  
  
Этот контрольный список содержит задачи, необходимые для развертывания службы федерации Active Directory (AD FS) \(AD FS @ no__t-1 в организации партнера по ресурсам. Он также включает задачи по настройке компонентов, необходимых для установки одной @ no__t-0half связи Федерации.  
  
Если вы развертываете [Дизайн веб-единого входа](https://technet.microsoft.com/library/dd807033.aspx), вам не нужно следовать этому контрольному списку. Однако для успешного развертывания [федеративного единого входа в Интернет](https://technet.microsoft.com/library/dd807050.aspx)необходимо выполнить задачи из этого контрольного списка.  
  
> [!IMPORTANT]  
> Убедитесь, что администратор организации партнера по учетным записям соответствует рекомендациям в [Checklist: Настройка организации партнера по учетным записям @ no__t-0, чтобы убедиться, что все необходимые задачи развертывания будут выполнены для успешного создания второй половины партнерства Федерации.

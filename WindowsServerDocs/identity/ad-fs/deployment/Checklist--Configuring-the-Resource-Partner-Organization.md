# <a name="checklist-configuring-the-resource-partner-organization"></a>Контрольный список: Настройка партнерской организации по ресурсам

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Организации партнера по ресурсам содержит веб-серверы, размещения веб-приложений, которые будут доступны для пользователей в партнера по учетным записям. Администраторы в этой организации необходимо использовать управления AD FS оснастка для создания отношения доверия с поставщиком утверждений для представления их отношения доверия с организаций партнера по учетным записям. В свою очередь Администратор партнерской учетной записи необходимо создать отношений доверия с проверяющей стороной для каждой учетной записи партнерской организации, они должны доверять.  
  
Этот контрольный список содержит задачи, которые необходимы для развертывания служб федерации Active Directory \(AD FS\) в организации партнера по ресурсам. Он также включает задачи по настройке компонентов, которые необходимы для установления one\ половине партнерства федерации.  
  
Если вы развертываете [Web SSO Design](https://technet.microsoft.com/library/dd807033.aspx), необходимо выполнить этот контрольный список. Тем не менее, необходимо выполнить задачи в контрольном списке для успешного развертывания [Federated Web SSO Design](https://technet.microsoft.com/library/dd807050.aspx).  
  
> [!IMPORTANT]  
> Убедитесь, что администратор организации партнера по учетным записям следующим инструкциям, приведенным в [контрольный список: Настройка партнерской организации по учетной записи](Checklist--Configuring-the-Account-Partner-Organization.md) чтобы убедиться, что все необходимые развертывания задачи будут выполнены успешно создать второй половине партнерстве по федерации  
  
> [!NOTE]  
> Задачи в контрольном списке в порядке. Перехода по ссылке процедуры, вернитесь к данном разделу, после выполнения действия, описанные в этой процедуре, чтобы продолжить выполнение оставшихся задач контрольного списка.

# <a name="checklist-configuring-the-resource-partner-organization"></a>Контрольный список: Настройка партнерской организации по ресурсам

Организации партнера по ресурсам содержит веб-серверов, размещение в Интернете\-приложений, которые будут доступны для пользователей в партнере по учетным записям. Администраторы в этой организации должны использовать оснастку управления AD FS\-в для создания отношения доверия поставщика утверждений для представления их отношения доверия с партнерских организациях по ресурсам учетной записи. В свою очередь администратор учетной записи партнера необходимо создать доверия с проверяющей стороной для каждой партнерской организации, они будут доверять.  
  
Этот контрольный список содержит задачи, которые необходимы для развертывания служб федерации Active Directory \(AD FS\) в организации партнера по ресурсам. Он также включает задачи по настройке компонентов, которые необходимы для установления\-половины партнерство федерации.  
  
Если вы развертываете [Web SSO Design](https://technet.microsoft.com/library/dd807033.aspx), нет необходимости этот контрольный список. Тем не менее, необходимо выполнить задачи в этом контрольном списке для успешного развертывания [Federated Web SSO Design](https://technet.microsoft.com/library/dd807050.aspx).  
  
> [!IMPORTANT]  
> Убедитесь, что администратор организации партнера по учетным записям соответствует рекомендации [контрольный список: Настройка партнерской организации по](Checklist--Configuring-the-Account-Partner-Organization.md) чтобы убедиться, что все необходимые задачи по развертыванию будет завершена успешно создать второй половины партнерстве по федерации  
  
> [!NOTE]  
> Выполните по порядку задачи в контрольном списке. После выполнения всех шагов процедуры, на которую указывает ссылка, следует вернуться к этому разделу и продолжить выполнение оставшихся задач контрольного списка.

# <a name="checklist-implementing-a-web-sso-design"></a>Контрольный список: реализация проекта единого входа для интернет-решений

Этот родительский контрольный список включает ссылки на\-ссылки на важные понятия, касающиеся единого входа в\-Интернет\-\(единого входа\) проектировании службы федерации Active Directory (AD FS) \(AD FS\). Кроме того, в списке присутствуют ссылки на вложенные контрольные списки, которые помогут выполнить действия, необходимые для внедрения разработки.  
  
> [!NOTE]  
> Выполните по порядку задачи в контрольном списке. В случае перехода по ссылке в раздел общих понятий или вложенный контрольный список вернитесь после просмотра раздела общих понятий или выполнения действий из вложенного контрольного списка, чтобы можно было перейти к остальным задачам этого контрольного списка.

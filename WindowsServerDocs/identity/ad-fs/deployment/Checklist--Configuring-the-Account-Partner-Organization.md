# <a name="checklist-configuring-the-account-partner-organization"></a>Контрольный список: Настройка партнерской организации по учетным записям

Организация партнера по учетным записям содержит пользователей, которые будут получать доступ к приложениям Web @ no__t-0based в партнере по ресурсам. Администраторы в этой организации должны использовать оснастку управления AD FS @ no__t-0in для создания отношений доверия проверяющей стороны для представления отношений доверия с организациями партнера по ресурсам. Администратор партнера по ресурсам должен создать отношения доверия поставщиков утверждений для каждой организации партнера по учетным записям, которой они хотят доверять.  
  
Этот контрольный список включает задачи по развертыванию службы федерации Active Directory (AD FS) \(AD FS @ no__t-1 в организации партнера по учетным записям. Он также включает задачи по настройке компонентов, необходимых для установки одной @ no__t-0half связи Федерации.  
  
Если вы развертываете [Дизайн веб-единого входа](https://technet.microsoft.com/library/dd807033.aspx), вам не нужно следовать этому контрольному списку. Однако для успешного развертывания [федеративного единого входа в Интернет](https://technet.microsoft.com/library/dd807050.aspx)необходимо выполнить задачи из этого контрольного списка.  
  
> [!IMPORTANT]  
> Убедитесь, что администратор в организации партнера по ресурсам соответствует рекомендациям в [Checklist: Настройка организации партнера по ресурсам @ no__t-0, чтобы убедиться, что все необходимые задачи развертывания будут выполнены для успешного создания второй половины партнерства Федерации.

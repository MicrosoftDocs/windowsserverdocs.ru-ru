# <a name="checklist-configuring-the-account-partner-organization"></a>Контрольный список: Настройка партнерской организации по учетным записям

Организации партнера по учетным записям содержит пользователей, которые будет доступ к веб-\-приложениям в организации партнера по ресурсам. Администраторы в этой организации должны использовать оснастку управления AD FS\-в для создания отношений доверия проверяющих сторон для представления их отношения доверия с партнерских организациях по ресурсам. В свою очередь администратор партнера по ресурсам необходимо создать отношения доверия поставщика утверждений для каждой партнерской организации, они будут доверять.  
  
Этот контрольный список содержит задачи по развертыванию служб федерации Active Directory \(AD FS\) в организации партнера по учетным записям. Он также включает задачи по настройке компонентов, которые необходимы для установления\-половины партнерство федерации.  
  
Если вы развертываете [Web SSO Design](https://technet.microsoft.com/library/dd807033.aspx), нет необходимости этот контрольный список. Тем не менее, необходимо выполнить задачи в этом контрольном списке для успешного развертывания [Federated Web SSO Design](https://technet.microsoft.com/library/dd807050.aspx).  
  
> [!IMPORTANT]  
> Убедитесь, что администратор в партнерской организации по ресурсам следует рекомендациям в [контрольный список: Настройка партнерской организации по ресурсам](Checklist--Configuring-the-Resource-Partner-Organization.md) чтобы убедиться, что все необходимые задачи по развертыванию будет завершена успешно создать второй половины партнерстве по федерации.  
  
> [!NOTE]  
> Выполните по порядку задачи в контрольном списке. После выполнения всех шагов процедуры, на которую указывает ссылка, следует вернуться к этому разделу и продолжить выполнение оставшихся задач контрольного списка.

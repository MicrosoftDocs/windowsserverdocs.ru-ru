# <a name="checklist-configuring-the-account-partner-organization"></a>Контрольный список: Настройка партнерской организации по учетной записи

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Организации партнера по учетным записям содержит пользователей, которые будут обращаться к веб-приложений в организации партнера по ресурсам. Администраторы в этой организации должны использовать управления AD FS оснастка для создания отношений доверия с проверяющей стороной для представления их отношения доверия с партнерских организациях по ресурсам. В свою очередь администратор партнера по ресурсам необходимо создать отношения доверия с поставщиком утверждений для каждой учетной записи партнерской организации, они должны доверять.  
  
Этот контрольный список содержит задачи по развертыванию служб федерации Active Directory \(AD FS\) в организации партнера по учетным записям. Он также включает задачи по настройке компонентов, которые необходимы для установления one\ половине партнерства федерации.  
  
Если вы развертываете [Web SSO Design](https://technet.microsoft.com/library/dd807033.aspx), необходимо выполнить этот контрольный список. Тем не менее, необходимо выполнить задачи в контрольном списке для успешного развертывания [Federated Web SSO Design](https://technet.microsoft.com/library/dd807050.aspx).  
  
> [!IMPORTANT]  
> Убедитесь, что администратор в партнерской организации по ресурсам следующим инструкциям, приведенным в [контрольный список: Настройка партнерской организации по ресурсам](Checklist--Configuring-the-Resource-Partner-Organization.md) чтобы убедиться, что все необходимые развертывания задачи будут выполнены успешно создать второй половине партнерстве по федерации.  
  
> [!NOTE]  
> Задачи в контрольном списке в порядке. Перехода по ссылке процедуры, вернитесь к данном разделу, после выполнения действия, описанные в этой процедуре, чтобы продолжить выполнение оставшихся задач контрольного списка.

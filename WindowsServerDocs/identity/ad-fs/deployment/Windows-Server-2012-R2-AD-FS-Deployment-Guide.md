# <a name="windows-server-2016-and-2012-r2-ad-fs-deployment-guide"></a>Руководство по развертыванию R2 AD FS Windows Server 2016 и 2012 г.

>Область применения: Windows Server 2016, Windows Server 2012 R2

\(AD FS\) служб федерации Active Directory с Windows Server 2016 и 2012 R2 операционной системы можно использовать для построения решения по управлению федеративными удостоверениями, расширяющие распределенной идентификации, проверки подлинности и авторизации службы веб-приложениям через границы организации и платформы. При развертывании Служб федерации Active Directory, можно расширить существующие возможности управления удостоверениями вашей организации к Интернету.

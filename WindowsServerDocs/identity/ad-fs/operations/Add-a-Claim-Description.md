# <a name="add-a-claim-description"></a>Добавление описания утверждения


В организации партнера по учетным записям администраторы создают утверждений для представления пользователя членства в группе или роли или для представления некоторые данные о пользователе, например, пользователя идентификационный номер сотрудника.

В партнерской организации администраторы создают соответствующие заявки для представления групп и пользователей, которые могут быть распознаны как пользователей ресурсов. Так как исходящие утверждения в учетной записи партнера организации сопоставляются входящие утверждения в организации партнера по ресурсам, партнер по ресурсам может принимать учетные данные, предоставленные партнера по учетным записям. 

Чтобы добавить утверждение, можно использовать следующую процедуру.

Для выполнения этой процедуры минимальным требованием является членство в группе **Администраторы** или эквивалентные права на локальном компьютере.  Просмотрите сведения об использовании соответствующих учетных записей и членства в группах в [локальные и доменные группы по умолчанию](https://go.microsoft.com/fwlink/?LinkId=83477).

## <a name="to-add-a-claim-description"></a>Добавление описания утверждения

1. В диспетчере серверов щелкните **средства**, а затем выберите **управления AD FS**. 

2.  Разверните **службы** и в контекстном меню **Добавление описания утверждения**.
![Добавление описания утверждения](media\Add-a-Claim-Description\claimdesc1.png)

3.  Добавить диалоговое окно описания утверждения в поле **отображаемое имя**, введите уникальное имя, которое идентифицирует группу или роль для этого утверждения.

4.  Добавить **краткое имя**.

5.  В **идентификатор утверждений**, введите URI, который связан с группы или роли утверждения, которое вы будете использовать.

6.  В разделе **описание**, введите текст, который лучше всего описывает ее назначение этого утверждения.

7.  В зависимости от потребностей вашей организации выберите один из следующих флажков, соответствующим образом опубликовать это утверждение в метаданных федерации:


    - Чтобы опубликовать это утверждение, чтобы помнить, что этот сервер может принять это утверждение партнеров, щелкните **опубликовать это утверждение в метаданных федерации как тип утверждения, принимаемый этой службой федерации**.
    - Чтобы опубликовать это утверждение, чтобы помнить, что этот сервер может выдавать это утверждение партнеров, щелкните **опубликовать это утверждение в метаданных федерации как тип утверждения, отправляемый этой службой федерации**.

8.  Нажмите кнопку **ОК**.

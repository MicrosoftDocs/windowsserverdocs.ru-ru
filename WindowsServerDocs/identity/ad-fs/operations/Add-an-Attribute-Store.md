# <a name="add-an-attribute-store"></a>Добавление хранилища атрибутов

>Область применения: Windows Server 2016, Windows Server 2012 R2

Учетные записи пользователей и учетные записи компьютеров, которым требуется доступ к ресурсу, который защищен \(AD FS\) служб федерации Active Directory, хранятся в хранилище атрибутов, таких как \(AD DS\) доменных служб Active Directory. Хранилища атрибутов используется подсистемой выдачи утверждений для сбора данных, необходимых для выдачи утверждений. Данные из хранилища атрибутов затем проецируется в виде утверждений.  
  
Можно использовать следующую процедуру для добавления в службе федерации хранилище атрибутов.  
  
Членство в группе **Администраторы**, или эквивалентной группе на локальном компьютере минимальным требованием для выполнения этой процедуры.  Подробные сведения об использовании соответствующих учетных записей и членстве в группах в [локальные и доменные группы по умолчанию ](https://go.microsoft.com/fwlink/?LinkId=83477).   
  
#### <a name="to-add-an-attribute-store"></a>Добавление хранилища атрибутов  
  
1.  Откройте **управления AD FS**.  
  
2.  В разделе **действия** щелкните **Добавление хранилища атрибутов**.  

![Добавление хранилища атрибутов](media/Add-an-Attribute-Store/addstore1.PNG)
  
3.  В **Добавление хранилища атрибутов** диалоговом поле, настроить следующие свойства для хранилища атрибутов, которое вы хотите добавить:  
  
    -   В **отображаемое имя**, введите имя, которое вы хотите использовать для идентификации хранилище атрибутов.  
  
    -   В **тип хранилища атрибутов**, либо выберите тип магазина поддерживаемых атрибутов, **Active Directory**, **LDAP**, или **SQL**.  
  
    -   В **строка подключения**, если вы выбрали магазин \(LDAP\) протокола LDAP или хранилище с \(SQL\) языка, введите строку, которая использовалась для подключения к хранилищу атрибутов. Для хранилища атрибутов Active Directory не строки подключения не требуется; Таким образом данное поле отключено.  
  
        > [!NOTE]  
        > AD FS автоматически создает хранилище атрибутов Active Directory по умолчанию.  
 
![Добавление хранилища атрибутов](media/Add-an-Attribute-Store/addstore2.PNG) 

4.  Нажмите кнопку **ОК**.

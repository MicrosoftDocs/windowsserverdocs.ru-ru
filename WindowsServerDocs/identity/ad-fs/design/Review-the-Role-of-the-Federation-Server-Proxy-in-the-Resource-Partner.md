# <a name="review-the-role-of-the-federation-server-proxy-in-the-resource-partner"></a>Сведения о роли прокси-сервера федерации в организации партнера по ресурсам

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Прокси-сервера федерации в \(AD FS\) служб федерации Active Directory могут работать в одном или нескольких из следующих ролей, в зависимости от того, как настроить сервер для удовлетворения потребностей организации партнера по ресурсам:  
  
-   **Обнаружение партнера по учетным записям**: клиентский компьютер в Интернете необходимо определить, какая учетная запись партнера будет выполнять его проверку подлинности. Клиент находит партнера по учетным записям с помощью учетной записи партнера обнаружения Web формы \(discoverclientrealm.aspx\), которая хранится на прокси-сервера федерации в организации партнера по ресурсам. Если более одного партнера по учетным записям настроен управления AD FS оснастка, раскрывающиеся меню отображается клиенту со всех доступными партнерами по учетным записям, видимыми для компьютеров клиента Интернета, получить доступ к веб-формы обнаружения партнера учетной записи. Можно изменить способ представления веб-формы обнаружения партнера учетной записи на клиентских компьютерах, настроив файл discoverclientrealm.aspx.  
  
-   **Перенаправление токенов безопасности**: прокси-сервер федерации партнера по учетным записям отправляет токены безопасности партнеру по ресурсам. Прокси-сервер федерации ресурсов принимает эти токены и передает их на сервер федерации в организации партнера по ресурсам. Затем сервер федерации ресурсов выдает токен безопасности, привязанный к определенному ресурсу веб-сервера. Прокси-сервер федерации ресурсов перенаправляет этот токен клиенту.  
  
Итак, прокси-сервер федерации ресурсов облегчает процесс федеративного входа в систему, перенаправляя клиентские компьютеры на сервер федерации, который может проверять подлинность клиентов. Прокси-сервер федерации ресурсов также действует как прокси для токенов безопасности клиента на серверах федерации ресурсов.  
  
> [!NOTE]  
> При необходимости сократить объем оборудования и число требуемых сертификатов, прокси-сервер федерации может располагаться на том же компьютере, и веб-сервер.

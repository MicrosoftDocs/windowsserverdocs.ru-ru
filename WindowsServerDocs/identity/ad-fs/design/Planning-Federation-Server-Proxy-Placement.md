# <a name="planning-federation-server-proxy-placement"></a>Планирование размещения прокси-серверов федерации

Создание прокси-серверов в новом проекте. Сведения в следующих разделах помогут определить, когда и где следует поместить прокси-сервер федерации, а также нужно ли настроить его для роли партнера по учетным записям или роли партнера по ресурсам.

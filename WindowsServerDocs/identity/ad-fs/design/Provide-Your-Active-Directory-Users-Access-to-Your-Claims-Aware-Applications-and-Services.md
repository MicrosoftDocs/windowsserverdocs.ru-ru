-   **Домен Active Directory Services \(AD DS @ no__t-2:** Доменные службы Active Directory содержат учетные записи пользователей сотрудников, используемые для формирования токенов AD FS. -   **Сервер федерации партнера по учетным записям:** Этот сервер федерации присоединен к домену в лесу партнера по учетным записям.     Сервер федерации партнера по учетным записям может выполнять проверку подлинности следующих пользователей:

    -   сотрудников с учетными записями в любом месте леса;

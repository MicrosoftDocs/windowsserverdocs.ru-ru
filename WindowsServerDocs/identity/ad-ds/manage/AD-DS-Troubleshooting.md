# <a name="ad-ds-troubleshooting"></a>Устранение неполадок AD DS

>Область применения: Windows Server 2016, Windows Server 2012 R2

В данную документацию включены, устранение неполадок рекомендации и процедуры для диагностики и устранения проблем, которые могут возникнуть при использовании репликации Active Directory. 

Это содержимое в основном ориентировано на отклики на сообщения журнала событий службы каталогов и сообщения об ошибке на основе инструментов, может сообщить средства Repadmin.exe и Dcdiag.exe. Эти средства доступны на все контроллеры домена, работающих под управлением Windows Server 2016 или 2012 R2. Также можно установить средства удаленного администрирования сервера (RSAT) на рядовом сервере под управлением Windows 10. 

Сведения об установке RSAT см. в разделе [установки средств удаленного сервера]. (https://www.microsoft.com/en-us/download/details.aspx?id=45520)

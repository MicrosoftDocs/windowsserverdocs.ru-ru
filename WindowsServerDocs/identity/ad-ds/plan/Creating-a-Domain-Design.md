# <a name="creating-a-domain-design"></a>Создание проекта домена

>Область применения. Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Владелец леса несет ответственность за создание проекта домена для леса. Создание проекта домена включает в себя изучение требования к репликации и производительность существующей сетевой инфраструктуры и затем построения структуры домена, которая поддерживает доменных служб Active Directory (AD DS), чтобы функция наиболее эффективным способом. Домены используются для секционирования в каталог, чтобы данные в каталоге можно распространять и эффективное управление на всем предприятии. Для конструирования домена предназначена для обеспечения максимальной эффективности Active Directory, топологии репликации гарантией того, что репликация не использует слишком много доступную пропускную способность сети и не влияет на ежедневной работе сети.

# <a name="enabling-advanced-features-for-ad-ds"></a>Включение дополнительных возможностей для службы AD DS

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Доменных служб Active Directory (AD DS) позволяет вводят расширенные функции в вашей среде, создавая функционального уровня домена или леса. Чтобы использовать расширенные функции в Доменных службах Active Directory, необходимо определить операционных систем, работающих на контроллерах домена в вашей среде.   
  
Необходимо также определить наиболее режим работы для вашей организации, в зависимости от имеющейся инфраструктуры и затем повысить режим работы домена или леса соответствующим образом. Режим работы можно повысить, когда все контроллеры домена в домене или лесу работают соответствующую версию Windows. Несмотря на то, что повышение функционального уровня позволяет включить новые возможности, он также ограничивает версий операционных систем Windows, которые могут выполняться на контроллерах домена в вашей среде.

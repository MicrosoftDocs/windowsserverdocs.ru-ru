# <a name="enabling-advanced-features-for-ad-ds"></a>Включение дополнительных возможностей для доменных служб Active Directory

>Область применения. Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Доменных служб Active Directory (AD DS) позволяет вводят дополнительные функции в вашей среде, вызывая режимы работы домена или леса. Чтобы воспользоваться дополнительными функциями AD DS, необходимо определить операционные системы, которые выполняются на контроллерах домена в вашей среде.   
  
Необходимо также определить лучше всего функциональный уровень для вашей организации, в зависимости от существующей инфраструктуры и затем Повышение режима работы домена или леса соответствующим образом. Повышение режима работы можно, все контроллеры домена в домене или лесу при выполнении соответствующую версию Windows. Несмотря на то, что повышение функционального уровня позволяет включить новые функции, также ограничивается в версиях операционных систем Windows, которые могут выполняться на контроллерах домена в вашей среде.

# <a name="setting-site-link-properties"></a>Задание свойств связи сайтов

>Область применения. Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Межсайтовая репликация происходит в соответствии со свойствами объектов подключения. Когда средство проверки согласованности знаний (KCC) создает объекты соединения, расписание репликации наследует свойства объекта связи сайтов. Каждый объект связи сайтов представляет подключение глобальной сети (WAN) между двумя или несколькими сайтами.  
  
Задание свойств объекта связи сайтов включает следующие шаги:  
  
-   Определение стоимости, сопоставленный этому пути репликации. KCC использует цену для определения самого дешевого маршрута для репликации между двумя сайтами, которые реплицируются одного раздела каталога.  
  
-   Определение расписания, определяющий время которых межсайтовая репликация может произойти.  
  
-   Определение интервала репликации, который определяет, как часто должна происходить репликация периоды, когда может репликации, как определено в расписании.

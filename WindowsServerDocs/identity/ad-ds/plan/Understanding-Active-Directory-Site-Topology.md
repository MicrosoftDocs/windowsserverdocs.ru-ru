# <a name="understanding-active-directory-site-topology"></a>Основные сведения о топологии сайта Active Directory

>Область применения: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Топологии сайта существенно влияет на производительность сети и возможность пользователям доступ к сетевым ресурсам. Перед началом для проектирования топологии сайтов ознакомиться с функциями для сайтов в Windows Server 2008, другой сетевой топологии, которые организации часто используют, роль владельца топологии сайта и некоторые понятия репликации Active Directory.

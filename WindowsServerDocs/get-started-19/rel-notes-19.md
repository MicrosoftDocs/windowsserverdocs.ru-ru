# <a name="release-notes---important-issues-in-windows-server-2019"></a>Заметки о выпуске — важные проблемы в Windows Server 2019

>Область применения. Windows Server 2019

Эти заметки о выпуске суммировать самые важные проблемы в операционной системе Windows Server 2019, включая информацию о способах устранения или обхода этих проблем, если он известен. Сведения об изменениях структуры, новых функциях и исправлениях в этом выпуске см. в разделе [новые возможности в Windows Server 2019](whats-new-19.md) и в сообщениях групп разработчиков конкретных компонентов. Если не указано обратное, эти заметки распространяются на все выпуски и варианты установки Windows Server 2019.  

Этот документ постоянно обновляется. По мере обнаружения критических проблем, определения обходных путей и выпуска исправлений в документ добавляются соответствующие сведения.  

## <a name="release-notes"></a>Заметки о выпуске

Описанные ниже известные проблемы присутствуют в Windows Server 2019.

| Заголовок         | Описание                            |
| -----         | -----------                            |
| Меню вариант установки во время установки сервера усечен текст на немецком языке | При запуске программы установки с носителя на немецком сервере, в операционной системе окне выбора под названием «Выбор операционной системы, которую вы хотите установить,» описание варианты установки возможностей рабочего стола будет иметь отсутствующие или неверные символы в самом конце предложения. Ниже приведен полный немецкий текст, которое должно отображаться.<br/>      <br/>`Durch diese Option wird die vollständige grafische Umgebung von Windows installiert, wodurch zusätzlicher Speicherplatz verbraucht wird. Sie kann hilfreich sein, wenn Sie den Windows-Desktop verwenden möchten oder über eine App verfügen, die die grafische Umgebung benötigt.` <br><br>Это влияет только на немецком мультимедиа, снимаются в общей доступности из Windows Server 2019 г., Windows Server, версии 1809 и Microsoft Hyper-V Server 2019.|
| Фирменной настройки образа Windows Server неверные во время установки Windows Server версии 1809 | Во время установки для Windows Server версии 1809, фоновое изображение на некоторые начальный экраны показано &quot;Windows Server 2019&quot;.  Как система Windows Server версии 1709 и 1803, это просто сказать &quot;Windows Server&quot;.  Существует нет других изменениях, к любому другому месту в продукте, и никак не отражается на продукт Windows Server 2019.  Проблема ограничена этой один образ во время установки Windows Server версии 1809, доступные только для пользователей корпоративной лицензии, доступ к центра поддержки корпоративных лицензий.<br/> |

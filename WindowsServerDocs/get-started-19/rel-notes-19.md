# <a name="release-notes---important-issues-in-windows-server-2019"></a>Заметки о выпуске — важные проблемы в Windows Server 2019

>Область применения. Windows Server 2019

Эти заметки о выпуске суммировать самые важные проблемы в Windows Server&reg; 2019 операционной системы, включая информацию о способах устранения или обхода этих проблем, если он известен. Сведения об изменениях структуры, новых функциях и исправлениях в этом выпуске см. в разделе [новые возможности в Windows Server 2019](whats-new-19.md) и в сообщениях групп разработчиков конкретных компонентов. Если не указано обратное, эти заметки распространяются на все выпуски и варианты установки Windows Server 2019.  

Этот документ постоянно обновляется. По мере обнаружения критических проблем, определения обходных путей и выпуска исправлений в документ добавляются соответствующие сведения.

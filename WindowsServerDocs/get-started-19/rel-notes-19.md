# <a name="release-notes---important-issues-in-windows-server-2019"></a>Заметки о выпуске. Важные проблемы в Windows Server 2019

>Область применения. Windows Server 2019

В этих заметках о выпуске приведены сводные данные о наиболее важных проблемах в операционной системе Windows Server 2019, включая информацию о способах устранения или обхода этих проблем, если таковые известны. Информацию об изменениях структуры, новых компонентах и исправлениях в этом выпуске см. в статье [Новые возможности Windows Server 2019](whats-new-19.md) и в сообщениях групп разработчиков конкретных компонентов. Если не указано иное, эти заметки распространяются на все выпуски и варианты установки Windows Server 2019.  

Этот документ постоянно обновляется. По мере обнаружения критических проблем, определения обходных путей и выпуска исправлений в документ добавляются соответствующие сведения.  

## <a name="release-notes"></a>Заметки о выпуске

Описанные ниже известные проблемы присутствуют в Windows Server 2019.

| Заголовок         | Описание                            |
| -----         | -----------                            |
| В меню установки во время установки сервера усечен текст на немецком языке | При запуске программы установки с носителя на немецком сервере, в операционной системе, в окне выбора под названием "Выбор операционной системы, которую вы хотите установить", описание вариантов установки возможностей рабочего стола будет иметь отсутствующие или неверные символы в самом конце предложения. Ниже приведен полный немецкий текст, который должен отображаться.<br/>      <br/>`Durch diese Option wird die vollständige grafische Umgebung von Windows installiert, wodurch zusätzlicher Speicherplatz verbraucht wird. Sie kann hilfreich sein, wenn Sie den Windows-Desktop verwenden möchten oder über eine App verfügen, die die grafische Umgebung benötigt.` <br><br>Проблема влияет только на общедоступный немецкий носитель Windows Server 2019, Windows Server, версии 1809 и Microsoft Hyper-V Server 2019.|
| Изображение фирменной настройки Windows Server ошибочно отображается во время установки Windows Server версии 1809 | Во время установки Windows Server версии 1809 на некоторых начальных экранах показано фоновое изображение &quot;Windows Server 2019&quot;.  Как и в случае системы Windows Server версий 1709 и 1803, должно быть написано просто &quot;Windows Server&quot;.  Это никак не влияет на продукт и никак не отражается на продукте Windows Server 2019.  Проблема ограничена одним изображением во время установки Windows Server версии 1809, доступной только для пользователей корпоративной лицензии через центр VLSC.<br/> |

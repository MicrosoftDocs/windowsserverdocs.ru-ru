# <a name="features-removed-or-planned-for-replacement-in-windows-server"></a>Удаленные или подлежащие замене компоненты в Windows Server

> Относится к: Windows Server 2019, Windows Server 2016, Windows Server (Semi-Annual Channel)

Вы можете ознакомиться с удаленными и объявленными нерекомендуемыми функциями, реализованными в каждом из выпусков Windows Server.  

Выпуски Long-Term Servicing Channel (LTSC)

- [Удаленные или подлежащие удалению компоненты в Windows Server 2019](removed-features-19.md)
- [Компоненты, удаленные или не рекомендуемые к использованию в Windows Server 2016](../get-started/Deprecated-Features.md)

Выпуски Semi-Annual Channel

- [Удаленные или подлежащие замене компоненты в Windows Server версии 1903](removed-features-1903.md)
- [Удаленные или подлежащие замене компоненты в Windows Server версии 1803](../get-started/windows-server-1803-removed-features.md)
- [Удаленные или подлежащие замене компоненты в Windows Server версии 1709](../get-started/Removed-Features-1709.md) (неподдерживаемые начиная с 9 апреля 2019 г.)

В выпусках Windows Server Semi-Annual Channel новые функциональные возможности операционной системы появляются быстрее. Новый выпуск выходит каждые шесть месяцев и поддерживается в течение 18 месяцев. Сведения о сроках поддержки см. в статье [Информация о выпуске Windows Server](../get-started/windows-server-release-info.md) и разделе [Политика жизненного цикла поддержки Майкрософт](https://support.microsoft.com/lifecycle). По возможности всегда используйте последний доступный выпуск.

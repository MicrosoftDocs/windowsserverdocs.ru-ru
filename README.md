Этот репозиторий время активного управления и включен исходный код для Windows Server содержимое, публикуемое на (docs.microsoft.com) [https://docs.microsoft.com]:

- [Windows Server 2019](https://docs.microsoft.com/en-us/windows-server/get-started-19/get-started-19)
- [Windows Server 2016](https://docs.microsoft.com/en-us/windows-server/get-started/server-basics)
- [Essentials Server 2016 и сервера Essentials 2019 г.](https://docs.microsoft.com/en-us/windows-server-essentials/get-started/get-started)
- [Microsoft Hyper-V Server 2016](https://docs.microsoft.com/en-us/windows-server/get-started/server-basics)


Чтобы сделать небольшой обновления, измените файл и отправить запрос на Вытягивание. Ознакомьтесь с [редактирование существующей документации профессиональных Windows](https://docs.microsoft.com/en-us/windows/whats-new/contribute-to-a-topic) для получения дополнительных сведений.

Для внутренних, участников Microsoft — информация о том, как добавить или обновить качества содержимого, не менее, стандартов, форматирование справки и многое другое доступно в [руководства участника](https://aka.ms/windowsauthoring).
